一般パラメータ: 認証

「認証」タブでは、リスナーのセキュリティを構成できます。これによって、構成変更の保存やリスナーの停止など、リスナー制御ユーティリティから使用する権限を伴う操作には、パスワードが必要になります。Oracle Net Managerでは、暗号化されたパスワードを設定できます。

リスナー操作にパスワードは不要

管理作業用の認証を使用禁止にします。

リスナー操作にパスワードが必要

特定の管理作業用の認証を使用可能にします。この後で必須のパスワードを入力します。デフォルトのパスワードはoracleです。

注意: Oracle Netでは、1つのリスナーに1つのパスワードを構成できます。

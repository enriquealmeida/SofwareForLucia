Archivo listener.ora

Archivo de configuración del listener que especifica:

  • El nombre del listener

  • Las direcciones de protocolo en las que se aceptan solicitudes de conexión

  • Los servicios (servicio de base de datos o de otro tipo) que recibe

El archivo listener.ora debe residir en una de las siguientes ubicaciones:

  1. El directorio especificado por la variable de entorno TNS_ADMIN. Si la variable de entorno TNS_ADMIN no está definida como una variable en Windows NT, puede que esté en el registro.

  2. El directorio de configuración global del nodo. Para Sun Solaris, este directorio es /var/opt/oracle. Windows NT no tiene un directorio central.

  3. El directorio $ORACLE_HOME/network/admin en UNIX o el directorio ORACLE_HOME\network\admin en sistemas operativos Windows.

Nota: Oracle9i y Oracle8i no necesita identificación del servicio de base de datos debido al registro del servicio. No obstante, se requiere una configuración estática de los servicios si desea utilizar Oracle Enterprise Manager para supervisar la base de datos.

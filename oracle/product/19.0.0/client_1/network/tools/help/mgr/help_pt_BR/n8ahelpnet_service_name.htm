Nome do Serviço de Rede

Um nome simples de um serviço que é resolvido em um descritor de conexão. Os usuários iniciam uma solicitação de conexão passando um nome de usuário e uma senha com um nome de serviço de rede em uma string de conexão para o serviço ao qual desejam se conectar:

CONNECT nome do usuário/senha@nome_do_serviço_de_rede

Dependendo de suas necessidades, os nomes de serviço de rede podem ser armazenados em vários lugares, incluindo o seguinte:

  • Arquivo de configuração local, arquivo tnsnames.ora, em cada cliente

  • Servidor de Diretórios

  • Servidor Oracle Names

  • Serviço de nomeação externa, incluindo NIS ("Network Information Service", Serviço de Informações de Rede)

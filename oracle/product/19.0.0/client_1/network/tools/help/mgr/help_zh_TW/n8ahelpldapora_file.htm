ldap.ora 檔案

這是由「Oracle Net 組態輔助程式」所建立的檔案，內含下列目錄組態資訊：

  • 目錄類型

  • 目錄位置

  • 從屬端或伺服器連線至資料庫服務時，用來查詢或設定連線 ID 的預設 Oracle Context

ldap.ora 檔案必須位於下列其中一個位置：

  1. TNS_ADMIN 環境變數指定的目錄。如果 TNS_ADMIN 環境變數不是定義為 Windows NT 的變數，它可能就在登錄中。

  2. UNIX 的 $ORACLE_HOME/network/admin 目錄，或 Windows 作業系統的 ORACLE_HOME\network\admin 目錄。

Nome do Banco de Dados Global

O nome completo do banco de dados que o diferencia de forma exclusiva de qualquer outro banco de dados. O nome do banco de dados global tem o formato nome_do_banco_de_dados_domínio, por exemplo, sales.us.example.com.

A parte do nome do banco de dados, sales, é um nome simples que você deseja atribuir ao seu banco de dados. A parte do domínio do banco de dados, us.example.com , especifica o domínio em que o banco de dados está localizado, o que torna o nome de banco de dados global um nome exclusivo. Quando possível, a Oracle recomenda que o domínio de banco de dados corresponda ao domínio de rede.

O nome de banco de dados global é o nome de serviço padrão do banco de dados, conforme especificado pelo parâmetro SERVICE_NAMES no arquivo de parâmetros de inicialização.

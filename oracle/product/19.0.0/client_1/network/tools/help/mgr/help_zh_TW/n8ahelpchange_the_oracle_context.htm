變更 Oracle Context

啟動 Oracle Net Manager 時，目錄 > 服務命名資料夾中顯示的物件，即是使用 Oracle Net 組態輔助程式設定目錄用途組態時建立之預設 Oracle Context 所含的 Oracle 服務項目。您可以建立並修改您具有認證的其他 Oracle Context 項目。

變更目前的 Oracle Context：

  1. 選擇命令 > 目錄 > 變更目前的相關資訊環境。選擇新的相關資訊環境對話方塊就會顯示。

  2. 從目錄命名相關資訊環境清單中選取包含您要檢視之 Oracle Context 的目錄項目。

  3. 從 Oracle Context 清單中選取 Oracle Context，然後選擇確定。如果所做的修改並未儲存至目前的 Oracle Context，便會出現「儲存」對話方塊，提示您儲存變更。

Fichier cman.ora

Fichier de configuration qui indique les adresses de protocole des demandes entrantes et des commandes locales et distantes, ainsi que les paramètres d'Oracle Connection Manager. Oracle Net Manager ne prend pas en charge la configuration de ce fichier. Il doit être configuré manuellement.

Le fichier cman.ora doit résider dans l'un des emplacements suivants :

  1. Le répertoire indiqué par la variable d'environnement TNS_ADMIN . Si la variable d'environnement TNS_ADMIN n'est pas définie en tant que variable dans Windows NT, il se peut qu'elle soit dans la base de registres.

  2. Le répertoire de configuration global du noeud. Pour Sun Solaris, ce répertoire est /var/opt/oracle. Windows NT n'a pas de répertoire central.

  3. Le répertoire $ORACLE_HOME/network/admin dans UNIX ou le répertoire ORACLE_HOME\network\admin dans les systèmes d'exploitation Windows.

Pour plus d'informations, reportez-vous au chapitre Configuring and Administering Oracle Connection Manager dans le Guide de l'administrateur Oracle10i Net.

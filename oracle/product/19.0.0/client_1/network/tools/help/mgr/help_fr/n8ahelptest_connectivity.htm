Test des connexions

Un test de connexion vous permet de vérifier la connexion d'un objet de résolution de noms de service sélectionné à un service de base de données. Ce test effectue une connexion au service de base de données en utilisant des informations sur un descripteur de connexion configuré.

Pour vérifier les informations sur la connexion d'un réseau :

  1. Assurez-vous que le processus d'écoute et la base de données sont en cours d'exécution.

  2. Dans le panneau du navigateur, développez Oracle Net Configuration > Annuaire ou Local > Résolution de noms de service.

  3. Choisissez un nom de service réseau ou un service de base de données existant.

  4. Dans la barre d'outils, sélectionnez le symbole de base de données ou choisissez Commande > Tester le service. La boîte de dialogue Test de connexion apparaît et est mise à jour au fur et à mesure de la génération des résultats du test. La réussite du test est indiquée par le message suivant :

Le test a réussi.

Pour obtenir des informations sur le démarrage des composants, reportez-vous au chapitre Establishing a Connection and Testing the Network dans le Guide de l'administrateur Oracle10i Net.

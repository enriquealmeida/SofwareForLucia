Configurazione delle funzioni di log e trace del profilo

È possibile configurare le funzioni di log e trace di un client.

Le informazioni di log sono in genere memorizzate nel file sqlnet.log nella directory $ORACLE_HOME/network/log sulle piattaforme UNIX e nella directory ORACLE_HOME \network\log sulle piattaforme Windows.

Le informazioni di trace sono memorizzate per impostazione predefinita nel file sqlnet.trc per il client e nel file svr_pid.trc per il server nella directory $ORACLE_HOME/network/trace sulle piattaforme UNIX e nella directory ORACLE_HOME \network\trace sulle piattaforme Windows.

Per abilitare la funzione di log o trace, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Generale.

  3. Selezionare la scheda Log o Trace.

  4. Specificare le impostazioni di log o trace.

  5. Scegliere File > Salva configurazione di rete.

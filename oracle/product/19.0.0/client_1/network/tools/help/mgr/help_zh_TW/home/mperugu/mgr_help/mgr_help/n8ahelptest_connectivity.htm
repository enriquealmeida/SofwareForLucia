測試連線

您可以執行連線測試，驗證所選服務命名物件與資料庫服務的連線。連線測試時，會使用設定的連線描述區資訊，建立資料庫服務連線。

驗證網路連線資訊：

  1. 確定監聽器與資料庫執行中。

  2. 在瀏覽窗格中，展開 Oracle net 組態 > 目錄或區域 > 服務命名。

  3. 選取現有的網路服務名稱或資料庫服務。

  4. 從工具列選取資料庫符號，或者選取命令 > 測試服務。「連線測試」對話方塊就會顯示，並在產生結果時更新。測試成功時會產生以下訊息：

連線測試成功。

如需有關啟動元件的資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Establishing a Connection and Testing the Network 章節。

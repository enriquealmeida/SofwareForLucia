Archivo ldap.ora

Archivo creado por el Asistente de Configuración de Red de Oracle que contiene la siguiente información de configuración de directorios:

  • Tipo de directorio

  • Ubicación del directorio

  • Oracle Context por defecto que el cliente o el servidor utilizarán para consultar o configurar identificadores de conexión para conexiones a servicios de base de datos.

El archivo ldap.ora debe residir en una de las siguientes ubicaciones:

  1. El directorio especificado por la variable de entorno TNS_ADMIN . Si la variable de entorno TNS_ADMIN no está definida como una variable en Windows NT, puede que esté en el registro.

  2. El directorio $ORACLE_HOME/network/admin en UNIX o el directorio ORACLE_HOME \network\admin en sistemas operativos Windows.

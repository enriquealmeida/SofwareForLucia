Visão Geral de Perfis

Um perfil especifica preferências para ativar e configurar recursos do Oracle Net no cliente ou servidor. Com um perfil, você pode:

  • Especifique o domínio do cliente a ser acrescentado a nomes desqualificados

  • Priorizar métodos de nomeação.

  • Ative os recursos de rastreamento e de log.

  • Rotear conexões através de processos específicos

  • Configurar parâmetros para nomeação externa

  • Configurar o Oracle Advanced Security

Para acessar um perfil, no painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

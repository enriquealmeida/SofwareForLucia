Arquivo ldap.ora

Um arquivo criado pelo Assistente de Configuração do Oracle Net que contém as seguintes informações de configuração de diretório:

  • Tipo de diretório

  • Localização do diretório

  • O Oracle Context default que o cliente ou servidor utilizará para consultar ou configurar identificadores de conexão para conexões com serviços de bancos de dados

O arquivo ldap.ora deverá residir em uma das seguintes localizações:

  1. O diretório especificado pela variável de ambiente TNS_ADMIN. Se a variável de ambiente de rede TNS_ADMIN não tiver sido definida como uma variável no Windows NT, ela poderá estar no registro.

  2. O diretório $ORACLE_HOME/network/admin do UNIX ou o diretório ORACLE_HOME \network\admin de sistemas operacionais do Windows.

Listener Control-Utility

Ein in Oracle Net Services enthaltenes Utility, mit dem verschiedene Listener-Funktionen kontrolliert werden, wie z.B. Starten und Stoppen des Listeners sowie Ermitteln des Listener-Status. Das Utility Listener Control wird folgendermaßen ausgeführt:

lsnrctl-Befehl [listener_name]

listener_name - Der Name des Listeners, wie in der Datei listener.ora definiert. Der Listener muss nicht identifiziert werden, wenn der standardmäßige Listener-Name, LISTENER, verwendet wird.

Weitere Informationen finden Sie im Kapitel Listener Control Utility im Oracle10i Net Services Reference Guide.

Configuration statique d'informations sur le service de base de données

Pour que le processus d'écoute puisse accepter des demandes de connexion client à une base de données Oracle8 ou Oracle7, le fichier listener.ora doit être configuré avec les informations concernant la base de données.

Si la base de données est une base de données Oracle9i ou Oracle8i, il n'est pas nécessaire que le fichier listener.ora contienne des informations de service statiques. Ces informations sont inscrites automatiquement auprès du processus d'écoute par le biais de l'inscription de service. Toutefois, si vous utilisez Oracle Enterprise Manager, les informations de service statique correspondant à une base de données Oracle8i sont obligatoires.

Pour configurer des informations sur le service de base de données pour un processus d'écoute :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Processus d'écoute.

  2. Sélectionnez un processus d'écoute.

  3. Dans la liste du panneau de droite, sélectionnez Services de base de données.

  4. Choisissez Ajouter une base de données. Un nouvel onglet Base de données apparaît.

  5. Entrez le nom global de base de données dans le champ Nom global de base de données, l'emplacement du répertoire d'origine Oracle Home dans le champ Répertoire d'origine Oracle Home et l'identificateur système (SID) Oracle de la base de données dans le champ SID.

  6. Choisissez Fichier > Enregistrer la configuration réseau.

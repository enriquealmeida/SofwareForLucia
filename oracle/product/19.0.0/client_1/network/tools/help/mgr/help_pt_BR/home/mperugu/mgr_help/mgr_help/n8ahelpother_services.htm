Outros Serviços

A guia Outros Serviços permite que você especifique serviços que não sejam de banco de dados, para os quais o listener deverá receber solicitações de conexão. Esses serviços podem ser procedimentos externos ou Serviços Heterogêneos.

Nome do Serviço Global

Informe o nome do serviço e o nome do domínio do serviço.

Nome do Programa

Informe o nome do programa executável. Por exemplo, extproc.exe.

Argumento Zero do Programa

Informe o primeiro argumento interno, que em geral é usado como nome de programa alternativo. Por exemplo: argv0

Argumentos do Programa

Informe os argumentos da linha de comandos que devem ser especificados quando o serviço é iniciado. Por exemplo, -mode=5.

Ambiente

Informe as variáveis de ambiente que deverão ser definidas antes de o serviço ser iniciado. As variáveis de ambiente são especificadas no seguinte formato:

variável_ambiente=valores

Por exemplo, user=scott, start=immediate.

Observe que as variáveis são separadas por vírgulas.

SID

Informe o nome do programa ou um Identificador de Sistema Oracle (SID) para o serviço.

Diretório Oracle Home

Informe a localização do programa executável.

Botão Adicionar Serviço

Escolha para configurar informações sobre um serviço que não seja de banco de dados. Uma nova guia Serviço é exibida para que você especifique informações para outro serviço.

Botão Remover Serviço

Escolha para deletar a guia Serviço selecionada e suas respectivas informações sobre o serviço.

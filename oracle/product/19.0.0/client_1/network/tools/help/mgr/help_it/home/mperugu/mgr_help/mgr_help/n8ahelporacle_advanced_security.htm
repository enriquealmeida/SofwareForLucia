Oracle Advanced Security

Si tratta di un prodotto che fornisce una serie completa di funzioni di sicurezza per l'ambiente Oracle. Attraverso queste funzioni il prodotto garantisce la protezione delle reti aziendali, anche durante la connessione a Internet. Sono inoltre disponibili soluzioni integrate di cifratura e autenticazione di rete, servizi Single Sign-On e protocolli di sicurezza. Oracle Advanced Security integra gli standard industriali, garantisce la massima protezione della rete Oracle e molto di più.

Per ulteriori informazioni, vedere il manuale Oracle Advanced Security Administrator's Guide.

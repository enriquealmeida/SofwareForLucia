Roteiro de Origem

Especifica que todas as rotas de rede são usadas. O roteamento de origem geralmente é utilizado em ambientes nos quais um grande número de conexões é necessário para acessar o mesmo dispositivo. O Oracle Net oferece um roteamento através doOracle Connection Manager.

Para obter informações sobre o Oracle Connection Manager, consulte o capítulo Networking Challenges in Internet Age no Oracle10i Net Administrator's Guide.

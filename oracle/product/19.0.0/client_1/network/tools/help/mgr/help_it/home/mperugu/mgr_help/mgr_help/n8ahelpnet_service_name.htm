Nome del servizio di rete

Un nome semplice per un servizio che viene risolto in un descrittore di connessione. Per avviare una richiesta di connessione, gli utenti devono specificare in una stringa di connessione un nome utente, una password e il nome di un servizio di rete al quale desiderano connettersi:

CONNECT nomeutente/password@nome_servizio_di rete

In base alle necessità, i nomi di servizi di rete possono essere memorizzati in diverse posizioni, tra cui:

  • il file di configurazione locale, file tnsnames.ora, su ogni client;

  • il server delle directory;

  • il server Oracle Names;

  • Servizio di denominazione esterna, compreso NIS (Network Information Service)

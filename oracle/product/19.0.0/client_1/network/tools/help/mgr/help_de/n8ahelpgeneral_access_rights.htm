Allgemein: Zugriffsrechte

Im Register Zugriffsrechte können Sie angeben, welchen Clients die Zugriffsberechtigung zu der Datenbank erteilt oder entzogen werden soll.

Zugriffsrechte für TCP/IP-Client prüfen

Wählen Sie dieses Kontrollkästchen, wenn Sie angeben möchten, welchen Clients eine Zugriffsberechtigung zu der Datenbank erteilt oder entzogen werden soll. Heben Sie die Auswahl des Kontrollkästchens auf, wenn Sie diese Funktionalität deaktivieren möchten. Wenn das Kontrollkästchen nicht ausgewählt ist, haben alle Clients Zugriff auf die Datenbank.

Vom Zugriff ausgeschlossene Clients

Geben Sie einen Hostnamen oder eine IP-Adresse ein, um anzugeben, welche Clients vom Zugriff auf die Datenbank ausgeschlossen sind. Mit einem Komma trennen Sie Einträge auf derselben Zeile.

Zum Zugriff zugelassene Clients

Geben Sie einen Hostnamen oder eine IP-Adresse ein, um anzugeben, welche Clients auf die Datenbank zugreifen können. Mit einem Komma trennen Sie Einträge auf derselben Zeile.

Hinweis:Zum Zugriff zugelassene Clients ist vorrangig gegenüber Vom Zugriff ausgeschlossene Clients.

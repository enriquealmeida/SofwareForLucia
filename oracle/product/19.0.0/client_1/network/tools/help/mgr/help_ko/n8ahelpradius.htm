RADIUS

RADIUS(Remote Authentication Dial-In User Service)는 원격 인증과 액세스를 사용으로 설정하기 위해 가장 잘 알려진 클라이언트/서버 보안 프로토콜입니다. RADIUS를 설치한 후 RADIUS 방법을 구성하면 네트워크에서 RADIUS 표준을 지원하는 모든 인증 방법(토큰 카드와 스마트 카드 포함)을 사용할 수 있습니다. RADIUS 인증 방법을 사용하려면 다음 매개변수를 설정해야 합니다.

요소 설명
호스트 이름 기본 RADIUS 서버의 호스트 이름입니다. 기본값 = localhost
포트 번호 기본 RADIUS 서버의 수신 포트 번호입니다. 기본값 = 1645
시간 초과(초) Oracle 서버가 기본 RADIUS 서버에서 응답을 기다리는 시간(초)입니다. 기본값 = 15초
재시도 횟수 Oracle 데이터베이스가 기본 RADIUS 서버에 메시지를 보내는 횟수입니다.

기본값 = 3
보안 파일 Oracle 서버에서 보안 키의 파일 위치입니다. 보안상의 이유로 Oracle은 이 파일을 루트 액세스 전용으로 변경할 것을 권장합니다.
계정 보내기 RADIUS 계정은 Oracle 데이터베이스 서버 액세스 정보를 기록하고 RADIUS 계정 서버의 파일에 이 정보를 저장합니다. RADIUS 인증 서버가 모두 이 기능을 지원하는 경우만 이 기능을 사용할 수 있습니다. 값 = on 또는 off
경쟁 응답 사용자에게 인증 문자 확인 GUI를 제공하려는 경우 이 필드의 값은 on입니다.
기본 키워드 경쟁 응답 방식을 통해 사용자 ID를 확인하는 키워드입니다. 기본값 = challenge
인터페이스 클래스 이름 Oracle 클라이언트와 RADIUS 서버 간 경쟁 응답 컨버세션을 처리하는 클래스의 이름입니다. 기본값 = DefaultRadiusInterface

Endereço IP

Usado para identificar um nó em uma rede. Cada computador da rede recebe um endereço IP exclusivo, que se compõe do ID da rede e de um ID de host exclusivo. Esse endereço é normalmente representado em notação decimal separada por pontos, com o valor decimal de cada octeto separado por um ponto, por exemplo, 144.45.9.22.

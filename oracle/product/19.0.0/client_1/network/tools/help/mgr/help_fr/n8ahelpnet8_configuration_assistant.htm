Assistant Configuration Oracle Net

Outil utilisé après l'installation pour configurer des composants de base de réseau, notamment :

  • noms de processus d'écoute et adresses de protocole

  • méthodes de résolution de noms que le client utilisera pour résoudre des identificateurs de connexion

  • noms de service réseau dans un fichier tnsnames.ora

  • Utilisation du serveur d'annuaire

L'assistant Configuration Oracle Net est lancé automatiquement après l'installation du logiciel, comme indiqué dans le guide d'installation Oracle. Il peut être utilisé sur le client ou sur le serveur de base de données.

Il peut également être exécuté en mode autonome pour configurer l'utilisation des méthodes de résolution de noms, le processus d'écoute, les noms de service réseau dans le fichier tnsnames.ora et l'utilisation du serveur d'annuaire.

Pour exécuter l'assistant Configuration Oracle Net en mode autonome :

  • Sous UNIX, exécutez netmgr dans $ORACLE_HOME/bin.

  • Sous Windows, sélectionnez Démarrer > Programmes > Oracle - HOME_NAME > Outils de configuration et de migration > Assistant Configuration Oracle Net.

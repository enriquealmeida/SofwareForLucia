Configuration du chiffrement

Le chiffrement est l'opération consistant à rendre un message illisible, sauf pour le destinataire prévu.

Pour configurer les paramètres de cryptage :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Oracle Advanced Security.

  3. Sélectionnez l'onglet Cryptage.

  4. Sélectionnez Client ou Serveur dans la liste Chiffrement.

  5. Sélectionnez une valeur dans la liste Type de cryptage pour indiquer le niveau de cryptage du client ou du serveur lors de la négociation du cryptage et de l'intégrité. Les quatre valeurs possibles sont les suivantes :

    Elément Description
    accepté Le service n'est actif que si l'autre machine participant à la connexion indique "requis" ou "demandé" et qu'elle dispose d'un algorithme compatible.
    rejeté Le service ne doit pas être actif : la connexion échoue si l'autre machine indique "requis".
    demandé Le service n'est actif que si l'autre machine participant à la connexion indique "accepté", "requis" ou "demandé" et qu'elle dispose d'un algorithme compatible.
    requis Le service doit être actif et la connexion échoue si l'autre machine indique "rejeté" ou ne dispose pas d'un algorithme compatible.

  6. Dans le champ Valeurs de départ de cryptage, entrez une chaîne aléatoire de 256 caractères maximum. Cette chaîne est utilisée avec l'algorithme de cryptage que vous sélectionnez à l'étape suivante.

  7. Dans la liste Méthodes disponibles, sélectionnez un algorithme de cryptage, puis cliquez sur le bouton fléché vers la droite (<) pour le déplacer vers la liste Méthodes sélectionnées. L'ordre d'apparition des méthodes dans cette liste détermine l'ordre de préférence pour la négociation. Oracle Advanced Security prend en charge les algorithmes suivants :

    Elément Description
    RC4_256 clé sur 256 bits
    3DES168 DES triple avec option trois clés (168 bits)
    RC4_128 clé sur 128 bits
    3DES112 DES triple avec option deux clés (112 bits)
    RC4_56 clé sur 56 bits
    DES clé standard sur 56 bits
    RC4_40 clé sur 40 bits
    DES40 clé sur 40 bits

  8. Choisissez Fichier > Enregistrer la configuration réseau.

Remarque : Les différents algorithmes de cryptage et les différentes tailles de clé d'Oracle Advanced Security ne sont utilisables qu'aux Etats-Unis.

配置监听程序的口令验证

可以配置监听程序的安全性。这样，从监听程序控制实用程序使用的权限操作 (例如，保存配置更改或停止监听程序) 将需要口令。Oracle Net Manager 使您能够设置加密口令。

要为监听程序设置口令，请执行以下操作：

  1. a. 在导航器窗格中，展开 Oracle Net 配置 > 本地 > 监听程序。

  2. 选择监听程序。

  3. 在右侧窗格的列表中，选择一般参数。

  4. 选择验证选项卡。

  5. 选择需要监听程序操作口令。

  6. 在口令字段中输入口令，并在确认口令字段中再次输入此口令。

  7. 选择文件 > 保存网络配置。

Menu Commande

Le menu Commande vous permet d'effectuer les opérations suivantes :

Répertoire

Ce sous-menu contient des commandes permettant d'effectuer des opérations sur un serveur d'annuaire :

Elément Description
Changer contexte courant Vous permet de sélectionner un contexte différent du contexte Oracle en cours. Le contexte Oracle est une sous-arborescence de l'annuaire dans lequel sont stockées les entrées Oracle, notamment les services de base de données et les noms de service réseau à utiliser pour la résolution de noms d'annuaire. Lorsque vous sélectionnez un nouveau contexte Oracle, les objets qui se trouvent dans le dossier Configuration Oracle Net > Annuaire > Résolution de noms de service du navigateur s'affichent pour le nouveau contexte Oracle.
Définir l'authentification Vous permet de définir les éléments d'authentification du serveur d'annuaire.
Exporter les noms de service réseau Vous permet de lancer l'assistant Migration du serveur d'annuaire. Ce dernier exporte les noms de service réseau enregistrés dans un fichier tnsnames.ora vers un serveur d'annuaire.

Remarque : le menu Annuaire ne s'affiche que si l'accès à l'annuaire a été configuré. Pour plus d'informations sur la configuration de l'accès à l'annuaire et les autres conditions, reportez-vous à Exigences relatives à la résolution de noms d'annuaire pour Oracle Net Manager.

Tester un service

Vous permet de vous connecter à un service de base de données à l'aide des informations de descripteur de connexion enregistrées dans un fichier tnsnames.ora ou d'un serveur d'annuaire.

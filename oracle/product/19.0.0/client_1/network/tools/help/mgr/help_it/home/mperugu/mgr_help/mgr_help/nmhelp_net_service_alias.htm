Alias di servizio di rete

Un nome alternativo per un oggetto di denominazione delle directory in un server delle directory. Un server delle directory memorizza alias del servizio di rete per ciascun nome di servizio di rete o di database definito. Una voce alias del servizio di rete non dispone di informazioni del descrittore di connessione. Fa invece riferimento alla posizione dell'oggetto per il quale è stato creato. Quando un client richiede una ricerca all'interno di una directory di un alias del servizio di rete, la directory determina che la voce è un alias del servizio di rete e completa la ricerca come se si trattasse della voce di riferimento.

Microsoft Active Directory

Service d'annuaire conforme à LDAP et fourni avec le serveur Windows 2000. Il stocke des informations sur les objets de réseau et les met à la disposition des utilisateurs et des administrateurs de réseau. Active Directory permet également aux utilisateurs d'accéder aux ressources du réseau via un simple processus de connexion.

Active Directory peut être configuré comme une méthode de résolution de noms d'annuairee pour stocker des informations de service auxquelles les clients peuvent accéder.

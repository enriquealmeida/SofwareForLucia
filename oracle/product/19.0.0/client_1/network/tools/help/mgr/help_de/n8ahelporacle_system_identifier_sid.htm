Oracle-System-ID (SID)

Ein Name, der eine bestimmte Instanz einer laufenden Datenbank vor Oracle8i identifiziert. Bei jeder Datenbank ist mindestens eine Instanz vorhanden, die die Datenbank referenziert.

Bei Datenbanken vor Oracle 8i wurde die Datenbank mit einem SID identifiziert. Der SID war in dem Teil mit dem Verbindungsdeskriptor in einer Datei tnsnames.ora und in der Definition des Listeners in der Datei listener.ora enthalten. Verbindungsdeskriptoren, die von Clients verwendet werden, um die Verbindung zu einer Oracle8- oder Oracle7-Datenbank herzustellen, sollten mit dem Parameter SID konfiguriert werden.

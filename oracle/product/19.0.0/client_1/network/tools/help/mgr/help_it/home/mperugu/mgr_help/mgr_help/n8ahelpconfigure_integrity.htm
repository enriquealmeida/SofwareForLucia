Configurazione dell'integrità dei dati

La verifica dell'integrità dei dati è il processo mediante il quale una quantità viene calcolata in base al contenuto di un messaggio. Il mittente del messaggio calcola il checksum. Il valore del checksum viene incluso nel pacchetto del messaggio. Alla ricezione, il destinatario ricalcola il checksum del messaggio. La verifica dell'integrità dei dati{1} {2}è il processo mediante il quale una quantità viene calcolata in base al contenuto di un messaggio. Il mittente del messaggio calcola il checksum, il cui valore viene incluso nel pacchetto del messaggio. Alla ricezione, il destinatario ricalcola il checksum del messaggio. Per avere la certezza che il messaggio non sia stato alterato durante la trasmissione, i due valori di checksum devono corrispondere.

Per configurare l'integrità dei dati, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Oracle Advanced Security.

  3. Selezionare la scheda Integrità.

  4. Abilitare l'integrità dei dati del client o del server scegliendo rispettivamente la casella di controllo Client o Server.

  5. Specificare il livello di integrità adottato dal client e dal server. Selezionare Livello checksum sia per il client che per il server. I diversi livelli sono descritti di seguito.

    Elemento Descrizione
    accettato Il servizio sarà attivo se dall'altra parte della connessione viene specificato "necessario" o "richiesto" ed è disponibile un algoritmo compatibile; in caso contrario, il servizio sarà inattivo.
    rifiutato Il servizio non deve essere attivo. Se dall'altra parte è specificato "obbligatorio", la connessione non viene stabilita.
    richiesto Il servizio sarà attivo se dall'altra parte della connessione viene specificato "accettato", "necessario" o "richiesto" ed è disponibile un algoritmo compatibile; in caso contrario, il servizio sarà inattivo.
    necessario Il servizio deve essere attivo. Se dall'altra parte viene specificato "rifiutato" o se non è disponibile un algoritmo compatibile, la connessione non viene stabilita.

  6. Fare clic su File > Salva configurazione di rete.

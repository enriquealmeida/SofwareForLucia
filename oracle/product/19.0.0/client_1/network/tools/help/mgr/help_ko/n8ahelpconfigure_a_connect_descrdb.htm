Oracle Rdb 데이터베이스에 대한 접속 기술자 구성

Oracle RDB 데이터베이스는 Digital의 64비트 플랫폼용 데이터베이스입니다. Oracle RDB 서버는 Oracle 데이터베이스와 동일한 방식으로 클라이언트에 나타납니다. Oracle RDB 데이터베이스는 자체 리스너를 포함하고 있습니다. 클라이언트 구성은 Oracle RDB의 특정 매개변수로 네트 서비스 이름을 생성하여 데이터베이스에 액세스하는 작업으로 구성됩니다.

Oracle RDB 데이터베이스에 대한 클라이언트를 구성하려면 다음과 같이 하십시오.

  1. 네트 서비스 이름 생성에 설명된 대로 네트 서비스 이름을 생성합니다.

  2. 오른쪽 창의 서비스 식별 상자에서 고급을 선택합니다. 고급 서비스 옵션 대화상자가 나타납니다.

  3. RDB 데이터베이스 필드에 Oracle RDB 데이터베이스 파일 이름을 입력합니다.

  4. 선택적으로 전역 데이터베이스 이름 필드에 전역 데이터베이스 이름을 입력하고, 필요에 따라 서비스 유형 필드에 서비스 유형을 입력합니다.

  5. [고급 서비스 옵션] 대화상자에서 확인을 선택합니다.

  6. 오른쪽 창에서 적용을 선택합니다.

  7. 파일 > 네트워크 구성 저장을 선택합니다.

자세한 내용은 Oracle Rdb 설명서를 참조하십시오.

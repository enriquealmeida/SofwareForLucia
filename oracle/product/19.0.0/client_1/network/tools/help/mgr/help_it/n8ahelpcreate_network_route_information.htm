Modifica delle informazioni di instradamento di rete per un servizio di database

Nota: solo gli utenti che sono membri del gruppo OracleNetAdmins o OracleContextAdmins possono modificare le informazioni di rete per un servizio di database in una directory.

Dopo il completamento della registrazione del database con la directory, l'Assistente alla configurazione del database crea una voce del servizio di database nella directory. Per impostazione predefinita, questa voce contiene le informazioni dell'instradamento di rete, compresa la posizione del listener tramite un indirizzo di protocollo. È possibile creare nuovamente queste informazioni, nel caso in cui siano state rimosse, oppure modificare le informazioni dell'instradamento di rete esistenti.

Per creare o modificare le informazioni di instradamento di rete per un servizio di database, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Directory > Denominazione dei servizi.

  2. Selezionare il servizio di database. Nel riquadro di destra è visualizzato il nome di servizio di destinazione corrente.

  3. Nella casella Configurazione indirizzi fare clic sul segno più (+). Viene visualizzata una nuova scheda Indirizzo.

  4. Selezionare un protocollo, quindi immettere i parametri appropriati per il protocollo selezionato.

  5. Nel riquadro di destra, scegliere Applica.

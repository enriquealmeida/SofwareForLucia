Configurando Autenticação de Senha para o Listener

O listener pode ser configurado com segurança. Dessa forma, operações privilegiadas, como salvar alterações de configuração ou interromper o listener, utilizadas a partir do utilitário Listener Control exigirão uma senha. O Oracle Net Manager permite que você defina uma senha criptografada.

Para definir uma senha para o listener:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

  2. Selecione um listener.

  3. Na lista do painel direito, selecione Parâmetros Gerais.

  4. Selecione a guia Autenticação.

  5. Escolha Exigir uma Senha para as Operações de Listener.

  6. Informe a senha no campoSenha e digite-a novamente no campo Confirmar Senha.

  7. Escolha Arquivo > Salvar Configuração de Rede.

Assistente de Configuração do Oracle Net

Uma ferramenta pós-instalação que configura componentes de rede básicos após a instalação, incluindo:

  • Nomes de listeners e endereço de protocolo

  • Métodos de nomeação que o cliente utilizará para resolver identificadores de conexão

  • Nomes de serviço de rede em um arquivo tnsnames.ora

  • Uso do Servidor de diretórios

O Assistente de Configuração do Oracle Net é executado automaticamente após a instalação do software, conforme descrito no seu guia de instalação Oracle. Ele pode ser utilizado no cliente ou no servidor.

Também pode ser executado no modo independente para configurar o uso de métodos de nomeação, o listener, os nomes de serviço de rede no arquivo tnsnames.ora e o uso do serviço de diretório.

Para executar o Assistente de Configuração do Oracle Net no modo independente:

  • No UNIX, execute netmgr em $ORACLE_HOME/bin.

  • Nos sistemas operacionais Windows, escolha Iniciar > Programas > Oracle - NOME_DO_HOME > Ferramentas de Configuração e de Migração > Assistente de Configuração de Rede.

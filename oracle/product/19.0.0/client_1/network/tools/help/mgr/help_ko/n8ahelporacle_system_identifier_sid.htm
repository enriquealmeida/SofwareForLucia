Oracle SID(시스템 식별자)

실행 중인 Oracle8i 이전 버전 데이터베이스의 특정 인스턴스를 식별하는 이름입니다. 모든 데이터베이스에는 데이터베이스를 참조하는 하나 이상의 인스턴스가 있습니다.

Oracle8i 이전 버전 데이터베이스의 경우 SID는 데이터베이스를 식별하는 데 사용되었습니다. SID는 tnsnames.ora 파일의 접속 기술자 부분 및 listener.ora 파일의 리스너 정의에 포함되어 있습니다. 클라이언트가 Oracle8 또는 Oracle7 데이터베이스에 접속하는 데 사용하는 접속 기술자는 SID 매개변수로 구성되어야 합니다.

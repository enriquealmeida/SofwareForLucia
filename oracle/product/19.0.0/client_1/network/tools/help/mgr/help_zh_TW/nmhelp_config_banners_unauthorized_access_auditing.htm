設定用於未經授權的存取與稽核使用者動作的標幟

您應建立並設定標幟來警告使用者發生未經授權的存取與可能的使用者動作稽核情況。當從屬端應用程式登入資料庫時，系統變會將這些通知提供給從屬端應用程式。

若要設定顯示這些標幟，請在資料庫伺服器端設定下列 sqlnet.ora 參數指向包含標幟資訊的文字檔：

  • SEC_USER_UNAUTHORIZED_ACCESS_BANNER。例如：SEC_USER_UNAUTHORIZED_ACCESS_BANNER = /opt/Oracle/11g/dbs/unauthaccess.txt

  • SEC_USER_AUDIT_ACTION_BANNER。例如：SEC_USER_AUDIT_ACTION_BANNER = /opt/Oracle/11g/dbs/auditactions.txt

預設並不會設定這些參數。

設定這些參數後，Oracle Call Interface 應用程式必須使用適當的 OCI API 來擷取這些標幟並呈現給一般使用者。

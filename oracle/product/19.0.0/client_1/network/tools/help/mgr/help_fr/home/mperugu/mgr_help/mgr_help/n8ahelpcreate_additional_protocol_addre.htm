Création d'adresses de protocole supplémentaires

Plusieurs itinéraires réseau ou adresses de protocole peuvent accéder à un service de base de données.

Pour ajouter une adresse de protocole réseau à un service de base de données ou à un nom de service réseau existant :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Annuaire ou Local > Résolution de noms de service.

  2. Sélectionnez un nom de service réseau ou un service de base de données. Le panneau de droite affiche le service de destination et la liste d'adresses courants.

  3. Dans la case Configuration de l'adresse, choisissez plus (+). Un nouvel onglet Adresse apparaît.

  4. Sélectionnez un protocole et indiquez les informations appropriées sur les paramètres de protocole du protocole sélectionné.

  5. A l'aide du bouton "flèche vers le haut" ou "flèche vers le bas", modifiez l'ordre des adresses en fonction de l'emplacement qu'elles doivent occuper dans la liste. A moins que les options de plusieurs adresses aient été configurées, seule la première adresse de la liste est contactée. Pour choisir un ordre différent, reportez-vous à Configuration des options de plusieurs adresses.

  6. Dans le panneau de droite, choisissez Appliquer.

  7. Si vous effectuez ces modifications dans le dossier Local, choisissez Fichier > Enregistrer la configuration réseau.

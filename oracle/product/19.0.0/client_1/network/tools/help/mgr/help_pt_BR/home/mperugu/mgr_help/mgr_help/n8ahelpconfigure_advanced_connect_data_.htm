Configurando Opções Avançadas para Dados de Conexão

Além do nome do serviço ou do Oracle System Identifier (SID) do banco de dados, você pode, opcionalmente, configurar a seção CONNECT_DATA de um descritor de conexão com as seguintes opções:

  • O Nome da Instância para estabelecer conexão com uma instância de banco de dados específica

  • O Oracle home para iniciar o banco de dados na rede

  • O Tamanho da Unidade de Dados de Sessão para especificar o tamanho do buffer

  • O Servidor Dedicado para estabelecer conexão com um processo de servidor dedicado

  • Serviços Heterogêneos para acessar um sistema não Oracle

  • Definições do Oracle Rdb para acessar um banco de dados Oracle Rdb

Para configurar as opções de conexão avançadas para um nome de serviço de rede, faça o seguinte:

  1. No painel do navegador, expanda Configuração do Oracle Net > Diretório > Local ou Nomeação de Serviço.

  2. Selecione um nome do serviço de rede ou um serviço de banco de dados. O painel direito exibirá o serviço de destino atual e uma lista de endereços.

  3. Na caixa Identificação de Serviço, escolha Avançado. A caixa de diálogo Opções Avançadas de Serviço será exibida.

  4. Preencha campos ou selecione opções conforme apropriado e, em seguida, escolha OK.

  5. No painel direito, escolha Aplicar.

  6. Se você estiver fazendo essas alterações na pasta Local, escolha Arquivo > Salvar Configuração de Rede.

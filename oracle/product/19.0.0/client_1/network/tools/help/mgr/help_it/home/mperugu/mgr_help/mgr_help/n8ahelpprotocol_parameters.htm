Parametri di protocollo

Il listener, il server Oracle Names e Oracle Connection Manager sono identificati da un indirizzo di protocollo. Anche i client che desiderano utilizzare questi servizi devono essere configurati con lo stesso indirizzo. Nella tabella riportata di seguito sono descritti i parametri utilizzati dai protocolli Oracle supportati.

Protocollo Parametro Descrizione
IPC PROTOCOL Specificare ipc come valore.
IPC KEY Specificare un nome univoco per il servizio. Oracle Corporation consiglia di utilizzare il nome di servizio o il SID (ID di sistema) Oracle del servizio.
Named Pipes PROTOCOL Specificare nmp come valore.
Named Pipes SERVER Specificare il nome del computer server Oracle.
Named Pipes PIPE Specificare il nome pipe da utilizzare per la connessione al server. Può trattarsi di un nome qualsiasi.
TCP/IP e TCP/IP con SSL PROTOCOL Specificare tcp come valore per TCP/IP oppure tcps come valore per TCP/IP con SSL.
TCP/IP e TCP/IP con SSL HOST Specificare il nome host o l'indirizzo IP del computer.
TCP/IP e TCP/IP con SSL PORT Specificare il numero della porta di ascolto.

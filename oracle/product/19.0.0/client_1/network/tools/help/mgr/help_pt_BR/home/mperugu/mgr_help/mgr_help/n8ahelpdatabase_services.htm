Serviços de Banco de Dados

A guia Serviços de Banco de Dados permite que você especifique os serviços de banco de dados para os quais o listener deverá receber solicitações de conexão. O listener precisa dessas informações para estabelecer conexão de um cliente com o banco de dados.

As instâncias do Oracle9i e do Oracle8i utilizam registro de serviço dinâmico para informar o listener sobre seus serviços de banco de dados. Os bancos de dados Oracle7 e Oracle8 release 8.0, e as ferramentas do Oracle Enterprise Manager, requerem a configuração de serviço estático no arquivo listener.ora .

Escolha Adicionar Banco de Dados para adicionar informações sobre serviço de banco de dados.

Nome de Banco de Dados Global

Informe o nome e o nome de domínio do banco de dados fornecidos pelo valor do parâmetro SERVICE_NAMES ou dos parâmetros DB_NAME e DB_DOMAIN do arquivo de parâmetro de inicialização.

Diretório Oracle Home

Informe a localização do diretório Oracle Home do banco de dados.

Observação: O Windows NT já conhece o Oracle home. Portanto, este campo não é necessário para uma conexão com um serviço do Windows NT.

SID

Informe o SID (Identificador de Sistema Oracle) da instância.

Botão Adicionar Banco de Dados

Escolha para configurar informações de serviço para outro banco de dados. Uma nova guia Banco de Dados é exibida, na qual você pode especificar informações sobre serviço para outro banco de dados.

Botão Remover Banco de Dados

Escolha para deletar a guia Banco de Dados selecionada e suas respectivas informações de serviço.

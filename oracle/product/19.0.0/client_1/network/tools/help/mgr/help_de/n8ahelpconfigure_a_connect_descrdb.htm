Verbindungsdeskriptor für eine Oracle Rdb-Datenbank konfigurieren

Eine Oracle Rdb-Datenbank ist eine Datenbank für die 64-Bit-Plattform von Digital. Oracle Rdb-Server erscheinen den Clients wie Oracle-Datenbanken. Eine Oracle Rdb-Datenbank verfügt über einen eigenen Listener. Bei der Client-Konfiguration wird ein Net Service-Name mit Oracle Rdb-spezifischen Parametern für den Zugriff zu der Datenbank erstellt.

So konfigurieren Sie den Client für eine Oracle Rdb-Datenbank:

  1. Erstellen Sie einen Net Service Name, wie unter Net Service Name erstellen beschrieben.

  2. Wählen Sie im rechten Fensterbereich Dienst-Identifikation die Option Erweitert. Daraufhin wird das Dialogfeld Erweiterte Dienstoptionen angezeigt.

  3. Geben Sie den Namen einer Oracle Rdb-Datenbank in das Feld Rdb-Datenbank ein.

  4. Optional geben Sie den globalen Datenbanknamen in das Feld Globaler Datenbankname und gegebenenfalls den Diensttyp in das Feld Diensttyp ein.

  5. Klicken Sie im Dialogfeld "Erweiterte Dienstoptionen" auf OK.

  6. Wählen Sie Anwenden im rechten Fensterbereich.

  7. Wählen Sie Datei >Netzwerkkonfiguration speichern.

Weitere Informationen finden Sie in der Oracle Rdb-Dokumentation.

Oracle Names-Einträge in Directory-Server exportieren

Oracle Names wird in diesem Release nicht mehr unterstützt. Mit einem 9.2 Release von Oracle Names müssen Sie Datenbank- und Aliasobjekte, die in einem Oracle Names Server gespeichert sind, in einen Directory-Server oder indirekt in eine Datei im LDAP Data Interchange Format (LDIF) exportieren, die dann in den Directory-Server geladen werden kann. Wenn Sie die Datenbankobjekte exportieren, wird nur der Adress-Datensatz (A.SMD) exportiert. Der Adress-Datensatz enthält die Verbindungsdeskriptor-Informationen für das Datenbankobjekt. Datenbankobjekte werden als Net Service-Namen exportiert. Aliasnamen werden als Net Service Alias exportiert. Daten werden aus einer angegebenen Domain exportiert.

Weitere Informationen finden Sie im Kapitel Configuring Naming Methods im Oracle10i Net Administrator's Guide.

Genaue Anweisungen für das Exportieren von Daten aus einem Oracle Names Server finden Sie im Oracle9i Net Services Administrator's Guide für Release 9.2.

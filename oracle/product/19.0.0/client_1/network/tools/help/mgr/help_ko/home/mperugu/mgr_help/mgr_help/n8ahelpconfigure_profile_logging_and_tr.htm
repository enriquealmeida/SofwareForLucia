프로파일 로깅 및 추적 구성

로깅과 추적에 대해 클라이언트를 구성할 수 있습니다.

로그 정보는 일반적으로 UNIX의 경우 $ORACLE_HOME/network/log , Windows 플랫폼의 경우 ORACLE_HOME\network\log 에 있는 sqlnet.log 파일에 저장됩니다.

추적 정보는 기본적으로 클라이언트의 경우 sqlnet.trc에 저장되고 서버의 경우 svr_pid..trc에 저장되며, 이러한 파일은 UNIX의 경우 $ORACLE_HOME/network/trace, Windows 플랫폼의 경우 ORACLE_HOME \network\trace에 있습니다.

로깅 및 추적을 사용으로 설정하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창의 목록에서 일반 사항을 선택합니다.

  3. 로깅 또는 추적 탭을 선택합니다.

  4. 로깅 또는 추적 설정을 지정합니다.

  5. 파일 > 네트워크 구성 저장을 선택합니다.

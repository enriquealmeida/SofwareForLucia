設定監聽器位址

監聽器必須設定一或多個協定位址。監聽器會從設定相同位址的從屬端接收連線要求。

設定監聽器的協定位址：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

  2. 選取監聽器。

  3. 從右窗格的清單中，選取監聽位置。

  4. 選擇新增位址。即會顯示一個新的位址頁籤。

  5. 選取一個協定，然後輸入選取之協定的適當協定參數資訊。

    設定監聽器使用 TCP/IP 監聽 Oracle Net 從屬端時，應選擇預設的連接埠或 1521。如果沒這麼做，就必須在初始化參數檔設定 LOCAL_LISTENER 參數，並透過命名方法解析監聽器名稱。

  6. 如果電腦有一個以上的 IP 位址，而您希望監聽器監聽所有可用的 IP 位址，請設定 TCP/IP 或 TCP/IP 與 SSL，然後在主機欄位輸入電腦的名稱。

  7. 或者，按一下顯示進階來指定設定傳送和接收階段作業之緩衝區的設定值。

  8. 選擇檔案 > 儲存網路組態。

如需有關設定非預設連接埠的詳細資訊，請參閱 Oracle 10i Net Administrator's Guide 中的 Configuring and Administering the Listener 章節。

如需有關設定 I/O 緩衝區空間的詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Optimizing Performance 章節。

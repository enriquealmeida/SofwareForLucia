Struttura di directory e gerarchie di cartelle

Oracle Net Manager rappresenta graficamente una struttura degli oggetti di rete e del loro contenuto, organizzata in gerarchie di cartelle. È possibile utilizzare il riquadro del Navigator per visualizzare, modificare, aggiungere ed eliminare gli oggetti nelle cartelle.

Quando si espande una cartella, viene visualizzato una lista nidificata di oggetti e cartelle. Quando si seleziona un oggetto, vengono visualizzate le informazioni corrispondenti nel riquadro destro di Oracle Net Manager.

Directory

Visualizza i nomi semplici, definiti identificativi di connessione, configurati in un server delle directory.

Nota: la cartella Directory viene visualizzata soltanto se è stato configurato l'accesso alle directory. Per ulteriori informazioni sulla configurazione dell'accesso alle directory e sugli altri requisiti, vedere Requisiti di denominazione delle directory per Oracle Net Manager.

Locale

Visualizza gli elementi della rete configurati nei file di configurazione locali:

  • nomi di servizi di rete nel file tnsnames.ora;

  • listener nel file listener.ora;

  • profili nel file sqlnet.ora.

Panoramica sulla configurazione del listener

Questo argomento è costituito dalle sezioni seguenti.

Configurazione durante l'installazione

Durante un'installazione Enterprise Edition o Standard Edition sul database server, l'Assistente alla configurazione di Oracle Net configura un listener denominato LISTENER associato a un indirizzo di protocollo di ascolto TCP/IP per il database Oracle. Durante un'installazione personalizzata, l'Assistente alla configurazione di Oracle Net richiede di configurare un nome di listener e un indirizzo di protocollo per la propria scelta.

Inoltre, viene configurato automaticamente un indirizzo di protocollo IPC di ascolto per le chiamate alle procedure esterne, indipendentemente dal tipo di installazione.

L'Assistente alla configurazione di Oracle Net inoltre configura automaticamente le informazioni sul servizio per le procedure esterne nel file listener.ora.

Configurazione con Oracle Net Manager

Oracle Net Manager consente di:

  • creare un listener;

  • configurare le informazioni sul servizio per i database Oracle8 o versioni precedenti;

  • configurare le informazioni sul servizio per servizi del database non Oracle, tra cui procedure esterne e servizi eterogenei;

  • abilitare le funzioni di trace e log;

  • abilitare l'autenticazione tramite password per la utility Controllo del listener.

Per accedere alla configurazione del listener:

Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

Per ulteriori informazioni, vedere il capitolo Configuring and Administering the Listener del manuale Oracle10i Net Administrator's Guide.

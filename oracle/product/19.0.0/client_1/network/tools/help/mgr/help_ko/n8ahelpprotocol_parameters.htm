프로토콜 매개변수

리스너, Oracle Names Server 및 Oracle Connection Manager는 프로토콜 주소로 식별됩니다. 이러한 서비스를 사용하려면 클라이언트도 동일한 주소로 구성되어야 합니다. 다음 표는 지원되는 Oracle 프로토콜에서 사용하는 매개변수에 대한 설명합니다.

프로토콜 매개변수 설명
IPC PROTOCOL ipc를 값으로 지정합니다.
IPC KEY 서비스에 대한 고유 이름을 지정합니다. Oracle은 서비스 이름 또는 서비스의 Oracle SID(시스템 식별자) 사용을 권장합니다.
Named Pipes PROTOCOL nmp를 값으로 지정합니다.
Named Pipes SERVER Oracle 서버 컴퓨터의 이름을 지정합니다.
Named Pipes PIPE 데이터베이스 서버에 접속할 파이프 이름을 지정합니다. 이 이름은 모든 임의의 이름이 될 수 있습니다.
TCP/IP 및 SSL 사용 TCP/IP PROTOCOL TCP/IP의 경우 tcp를, SSL을 사용하는 TCP/IP의 경우 tcps를 값으로 지정합니다.
TCP/IP 및 SSL 사용 TCP/IP HOST 호스트 이름 또는 컴퓨터의 IP 주소를 지정합니다.
TCP/IP 및 SSL 사용 TCP/IP PORT 수신 포트 번호를 지정합니다.

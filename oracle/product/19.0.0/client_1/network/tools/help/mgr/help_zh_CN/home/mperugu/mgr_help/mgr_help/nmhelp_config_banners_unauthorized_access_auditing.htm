配置用于未授权访问和审计用户操作的标帜

应创建并配置标帜以提醒用户留意未授权的访问以及可能会对用户操作进行审计。客户机应用程序登录数据库时，这些通知将可供该应用程序使用。

要配置要显示的这些标帜，请将数据库服务器端的以下 sqlnet.ora 参数设置为指向包含标帜信息的文本文件：

  • SEC_USER_UNAUTHORIZED_ACCESS_BANNER。例如：SEC_USER_UNAUTHORIZED_ACCESS_BANNER = /opt/Oracle/11g/dbs/unauthaccess.txt

  • SEC_USER_AUDIT_ACTION_BANNER。例如：SEC_USER_AUDIT_ACTION_BANNER = /opt/Oracle/11g/dbs/auditactions.txt

默认情况下不会设置这些参数。

设置这些参数之后，Oracle Call Interface 应用程序必须使用相应的 OCI API 检索这些标帜并将其显示给最终用户。

工具栏

位于 Oracle Net Manager 左侧的工具栏包含与文件夹层次结构中的网络对象相对应的图标。是否启用工具栏按钮取决于窗格中所查看的或所选择的对象。将鼠标移动到工具栏的图标上来显示该图标功能的说明。

工具栏包含以下图标：

图标 说明
添加 选择此项以便为此计算机创建新的网络服务名、监听程序和 Oracle Names Server。
删除 选择此项以删除服务命名和监听程序对象。
验证连接 选择以验证所选的服务命名对象到数据库服务的连接。在连接测试过程中，通过使用存储在 tnsnames.ora 文件或目录服务器中的连接描述符信息来建立与数据库服务的连接。
帮助 选择此项以显示联机帮助。

注：有些命令只有在选择合适类型的对象时才能使用。例如，如果需要创建监听程序，首先必须选择监听程序文件夹。

고급 서비스 옵션 대화상자

고급 서비스 옵션 대화상자를 사용하여 고급 CONNECT_DATA 매개변수를 구성할 수 있습니다.

추가 서비스 설정

요소 설명
인스턴스 이름 액세스할 데이터베이스 인스턴스의 이름을 입력합니다. 인스턴스 이름은 초기화 매개변수 파일의 INSTANCE_NAME 매개변수 값에서 얻을 수 있습니다.
SDU(세션 데이터 단위) 세션 데이터 단위 크기를 입력하여 네트워크를 통해 전송되는 패킷 관련 성능 특성을 변경합니다. SDU 크기를 일반 전송 프레임 크기의 배수로 설정합니다. 이더넷 프레임의 일반 크기는 1500이지만 사용 가능한 크기는 1460입니다. 암호화를 사용하지 않으면 1바이트가 예약됩니다. 사용할 알고리즘은 암호화를 사용하지 않는 경우 1460에 1을 더한 값의 배수이고, 이더넷의 경우 1460의 배수입니다.
이기종 서비스에 사용 Oracle 서버가 이기종 서비스를 통해 비Oracle 시스템에 액세스하도록 하려면 선택합니다.

Oracle RDB 설정

요소 설명
Oracle RDB 설정 Oracle RDB 데이터베이스의 파일 이름을 지정합니다.
서비스 유형 Oracle RDB 데이터베이스에 사용할 서비스 유형을 입력합니다.
전역 데이터베이스 이름 Oracle RDB 데이터베이스에 대한 식별자를 입력합니다.

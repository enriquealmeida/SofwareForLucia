General: Direccionamiento

El separador Direccionamiento permite dirigir solicitudes de conexión desde esta computadora a un proceso concreto en cualquier momento.

Usar Siempre Servidor Dedicado

Seleccione esta opción para hacer que el listener origine un servidor dedicado para todas las sesiones de red de este cliente. Para ello, agrega (SERVER= DEDICATED) a la parte CONNECT_DATA de los descriptores de conexión. Las conexiones de este cliente utilizarán servidores dedicados, incluso si se ha configurado un servidor compartido.

Con esta opción, las conexiones serán más rápidas, pero el servidor será más lento. Oracle Corporation recomienda utilizar esta opción si realiza transacciones de base de datos masivas que dejen muy poco tiempo libre.

Nota: Si esta opción está activada, anula la opción Tipo de Conexión de la hoja de propiedades Identificación de Servicios.

Dar Preferencia a Direccionamientos de Connection Manager

Seleccione esta opción para enviar el cliente a Oracle Connection Manager siempre que sea posible. Si no hay direcciones de Oracle Connection Manager disponibles, las conexiones se distribuyen mediante la dirección del listener disponible.

Routage de la source

Indique que tous les itinéraires réseau sont utilisés. Le routage de la source est généralement utilisé dans les environnements dans lesquels de nombreuses connexions doivent accéder au même service. Oracle Net offre un routage via Oracle Connection Manager.

Pour plus d'informations sur Oracle Connection Manager, reportez-vous au chapitre Networking Challenges in the Internet Age dans le Guide de l'administrateur Oracle10i Net.

Configurando os Métodos de Autenticação do Oracle Advanced Security

Métodos de autenticação centralizados e seguros permitem que você tenha uma alta confiança na identidade dos usuários, clientes e servidores em ambientes distribuídos. Os métodos de autenticação de rede também podem oferecer o benefício do sign-on único para os usuários.

Use o Oracle Net Manager para configurar os seguintes adaptadores de autenticação:

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • Autenticação nativa do Windows NT

Observação: Se você tiver configurado o protocolo TCP/IP com SSL, a SSL (Secure Sockets Layer) será configurada por default. Qualquer método de autenticação escolhido nesta guia substituirá os recursos de autenticação da SSL. Consulte a guia SSL para configurar as opções de SSL adicionais. Se você estiver usando o protocolo TCP/IP com SSL, não use nenhum outro método de autenticação.

Para configurar os métodos de autenticação:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

  2. Na lista do painel direito, selecione Oracle Advanced Security.

  3. Selecione a guia Autenticação.

  4. Na lista Métodos Disponíveis, selecione o método de autenticação que deseja usar e, em seguida, escolha o botão de seta para a direita (>).

  5. Ordene os métodos de autenticação de acordo com o método que deseja usar. Selecione o método de autenticação na lista Métodos Selecionados e use o botão Promover ou Rebaixar para mover a seleção para cima ou para baixo na lista.

  6. Escolha a guia Outros Parâmetros para configurar parâmetros adicionais para os seguintes métodos de autenticação:

    • Kerberos

    • CyberSAFE

    • RADIUS

  7. Escolha Arquivo > Salvar Configuração de Rede.

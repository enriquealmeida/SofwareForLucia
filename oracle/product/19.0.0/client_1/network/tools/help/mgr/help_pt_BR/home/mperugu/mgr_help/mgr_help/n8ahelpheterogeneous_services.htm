Serviços Heterogêneos

Um componente integrado que fornece a tecnologia genérica para acessar sistemas não Oracle a partir do servidor de banco de dados Oracle. Os serviços heterogêneos permitem que você:

  • Use o Oracle SQL para acessar de forma transparente dados armazenados em sistemas não Oracle se os dados residirem em um servidor Oracle.

  • Use chamadas de procedimento Oracle para acessar de forma transparente serviços, APIs (interfaces de programas aplicativos) ou sistemas não Oracle a partir do seu ambiente distribuído Oracle.

Para iniciar uma conexão com o sistema não Oracle, o banco de dados Oracle8i inicia um processo agente através do listener do gateway. Para que o banco de dados Oracle8i estabeleça conexão com o agente, o servidor deverá ser configurado com um nome do serviço de rede para acessar o agente, e o listener deverá ser configurado com informações de serviço sobre o agente.

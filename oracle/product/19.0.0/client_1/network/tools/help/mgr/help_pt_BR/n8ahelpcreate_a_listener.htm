Criando um Listener

Deve haver um listener no servidor, para que o banco de dados receba conexões.

Observação: durante o processo de instalação, é criado um listener padrão cujo nome é LISTENER.

Para criar um listener:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

  2. Escolha o sinal de adição (+) na barra de ferramentas ou Editar > Criar. A caixa de diálogo Escolher Nome de Listener será exibida.

  3. Aceite o nome do listener fornecido no campo Nome do Listener ou edite o campo com outro nome. Por default, o nome do listener é LISTENER para o primeiro listener. Os outros nomes utilizarão LISTENER n como default. O nome do listener deve ser exclusivo em um nó.

  4. Escolha OK.

  5. Execute as etapas de 4 a 6 em Configurar Endereços de Listener.

Serveur partagé

Serveur configuré pour permettre à un grand nombre de processus utilisateur de partager un petit nombre de processus serveur, de sorte que le nombre d'utilisateurs puisse évoluer. Avec cette configuration, de nombreux processus utilisateur se connectent à un répartiteur. Ce répartiteur achemine les demandes de session réseau entrantes vers une file d'attente commune. Un processus serveur partagé inactif, faisant partie d'un pool partagé de processus serveur, prend une demande dans la file d'attente. Cela signifie qu'un petit pool de processus serveur peut desservir un grand nombre de clients. Voir aussi serveur dédié.

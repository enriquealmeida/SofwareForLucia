Arquivo sqlnet.ora

Um arquivo de configuração para o cliente ou servidor, que especifica:

  • O domínio do cliente a ser anexado a nomes de serviços não-qualificados ou nomes de serviços de rede

  • A ordem dos métodos de nomeação que o cliente deverá usar ao resolver um nome

  • Os recursos de log e rastreamento a serem usados

  • Rota das conexões

  • servidores Preferenciais Oracle Names

  • Parâmetros de nomeação externa

  • Os parâmetros do Oracle Advanced Security

O arquivo sqlnet.ora deverá residir em uma das seguintes localizações:

  1. O diretório especificado pela variável de ambiente TNS_ADMIN. Se a variável de ambiente de rede TNS_ADMIN não tiver sido definida como uma variável no Windows NT, ela poderá estar no registro.

  2. O diretório $ORACLE_HOME/network/admin do UNIX ou o diretório ORACLE_HOME\network\admin de sistemas operacionais do Windows.

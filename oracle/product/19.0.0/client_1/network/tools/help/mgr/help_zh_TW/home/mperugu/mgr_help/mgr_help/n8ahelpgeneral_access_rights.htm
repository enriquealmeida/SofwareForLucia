一般：存取權

您可以使用存取權頁籤，指定允許或拒絕哪些從屬端存取資料庫。

檢查 TCP/IP 從屬端存取權

如果您想指定哪些從屬端允許或拒絕資料庫存取，請選取此核取方塊。如果要停用此功能，請取消選取此核取方塊。如果取消選取核取方塊，則允許所有從屬端存取。

無法存取的從屬端

輸入主機名稱或 IP 位址，指定拒絕哪些從屬端存取資料庫。請使用逗號來分隔放在同一行上的項目。

允許存取的從屬端

輸入主機名稱或 IP 位址，指定允許哪些從屬端存取資料庫。請使用逗號來分隔放在同一行上的項目。

注意：允許存取的從屬端優先順序高於無法存取的從屬端。

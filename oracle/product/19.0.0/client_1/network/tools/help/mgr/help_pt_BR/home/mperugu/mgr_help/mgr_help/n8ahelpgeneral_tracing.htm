Geral: Rastreamento

A tab Rastreamento permite que você defina parâmetros de rastreamento para este computador. Se este computador for um cliente, use os campos Informações sobre o Cliente. Se for um servidor, use os campos de Informações sobre o Servidor.

O rastreamento do cliente controla as operações que o cliente executa. O rastreamento do servidor acompanha as operações que o servidor executa durante uma conexão de rede.

Nível de Rastreamento

A partir da lista, selecione o nível de rastreamento:

OFF -- O rastreamento não está ativado. Desativado é a definição padrão.

USER -- O rastreamento é definido em um nível apropriado para os usuários. Os rastreamentos identificam condições de erro induzidas pelo usuário.

ADMIN -- O rastreamento é definido como um nível apropriado para os administradores de bancos de dados. Os rastreamentos identificam problemas específicos de instalação.

SUPPORT -- O rastreamento é definido como um nível adequado para suporte ao cliente.

Os arquivos de log podem se tornar muito grandes. A Oracle Corporation recomenda que você mantenha o recurso de rastreamento desativado quando não estiver diagnosticando problemas de rede.

Diretório de Rastreamento

Informe o diretório em que os arquivos de rastreamento serão gravados. Apesar de os arquivos de rastreamento de clientes e servidores conterem informações sobre o cliente ou o servidor, ambos serão gravados no computador local. O default é $ORACLE_ HOME/network/trace em plataformas UNIX e ORACLE_HOME\network\trace em plataformas Windows.

Arquivo de Rastreamento

Informe o nome do arquivo de rastreamento. O nome do arquivo padrão é sqlnet.trc para o cliente e svr_pid.trc para o servidor.

Nome do Arquivo de Rastreamento Exclusivo:

Escolha para criar nomes exclusivos para cada arquivo de rastreamento. Um identificador de processo é acrescentado ao nome de cada arquivo de rastreamento, de modo que vários arquivos de rastreamento possam coexistir. Desmarque esta opção se não quiser que sejam criados arquivos de rastreamento exclusivos, e desejar que todas as informações de rastreamento sejam armazenadas no mesmo arquivo de rastreamento.

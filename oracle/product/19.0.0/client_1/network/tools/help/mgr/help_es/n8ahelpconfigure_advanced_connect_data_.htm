Configuración de Opciones Avanzadas de Datos de Conexión

Además del nombre de servicio o identificador del sistema Oracle (SID) de la base de datos, puede configurar de forma opcional la sección CONNECT_DATA del descriptor de conexión mediante las siguientes opciones:

  • Nombre de la Instancia para conectarse a una instancia de base de datos específica.

  • Directorio raíz de Oracle de la base de datos para iniciar la base de datos en toda la red

  • Tamaño de la Unidad de Datos de la Sesión para especificar el tamaño del buffer

  • Servidor Dedicado para conectarse a un proceso del servidor dedicado.

  • Servicios Heterogéneos para acceder a un sistema que no sea de Oracle.

  • Valores de Oracle Rdb para acceder a una base de datos Oracle Rdb.

Para configurar las opciones avanzadas de conexión para un nombre del servicio de red:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Directorio > Local o Nomenclatura de Servicios.

  2. Seleccione un nombre de servicio de red o de base de datos. El panel de la derecha muestra el servicio de destino actual y una lista de direcciones.

  3. En el recuadro Identificación de Servicios, seleccione Avanzado. Aparece el recuadro de diálogo Opciones de Servicio Avanzadas.

  4. Introduzca los campos o seleccione las opciones adecuadas y, a continuación, haga clic en Aceptar.

  5. Haga clic en Aplicar en el panel de la derecha.

  6. Si desea realizar estos cambios en la carpeta Local, seleccione Archivo > Guardar Configuración de Red.

Configurazione della cifratura

Il processo di cifratura consiste nel rendere un messaggio illeggibile a chiunque, ad eccezione del destinatario.

Per configurare i parametri di cifratura, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Oracle Advanced Security.

  3. Selezionare la scheda Cifratura.

  4. Selezionare Client o Server dalla lista Cifratura.

  5. Dalla lista Tipo di cifratura, selezionare un valore per specificare il livello di cifratura e integrità adottato dal client o dal server. I quattro valori possibili sono i seguenti:

    Elemento Descrizione
    accettato Il servizio sarà attivo se dall'altra parte della connessione viene specificato "obbligatorio" o "richiesto" ed è disponibile un algoritmo compatibile; in caso contrario, il servizio sarà inattivo.
    rifiutato Il servizio non deve essere attivo. Se dall'altra parte è specificato "obbligatorio", la connessione non viene stabilita.
    richiesto Il servizio sarà attivo se dall'altra parte della connessione viene specificato "accettato", "obbligatorio" o "richiesto" ed è disponibile un algoritmo compatibile; in caso contrario, il servizio sarà inattivo.
    necessario Il servizio deve essere attivo. Se dall'altra parte viene specificato "rifiutato" o se non è disponibile un algoritmo compatibile, la connessione non viene stabilita.

  6. Immettere una base di cifratura nel campo Base di cifratura: una stringa casuale composta da un massimo di 256 caratteri. Questa base viene utilizzata con l'algoritmo di cifratura selezionato al passo successivo.

  7. Dalla lista Metodi disponibili, selezionare un algoritmo di cifratura, quindi scegliere il pulsante freccia destra (<) per spostarlo nella lista Metodi selezionati. L'ordine dei metodi nella lista Metodi selezionati determina l'ordine preferito per la negoziazione. Gli algoritmi riportati di seguito sono supportati da Oracle Advanced Security.

    Elemento Descrizione
    RC4_256 Dimensione chiave a 256 bit
    3DES168 Triple DES con un'opzione a tre chiavi (a 168 bit)
    RC4_128 Dimensione chiave a 128 bit
    3DES112 Triple DES con un'opzione a due chiavi (a 112 bit)
    RC4_56 Dimensione chiave a 56 bit
    DES Dimensione chiave a 56 bit standard
    RC4_40 Dimensione chiave a 40 bit
    DES40 Dimensione chiave a 40 bit

  8. Scegliere File > Salva configurazione di rete.

Nota: tutti gli algoritmi di cifratura di Oracle Advanced Security e le relative lunghezze delle chiavi possono essere utilizzati soltanto all'interno degli Stati Uniti.

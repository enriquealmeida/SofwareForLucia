Directory-Server

Ein Directory Server, auf den mit dem Lightweight Directory Access Protocol (LDAP) zugegriffen wird. Ein LDAP-konformes Directory ermöglicht die zentralisierte Speicherung und den zentralisierten Abruf von Datenbank-Netzwerkkomponenten, Voreinstellungen für Benutzer- und Unternehmensrichtlinien, Informationen über Benutzerberechtigungsprüfung und Sicherheit, wobei client-seitige und server-seitige lokalisierte Dateien ersetzt werden.

Oracle Net unterstützt die folgenden Directory Services:

  • Oracle Internet Directory

  • Microsoft Active Directory

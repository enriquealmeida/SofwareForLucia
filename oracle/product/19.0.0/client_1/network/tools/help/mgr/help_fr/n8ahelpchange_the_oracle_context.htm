Changer le contexte Oracle

Lorsqu'Oracle Net Manager est lancé, les objets affichés dans le dossier Répertoire > Résolution de noms de service sont les entrées de service Oracle qui se trouvent dans le contexte Oracle par défaut établi lors de la configuration du service d'utilisation du répertoire à l'aide de l'assistant Configuration Oracle Net. Vous pouvez créer ou modifier les entrées d'un autre contexte Oracle pour lequel vous avez l'authentification.

Pour changer le contexte Oracle courant :

  1. Choisissez Commande > Annuaire > Modifier le contexte actuel. La boîte de dialogue Choisir un nouveau contexte apparaît.

  2. Dans la liste Contexte de résolution de noms d'annuaire, sélectionnez l'entrée d'annuaire qui contient le contexte Oracle que vous voulez afficher.

  3. Dans la liste Contexte Oracle, sélectionnez le contexte Oracle, puis choisissez OK. Si vos modifications n'ont pas été enregistrées dans le contexte Oracle en cours, une boîte de dialogue d'enregistrement vous invite à le faire.

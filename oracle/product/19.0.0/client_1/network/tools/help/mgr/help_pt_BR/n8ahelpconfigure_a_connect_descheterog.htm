Configurando um Descritor de Conexão para Serviços Heterogêneos

Os Serviços Heterogêneos permitem que você acesse sistemas não Oracle a partir do servidor Oracle. Para iniciar uma conexão com um sistema não Oracle, o servidor Oracle inicia um processo agente por intermédio do listener do gateway. Para que o banco de dados Oracle estabeleça conexão com um sistema não Oracle, configure o servidor com um nome de serviço de rede.

Para configurar o servidor para Serviços Heterogêneos:

  1. Verifique se o listener está configurado para Serviços Heterogêneos, conforme descrito em Configurar Serviços Heterogêneos para o Listener.

  2. Crie um nome do serviço de rede no arquivo tnsnames.ora que possa ser usado para conexões do servidor de banco de dados Oracle com um sistema não Oracle. Certifique-se de que as informações sobre descritor de conexão correspondem às informações configuradas para o listener.

  3. Selecione o nome do serviço de rede na pasta Local > Nomeação de Serviço.

  4. Na caixa Identificação de Serviço do painel direito, escolha Avançado. A caixa de diálogo Opções Avançadas de Serviço será exibida.

  5. Escolha Usar para Serviços Heterogêneos e, em seguida, OK.

  6. No painel direito, escolha Aplicar.

  7. Escolha Arquivo > Salvar Configuração de Rede.

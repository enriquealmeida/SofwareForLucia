ネーミング・メソッドの優先順位付け

ネーミング・メソッドを構成した後に、その優先順位をプロファイル内で設定する必要があります。リストの先頭のネーミング・メソッドは、最初にネット・サービス名またはデータベース・サービス名から接続記述子への変換に使用されます。名前を変換できない場合は、リストで次に指定されているメソッドが使用されます。

ネーミング・メソッドを指定する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「プロファイル」を展開します。

  2. 右ペインのリストから「ネーミング」を選択します。

  3. 「メソッド」タブを選択します。

  4. 「使用可能なメソッド」リストからネーミング・メソッドを選択し、右矢印ボタンを選択します。選択したネーミング・メソッドが「選択メソッド」リストに移動します。

  5. ネーミング・メソッドを並べ替えます。Oracle Netではネット・サービス名またはデータベース・サービス名を変換する際に、この順でメソッドを試行します。「選択メソッド」リストからネーミング・メソッドを選択し、「上へ」または「下へ」を選択してリスト内で選択項目を上下に移動します。

  6. 「ファイル」→「ネットワーク構成の保存」を選択します。

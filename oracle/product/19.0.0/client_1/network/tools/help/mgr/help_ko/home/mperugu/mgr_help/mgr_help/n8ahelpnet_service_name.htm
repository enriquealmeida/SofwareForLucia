네트 서비스 이름

접속 기술자를 분석하는 서비스의 간단한 이름입니다. 사용자는 다음과 같이 접속할 서비스에 대한 접속 문자열에 네트 서비스 이름과 함께 사용자 이름과 암호를 전달하여 접속 요청을 시작합니다.

CONNECT username/password@net_service_name

필요에 따라 네트 서비스 이름을 다음과 같이 다양한 위치에 저장할 수 있습니다.

  • 각 클라이언트에서 로컬 구성 파일, tnsnames.ora 파일

  • 디렉토리 서버

  • Oracle Names Server

  • NIS(네트워크 정보 서비스)를 포함한 외부 이름 지정 서비스

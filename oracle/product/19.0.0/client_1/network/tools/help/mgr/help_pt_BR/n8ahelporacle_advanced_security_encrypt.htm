Oracle Advanced Security: Criptografia

A guia Criptografia permite que você configure parâmetros de criptografia para os métodos de autenticação selecionados.

Observação: Use a guiaSSL para configurar conjuntos de cifras para a SSL (Secure Sockets Layer).

Criptografia

Na lista, selecione o computador que você está configurando: Cliente ou Servidor.

Tipo de Criptografia

Selecione um valor para especificar o comportamento do cliente ou servidor durante a negociação de criptografia e integridade. Os quatro tipos possíveis de criptografia são:

Elemento Descrição
aceito O serviço estará ativo se o outro lado da conexão especificar "necessário" ou "solicitado" e houver um algoritmo compatível disponível do outro lado; caso contrário, ele estará inativo.
rejeitado O serviço não deve estar ativo e a conexão falhará se o outro lado especificar "necessário".
solicitado O serviço estará ativo se o outro lado da conexão especificar "aceito", "necessário" ou "solicitado" e houver um algoritmo compatível disponível do outro lado; caso contrário, ele estará inativo.
necessário O serviço deverá estar ativo e a conexão falhará se o outro lado especificar "rejeitado" ou se não houver um algoritmo compatível do outro lado.

EncryptionSeed

Informe uma semente de criptografia, uma string aleatória de até 256 caracteres, no campo Semente de Criptografia. A semente de criptografia é usada para gerar chaves criptográficas. Isso é necessário quando a criptografia ou a integridade é ativada.

Observação : Se você optar por usar caracteres especiais, como, por exemplo, uma vírgula [,] ou um parêntese direito [)] como parte do parâmetro Encryption Seed, a Oracle recomenda que o valor esteja entre aspas simples.

Lista Métodos Disponíveis/Métodos Selecionados

Exibe os métodos de autenticação disponíveis para os métodos de seleção e criptografia escolhidos. A ordem dos métodos na lista determina a ordem de negociação preferencial.

Use os botões de seta para a esquerda (<) e para a direita (>) para mover um algoritmo de criptografia selecionado entre as listas. Os seguintes algoritmos de criptografia são suportados pelo Oracle Advanced Security:

Elemento Descrição
AES256 Chave AES de 256 bits
RC4_256 Chave RC4 de 256 bits
AES192 Chave AES de 192 bits
3DES168 DES triplo com uma opção de chave tripla (168 bits)
AES128 Chave AES de 128 bits
RC4_128 Chave RC4 de 128 bits
3DES112 DES triplo com uma opção de chave dupla (112 bits)
DES Chave DES de 56 bits
RC4_40 Chave RC4 de 40 bits
DES40 Chave DES de 40 bits

Cuadro de Diálogo Seleccionar Nuevo Contexto

El recuadro de diálogo Seleccionar Nuevo Contexto Oracle permite definir otro Contexto de Oracle, contenedor denominado cn=OracleContext de un Árbol de Información de Directorios (DIT) que contiene entradas que se pueden utilizar con el software Oracle. Estas entradas pueden incluir nombres de servicios de red y nombres de servicios de base de datos para las conexiones a los servicios de base de datos. Por defecto, Oracle Net Manager utiliza el Oracle Context configurado por el Asistente de Configuración de Red de Oracle.

Contexto de Nomenclatura de Directorios

En la lista, seleccione una entrada de directorio que contenga el Oracle Context que desea ver.

Oracle Context

Seleccione un Oracle Context de la lista. El contexto que seleccione debe contener las entradas del árbol de directorios cuya información de red desee modificar o agregar.

La carpeta Directorio > Nomenclatura de Servicios del panel del navegador se actualiza con los objetos de red del Oracle Context seleccionado.

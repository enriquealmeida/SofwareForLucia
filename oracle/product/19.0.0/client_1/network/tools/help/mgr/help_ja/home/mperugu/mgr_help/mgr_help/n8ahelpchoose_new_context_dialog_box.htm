新規コンテキストの選択ダイアログ・ボックス

新規コンテキストの選択ダイアログ・ボックスでは、別のOracleコンテキストを設定できます。Oracleコンテキストは、Oracleソフトウェアで使用するエントリを含むディレクトリ情報ツリー(DIT)のcn=OracleContextというコンテナです。このエントリには、データベース・サービスに接続するためのネット・サービス名やデータベース・サービス名が格納されます。デフォルトでは、Oracle Net Managerは、Oracle Net Configuration Assistantによって構成されたOracleコンテキストを使用します。

ディレクトリ・ネーミング・コンテキスト

リストから、表示するOracleコンテキストを含むディレクトリ・エントリを選択します。

Oracleコンテキスト

リストからOracleコンテキストを選択します。ここでは、ディレクトリ・ツリー内でネットワーク情報を変更または追加するエントリを含むOracleコンテキストを選択する必要があります。

ナビゲータ・ペインで「ディレクトリ」→「サービス・ネーミング」フォルダにアクセスすると、選択したOracleコンテキストのネットワーク・オブジェクトが更新されます。

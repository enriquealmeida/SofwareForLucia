데이터베이스 서비스

데이터베이스 서비스 탭에서는 리스너가 접속 요청을 수신해야 하는 데이터베이스 서비스를 지정할 수 있습니다. 리스너는 이 정보를 사용하여 클라이언트를 데이터베이스에 접속합니다.

Oracle9i 및 Oracle8i 인스턴스는 동적 서비스 등록을 사용하여 리스너에 데이터베이스 서비스를 알립니다. Oracle7 및 Oracle8 릴리스 8.0 데이터베이스 및 Oracle Enterprise Manager 툴은 listener.ora 파일에서 정적 서비스를 구성해야 합니다.

데이터베이스 추가를 선택하여 데이터베이스 서비스 정보를 추가합니다.

전역 데이터베이스 이름

초기화 매개변수 파일의 SERVICE_NAMES 매개변수 값 또는 DB_NAME 및 DB_DOMAIN 매개변수 값에 제공된 대로 데이터베이스 이름 및 데이터베이스의 도메인 이름을 입력합니다.

Oracle 홈 디렉토리

데이터베이스의 Oracle 홈 위치를 입력합니다.

주: Windows NT의 경우 이미 Oracle 홈이 지정되어 있습니다. 따라서 Windows NT에서 서비스에 접속할 경우 이 필드를 지정할 필요 없습니다.

SID

해당 인스턴스의 Oracle SID(시스템 식별자)를 입력합니다.

데이터베이스 추가 단추

다른 데이터베이스에 대한 서비스 정보를 구성합니다. 다른 데이터베이스에 대한 서비스 정보를 입력할 수 있는 새 데이터베이스 탭이 나타납니다.

데이터베이스 제거 단추

선택된 데이터베이스 탭과 해당 서비스 정보를 삭제합니다.

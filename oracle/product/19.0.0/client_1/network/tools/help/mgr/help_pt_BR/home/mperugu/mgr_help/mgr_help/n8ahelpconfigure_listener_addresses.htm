Configurando Endereços de Listener

O listener deve ser configurado com um ou mais endereços de protocolo. O listener recebe solicitações de conexão de clientes configurados com o mesmo endereço.

Para configurar um endereço de protocolo para o listener:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

  2. Selecione um listener.

  3. Na lista do painel direito, selecioneLocais de Listening.

  4. Escolha Adicionar Endereço. Uma nova guia Endereço será exibida.

  5. Selecione um protocolo e especifique informações apropriadas sobre parâmetros de protocolo para o protocolo selecionado.

    Se você estiver configurando o listener para atender em TCP/IP para clientes Oracle Net, deverá escolher a porta default 1521. Caso contrário, você deverá configurar o parâmetro LOCAL_LISTENER no arquivo de parâmetros de inicialização e resolver o nome do listener por meio de um método de nomeação.

  6. Se o computador tiver mais de um endereço IP e você desejar que o listener faça listening em todos os endereços IP disponíveis, configure TCP/IP ou TCP/IP com SSL e informe o nome do host do computador no campo Host.

  7. Opcionalmente, clique em Mostrar Avançado para especificar configurações para definir o espaço no buffer de Entrada/Saída de operações de envio e recepção da sessão.

  8. Escolha Arquivo > Salvar Configuração de Rede.

Para obter mais informações sobre como configurar uma porta não padrão, consulte o Capítulo Configuring and Administering the Listener no Oracle10i Net Administrator's Guide

Para obter mais informações sobre como configurar o espaço de buffer de Entrada/Saída, consulte o capítulo Optimizing Performance no Oracle10i Net Administrator's Guide.

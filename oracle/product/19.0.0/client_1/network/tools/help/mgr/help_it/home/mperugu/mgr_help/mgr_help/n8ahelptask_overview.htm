Panoramica sulle operazioni

Di seguito viene riportata una lista che riassume le operazioni che consentono di configurare Oracle Advanced Security per la rete.

  1. Selezionare i metodi di autenticazione.

  2. Assegnare le priorità per quanto riguarda l'ordine di utilizzo dei metodi di autenticazione durante la negoziazione client/server.

  3. Configurare gli altri parametri per ogni metodo di autenticazione selezionato.

  4. Configurare l'integrità sia dal lato client che dal lato server (livello checksum).

  5. Configurare la cifratura sia dal lato client che dal lato server.

  6. Configurare le impostazioni di comunicazione SSL (Secure Socket Layer) sia dal lato client che dal lato server.

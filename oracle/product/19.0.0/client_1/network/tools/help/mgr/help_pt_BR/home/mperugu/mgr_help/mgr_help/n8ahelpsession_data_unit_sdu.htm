SDU (Session Data Unit)

Um buffer que o Oracle Net utiliza para incluir dados antes de transmiti-los pela rede. O Oracle Net envia os dados contidos no buffer quando solicitado ou quando está cheio. Quando grandes volumes de dados são recuperados e quando o tamanho do pacote é consistentemente o mesmo, a recuperação pode ser agilizada para ajustar o tamanho default da SDU.

Servicios Heterogéneos

Componente integrado que proporciona la tecnología genérica para acceder a sistemas que no sean de Oracle desde el servidor de la base de datos Oracle. Los Servicios Heterogéneos permiten:

  • Utilizar Oracle SQL para acceder de forma transparente a los datos almacenados en sistemas que no sean de Oracle como si los datos estuvieran en un servidor de Oracle.

  • Utilizar llamadas de procedimientos de Oracle para acceder de forma transparente a sistemas, servicios o interfaces de programación de aplicaciones (API) que no sean de Oracle, desde un entorno distribuido de Oracle.

Para iniciar una conexión a un sistema que no sea de Oracle, la base de datos Oracle8i inicia un proceso de agente mediante el listener del gateway. Para que la base de datos Oracle8i se conecte al agente, el servidor debe estar configurado con un nombre de servicio de red para acceder al agente y el listener debe estar configurado con información de servicio sobre el agente.

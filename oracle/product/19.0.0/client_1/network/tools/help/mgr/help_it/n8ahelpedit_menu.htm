Menu Modifica

Il menu Modifica consente di aggiungere, rimuovere o rinominare un nome di servizio di rete, un listener o un server Oracle Names dalle cartelle aperte.

Crea

Scegliere questa opzione per creare un oggetto di rete.

Elimina

Scegliere questa opzione per eliminare un oggetto di rete.

Rinomina

Scegliere questa opzione per rinominare un oggetto di rete.

Dialogfeld "Neuen Kontext wählen"

Mit dem Dialogfeld "Neuen Kontext wählen" können Sie einen anderen Oracle Context festlegen, bei dem es sich um einen Container namens "cn=OracleContext" in einem Directory Information Tree (DIT) handelt, der Einträge zur Verwendung mit der Oracle-Software enthält. Diese Einträge können Net Service Names und Datenbank-Servicenamen für Verbindungen mit Datenbankdiensten enthalten. Standardmäßig verwendet Oracle Net Manager den Oracle Context, der mit dem Oracle Net-Konfigurationsassistent konfiguriert wurde.

Directory-Benennungskontext

Wählen Sie einen Directory-Eintrag aus der Liste, der den Oracle Context enthält, der angezeigt werden soll.

Oracle Context

Wählen Sie einen Oracle Context aus der Liste. Der ausgewählte Oracle Context muss die Einträge in dem Verzeichnisbaum enthalten, deren Netzwerkinformationen geändert oder hinzugefügt werden sollen.

Der Ordner Directory > Dienstebenennung im Navigator-Fensterbereich wird mit den Netzwerkobjekten des ausgewählten Oracle Context aktualisiert.

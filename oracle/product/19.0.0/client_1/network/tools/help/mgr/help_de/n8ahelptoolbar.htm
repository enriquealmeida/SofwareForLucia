Symbolleiste

Die Symbolleiste auf der linken Seite von Oracle Net Manager enthält Symbole, die dem Netzwerkobjekt in den Ordnerhierarchien entsprechen. Die Schaltflächen der Symbolleiste werden aktiviert, je nachdem, welche Objekte in einem Fensterbereich angezeigt oder ausgewählt werden. Setzen Sie den Mauszeiger über ein Symbol in einer Symbolleiste, um die Beschreibung der Funktion des Symbols aufzurufen.

Die Symbolleiste enthält folgende Symbole:

Symbol Beschreibung
Hinzufügen Erstellt neue Net Service-Namen, Listener und einen Oracle Names Server für diesen Rechner.
Löschen Löscht Dienstebenennungs- und Listener-Objekte.
Verbindung prüfen Wählen Sie diese Option, um die Konnektivität eines ausgewählten Dienstebenennungsobjekte zu einem Datenbankdienst zu überprüfen. Während des Konnektivitätstests wird mit den Verbindungsdeskriptor-Informationen, die in einer Datei tnsnames.ora oder einem Directory-Server gespeichert sind, eine Verbindung zu einem Datenbankdienst hergestellt.
Hilfe Zeigt die Onlinehilfe an.

Hinweis: Einige der Befehle sind nur verfügbar, wenn Sie den entsprechenden Objekttyp auswählen. Wenn Sie z.B. einen Listener erstellen möchten, müssen Sie zuerst den Ordner Listener auswählen.

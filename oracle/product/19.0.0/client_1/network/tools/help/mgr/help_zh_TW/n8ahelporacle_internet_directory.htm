Oracle Internet Directory

在 Oracle 資料庫上作為應用程式實行的目錄服務。它可用來擷取分散各處的使用者和網路資源的相關資訊。其結合了 Lightweight Directory Access Protocol (LDAP) 版本 3 (開放式網際網路標準目錄存取協定) 與 Oracle 資料庫的高效能、擴充性、穩健性以及可用性。

如需詳細資訊，請參閱 Oracle Internet Directory Administrator's Guide。

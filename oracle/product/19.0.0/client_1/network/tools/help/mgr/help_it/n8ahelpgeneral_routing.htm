Generale: Instradamento

La scheda Instradamento consente di indirizzare sempre le richieste di connessione dal computer verso un processo specifico.

Utilizza sempre server dedicato

Scegliere questa opzione per obbligare il listener a generare dinamicamente un server dedicato per tutte le sessioni di rete da questo client. Ciò avviene tramite l'aggiunta di (SERVER= dedicated) alla porzione CONNECT_DATA dei descrittori di connessione. Le connessioni da questo client utilizzano i server dedicati, anche se è configurato il server condiviso.

Questa opzione rende più veloce la connessione, ma rallenta il server. Si consiglia pertanto di utilizzarla se si stanno eseguendo transazioni di database molto intense con tempi di inattività ridotti.

Nota: se abilitata, questa opzione ha la priorità sull'opzione Tipo di connessione nella finestra delle proprietà Identificazione servizio.

Preferisci instradamento Connection Manager

Scegliere questa opzione per inviare il client a un indirizzo di Oracle Connection Manager ogni volta che è possibile. Se non è disponibile nessun indirizzo di Oracle Connection Manager, le connessioni vengono instradate tramite un qualsiasi indirizzo disponibile del listener.

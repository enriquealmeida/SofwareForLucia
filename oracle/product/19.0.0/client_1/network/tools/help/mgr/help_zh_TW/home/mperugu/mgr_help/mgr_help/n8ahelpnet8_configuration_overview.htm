Oracle Net Services 組態簡介

您可以使用 Oracle Net Manager 來設定許多網路元素，包括命名方法、從屬端與伺服器設定檔以及監聽器。

命名方法

定義簡單名稱以識別服務的位置，例如資料庫。這些簡單名稱稱為連線 ID，會對應至包含服務的網路位置以及識別的連線描述區。

您可以使用 Oracle Net Manager 來設定連線 ID 透過命名方法解析成連線描述區的各種方法。

從屬端和伺服器設定檔

設定透過設定檔啟用與設定從屬端和伺服器之 Oracle Net 功能的偏好設定。您可以使用設定檔來設定決定命名方法優先順序、啟用 Oracle Advanced Security 功能、啟用記錄日誌與追蹤或路由連線的偏好設定。

伺服器的監聽器組態

設定監聽器接收從屬端連線。監聽器應該設定：

  • 一或多個監聽協定位址

  • 目的地服務的服務資訊

協定位址是在監聽器組態檔 listener.ora 中設定。服務資訊則不一定在 listener.ora 檔案中設定：

  • Oracle8i 或更新版本的資料庫服務會自動將特定資訊註冊至監聽器，例如其服務名稱、執行處理名稱、服務處理程式以及負載資訊。此功能 (稱為服務註冊) 不需您在 listener.ora 檔案中設定服務組態。

  • 其他服務，包括 Oracle8 資料庫，需要在listener.ora檔案中設定有服務組態。

如需有關 Oracle Names 的資訊，請參閱「Oracle Names 簡介」。

如需詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Configuring Naming Methods 章節。

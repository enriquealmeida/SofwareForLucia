승인됨

상대편에서 원할 경우 보안 서비스를 사용합니다.

접속 당사자 쪽에서 보안 서비스를 원하지 않아도 상대편에서 보안 서비스를 필요함이나 요청됨으로 설정하여 요청하는 경우 이를 허용합니다. 상대편이 필요함이나 요청됨으로 설정되어 있고 일치하는 알고리즘이 있으면 보안 서비스가 설정된 상태로 접속이 오류 없이 유지됩니다. 상대편이 필요함으로 설정되어 있지만 일치하는 알고리즘이 없는 경우 오류 메시지 ORA- 2650이 표시되고 접속이 종료됩니다.

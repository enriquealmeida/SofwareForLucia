Parámetros de Protocolo

El listener, Oracle Names Server y Oracle Connection Manager se identifican por una dirección de protocolo. Los clientes que deseen utilizar estos servicios también se deben configurar con la misma dirección. En la siguiente tabla se describen los parámetros que utilizan los protocolos soportados por Oracle:

Protocolo Parámetro Descripción
IPC PROTOCOL Especifica ipc como el valor.
IPC KEY Especifica un nombre único para el servicio. Oracle Corporation recomienda utilizar el nombre de servicio o el identificador del sistema Oracle (SID) del servicio.
Named Pipes PROTOCOL Especifica nmp como el valor.
Named Pipes SERVER Especifica el nombre de la computadora de Oracle Server.
Named Pipes PIPE Especifica el nombre de canal para conectar al servidor de la base de datos. Puede ser cualquier nombre arbitrario.
TCP/IP y TCP/IP con SSL PROTOCOL Especifica tcp como el valor para TCP/IP o tcps como el valor para TCP/IP con SSL.
TCP/IP y TCP/IP con SSL HOST Especifica el nombre del host o la dirección IP de la computadora.
TCP/IP y TCP/IP con SSL PORT Especifica el número de puerto del listener.

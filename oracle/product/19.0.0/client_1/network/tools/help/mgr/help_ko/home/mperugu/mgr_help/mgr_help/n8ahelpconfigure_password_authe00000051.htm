리스너에 대한 비밀번호 인증 구성

보안을 사용하여 리스너를 구성할 수 있습니다. 이렇게 리스너 제어 유틸리티에서 사용되는 구성 변경 사항 저장이나 리스너 정지 등 권한이 필요한 작업에는 비밀번호가 필요합니다. Oracle Net Manager를 사용하여 암호화된 비밀번호를 설정할 수 있습니다.

리스너에 대한 비밀번호를 설정하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬> 리스너를 확장합니다.

  2. 리스너를 선택합니다.

  3. 오른쪽 창에 있는 목록에서 일반 매개변수를 선택합니다.

  4. 인증 탭을 선택합니다.

  5. 리스너 작업을 위한 비밀번호 요구를 선택합니다.

  6. 비밀번호 필드에 비밀번호를 입력하고 비밀번호 확인 필드에 비밀번호를 다시 입력합니다.

  7. 파일 > 네트워크 구성 저장을 선택합니다.

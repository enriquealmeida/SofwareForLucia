ホスト・ネーミング

ユーザーがTCP/IP環境でホスト名の別名を使用してOracleデータベース・サーバーに接続できるようにするネーミング・メソッド。ホスト名は、ドメイン・ネーム・システム(DNS)、ネットワーク・インフォメーション・サービス(NIS)または集中管理される/etc/hostsファイルの集合など、既存の名前変換サービス内のサーバーのグローバル・データベース名にマップされます。

ホスト・ネーミング・メソッドの構成の詳細は、Oracle10i Net Services管理者ガイドの「ネーミング・メソッドの構成」の章を参照してください。

Verbindungsanforderungen umleiten

Clients können so konfiguriert werden, dass Verbindungsanforderungen stets an einen dedizierten Server-Prozess oder einen Oracle Connection Manager umgeleitet werden.

So leiten Sie Verbindungsanforderungen um:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal > Profil im Navigator-Fensterbereich ein.

  2. Wählen Sie Allgemein aus der Liste im rechten Fensterbereich.

  3. Wählen Sie das Register Routing.

  4. Wählen Sie die bevorzugte Umleitung für die Verbindungsanforderungen.

  5. Wählen Sie Datei >Netzwerkkonfiguration speichern.

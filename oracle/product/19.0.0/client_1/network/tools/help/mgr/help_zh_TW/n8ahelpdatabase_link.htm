資料庫連結

定義從 Oracle 資料庫伺服器到另一個資料庫伺服器單向通訊的指標。連結指標實際上是定義為資料說明表格中的項目。若要存取連結，您必須連線至包含資料說明項目的區域資料庫。

資料庫連結連線是單向的概念，即連線至本機資料庫 A 的從屬端可以使用資料庫 A 中所儲存的連結，存取遠端資料庫 B 中的資訊，但是連線至資料庫 B 的使用者無法使用相同連結來存取資料庫 A 中的資料。如果資料庫 B 中的本機使用者想要存取資料庫 A 中的資料，那麼他們必須定義要儲存於資料庫 B 的資料字典中的連結。

Oracle 支援下列資料庫連結類型：

  • 資料庫特定綱要中的專用資料庫連結。只有專用資料庫連結的擁有者可以使用該連結。

  • 資料庫的公用資料庫連結。資料庫中的所有使用者都可以使用該連結。

如需有關建立專用和公用資料庫連結的詳細資訊，請參閱 Oracle10i Database Administrator's Guide。

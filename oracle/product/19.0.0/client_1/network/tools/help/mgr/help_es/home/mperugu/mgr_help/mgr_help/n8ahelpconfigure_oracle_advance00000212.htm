Configuración de Métodos de Autenticación de Oracle Advanced Security

Los métodos de autenticación de seguridad centralizados permiten conseguir un alto nivel de confianza en la identidad de los usuarios, clientes y servidores de los entornos distribuidos. Además, los métodos de autenticación de red ofrecen la ventaja de un único proceso de inicio de sesión para los usuarios.

Utilice el Gestor de Red de Oracle para configurar los siguientes adaptadores de autenticación:

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • Autenticación Nativa de Windows NT

Nota: Si ha configurado el protocolo TCP/IP con SSL, SSL (Secure Sockets Layer) se configurará por defecto. Cualquier método de autenticación que seleccione en este separador sustituirá las funciones de SSL. Acceda al separador SSL para configurar otras opciones de SSL. Si utiliza el protocolo TCP/IP con SSL, no utilice ningún otro método de autenticación.

Para configurar los métodos de autenticación:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione Oracle Advanced Security de la lista del panel de la derecha.

  3. Seleccione el separador Autenticación.

  4. Seleccione de la lista Métodos Disponibles el método de autenticación que desee utilizar y haga clic en el botón de flecha a la derecha (>).

  5. Ordene los métodos de autenticación según el método que desee utilizar. Seleccione el método de autenticación en la lista Métodos Seleccionados y utilice los botones Aumentar o Reducir para que la selección ascienda o descienda en la lista.

  6. Seleccione el separador Otros Parámetros para configurar parámetros adicionales correspondientes a los siguientes métodos de autenticación:

    • Kerberos

    • CyberSAFE

    • RADIUS

  7. Seleccione Archivo > Guardar Configuración de Red.

Dettagli sull'alias di servizio di rete

Questo foglio delle proprietà consente di visualizzare i dettagli dell'alias di servizio di rete selezionato.

Alias di servizio di rete

Consente di visualizzare o modificare il nome dell'alias di servizio di rete.

Nome servizio di rete o Servizio di database

Consente di visualizzare o modificare le informazioni riportate di seguito per il servizio di database o il nome di servizio di rete.

  • Contesto Oracle: consente di visualizzare o modificare il contesto Oracle del servizio di database o del nome di servizio di rete.

  • Nome: consente di visualizzare il nome distinto (DN) del servizio di database o del nome di servizio di rete.

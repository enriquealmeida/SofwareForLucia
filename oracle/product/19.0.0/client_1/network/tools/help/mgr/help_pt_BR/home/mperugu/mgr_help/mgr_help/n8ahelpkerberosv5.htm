KERBEROS(V5)

O método de autenticação KERBEROS(V5) requer que você defina os seguintes parâmetros:

Elemento Descrição
Serviço Informe o nome do serviço de autenticação: KERBEROS.
Arquivo de Cache de Credencial O caminho completo do arquivo de cache de credencial do Kerberos: o default depende do sistema operacional.
Arquivo de Configuração O caminho completo do arquivo de configuração do Kerberos. Esse arquivo contém o reino do KDC (key distribution center) e mapeia hosts para hosts KDC: o default depende do sistema operacional.
Arquivo de Tradução de Realm O caminho completo do arquivo de tradução de realm do Kerberos. Esse arquivo mapeia um nome de host, ou um nome de domínio, para um realm: o default depende do sistema operacional.
Tabela-chave O caminho completo do arquivo de mapeamento principal/de chave secreta Kerberos. Esse caminho é usado pelo servidor Oracle para extrair sua chave e decriptografar informações de autenticação recebidas do cliente. O default depende do sistema operacional.
Desvio do Clock O número de segundos após o qual a credencial do Kerberos é considerada desatualizada. O default = 300 segundos.

修改資料庫服務的網路路由資訊

注意：只有身為 OracleNetAdmins 或 OracleContextAdmins 群組成員的使用者，才可以修改目錄中資料庫服務的網路資訊。

當資料庫完成目錄註冊時，「資料庫組態輔助程式」會在目錄中建立資料庫服務項目。依照預設，此項目會包含網路路由資訊，內含透過協定位址表示的監聽器位置。如果此資訊已被移除，您可以重新建立此資訊，或修改現有的網路路由資訊。

建立或修改資料庫服務的網路路由資訊：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 目錄 > 服務命名。

  2. 選取資料庫服務。右窗格會顯示目前的目的地服務名稱。

  3. 在位址組態方塊中，選擇加號 (+)。即會顯示一個新的位址頁籤。

  4. 選取一個協定，然後輸入選取之協定的適當協定參數。

  5. 在右窗格中，選擇套用。

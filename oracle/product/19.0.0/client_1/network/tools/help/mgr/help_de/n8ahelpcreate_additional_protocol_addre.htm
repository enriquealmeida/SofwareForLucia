Zusätzliche Protokolladressen erstellen

Auf einen Datenbankdienst kann von mehr als einer Netzwerk-Route oder Protokolladresse zugegriffen werden.

So fügen Sie eine Netzwerk-Protokolladresse zu einem bestehenden Net Service-Name oder Datenbankdienst hinzu:

  1. Blenden Sie im Navigator-Fensterbereich Oracle Net-Konfiguration > Verzeichnis oder Lokal >Dienstebenennung ein.

  2. Wählen Sie einen Net Service-Namen oder einen Datenbankdienst. Im rechten Fensterbereich werden der aktuelle Zieldienst und die Adressliste angezeigt.

  3. Klicken Sie im Feld Adresskonfiguration auf das Pluszeichen (+). Daraufhin wird ein neues Register Adresse angezeigt.

  4. Wählen Sie ein Protokoll, und geben Sie die entsprechenden Protokollparameter-Informationen für das ausgewählte Protokoll ein.

  5. Ordnen Sie die Adressen mit den Pfeiltasten nach links bzw. rechts in der Reihenfolge an, in der sie in der Adressliste auftreten sollen. Wenn nicht mehrere Adressoptionen konfiguriert sind, wird nur Verbindung mit der ersten Adresse in der Liste aufgenommen. Unter Mehrere Adressoptionen konfigurieren wird beschrieben, wie eine andere Reihenfolge konfiguriert wird.

  6. Wählen Sie Anwenden im rechten Fensterbereich.

  7. Wenn Sie diese Änderungen an dem Ordner "Lokal" vornehmen, wählen Sie Datei > Netzwerkkonfiguration speichern.

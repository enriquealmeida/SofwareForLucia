Endereço

Um endereço de protocolo de rede de um objeto de rede.

Quando uma conexão é estabelecida, o cliente e o receptor da solicitação (listener, o servidor Oracle Names ou o Oracle Connection Manager) são configurados com endereços de protocolo idênticos. O cliente utiliza esse endereço para enviar a solicitação de conexão à localização de um objeto de rede específico e o destinatário "atende" solicitações nesse endereço.

O seguinte endereço define um endereço de atendimento para o listener.

ADDRESS=

(PROTOCOL=tcp)

(HOST=sales-server)

(PORT=1521))

Os parâmetros de protocolo do endereço dependem do protocolo selecionado.

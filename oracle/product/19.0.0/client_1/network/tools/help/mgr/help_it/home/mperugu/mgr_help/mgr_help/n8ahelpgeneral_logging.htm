Generale: Log

La scheda Log consente di impostare i parametri di log per il computer. Se si tratta di un client, utilizzare i campi Informazioni client. Se si tratta di un server, utilizzare i campi Informazioni server.

Directory di log

Immettere il percorso della directory in cui verranno scritti i file di log. L'impostazione predefinita è $ORACLE_HOME/network/log sulle piattaforme UNIX e ORACLE_HOME \network\log sulle piattaforme Windows.

File di log

Immettere il nome del file di log sul client. Il nome predefinito è sqlnet.log.

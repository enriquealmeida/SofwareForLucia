Recursos Principais do Oracle Net Manager

O Oracle Net Manager é uma ferramenta que o ajuda a configurar e gerenciar o ambiente de rede Oracle. Os seguintes componentes e recursos do Oracle Net estão disponíveis para configuração e gerenciamento com o Oracle Net Manager:

Nomeação de Serviço

Você pode criar ou modificar descritores de rede para serviços de banco de dados em um arquivo tnsnames.ora, servidor de diretórios ou um servidor Oracle Names. Descrições de rede, chamados descritores de conexão, são mapeados para identificadores de conexão que os clientes utilizam em suas strings de conexão durante uma conexão de banco de dados.

Listener

Você pode criar ou modificar umlistener, um processo do servidor que recebe e responde a solicitações de conexão do cliente para um serviço de banco de dados.

Perfil

Você pode criar ou modificar um perfil, um conjunto de parâmetros que determinam como o cliente se conecta à rede Oracle. Você pode usar·parâmetros de cliente para métodos de nomeação, log, rastreamento e nomeação externa, além do Oracle Advanced Security.

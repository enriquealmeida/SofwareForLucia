Net Service-Alias erstellen

Hinweise

  • Nur Benutzer, die Mitglieder der Gruppe OracleNetAdmins oder OracleContextAdmins sind, können Net Service Alias-Einträge in einem Directory erstellen.

  • Net Service-Aliasnamen werden bei Microsoft Active Directory nicht unterstützt.

  • Sie können bestehende Aliasobjekte aus einem Oracle Names Server exportieren. Weitere Informationen finden Sie im Kapitel Exporting Naming Data to a Directory Server im Oracle10i Net Services Administrator's Guide.

Sie können Net Service-Aliasnamen für Directory-Benennungsobjekte – Net Service Names oder Datenbankdienste – in einem Directory-Server erstellen. In einem Net Service-Alias wird der Name des referenzierten Objektes gespeichert.

Wenn ein Client eine Directory-Suche nach einem Net Service-Aliasnamen anfordert, bestimmt das Directory, dass der Eintrag ein Net Service-Aliasname ist und schließt die Suche so ab, als handle es sich um den referenzierten Eintrag. Beispiel: Ein Net Service Alias salesalias kann für einen Net Service Name sales erstellt werden. Wenn salesalias verwendet wird, um eine Verbindung zu einem Datenbankservice herzustellen, wie in CONNECT scott/tiger@salesalias, wird er als sales aufgelöst und verwendet dessen Verbindungsdeskriptorinformationen.

Es gibt zwei Hauptverwendungszwecke für Net Service-Aliasnamen:

  • Mit einem Net Service-Aliasnamen können Clients einen Datenbankdienst oder Net Service-Namen mit einem anderen Namen referenzieren.

  • Ein Net Service-Aliasname kann in einem Oracle Context für einen Datenbankdienst oder Net Service-Namen in einem anderen Oracle Context verwendet werden. Dadurch kann dieser Datenbankdienst oder Net Service-Name einmal in dem Directory-Server definiert werden, jedoch aus anderen Oracle Contexts verwendet werden.

So erstellen Sie einen Net Service-Aliasnamen:

  1. Blenden Sie Oracle Net-Konfiguration >Directory > Dienstebenennung> Aliasnamen im Navigator-Fensterbereich ein.

  2. Klicken Sie auf das Pluszeichen (+) in der Symbolleiste, oder wählen Sie Bearbeiten > Erstellen aus der Menüleiste. Daraufhin wird das Dialogfeld Net Service-Aliasname erstellen angezeigt.

  3. Geben Sie einen Namen für den Net Service-Alias in das Feld Net Service-Alias ein.

  4. Geben Sie die folgenden Informationen über den Net Service-Namen oder Datenbankdienst in das Feld Net Service oder Datenbankdienst ein:

    • Oracle Context:Geben Sie den Oracle Context des Datenbankdienstes oder Net Service-Namens an, indem Sie einen Oracle Context aus der Liste wählen oder in das Feld eingeben.

    • Name: Geben Sie den Distinguished Name (DN) des Datenbankdienstes oder Net Service Namens an, indem Sie einen Oracle Context aus der Liste wählen oder in das Feld eingeben.

  5. Wählen Sie Erstellen.

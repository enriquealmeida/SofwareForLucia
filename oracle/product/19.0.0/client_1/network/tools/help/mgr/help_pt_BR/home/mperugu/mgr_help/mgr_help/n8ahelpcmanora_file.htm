Arquivo cman.ora

Um arquivo de configuração que especifica endereços de protocolo para solicitações recebidas e comandos locais e remotos, além de parâmetros do Oracle Connection Manager. O Oracle Net Manager não suporta a configuração desse arquivo. Ele deve ser configurado manualmente.

O arquivo cman.ora deverá residir em uma das seguintes localizações:

  1. O diretório especificado pela variável de ambiente TNS_ADMIN. Se a variável de ambiente de rede TNS_ADMIN não tiver sido definida como uma variável no Windows NT, ela poderá estar no registro.

  2. O diretório de configuração global do nó. Para Sun Solaris, este diretório é /var/opt/oracle. O Windows NT não tem um diretório central.

  3. O diretório $ORACLE_HOME/network/admin do UNIX ou o diretório ORACLE_HOME \network\admin de sistemas operacionais do Windows.

Para obter mais informações, consulte o capítulo Configuring and Administering Oracle Connection Manager no Oracle10i Net Administrator's Guide.

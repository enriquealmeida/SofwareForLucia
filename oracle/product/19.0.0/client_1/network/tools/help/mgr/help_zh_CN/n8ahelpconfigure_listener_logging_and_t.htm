配置监听程序的事件记录和跟踪

可以使用事件记录和跟踪参数配置监听程序。

日志信息通常存储在listener.log文件中。在 UNIX 上，该文件在$ORACLE_HOME/network/log目录中；在 Window 平台上，该文件在ORACLE_HOME\network\log目录中。

跟踪信息默认存储在listener.trc文件中。在 UNIX 上，该文件在$ORACLE_HOME/network/trace目录中；在 Window 平台上，该文件在ORACLE_HOME\network\trace目录中。

要启用事件记录或跟踪，请执行以下操作：

  1. a. 在导航器窗格中，展开 Oracle Net 配置 > 本地 > 监听程序。

  2. 选择监听程序。

  3. 在右侧窗格的列表中，选择一般参数。

  4. 选择事件记录和跟踪选项卡。

  5. 根据需要调整事件记录和跟踪设置。

  6. 选择文件 > 保存网络配置。

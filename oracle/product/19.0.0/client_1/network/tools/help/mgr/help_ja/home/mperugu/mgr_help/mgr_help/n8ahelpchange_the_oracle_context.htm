Oracleコンテキストの変更

Oracle Net Managerの開始時に、「ディレクトリ」→「サービス・ネーミング」フォルダを選択すると、デフォルトのOracleコンテキストに含まれるOracleサービス・エントリが表示されます。デフォルトのOracleコンテキストは、Oracle Net Configuration Assistantによるディレクトリ使用の構成時に確立されたものです。認証を持つ別のOracleコンテキストのエントリを作成および変更できます。

現在のOracleコンテキストを変更する手順は、次のとおりです。

  1. 「コマンド」→「ディレクトリ」→「カレント・コンテキストの変更」を選択します。「新規コンテキストの選択」ダイアログ・ボックスが表示されます。

  2. 「ディレクトリ・ネーミング・コンテキスト」リストから、表示するOracleコンテキストを含むディレクトリ・エントリを選択します。

  3. 「Oracleコンテキスト」リストから使用するOracleコンテキストを選択し、「OK」を選択します。変更結果が現在のOracleコンテキストに保存されていない場合は、「保存」ダイアログ・ボックスに、変更結果の保存を求めるプロンプトが表示されます。

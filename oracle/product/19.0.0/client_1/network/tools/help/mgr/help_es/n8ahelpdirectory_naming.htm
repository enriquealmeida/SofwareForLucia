Nomenclatura de Directorios

Método de nomenclatura que resuelve un servicio de base de datos o un nombre de servicio de red en un descriptor de conexión, almacenado en un servidor de directorios central.

El directorio proporciona una administración central de los servicios de base de datos y de los nombres de servicio de red, reduciendo, así, el esfuerzo asociado a la adición o reubicación de los servicios. Aunque los nombres de servicio de red se pueden configurar como alias de un servicio, el directorio puede hacer referencia a un servicio de base de datos directamente sin utilizar un nombre de servicio de red. El servicio de base de datos se agrega de forma automática como entrada al directorio durante la instalación.

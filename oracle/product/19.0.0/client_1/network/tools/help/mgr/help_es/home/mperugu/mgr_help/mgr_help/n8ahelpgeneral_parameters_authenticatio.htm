Parámetros Generales: Autenticación

El separador Autenticación permite configurar la seguridad de un listener. De este modo, será necesaria la contraseña para operaciones con privilegios, como guardar los cambios de configuración o detener el listener, que se realicen desde la Utilidad de Control de Listener. El Gestor de Red de Oracle permite definir una contraseña cifrada.

No Requerir ninguna Contraseña para las Operaciones del Listener

Seleccione esta opción para desactivar la autenticación para las tareas administrativas.

Requerir una Contraseña para las Operaciones del Listener

Seleccione para permitir la autenticación para determinadas tareas de administrador. A continuación, puede introducir la contraseña requerida. La contraseña por defecto es "oracle".

Nota: La Red de Oracle permite la configuración de una contraseña por cada listener.

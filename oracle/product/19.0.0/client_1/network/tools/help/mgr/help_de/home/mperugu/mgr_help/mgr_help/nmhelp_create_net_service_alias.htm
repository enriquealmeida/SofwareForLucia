Dialogfeld "Net Service-Alias erstellen"

Im Dialogfeld "Net Service-Alias erstellen" können Sie Net Service-Alias in einem Directory-Server erstellen.

Ein Net Service-Aliasname ist ein alternativer Name für einen Net Service-Namen oder Datenbankdienst. Ein Net Service-Aliaseintrag enthält keine Verbindungsdeskriptor-Informationen. Stattdessen referenziert er nur den Speicherort des Objektes, für das er den Alias darstellt. Wenn ein Client in einem Verzeichnis nach einem Net Service-Alias sucht, bestimmt das Verzeichnis, ob es sich bei dem Eintrag um einen Net Service-Alias handelt und nimmt die Durchsuchung so vor, als wäre der Alias der referenzierte Eintrag. Beispiel: Ein Alias db1alias kann für einen Datenbankservice db1 erstellt werden. Wenn db1alias für die Anmeldung bei einem Datenbankservice verwendet wird, wie in CONNECT scott/tiger@db1alias, wird der Alias als Verbindungsdeskriptorinformation für db1 aufgelöst und verwendet diese.

Es gibt zwei Hauptverwendungszwecke für Net Service-Alias:

  • Mit einem Net Service-Alias können Clients einen Datenbankdienst oder Net Service-Namen mit einem anderen Namen referenzieren.

  • Ein Net Service-Aliasname kann in einem Oracle Context für einen Datenbankdienst oder Net Service-Namen in einem anderen Oracle Context verwendet werden. Dadurch kann der Datenbankdienst oder Net Service-Name einmal in dem Directory-Server definiert, jedoch von Clients referenziert werden, die andere Oracle Contexts verwenden.

Net Service-Alias

Geben Sie den Namen des Net Service-Alias ein.

Net Service-Name oder Datenbankdienst

Im Feld Net Service oder Datenbankdienst geben Sie Informationen über den Datenbankdienst oder Net Service-Namen ein:

Oracle Context:Geben Sie den Oracle Context des Datenbankdienstes oder Net Service-Namens an, indem Sie einen Oracle Context aus der Liste wählen oder in das Feld eingeben.

Name: Geben Sie den Distinguished Name (DN) des Datenbankdienstes oder Net Service Namens an, indem Sie einen Oracle Context aus der Liste wählen oder in das Feld eingeben.

Allgemeine Parameter: Logging und Tracing

Mit dem Register Logging & Tracing können Sie das Logging und Tracing für den Listener aktivieren. Mit Oracle Net Manager können Sie Namen und Speicherort angeben, in den die Listener-Logdateien geschrieben werden.

Logging deaktiviert

Wählen Sie diese Option, um das Logging auf dem Listener zu deaktivieren.

Logging aktiviert

Wählen Sie diese Option, um das Logging auf dem Listener zu aktivieren. Wenn diese Option ausgewählt ist, können Sie angeben, wohin die Loginformationen geschrieben werden sollen.

Logdatei

Geben Sie den Pfad- und Dateinamen ein, in den die Loginformationen aus dem Listener geschrieben werden. Beachten Sie, dass die Dateinamenerweiterung für diese Datei stets .log ist. Der Standardname ist listener.log. Das Standardverzeichnis ist $ORACLE_HOME/network/log bei UNIX- und ORACLE_HOME\network\log bei Windows-Plattformen.

Tracing deaktiviert

Wählen Sie diese Option, um das Tracing auf dem Listener zu deaktivieren.

Tracing aktiviert

Wählen Sie diese Option, um das Tracing auf dem Listener zu aktivieren. Sobald das Tracing aktiviert ist, können Sie die gewünschte Tracing-Ebene angeben und festlegen, wohin die Trace-Informationen geschrieben werden, indem Sie die standardmäßige Trace-Datei ändern.

Tracing-Ebene

Wählen Sie die Trace-Ebene aus der Liste:

Off - Tracing ist nicht aktiviert. Off ist die Standardeinstellung.

User - Das Tracing ist auf eine für Benutzer zutreffende Ebene festgelegt. Bei dieser Ebene wird auf vom Benutzer verursachte Fehlerbedingungen geprüft.

Administrator - Das Tracing ist auf eine für Datenbankadministratoren zutreffende Ebene festgelegt. Hier wird auf installationsspezifische Probleme geprüft.

Support - Das Tracing ist auf eine für den Support zutreffende Ebene eingestellt.

Trace-Dateien können sehr groß werden. Oracle Corporation empfiehlt, dass Sie das Tracing ausschalten, wenn keine Netzwerkprobleme behoben werden.

Trace-Datei

Geben Sie den Pfad- und Dateinamen ein, in den Trace-Informationen aus dem Listener geschrieben werden. Beachten Sie, dass die Dateinamenerweiterung für diese Datei stets .trc ist. Der Standardname ist listener.trc. Das Standardverzeichnis ist $ORACLE_HOME/network/trace bei UNIX und ORACLE_HOME\network\trace bei Windows-Plattformen.

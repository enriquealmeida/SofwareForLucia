監聽器控制公用程式

包括在 Oracle Net Services 的公用程式，可控制不同的監聽器功能，例如啟動、停止以及取得監聽器的狀態。「監聽器控制」公用程式是用以下方式執行的：

lsnrctl 命令 [listener_name]

[listener_name] - listener.ora 檔案中定義的監聽器名稱。如果目前使用預設監聽器名稱 LISTENER，就不需要識別監聽器。

如需詳細資訊，請參閱 Oracle10i Net Services Reference Guide 中的 Listener Control Utility 章節。

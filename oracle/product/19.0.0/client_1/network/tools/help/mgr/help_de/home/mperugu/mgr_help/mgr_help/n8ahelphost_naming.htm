Hostbenennung

Eine Benennungsmethode, mit der sich Benutzer bei einem Oracle-Datenbankserver mit einem Hostaliasnamen in einer TCP/IP-Umgebung anmelden können. Hostnamen werden dem globalen Datenbanknamen des Servers in einem bestehenden Service zur Namensauflösung zugeordnet, wie z.B. Domain Name System (DNS), Network Information Service (NIS) oder einer zentral verwalteten Gruppe von /etc/hosts-Dateien.

Informationen zur Konfiguration der Hostbenennungsmethode finden Sie im Kapitel Configuring Naming Methods im Oracle10i Net Services Administrator's Guide.

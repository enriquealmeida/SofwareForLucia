Ubicaciones de Listeners

Los listeners se configuran para recibir las solicitudes de conexión de entrada a una base de datos en una dirección de protocolo de red específica. El separador Ubicaciones de Listeners permite configurar una o varias direcciones de ubicación en las que el listener debe esperar las conexiones cliente.

Protocolo

De la lista, seleccione el protocolo para la dirección. Según el protocolo seleccionado, introduzca la información sobre el parámetro de protocolo en los campos que corresponda.

Botón Mostrar Avanzadas/Ocultar Avanzadas

Utilice este botón para mostrar u ocultar los valores para especificar el límite de espacio en buffer de E/S para el listener.

Estos valores son compatibles con los protocolos TCP/IP, TCP/IP con SSL y SDP.

  • Tamaño Total de Buffer de Envío

Especifique, en bytes, el espacio de buffer para las operaciones de envío de las sesiones.

Ya que el servidor de bases de datos escribe datos en los clientes, normalmente, lo más adecuado es definir este valor en el servidor. Si el servidor de bases de datos está recibiendo un gran volumen de solicitudes, también debe definir el valor Tamaño Total de Buffer de Recepción.

  • Tamaño Total de Buffer de Recepción

Especifique, en bytes, el espacio de buffer para las operaciones de recepción de las sesiones.

Dedicar de forma estática esta dirección para conexiones JServer

Configure el archivo listener.ora con los puntos finales del protocolo General Inter-Orb Protocol (GIOP) para las conexiones de Oracle JServer versión 8.1.

Agregar Dirección

Seleccione esta opción para crear una nueva dirección de protocolo. Se mostrará un nuevo separador Dirección, donde podrá seleccionar el protocolo e introducir información del protocolo en los campos.

Eliminar Dirección

Haga clic en este botón para eliminar el separador Dirección seleccionado y su información de dirección.

Para obtener más información sobre la configuración del espacio del buffer de E/S, consulte el capítulo sobre optimización del rendimiento, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

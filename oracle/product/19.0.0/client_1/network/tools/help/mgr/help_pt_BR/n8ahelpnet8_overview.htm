Visão Geral do Oracle Net Services

O Oracle Net Services oferece soluções de conectividade que abrangem toda a empresa em ambientes de computação distribuídos e heterogêneos. O Oracle Net Services ameniza as complexidades de configuração e gerenciamento de rede, maximiza o desempenho e melhora os recursos de diagnóstico da rede.

O Oracle Net, um componente do Oracle Net Services, permite estabelecer uma sessão de rede de uma aplicação cliente para um servidor de banco de dados Oracle. Uma vez que a sessão de rede seja estabelecida, o Oracle Net atua como um transportador de dados para a aplicação cliente e o servidor de banco de dados. Ele é responsável por estabelecer e manter a conexão entre a aplicação cliente e o servidor de banco de dados, além de promover a troca de mensagens entre eles. O Oracle Net pode realizar essas tarefas porque ele está localizado em cada computador da rede.

As sessões de rede são estabelecidas com a ajuda de um listener. O listener é um processo separado que reside no servidor de banco de dados. O listener recebe solicitações de conexões de clientes e gerencia o tráfego dessas solicitações até o servidor.

O listener intermedia a solicitação do cliente, transmitindo-a ao servidor. Toda vez que um cliente solicita uma sessão de rede com um servidor, um listener recebe a solicitação real. Se houver correspondência entre as informações do cliente e do listener, o listener concederá uma conexão ao servidor.

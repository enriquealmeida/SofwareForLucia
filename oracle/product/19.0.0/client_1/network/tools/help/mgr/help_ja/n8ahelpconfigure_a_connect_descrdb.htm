Oracle RDBデータベースの接続記述子の構成

Oracle RDBデータベースは、Compaq Tru64 UNIXプラットフォーム用のデータベースです。Oracle RDBサーバーは、クライアントからはOracleデータベースと同様に見えます。Oracle RDBデータベースには、専用のリスナーがあります。クライアントの構成では、Oracle RDB固有のパラメータを使用してネット・サービス名を作成し、データベースにアクセスします。

Oracle RDBデータベース用にクライアントを構成する手順は、次のとおりです。

  1. 「ネット・サービス名の作成」の指示に従って、ネット・サービス名を作成します。

  2. 右ペインの「サービスの識別」ボックスで「詳細」を選択します。「詳細サービス・オプション」ダイアログ・ボックスが表示されます。

  3. 「RDBデータベース」フィールドにOracle RDBデータベースのファイル名を入力します。

  4. 必要に応じて、「グローバル・データベース名」フィールドにグローバル・データベース名、「サービス・タイプ」フィールドにサービス・タイプを入力します。

  5. 「詳細サービス・オプション」ダイアログ・ボックスで「OK」を選択します。

  6. 右ペインで「適用」を選択します。

  7. 「ファイル」→「ネットワーク構成の保存」を選択します。

詳細は、Oracle RDBのマニュアルを参照してください。

尋找

尋找 Oracle Names 伺服器的程序。在尋找期間，會先找出一個 Oracle Names 伺服器。找到第一個 Oracle Names 伺服器之後，此伺服器會偵測管理區域中的所有其他 Oracle Names 伺服器。然後建立 Oracle Names 伺服器清單並儲存於 UNIX 上的 .sdns.ora 和 Windows 上的 sdns.ora。此清單會依回應時間排序。

尋找會依下列順序搜尋第一個 Oracle Names 伺服器：

  1. sqlnet.ora 檔案 中的偏好 Oracle Names 伺服器

  2. 已知的 Oracle Names 伺服器

  3. 使用於連接埠 1575 的 TCP/IP 所設定的區域 Oracle Names 伺服器

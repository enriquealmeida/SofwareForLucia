設定連線 Oracle 連線管理程式的連線描述區

若要透過 Oracle 連線管理程式 將從屬端傳送至資料庫伺服器，請使用指定「Oracle 連線管理程式」協定位址的連線描述區，來設定 tnsnames.ora 檔案或目錄伺服器。從屬端可以利用這個位址來連線「Oracle 連線管理程式」電腦。

設定「Oracle 連線管理程式」的協定位址：

  1. 在瀏覽窗格中，展開 Oracle net 組態 > 目錄或區域 > 服務命名。

  2. 從工具列選取加號 (+)，或者從功能表列選擇編輯 > 建立。「網路服務名稱精靈」的「歡迎」頁面就會顯示。

  3. 在網路服務名稱欄位輸入名稱。

  4. 選擇下一步。「協定」頁面就會顯示。

  5. 選取設定讓「Oracle 連線管理程式」監聽的協定。預設的協定是 TCP/IP。

  6. 選擇下一步。「協定設定值」頁面就會顯示。

  7. 在出現的欄位中輸入選取之協定的適當協定參數。如果是使用 TCP/IP，預設連接埠為 1630。

  8. 選擇下一步。「服務」頁面就會顯示。

  9. 如果目的地服務是 Oracle8i 或更新的版本，請選取 Oracle8i 或更新的版本，然後在服務名稱欄位中輸入服務名稱。如果目的地服務是 Oracle8 或 Oracle7，請選取 Oracle8 或先前的版本，然後在資料庫 SID 欄位中輸入 Oracle 系統 ID。

  10. 選擇下一步。「服務」頁面就會顯示。

  11. 按一下完成儲存組態並關閉「網路服務名稱精靈」。

  12. 選擇檔案 > 儲存網路組態。

  13. 新的網路服務名稱便會新增至「服務命名」資料夾。

如需有關完全啟用「Oracle 連線管理程式」功能的詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中 Configuring and Administering Oracle Connection Manager 章節。

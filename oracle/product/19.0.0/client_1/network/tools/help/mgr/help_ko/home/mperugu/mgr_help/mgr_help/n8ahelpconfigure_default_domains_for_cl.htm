클라이언트의 기본 도메인 구성

클라이언트가 특정 도메인의 이름을 자주 요청하는 환경에서는 기본 도메인을 설정하는 것이 좋습니다. 기본 도메인이 프로파일에 설정되어 있으면 검증되지 않은 임의의 네트 서비스 이름이나 서비스 이름에 기본 도메인이 자동으로 추가됩니다. 예를 들어, 기본 도메인이 us.example.com으로 설정된 경우 다음 접속 문자열의 sales는 sales.us.example.com으로 검색됩니다.

CONNECT scott/tiger@sales

기본 도메인을 지정하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창의 목록에서 이름 지정을 선택합니다.

  3. 방법 탭을 선택합니다.

  4. 기본 도메인 필드에 도메인을 입력합니다.

  5. 파일 > 네트워크 구성 저장을 선택합니다.

Oracle Net Managerのディレクトリ・ネーミング要件

Oracle Net Managerでは、ディレクトリ・サーバー内のネット・サービス名の作成と同様に、データベース・サービスのネットワーク・ルート情報を作成できます。

Oracle Net Managerでディレクトリに書き込むには、次の前提条件があります。

  • Oracleスキーマを指定してディレクトリを構成する必要があります。

  • Oracle Net Managerを実行するコンピュータは、アクセスしてエントリを書き込むデフォルトのOracleコンテキストを指定して構成する必要があります。

これらの構成は、Oracle Net Configuration Assistantで設定します。詳細は、「ディレクトリ・ネーミング構成の概要」を参照してください。Oracle Net Managerには、この機能はありません。

ディレクトリ・アクセスが構成されていない場合、ナビゲータ・ペインの「Oracle Netの構成」→「ディレクトリ」フォルダ、および「ツール」→「ディレクトリ」メニューは使用できません。

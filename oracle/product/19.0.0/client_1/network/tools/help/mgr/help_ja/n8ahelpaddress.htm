アドレス

ネットワーク・オブジェクトのネットワーク・プロトコル・アドレス。

接続の確立時に、クライアントとリクエストの受信者(リスナー、Oracle Names ServerまたはOracle Connection Manager)は、同じプロトコル・アドレスを使用して構成されます。クライアントはこのアドレスを使用して特定のネットワーク・オブジェクト位置に接続リクエストを送信し、受信者はこのアドレスに基づきリクエストをリスニングします。

次のアドレスでは、リスナーのリスニング・アドレスを定義しています。

ADDRESS=

(PROTOCOL=tcp)

(HOST=sales-server)

(PORT=1521))

アドレスのプロトコル・パラメータは選択したプロトコルによって異なります。

Oracle Net Manager 的目錄命名需求

您可以使用 Oracle Net Manager 來建立資料庫服務的網路路由資訊，以及建立目錄伺服器中的網路服務名稱。

若要讓 Oracle Net Manager 寫入目錄：

  • 必須以 Oracle 綱要設定目錄。

  • 執行 Oracle Net Manager 的電腦必須設定預設的 Oracle Context 以存取和寫入項目。

這些組態設定值是以 Oracle Net 組態輔助程式設定的，如目錄命名組態簡介中的說明。Oracle Net Manager 不提供此功能。

如果未設定目錄存取，瀏覽窗格中的 Oracle Net 組態 > 目錄資料夾以及命令 > 目錄功能表就無法使用。

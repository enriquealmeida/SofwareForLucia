Priorizar Métodos de Nomeação

Após os métodos de nomeação terem sido configurados, eles deverão ser priorizados no perfil. O método de nomeação na parte superior da lista é usado para resolver um nome do serviço de rede ou nome de serviço de banco de dados em um descritor de conexão. Se não for possível resolver o nome, o segundo método da lista será usado.

Para especificar a ordem dos métodos de nomeação, faça o seguinte:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

  2. Na lista do painel direito, selecione Nomeação.

  3. Selecione a guia Métodos.

  4. Selecione métodos de nomeação na lista Métodos Disponíveis e escolha o botão de seta para a direita. O método de nomeação selecionado passa para a lista Métodos Selecionados.

  5. Ordene os métodos de nomeação de acordo com o método que deseja que o Oracle Net tente primeiro ao tentar resolver nomes de serviço de rede ou nomes de serviços de bancos de dados. Selecione o método de nomeação na lista Métodos Selecionados e escolha Promover ou Rebaixar, para mover a seleção para cima ou para baixo na lista.

  6. Escolha Arquivo > Salvar Configuração de Rede.

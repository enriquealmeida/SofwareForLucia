Acheminement des demandes de connexion

Les clients peuvent être configurés de manière que les demandes de connexion soient toujours transmises à un processus serveur dédié ou à Oracle Connection Manager.

Pour acheminer les demandes de connexion :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Général.

  3. Sélectionnez l'onglet Routage.

  4. Sélectionnez votre mode préféré d'acheminement des demandes de connexion.

  5. Choisissez Fichier > Enregistrer la configuration réseau.

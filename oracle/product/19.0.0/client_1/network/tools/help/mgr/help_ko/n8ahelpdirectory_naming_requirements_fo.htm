Oracle Net Manager에 대한 디렉토리 이름 지정 요구 사항

Oracle Net Manager를 사용하면 디렉토리 서버에 네트 서비스 이름을 생성하는 것은 물론 데이터베이스 서비스에 대한 네트워크 경로 정보를 생성할 수 있습니다.

Net8 Assistant가 디렉토리에 작성할 수 있도록 하려면 다음과 같이 하십시오.

  • 디렉토리를 Oracle 스키마를 사용하여 구성해야 합니다.

  • Oracle Net Manager를 실행 중인 시스템은 항목을 액세스하고 기록할 수 있도록 기본 Oracle Context로 구성되어야 합니다.

이러한 구성 설정은 디렉토리 이름 지정 구성 개요에 설명된 대로 Oracle Net Configuration Assistant를 사용하여 설정됩니다. Oracle Net Manager는 이 기능을 제공하지 않습니다.

디렉토리 액세스가 구성되지 않은 경우, 네비게이터 창의 Oracle Net 구성 > 디렉토리 폴더 및 명령 > 디렉토리 메뉴를 사용할 수 없습니다.

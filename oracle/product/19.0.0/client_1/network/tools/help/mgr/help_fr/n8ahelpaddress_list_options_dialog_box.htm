Boîte de dialogue Options de liste d'adresses

La boîte de dialogue Options de liste d'adresses vous permet d'indiquer l'ordre et la façon dont les adresses de protocole des descripteurs de connexion sont utilisées dans une connexion.

Essayer chaque adresse, dans l'ordre, jusqu'à ce que l'une réussisse

Sélectionnez cette option pour activer le changement d'adresse de connexion pour un client Oracle9i ou Oracle8i. Si le système ne parvient pas à accéder à la première adresse, le changement d'adresse de connexion permet au client d'utiliser l'adresse de protocole de processus d'écoute suivante. Sélectionnez cette option ainsi que Rendre l'adresse compatible avec Net8 8.0 Client pour désactiver le routage de la source sur les clients de version 8.0 ou antérieure.

Essayer chaque adresse, au hasard, jusqu'à ce que l'une réussisse

Sélectionnez cette option pour activer la répartition de la charge du client et la reprise sur délai de connexion pour un client Oracle9i ou Oracle8i.

Essayer une adresse au hasard

Sélectionner cette option pour activer la répartition de la charge du client pour un client Oracle9i ou Oracle8i.

Utiliser chaque adresse, dans l'ordre, jusqu'à ce que la destination soit atteinte

Sélectionner cette option pour activer le routage de la source sur les clients Oracle8i ou Oracle8.

Cette option est obligatoire lorsque plusieurs arrêts d'adresse sont nécessaires pour atteindre la destination. Elle est généralement utilisée pour activer les fonctionnalités d'Oracle Connection Manager.

Ce paramètre n'est pas obligatoire pour un client Oracle9i.

N'utiliser que la première adresse

Sélectionnez cette option pour désactiver les fonctions de routage de la source, de reprise sur délai de connexion et de répartition de la charge du client.

Rendre l'adresse compatible avec Net8 8.0 Client

Sélectionnez cette option si vous configurez un client de la version 8.0. Vous ne pourrez alors sélectionner que les options suivantes :

- Essayer chaque adresse, dans l'ordre, jusqu'à ce que l'une réussisse

Utiliser chaque adresse, dans l'ordre, jusqu'à ce que la destination soit atteinte

Désactivez cette option si vous configurez un client Oracle9i ou Oracle8i.

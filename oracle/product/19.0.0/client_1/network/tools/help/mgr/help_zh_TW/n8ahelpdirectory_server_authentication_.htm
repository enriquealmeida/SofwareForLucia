目錄伺服器認證對話方塊

您可以使用「目錄伺服器認證」對話方塊，為目錄伺服器中選取的 Oracle Context 設定認證證明資料。

您必須使用目錄伺服器來認證 Oracle Net Manager，以執行任何目錄伺服器作業，如檢視、新增或修改目錄中的 Oracle Net 項目。

使用者

輸入具有目錄存取權的授權使用者。

如果要新增或修改項目，請以辨別名稱 (DN) 的格式輸入具備新增和修改權限的目錄管理員使用者。例如 c=US,o=EXAMPLE,ou=admin,cn=scott 就是管理員 Scott 的 DN。

密碼

輸入使用者的密碼。

注意：如果只要檢視物件，而目錄已設定「匿名」權限，請將「使用者和密碼」欄位保留空白。

依照預設，Oracle Net Manager 會嘗試使用下列其中一種方法認證目錄：

  • 原生認證：使用 Windows NT 作業系統使用者名稱和密碼。

  • 強式認證：使用「安全通訊端層 (SSL)」憑證。

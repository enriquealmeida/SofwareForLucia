Alias de Serviço de Rede

Um nome alternativo para um objeto de nomeação de diretório em um servidor de diretório. Um servidor de diretório armazena apelidos de serviço de rede para um nome do serviço de rede ou um serviço de banco de dados. Uma entrada de apelido de serviço de rede não tem informações de descritor de conexão. Em vez disso, ela só faz referência à localização do objeto para o qual é um apelido. Quando um cliente solicita uma pesquisa de diretório para um apelido de serviço de rede, o diretório determina que a entrada seja um apelido de serviço de rede e conclui a pesquisa como se fosse a entrada referenciada.

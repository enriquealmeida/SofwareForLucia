Panoramica sulla configurazione di denominazione locale

Questo argomento è costituito dalle sezioni seguenti.

Installazione amministratore o runtime

Se l'utilizzo della directory non è stato configurato, l'Assistente alla configurazione di Oracle Net richiede di configurare i nomi di servizi di rete nel file tnsnames.ora per un'installazione client amministratore o runtime.

Installazione personalizzata

Se durante un'installazione server o client personalizzata si seleziona la denominazione locale come metodo di denominazione, l'Assistente alla configurazione di Oracle Net richiede all'utente di configurare i nomi di servizi di rete nel file tnsnames.ora .

Configurazione con Oracle Net Manager

Oracle Net Manager consente di configurare i nomi di servizi di rete, nonché:

  • creare nomi di servizi di rete per i servizi di database non Oracle;

  • configurare più instradamenti per un listener utilizzando più indirizzi di protocollo;

  • configurare informazioni CONNECT_DATA avanzate per un servizio di database o un nome di servizio di rete.

Per accedere alla configurazione della denominazione locale, effettuare le operazioni riportate di seguito.

Nel riquadro del Navigator, scegliere la cartella Locale > Denominazione dei servizi per creare i nomi di servizi di rete o modificare le informazioni del descrittore di connessione.

Per ulteriori informazioni, vedere il capitolo Configuring Naming Methods del manuale Oracle10i Net Administrator's Guide.

Servidor Compartido

Un servidor que está configurado para que varios procesos del usuario puedan compartir un número muy reducido de procesos del servidor, de forma que aumenta la cantidad de usuarios soportados. Con una configuración de Servidor Compartido, varios procesos del usuario se conectan a un distribuidor. El distribuidor dirige distintas solicitudes de sesiones de entrada en red a una cola común. Un proceso de servidor compartido inactivo de un pool compartido de procesos de servidor toma una solicitud de la cola. De este modo, un grupo pequeño de procesos del servidor puede atender a una gran cantidad de clientes. Compárelo con el servidor dedicado.

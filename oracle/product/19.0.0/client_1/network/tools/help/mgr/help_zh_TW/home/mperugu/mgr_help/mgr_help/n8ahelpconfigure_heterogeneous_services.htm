設定監聽器的異質服務

您可以使用「異質服務」從 Oracle 伺服器存取非 Oracle 系統。為了起始對非 Oracle 系統的連線，Oracle 伺服器會透過閘道上的監聽器啟動代理程式處理作業。設定監聽器「異質服務」的組態涉及建立「異質代理程式」的服務資訊。

設定監聽器的異質服務：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

  2. 選取監聽器。

  3. 從右窗格的清單中，選取其他服務。

  4. 選擇新增服務。即會顯示一個新的服務頁籤。

  5. 在程式名稱欄位輸入代理程式執行檔，在 SID 欄位輸入代理程式的 Oracle 系統 ID (SID)，並在 Oracle 本位目錄欄位輸入可執行的 Oracle 本位目錄。

  6. 確認伺服器設定有存取「異質服務」的連線描述區，請參閱設定外部程序的連線描述區。

  7. 選擇檔案 > 儲存網路組態。

Überblick über Listener

Dienste koordinieren ihre Sessions mithilfe eines Listeners, einem Prozess auf dem Server, der Verbindungsanforderungen für eine Clientanwendung empfängt. Listener sind so konfiguriert, dass sie auf Protokolladressen für einen Datenbankdienst oder Nicht-Datenbank-Dienst "hören".

Protokolladressen sind in der Listener-Konfigurationsdatei listener.ora für einen Datenbankdienst oder einen anderen Dienst konfiguriert. Ein Oracle9i- oder Oracle8i-Datenbankdienst registriert seine Informationen automatisch mit dem Listener, wie z.B. Servicenamen, Instanznamen und Lastinformationen. Dieses Feature, das als Serviceregistrierung bezeichnet wird, erfordert keine Servicekonfiguration in der Datei listener.ora. Andere Dienste, einschließlich der Datenbankdienste von Oracle8 und Oracle7 sowie die Verwaltungstools von Oracle Enterprise Manager, erfordern eine Dienstkonfiguration in der Datei listener.ora .

Nachdem der Listener konfiguriert wurde, kann er mit dem Utility Listener Control verwaltet werden.

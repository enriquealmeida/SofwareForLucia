Funciones Clave del Gestor de Red de Oracle

El Gestor de Red de Oracle es una herramienta que le ayudará a configurar y gestionar su entorno de redes Oracle. Con la Red de Oracle podrá configurar y gestionar las funciones y componentes del Gestor de Red de Oracle siguientes:

Nomenclatura de Servicios

Puede crear o modificar descripciones de red para servicios de base de datos en un archivo tnsnames.ora, servidor de directorios o un servidor de Oracle Names. Las descripciones de red, denominadas descriptores de conexión, se asignan a identificadores de conexión que los clientes utilizan en las cadenas de conexión durante la conexión a una base de datos.

Listener

Puede crear o modificar un listener, que es un proceso del servidor que recibe y responde a las solicitudes de conexión cliente a un servicio de bases de datos.

Perfil

Puede crear o modificar un perfil, que es una recopilación de parámetros que determinan el modo en que el cliente se conecta a la red Oracle. Puede utilizar los parámetros del cliente para métodos de nomenclatura, registro, rastreo, nomenclatura externa y Oracle Advanced Security.

sqlnet.ora 파일

다음을 지정하는 클라이언트나 서버의 구성 파일입니다.

  • 한정되지 않은 서비스 이름이나 네트 서비스 이름에 추가할 클라이언트 도메인

  • 이름을 분석할 때 클라이언트가 사용할 이름 지정 방법의 순서

  • 사용할 로깅 및 추적 기능

  • 접속 경로

  • 기본 Oracle Names Server

  • 외부 이름 지정 매개변수

  • Oracle Advanced Security 매개변수

sqlnet.ora 파일은 다음 위치 중 하나에 있어야 합니다.

  1. TNS_ADMIN 환경 변수에 의해 지정된 디렉토리입니다. TNS_ADMIN 환경 변수가 Windows NT에서 변수로 정의되지 않은 경우 레지스트리에 있을 수 있습니다.

  2. UNIX의 $ORACLE_HOME/network/admin 디렉토리 또는 Windows 운영 체제의 ORACLE_HOME\network\admin 디렉토리입니다.

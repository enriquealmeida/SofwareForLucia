Parametri generali: Generale

La scheda Generale consente di configurare i parametri generali per il listener.

Nome listener

Visualizza il nome del listener.

Tempo di attesa all'avvio

Specificare il tempo, in secondi, per l'attesa del listener prima della risposta a un comando START della utility Controllo del listener.

Nota: questa impostazione non è più valida e non sarà più supportata nelle release future. Se è necessario questo parametro per eseguire il listener, avvertire il Supporto Oracle.

Salva configurazione alla chiusura

Scegliere questa opzione per rendere permanenti le modifiche apportate dal comando SET della utility Controllo del listener.

Informazioni di contatto SNMP

Immettere il testo da inserire nel file snmp.ora da utilizzare con Oracle Enterprise Manager. Queste informazioni verranno inviate con qualsiasi connessione SNMP e possono essere composte da qualsiasi stringa. Non influiscono sul processo di creazione di una connessione SNMP. In genere, contengono contatti per l'amministratore o l'utente del client.

ADMIN_RESTRICTIONS_listener_name

Limita l'amministrazione in fase di esecuzione del listener.

L'impostazione predefinita è off. L'impostazione del parametro ADMIN_RESTRICTIONS_listener_name=on disabilita la modifica in fase di esecuzione dei parametri nel file listener.ora. Ovvero il listener non accetta i comandi SET che modificano i parametri inclusi in tale file. Per modificare i parametri nel file listener.ora, incluso ADMIN_RESTRICTIONS_listener_name, modificare manualmente il file listener.ora e ricaricare i parametri (mediante il comando RELOAD) per applicare le nuove modifiche senza interrompere e riavviare in modo esplicito il listener.

Procedimentos Externos

Um procedure PL/SQL em execução em um servidor Oracle pode chamar uma função ou procedimento externo que é elaborado em linguagem de programação C e armazenado em uma biblioteca compartilhada. Para que o banco de dados Oracle estabeleça conexão com procedimentos externos, o servidor deverá ser configurado com um nome do serviço de rede e o listener deverá ser configurado com informações sobre endereço de protocolo e informações de serviço.

目錄中的 Oracle Net 項目

目錄中的項目會按照其辨別名稱 (DN)，排列成階層式的樹狀結構。此目錄項目樹狀結構稱為目錄資訊樹狀結構 (DIT)。每一個項目均包含一或多個描述該項目的屬性。

DIT 中的 Oracle Net 項目可以是以下任何之一：

  • 建立資料庫時建立的資料庫服務

  • 網路服務名稱

  • 網路服務別名

所有 Oracle Net 項目會顯示在儲存 Oracle 項目的 Oracle Context 子樹狀結構。這個子樹狀結構是在伺服器上「自訂」安裝時，建立在使用者指定的 DIT 位置中。從屬端會設定目錄用途以及預設的 Oracle Context，用來查詢項目。

每一個項目包含一或多個描述項目的屬性。每一個屬性有一種類型和一個值。Oracle Net 網路目錄有以下的屬性：

  • 一般名稱

  • 項目版本

  • 連線描述區

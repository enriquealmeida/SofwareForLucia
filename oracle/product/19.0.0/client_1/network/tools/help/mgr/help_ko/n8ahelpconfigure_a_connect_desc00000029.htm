외부 프로시저에 대한 접속 기술자 구성

외부 프로시저는 다른 프로그램에서 호출되는 프로시저이지만 다른 언어로 작성되어 있습니다. 특별한 목적으로 처리하는 데 필요한 C 루틴을 하나 이상 호출하는 PL/SQL 프로그램을 예로 들 수 있습니다.

Oracle 데이터베이스에서 외부 프로시저에 접속할 수 있도록 하려면 데이터베이스 서버가 외부 프로시저에 액세스할 네트 서비스 이름으로 구성되어야 합니다.

주: 설치 시 Oracle Net Configuration Assistant는 외부 프로시저에 대한 네트 서비스 이름을 자동으로 생성합니다.

외부 프로시저에 대한 데이터베이스 서버를 구성하려면 다음과 같이 하십시오.

  1. 리스너에 대한 외부 프로시저 구성에 설명된 대로 리스너가 외부 프로시저에 대해 구성되어 있는지 확인합니다.

  2. 데이터베이스의 tnsnames.ora 파일에 네트 서비스 이름을 생성합니다. 접속 기술자 정보가 리스너에 대해 구성된 정보와 일치하는지 확인합니다.

  3. 파일 > 네트워크 구성 저장을 선택합니다.

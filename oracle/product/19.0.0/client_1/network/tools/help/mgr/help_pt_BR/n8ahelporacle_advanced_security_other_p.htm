Oracle Advanced Security: Outros Parâmetros

A guia Outros Parâmetros permite que você configure parâmetros de serviços de autenticação adicionais.

Na listaServiço de Autenticação, selecione o método de autenticação. Dependendo do método escolhido, especifique informações nos campos, conforme apropriado.

KERBEROS(V5)

Elemento Descrição
Serviço Informe qualquer string para especificar o nome de serviço Kerberos.
Arquivo de Cache de Credencial Informe qualquer nome de caminho válido para especificar o caminho do cache de credencial do Kerberos.
Arquivo de Configuração Informe qualquer nome de caminho válido para especificar o arquivo de configuração do Kerberos.
Arquivo de Tradução de Realm Informe um caminho válido para especificar o nome do host do Kerberos para o arquivo de tradução de realm.
Tabela-chave Especifique qualquer nome de caminho válido para especificar o arquivo de chave secreta do Kerberos.
Desvio do Clock Informe um inteiro positivo no para especificar a diferença aceitável no número de segundos entre o momento em que uma credencial é enviada e o momento em que ela é recebida.

CYBERSAFE

Serviço GSSAPI: Especifique um serviço CYBERSAFE. Informe uma string principal de serviço corretamente formatada.

NTS

Não são necessários parâmetros adicionais.

RADIUS

Elemento Descrição
Nome do Host Especifique o nome do servidor RADIUS principal.
Número da Porta Informe o número da porta do servidor RADIUS principal.
Timeout (segundos) Informe o número de segundos que o servidor de banco de dados Oracle deve esperar pela resposta do servidor RADIUS principal.
Número de Repetições Informe o número de vezes que o servidor de banco de dados Oracle deve reenviar as mensagens para o servidor RADIUS principal.
Arquivo Secreto Informe o caminho do arquivo de chave secreta.
Enviar Contabilidade Especifique on para ativar ou off para desativar a contabilidade.
Resposta a Desafio Especifique on para ativar ou off para desativar a resposta ao desafio.
Palavra-Chave Default Informe a palavra-chave para solicitar um desafio do servidor RADIUS.
Nome da Classe de Interface Informe o nome da classe que você criou para tratar a conversão de desafio a resposta para o cliente Oracle e o servidor RADIUS.

Para obter mais informações, consulte o Oracle Advanced Security Administrator's Guide.

Oracle Connection Manager

클라이언트 접속 요청을 다음 홉으로 전송하거나 데이터베이스 서버에 직접 전송할 수 있는 경로 지정 프로그램입니다. Oracle Connection Manager를 통해 접속 요청을 전송하는 클라이언트는 Oracle Connection Manager에 구성된 액세스 제어, 다중 전송 및 프로토콜 변환 기능을 사용할 수 있습니다.

Oracle Connection Manager는 다음과 같은 환경에서 접속 경로 지정을 제공합니다.

  • 많은 수의 사용자가 단일 프로토콜을 통해 단일 서비스에 액세스해야 할 경우

  • 클라이언트와 서버에 다른 프로토콜이 설치되어 있어 일반적인 방법으로는 접속할 수 없는 경우

  • 기본 Oracle Net 접속이 클라이언트 및 서버에 동일한 프로토콜을 설치하도록 요구하는 경우

  • TCP/IP 환경에서 지정된 서버에 대한 클라이언트 액세스 제어가 필요한 경우

Oracle Connection Manager에 대한 자세한 내용은 Oracle9i Net Services Administrator's Guide의 Networking Challenges in the Internet Age 장을 참조하십시오.

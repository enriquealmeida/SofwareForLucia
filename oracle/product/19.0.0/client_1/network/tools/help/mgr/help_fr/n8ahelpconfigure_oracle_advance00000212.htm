Configuration des méthodes d'authentification d'Oracle Advanced Security

Les méthodes d'authentification centralisées et sécurisées vous permettent de vous assurer de l'identité des utilisateurs, des clients et des serveurs en environnement distribué. Les méthodes d'authentification réseau offrent également l'avantage d'un accès avec connexion unique (SSO) pour les utilisateurs.

Oracle Net Manager permet de configurer les adaptateurs d'authentification suivants :

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • authentification native Windows NT

Remarque : Si vous avez configuré le protocole TCP/IP avec SSL, la couche SSL (Secure Sockets Layer) est configurée par défaut. La méthode d'authentification choisie dans cet onglet remplace les fonctions d'authentification de SSL. Voir l'onglet SSL pour configurer d'autres options SSL. Si vous utilisez le protocole TCP/IP avec SSL, n'employez pas une autre méthode d'authentification.

Pour configurer les méthodes d'authentification :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Oracle Advanced Security.

  3. Sélectionnez l'onglet Authentification.

  4. Sélectionnez dans la liste Méthodes disponibles la méthode d'authentification que vous souhaitez utiliser, puis choisissez le bouton "flèche vers la droite" (>).

  5. Modifiez l'ordre des méthodes d'authentification en fonction de celle que vous voulez utiliser. Choisissez une méthode dans la liste Méthodes sélectionnées, puis cliquez sur le bouton Promouvoir ou Rétrograder, selon que vous désirez la déplacer vers le haut ou vers le bas de la liste.

  6. Choisissez l'onglet Autres paramètres en vue de configurer des paramètres supplémentaires pour les méthodes d'authentification suivantes :

    • Kerberos

    • CyberSAFE

    • RADIUS

  7. Choisissez Fichier > Enregistrer la configuration réseau.

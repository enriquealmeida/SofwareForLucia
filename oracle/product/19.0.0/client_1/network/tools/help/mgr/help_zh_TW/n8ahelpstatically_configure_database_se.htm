靜態設定資料庫服務資訊

為了讓監聽器接受 Oracle8 或 Oracle7 資料庫的從屬端要求，listener.ora 檔案必須設定資料庫相關資訊。

如果資料庫是 Oracle9i 或 Oracle8i 資料庫，listener.ora 檔案中不需要靜態服務資訊。服務資訊會透過服務註冊，自動在監聽器註冊。不過，如果您使用的是 Oracle Enterprise Manager，就需要 Oracle8i 資料庫的靜態資訊。

設定監聽器的資料庫服務資訊：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

  2. 選取監聽器。

  3. 從右窗格的清單中，選取資料庫服務。

  4. 選取新增資料庫。即會顯示一個新的資料庫頁籤。

  5. 在全域資料庫名稱欄位中輸入全域資料庫名稱，在 Oracle 本位目錄欄位中輸入 Oracle 本位目錄的位置，以及在 SID 欄位中輸入資料庫的 Oracle 系統 ID (SID)。

  6. 選擇檔案 > 儲存網路組態。

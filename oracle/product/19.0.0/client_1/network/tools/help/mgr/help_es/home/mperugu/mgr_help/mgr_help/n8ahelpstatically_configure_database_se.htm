Configuración Estática de Información de Servicio de Base de Datos

Para que el listener acepte las solicitudes del cliente de una base de datos Oracle8 u Oracle7, el archivo listener.ora se debe configurar con la información de la base de datos.

Si la base de datos es Oracle9i u Oracle8i, no se necesita información estática de servicio en el archivo listener.ora. La información de servicio se registra con el listener de forma automática mediante el registro del servicio. Sin embargo, si utiliza Oracle Enterprise Manager, se necesita información estática para la base de datos Oracle8i.

Para configurar la información de servicio de base de datos para un listener:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Listeners.

  2. Seleccione un listener.

  3. Seleccione Servicios de Base de Datos de la lista del panel de la derecha.

  4. Haga clic en Agregar Base de Datos. Aparece un nuevo separador Base de Datos.

  5. Introduzca el nombre de la base de datos global en el campo Nombre de la Base de Datos Global, la ubicación del directorio raíz de Oracle en el campo Directorio Raíz de Oracle y el identificador del sistema Oracle (SID) de la base de datos en el campo SID.

  6. Seleccione Archivo > Guardar Configuración de Red.

Modificando Informações de Rota de Rede para um Serviço de Banco de Dados

Observação: Somente usuários membros do grupo OracleNetAdmins ou OracleContextAdmins podem modificar informações de rede para um serviço de banco de dados em um diretório.

Quando o registro do banco de dados junto ao diretório é concluído, o Assistente de Configuração de Banco de Dados cria uma entrada de serviço de banco de dados no diretório. Por default, esta entrada contém informações de rota da rede que incluem a localização do listener através de um endereço de protocolo. Você pode criar estas informações, caso tenham sido removidas, ou modificar as informações existentes sobre rota da rede.

Para criar ou modificar informações sobre rota de rede para um serviço de banco de dados:

  1. No painel do navegador, expanda Configuração do Oracle Net > Diretório > Nomeação de Serviço.

  2. Selecione o serviço de banco de dados. O painel direito exibe o nome do serviço de destino atual.

  3. Na caixa Configuração de Endereço, escolha o sinal de adição (+). Uma nova guia Endereço será exibida.

  4. Selecione um protocolo e especifique informações apropriadas sobre parâmetros de protocolo para o protocolo selecionado.

  5. No painel direito, escolha Aplicar.

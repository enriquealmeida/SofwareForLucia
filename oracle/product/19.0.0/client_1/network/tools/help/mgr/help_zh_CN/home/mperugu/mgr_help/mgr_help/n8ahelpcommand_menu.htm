“命令”菜单

“命令”菜单使您能够执行以下操作：

目录

此子菜单包含对目录服务器执行操作的命令：

元素 说明
更改当前上下文 选择此项以选择另一个 Oracle Context 而不使用当前的 Oracle Context。Oracle Context 是存储 Oracle 条目的目录中的子树，包含与目录命名一起使用的数据库服务名和网络服务名。当选定一个新的 Oracle Context 时，导航器的 Oracle Net 配置 > 目录 > 服务命名文件夹就会为新的 Oracle Context 显示对象。
设置验证 选择以便将验证身份证明设置为目录服务器。
导出网络服务名 选择以启动“目录服务器移植向导”。此向导将存储在tnsnames.ora文件中的 Net 服务名导出到目录服务器。

注：此目录菜单只在已配置目录访问后才显示。有关目录访问配置和其他要求的详细信息，请参阅 Oracle Net Manager 的目录命名要求。

测试服务

选择以使用存储在 tnsnames.ora 文件或目录服务器中的连接描述符信息连接到某个数据库服务。

Finestra di dialogo Opzioni lista indirizzi

La finestra di dialogo Opzioni lista indirizzi consente di specificare l'ordine e la modalità d'utilizzo degli indirizzi di protocollo presenti nei descrittori di connessione delle connessioni.

Provare con ciascun indirizzo, in ordine, fino a individuarne uno corretto

Selezionare questa opzione per abilitare il failover della fase di connessione per un client Oracle 9i o Oracle8i. Se l'indirizzo del campo è irraggiungibile, il failover consente al client di utilizzare il successivo indirizzo di protocollo del listener. Selezionare questa opzione e l'opzione Utilizzare opzioni compatibili con i client Net8 8.0 per disattivare l'instradamento sorgente per un client 8.0 o di una release precedente.

Provare con ciascun indirizzo, in ordine casuale, fino a individuarne uno corretto

Selezionare questa opzione per abilitare il bilanciamento del carico del client e il failover della fase di connessione per un client Oracle9i o Oracle8i.

Provare con un solo indirizzo, selezionato casualmente

Selezionare questa opzione per abilitare la funzione di bilanciamento del carico del client per un client Oracle9i o Oracle8i.

Utilizzare ciascun indirizzo in ordine fino a raggiungere la destinazione

Selezionare questa opzione per abilitare l'instradamento sorgente per un client Oracle8i o Oracle8.

Questa opzione è necessaria quando occorrono più indirizzi per raggiungere la destinazione. In genere è utilizzata per abilitare le funzioni di Oracle Connection Manager.

Non è un parametro obbligatorio per i client Oracle9i.

Utilizzare solo il primo indirizzo

Selezionare questa opzione per disabilitare le funzioni di instradamento sorgente, il failover della fase di connessione e il bilanciamento del carico del client.

Utilizzare opzioni compatibili con i client Net8 8.0

Selezionare questa opzione se si sta configurando un client di una release precedente alla 8.0. Sono disponibili soltanto le seguenti opzioni:

-Provare con ciascun indirizzo, in ordine, fino a individuarne uno corretto

-Utilizzare ciascun indirizzo in ordine fino a raggiungere la destinazione

Deselezionare questa opzione se si sta configurando un client Oracle9i o Oracle8i.

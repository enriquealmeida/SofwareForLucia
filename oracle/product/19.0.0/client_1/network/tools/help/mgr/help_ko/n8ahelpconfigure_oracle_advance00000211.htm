Oracle Advanced Security 구성

Oracle Net Manager를 사용하여 Oracle Advanced Security를 구성할 수 있습니다. 각 Oracle Advanced Security 탭 페이지에서 별도의 매개변수 집합 구성이 가능합니다. Oracle 고급 보안을 사용하면 데이터 암호화와 무결성 검사, 향상된 인증 및 Single Sign-On이 가능하며 LDAP 준수 디렉토리 서버의 중앙 집중화된 사용자 관리와 인증서를 기반으로 하는 Single sign-on(이 기능은 Secure Sockets Layer (SSL)에 의존함)을 제공합니다.

Oracle Advanced Security를 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창에 있는 목록에서 Oracle Advanced Security를 선택합니다.

  3. 다음 탭 패널이 나타납니다.

    • 인증

    • 기타 매개변수

    • 무결성

    • 암호화

    • SSL

  4. 구성을 변경합니다.

  5. 파일 > 네트워크 구성 저장을 선택합니다.

변경 사항은 sqlnet.ora 파일에 저장됩니다.

주: 목록에 Oracle Advanced Security가 표시되지 않으면 설치되지 않은 것입니다.

자세한 내용은 Oracle Advanced Security Administrator's Guide를 참조하십시오.

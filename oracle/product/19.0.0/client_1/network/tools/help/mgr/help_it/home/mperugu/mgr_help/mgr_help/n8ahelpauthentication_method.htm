Metodo di autenticazione

Un metodo di protezione che consente di stabilire chiaramente l'identità di utenti, client e server negli ambienti distribuiti. I metodi di autenticazione di rete offrono inoltre agli utenti i vantaggi della funzione Single Sign-On. I metodi di autenticazione elencati di seguito sono disponibili in Oracle9i e Oracle8i, a seconda che Oracle Advanced Security sia installato o meno:

  • CyberSafe

  • KERBEROS(v5)

  • RADIUS

  • SSL

  • Autenticazione originaria di Windows NT

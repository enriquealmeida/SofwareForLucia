Heterogeneous Services

Un componente integrato che fornisce la tecnologia necessaria per accedere a sistemi non Oracle dal database server Oracle. Heterogeneous Services consente di:

  • Utilizzare Oracle SQL per accedere in modo trasparente ai dati memorizzati su sistemi non Oracle come se risiedessero su un server Oracle.

  • Utilizzare le chiamate di procedura Oracle per accedere in modo trasparente a sistemi, servizi o interfacce API (application programming interfaces) non Oracle dal proprio ambiente distribuito Oracle.

Per stabilire una connessione a un sistema non Oracle, il database Oracle8i avvia un processo agente mediante il listener sul gateway. Affinché il database Oracle8i possa connettersi all'agente, è necessario che il server sia configurato con un nome di servizio di rete per accedere all'agente e che il listener sia configurato con le informazioni del servizio sull'agente.

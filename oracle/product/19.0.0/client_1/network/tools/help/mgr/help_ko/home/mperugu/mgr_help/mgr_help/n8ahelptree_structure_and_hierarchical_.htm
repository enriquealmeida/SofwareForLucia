트리 구조와 계층 폴더

Oracle Net Manager는 네트워크 객체 및 네트워크 객체가 포함하는 객체를 폴더 계층으로 구성된 그래픽적인 트리 뷰로 표시합니다. 네비게이터 창을 사용하여 각 폴더에서 객체를 보거나, 수정하거나, 추가할 수 있습니다.

폴더를 확장하면 객체와 폴더의 중첩 목록이 나타납니다. 객체를 선택하면 해당 객체에 대한 정보가 Oracle Net Manager의 오른쪽 창에 표시됩니다.

디렉토리

디렉토리 서버에서 구성된 접속 식별자라는 단순 이름을 표시합니다.

주: 디렉토리 폴더는 디렉토리에 대한 액세스가 구성된 경우 표시됩니다. 디렉토리 액세스 구성 및 다른 요구 사항에 대한 자세한 내용은 Oracle Net Manager에 대한 디렉토리 이름 지정 요구 사항을 참조하십시오.

로컬

로컬 구성 파일에 구성된 네트워크 구성 요소를 표시합니다.

  • tnsnames.ora 파일의 네트 서비스 이름

  • listener.ora 파일의 리스너

  • sqlnet.ora 파일의 프로파일

Oracle Names Server

Oracle Names Server를 표시합니다.

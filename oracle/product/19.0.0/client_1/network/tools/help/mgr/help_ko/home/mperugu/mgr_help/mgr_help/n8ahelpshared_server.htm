공유 서버

다수의 사용자 프로세스가 적은 수의 서버 프로세스를 공유하도록 구성하는 서버로 보다 많은 수의 사용자를 지원할 수 있습니다. 공유 서버 구성을 사용하면 많은 사용자 프로세스가 디스패처에 접속할 수 있습니다. 디스패처는 여러 개의 수신 네트워크 세션 요청을 공통 대기열로 보냅니다. 공유 풀의 서버 프로세스 중 유휴 공유 서버 프로세스가 대기열에서 요청을 가져옵니다. 이렇게 하면 서버 프로세스의 소형 풀이 많은 수의 클라이언트에게 서비스를 제공할 수 있습니다. 전용 서버와 대조됩니다.

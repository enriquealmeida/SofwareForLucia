一般信息：路由选择

路由选择选项卡使您能够在所有时刻将连接请求从本计算机引导到特定进程。

始终使用专用服务器

选择以强制监听程序从此客户机为所有网络会话衍生一个专用服务器。通过将(SERVER=dedicated)添加到连接描述符的CONNECT_DATA部分来完成该操作。即使配置了共享服务器，来自此客户机的连接仍会使用专用服务器。

使用此选项虽然可缩短连接时间，但会降低服务器的运行速度。因此 Oracle Corporation 建议在空闲时间很少的情况下处理大量数据库事务时，才使用此选项。

注：如果启用此选项，将覆盖服务标识属性工作表中的连接类型选项。

首选连接管理器路由

选择此选项以便在任何可能的时候将客户机发送到 Oracle Connection Manager。如果没有可用的 Oracle Connection Manager 地址，则连接将通过任何可用的监听程序地址发送。

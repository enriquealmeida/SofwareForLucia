プロファイル・ロギングおよびトレースの構成

クライアントは、ロギングおよびトレース用に構成できます。

通常、ログ情報は、UNIXの場合は$ORACLE_HOME/network/log、Windowsの場合はORACLE_HOME \network\logにあるsqlnet.logファイルに格納されます。

トレース情報は、UNIXの場合は$ORACLE_HOME/network/trace、Windowsの場合はORACLE_HOME\network\traceにある、sqlnet.trc(クライアントの場合)およびsvr_pid.trc(サーバーの場合)にデフォルトで格納されます

ロギングまたはトレースを使用可能にする手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「プロファイル」を展開します。

  2. 右ペインのリストから「一般」を選択します。

  3. 「ロギング」または「トレース」タブを選択します。

  4. ロギングまたはトレース設定を指定します

  5. 「ファイル」→「ネットワーク構成の保存」を選択します。

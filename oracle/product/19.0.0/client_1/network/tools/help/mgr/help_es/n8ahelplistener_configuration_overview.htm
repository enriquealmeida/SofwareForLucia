Visión General de la Configuración de Listener

En este tema, se incluyen las siguientes secciones.

Configuración durante la Instalación

Durante una instalación Enterprise Edition o Standard Edition en el servidor de bases de datos, el Asistente de Configuración de Red de Oracle configura un listener con el nombre LISTENER y con una dirección de protocolo TCP/IP para la base de datos Oracle. Durante una instalación personalizada, el Asistente de Configuración de Red de Oracle le solicitará que configure el nombre de listener y la dirección de protocolo que desee.

Además, se configura una dirección de protocolo IPC para llamadas de procedimientos externos, independientemente del tipo de instalación.

El Asistente de Configuración de Red de Oracle también configura de forma automática la información de servicios para los procedimientos externos en el archivo listener.ora.

Configuración con el Gestor de Red de Oracle

El Gestor de Red de Oracle permite:

  • Crear un listener.

  • Configurar la información de servicio para bases de datos Oracle8 o anteriores.

  • Configurar la información de servicio para bases de datos que no sean de Oracle, incluidos procedimientos externos y servicios heterogéneos.

  • Activar las funciones de rastreo y registro.

  • Activar la autenticación de contraseña para la Utilidad de Control de Listener.

Para acceder a la configuración del listener:

En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Listeners.

Para obtener más información, consulte el capítulo sobre configuración y administración del listener, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

Indirizzo protocollo

Un indirizzo che identifica l'indirizzo di rete di un oggetto di rete.

Quando si effettua una connessione, il client e il destinatario della richiesta, ad esempio il listener, il server Oracle Names oppure Oracle Connection Manager, vengono configurati con un indirizzo di protocollo identico. Il client utilizza questo indirizzo per inviare la richiesta di connessione alla posizione di un oggetto di rete particolare e il destinatario "ascolta" le richieste su questo indirizzo. È importante installare gli stessi protocolli per il client e il destinatario della connessione, nonché configurare gli stessi indirizzi.

Gli indirizzi di protocollo sono composti dagli elementi ADDRESS e ADDRESS_LIST.

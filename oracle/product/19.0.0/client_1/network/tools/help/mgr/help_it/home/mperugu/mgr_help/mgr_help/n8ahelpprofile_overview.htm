Panoramica sul profilo

Nel profilo sono specificate le preferenze di abilitazione e configurazione delle funzioni di Oracle Net sul client o sul server. Con un profilo è possibile:

  • specificare il dominio del client da aggiungere ai nomi non qualificati;

  • assegnare le priorità ai metodi di denominazione;

  • abilitare le funzioni di log e trace;

  • instradare le connessioni tramite processi specifici;

  • configurare parametri per ladenominazione esterna;

  • configurare Oracle Advanced Security.

Per accedere a un profilo, nel riquadro del Navigator espandere Configurazione di Oracle Net > Locale > Profilo.

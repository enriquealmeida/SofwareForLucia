Configuración del Cifrado

El cifrado es el proceso de disfrazar un mensaje para que no lo pueda leer nadie excepto el destinatario.

Para configurar los parámetros del cifrado:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione Oracle Advanced Security de la lista del panel de la derecha.

  3. Seleccione el separador Cifrado.

  4. Seleccione Cliente o Servidor de la lista Cifrado.

  5. Seleccione un valor de la lista Tipo de Cifrado para especificar el comportamiento del cliente o servidor al negociar el cifrado y la integridad. A continuación, se describen los cuatro valores posibles:

    Elemento Descripción
    aceptado El servicio estará activo si el otro lado de la conexión especifica "necesario" o "solicitado" y en el otro lado está disponible un algoritmo compatible. De lo contrario, estará inactivo.
    rechazado El servicio no debe estar activo y la conexión fracasará si el otro lado especifica "necesario".
    solicitado El servicio estará activo si el otro lado de la conexión especifica "aceptado", "necesario" o "solicitado" y en el otro lado está disponible un algoritmo compatible.
    necesario El servicio debe estar activo y la conexión fracasará si el otro lado especifica "rechazado" o si no existe un algoritmo compatible en el otro lado.

  6. Introduzca un cifrado inicial en el campo Cifrado Inicial: una cadena aleatoria con una longitud máxima de 256 caracteres. Este cifrado inicial se utiliza con el algoritmo de cifrado seleccionado en el siguiente paso.

  7. En la lista Métodos Disponibles, seleccione un algoritmo de cifrado y, a continuación, haga clic en el botón de flecha a la derecha (<) para moverlo a la lista Métodos Seleccionados. El orden de los métodos en la lista Métodos Seleccionados determina el orden de preferencia para la negociación. Oracle Advanced Security soporta los siguientes algoritmos:

    Elemento Descripción
    RC4_256 Tamaño de clave de 256 bits
    3DES168 Opción DES triple con clave triple (168 bits)
    RC4_128 Tamaño de clave de 128 bits
    3DES112 Opción DES triple con clave doble (112 bits)
    RC4_56 Tamaño de clave de 56 bits
    DES Tamaño de clave estándar de 56 bits
    RC4_40 Tamaño de clave de 40 bits
    DES40 Tamaño de clave de 40 bits

  8. Seleccione Archivo > Guardar Configuración de Red.

Nota: Todos los algoritmos de cifrado y longitudes de clave de Oracle Advanced Security están disponibles sólo para uso en EE.UU.

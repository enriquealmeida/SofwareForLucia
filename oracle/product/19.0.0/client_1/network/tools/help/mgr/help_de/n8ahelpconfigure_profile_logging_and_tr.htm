Listener-Logging und -Tracing konfigurieren

Der Client kann für Logging und Tracing konfiguriert werden.

Loginformationen werden im Allgemeinen in der Datei sqlnet.log in $ORACLE_HOME/network/log bei UNIX und ORACLE_HOME \network\log bei Windows-Plattformen gespeichert.

Traceinformationen werden standardmäßig in sqlnet.trc für den Client und svr_pid..trc für den Server in $ORACLE_HOME/network/trace bei UNIX and ORACLE_HOME \network\trace bei Windows-Plattformen gespeichert

So aktivieren Sie das Logging oder Tracing:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal > Profil im Navigator-Fensterbereich ein.

  2. Wählen Sie Allgemein aus der Liste im rechten Fensterbereich.

  3. Wählen Sie das Register Logging oder Tracing.

  4. Geben Sie die Einstellungen für das Logging oder Tracing an.

  5. Wählen Sie Datei >Netzwerkkonfiguration speichern.

CONNECT_DATA

대상 데이터베이스 서비스 이름 또는 Oracle SID(시스템 식별자)를 정의하는 접속 기술자의 일부분입니다. 다음 예에서 SERVICE_NAME은 sales.us.example.com이라는 데이터베이스 서비스를 지정합니다.

(DESCRIPTION=

(ADDRESS=(PROTOCOL=tcp)(HOST=sales-pc)(PORT=1521)

(CONNECT_DATA=(SERVICE_NAME=sales.us.example.com)))

CONNECT_DATA는 고급 접속 정보로 구성할 수 있습니다.

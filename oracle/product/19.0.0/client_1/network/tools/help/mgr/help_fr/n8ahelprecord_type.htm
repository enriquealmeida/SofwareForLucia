Type d'enregistrement

Type d'informations d'objet réseau stockées sur un serveur Oracle Names. Les serveurs Oracle Names prennent en charge les types d'enregistrement suivants :

- A.SMD : adresses réseau, y compris les adresses de service de base de données et les noms de service réseau.

- CNAME.SMD : alias d'un objet réseau.

- DL.RDBMS.OMD : liens globaux de base de données.

- DLCR.RDBMS.OMD : qualificatifs de lien.

- NS.SMD : adresses des serveurs Oracle Names.

- V1ADD.NPO.OMD : chaîne de connexion SQL*Net version 1

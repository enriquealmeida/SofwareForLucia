Überblick über Profile

Mit einem Profil werden Voreinstellungen für das Aktivieren und Konfigurieren von Oracle Net-Funktionen auf dem Client oder Server angegeben. Mit einem Profil können Sie folgende Schritte ausführen:

  • Client-Domain angeben, die an nicht vollständige Namen angehängt wird

  • Priorität der Benennungsmethoden festlegen

  • Aktivieren von Logging- und Tracing-Funktionen

  • Verbindungen über bestimmte Prozesse weiterleiten

  • Parameter für die externe Benennung konfigurieren

  • Konfigurieren von Oracle Advanced Security

Um auf ein Profil zuzugreifen, blenden Sie im Navigator-Fensterbereich "Oracle Net-Konfiguration > Lokal > Profil" ein.

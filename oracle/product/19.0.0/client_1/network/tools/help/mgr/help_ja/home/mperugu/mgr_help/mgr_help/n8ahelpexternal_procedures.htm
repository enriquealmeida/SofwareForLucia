外部プロシージャ

Oracleサーバー上で実行されるPL/SQLプロシージャは、Cプログラミング言語で作成され、共有ライブラリに格納された外部プロシージャやファンクションをコールできます。Oracleデータベースから外部プロシージャに接続するには、ネット・サービス名を指定してサーバーを構成し、プロトコル・アドレスおよびサービス情報を使用してリスナーを構成する必要があります。

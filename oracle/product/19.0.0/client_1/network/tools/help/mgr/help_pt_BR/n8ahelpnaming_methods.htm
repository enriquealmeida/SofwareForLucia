Nomeação: Métodos

A guia Métodos permite especificar quais métodos de nomeação este computador cliente deverá usar, além da ordem de prioridade. O método de nomeação na parte superior da lista é usado primeiro para resolver um determinado identificador de conexão, nome de serviço de rede ou serviço de banco de dados. Se não for possível resolver o nome, o próximo método especificado na lista será usado.

Durante uma solicitação de conexão, o cliente tenta entrar em contato com o primeiro método de nomeação listado para resolver identificadores de conexão em um descritor de conexão. Em seguida, o cliente encaminha a solicitação ao endereço de listener especificado no descritor de conexão.

O Oracle Net suporta os seguintes métodos de nomeação:

  • Nomenclatura local(TNSNAMES)

  • Nomenclartura de diretório(LDAP)

  • Nomenclatura de host(HOSTNAME)

  • Nomenclatura de conexão fácil(EZCONNECT)

  • NIS(Network Information Service)

Lista Métodos Selecionados

Exibe os métodos selecionados. A ordem dos métodos de nomeação na lista determina a ordem na qual o Oracle Net tenta utilizar os métodos de nomeação.

botão <

Escolha para mover um método de nomeação selecionado na lista Métodos Disponíveis para a lista Métodos Selecionados.

botão >

Escolha para mover um método de nomeação selecionado na lista Métodos Selecionados para a lista Métodos Disponíveis.

Botão Promover

Escolha para mover um método de nomeação para cima na lista Métodos Selecionados. O Oracle Net tenta usar o primeiro método para resolver um nome do serviço de rede ou um nome de serviço de banco de dados. Se não for possível usar o primeiro método, será utilizado o segundo método, e assim por diante.

Botão Rebaixar

Escolha para mover um método de nomeação selecionado para baixo na lista Métodos Selecionados.

Domínio Default

Informe o nome do domínio da rede a partir do qual ocorre a maioria das solicitações de clientes. Esse nome poderia ser o domínio em que o cliente reside ou um domínio a partir do qual o cliente geralmente solicita serviços de rede.

Quando definido, o nome do domínio é automaticamente anexado a qualquer nome não qualificado.

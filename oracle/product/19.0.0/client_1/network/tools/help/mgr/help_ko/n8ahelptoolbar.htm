도구 모음

Oracle Net Manager의 왼쪽에 있는 도구 모음은 폴더 계층의 네트워크 객체에 해당하는 아이콘을 포함합니다. 창에서 현재 보거나 선택한 객체에 따라 사용할 수 있는 도구 모음 단추가 달라집니다. 마우스 커서를 도구 모음 아이콘 위로 이동하면 아이콘 기능에 대한 설명이 나타납니다.

도구 모음을 구성하고 있는 아이콘은 다음과 같습니다.

아이콘 설명
추가 이 컴퓨터에 대한 새 네트 서비스 이름, 리스너 및 Oracle Names Server를 생성합니다.
삭제 서비스 이름 지정 및 리스너 객체를 삭제합니다.
접속 확인 선택한 서비스 이름 지정 객체의 데이터베이스 서비스에 대한 접속을 검증합니다. 접속 테스트를 수행하는 동안 데이터베이스 서비스에 대한 접속은 tnsnames.ora 파일이나 디렉토리 서버에 저장된 접속 기술자 정보를 사용하여 설정됩니다.
도움말 온라인 도움말을 표시합니다.

주: 일부 명령은 해당 객체 유형을 선택한 경우에만 사용할 수 있습니다. 예를 들어, 리스너를 생성하려는 경우 먼저 리스너 폴더를 선택해야 합니다.

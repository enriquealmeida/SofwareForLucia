區域命名組態簡介

本主題涵蓋下列段落：

管理員或程式實際執行安裝

如果未設定目錄用途，「Oracle Net 組態輔助程式」會提示您在 tnsnames.ora 檔案中，針對管理員或程式實際執行從屬端安裝，設定網路服務名稱。

自訂安裝

如果在自訂從屬端或伺服器安裝時選取區域命名作為命名方法，「Oracle Net 組態輔助程式」會提示您在tnsnames.ora檔案中設定網路服務名稱。

使用 Oracle Net Manager 進行設定

您可以使用 Oracle Net Manager 來設定網路服務名稱，以及：

  • 建立非 Oracle 資料庫服務的網路服務名稱

  • 設定透過多個協定位址連線監聽器

  • 設定資料庫服務或網路服務名稱的進階 CONNECT_DATA 資訊

存取區域命名組態：

從瀏覽窗格中，選擇區域 > 服務命名資料夾，以建立網路服務名稱或修改連線描述區資訊。

如需詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Configuring Naming Methods 章節。

接続リクエストのルーティング

クライアントは、接続リクエストが常に専用サーバー・プロセスまたはOracle Connection Managerにダイレクトされるように構成できます。

接続リクエストをルーティングする手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「プロファイル」を展開します。

  2. 右ペインのリストから「一般」を選択します。

  3. 「ルーティング」タブを選択します。

  4. 接続リクエストのルーティングに使用する方法を選択します。

  5. 「ファイル」→「ネットワーク構成の保存」を選択します。

数据库链接

定义从 Oracle 数据库服务器到另一个数据库服务器单向通信路径的指针。链接指针实际上被定义为数据字典表中的一个条目。要访问该链接，必须连接到包含数据字典条目的本地数据库。

“数据库链接”连接是单向的：它允许连接到本地数据库 A 的客户机使用存储在数据库 A 中的链接访问远程数据库 B 中的信息，但是，连接到数据库 B 的用户不能使用同一链接访问数据库 A 中的数据。如果数据库 B 上的本地用户要访问数据库 A 上的数据，他们必须定义存储在数据库 B 的数据字典中的链接。

支持下列数据库链接类型：

  • 在数据库特定方案中的私有数据库链接。只有私有数据库链接的所有者可以使用它。

  • 数据库的公共数据库链接。数据库的所有用户都可以使用它。

有关创建私有和公共数据库链接的详细信息，请参阅 Oracle10i Database Administrator's Guide。

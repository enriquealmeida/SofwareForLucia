Oracle Context 변경

Oracle Net Manager가 시작될 때 디렉토리 > 서비스 이름 지정 폴더에 표시되는 객체는 디렉토리 사용 구성 중에 Oracle Net Configuration Assistant로 설정된 기본 Oracle Context에 포함된 Oracle 서비스 항목입니다. 인증 권한을 갖는 다른 Oracle Context 항목을 생성 및 수정할 수 있습니다.

현재 Oracle Context를 변경하려면 다음과 같이 하십시오.

  1. 명령 > 디렉토리 > 현재 컨텍스트 변경을 선택합니다. 새 컨텍스트 선택 대화상자가 나타납니다.

  2. 디렉토리 이름 지정 컨텍스트 목록에서 확인할 Oracle Context를 포함하는 디렉토리 항목을 선택합니다.

  3. Oracle Context 목록에서 Oracle Context를 선택한 다음 확인을 선택합니다. 수정 사항이 현재 Oracle Context에 저장되지 않은 경우 변경 내용을 저장할 수 있는 [저장] 대화상자가 나타납니다.

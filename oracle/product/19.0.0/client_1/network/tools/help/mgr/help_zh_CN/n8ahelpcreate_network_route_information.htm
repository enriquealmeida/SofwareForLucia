修改数据库服务的网络路由信息

注：只有 OracleNetAdmins 或 OracleContextAdmins 组的成员用户可以在目录中修改数据库服务的网络信息。

将数据库注册到目录中之后，Database Configuration Assistant 将在该目录中创建数据库服务条目。默认情况下，此条目包含网络路由信息，其中涉及到利用协议地址的监听程序的位置。如果已将此信息删除可重新创建，还可以修改现有的网络路由信息。

要创建或修改数据库服务的网络路由信息，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle Net 配置 > 目录 > 服务命名。

  2. 选择数据库服务。右窗格显示当前目标服务名。

  3. 在地址配置框中，选取加号 (+)。将显示新的地址选项卡。

  4. 请选择一个协议并为所选协议输入相应的协议参数信息。

  5. 在右侧窗格中，选择应用。

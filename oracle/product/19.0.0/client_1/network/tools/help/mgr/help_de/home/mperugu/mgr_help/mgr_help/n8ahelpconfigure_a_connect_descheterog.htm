Verbindungsdeskriptor für Heterogene Dienste konfigurieren

Mit heterogenen Diensten können Sie von dem Oracle-Server auf fremde Systeme zugreifen. Um eine Verbindung zu einem fremden System herzustellen, startet der Oracle-Server einen Agent-Prozess über den Listener auf dem Gateway. Damit sich die Oracle-Datenbank bei einem fremden System anmelden kann, konfigurieren Sie den Server mit einem Net Service Name .

So konfigurieren Sie den Server für heterogene Dienste:

  1. Stellen Sie sicher, dass der Listener für heterogene Dienste konfiguriert ist, wie unter Heterogene Dienste für den Listener konfigurieren beschrieben.

  2. Erstellen Sie einen Net Service-Namen in der Datei tnsnames.ora, der für Verbindungen von dem Oracle Datenbank-Server zu einem fremden System verwendet werden kann. Stellen Sie sicher, dass die Verbindungsdeskriptor-Informationen mit den Informationen übereinstimmen, die für den Listener konfiguriert wurden.

  3. Wählen Sie den Net Service-Namen im Ordner Lokal > Dienstebenennung.

  4. Wählen Sie im rechten Fensterbereich Dienst-Identifikation die Option Erweitert. Daraufhin wird das Dialogfeld Erweiterte Dienstoptionen angezeigt.

  5. Wählen Sie Verwendung für heterogene Services, und klicken Sie dann auf OK.

  6. Klicken Sie auf Anwenden im rechten Fensterbereich.

  7. Wählen Sie Datei >Netzwerkkonfiguration speichern.

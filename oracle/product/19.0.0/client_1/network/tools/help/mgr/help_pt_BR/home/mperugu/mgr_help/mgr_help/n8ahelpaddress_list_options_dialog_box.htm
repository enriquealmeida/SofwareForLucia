Caixa de Diálogo Opções da Lista de Endereços

A caixa de diálogo Opções da Lista de Endereços permite que você especifique a ordem e como os endereços de protocolo em descritores de conexão serão utilizados em conexões.

Tentar cada endereço, em ordem, até ser bem-sucedido

Selecione para ativar failover para tempo de conexão para um cliente Oracle9i ou Oracle8i. Se o primeiro endereço não puder ser alcançado, o failover permitirá que o cliente use o próximo endereço de protocolo de listener. Selecione esta opção e a opção Use Opções Compatíveis com Clientes do Net8 8.0 para desativar o roteamento de origem de um cliente da release 8.0 ou anterior.

Tentar aleatoriamente cada endereço até obter êxito com um deles

Selecione para ativar os recursos de balanceamento de carga do cliente e failover de tempo de conexão para um cliente Oracle9i ou Oracle8i.

Tentar um endereço selecionado aleatoriamente

Selecione para ativar o recurso de balanceamento de carga para um cliente Oracle9i ou Oracle8i.

Usar cada endereço em ordem até atingir o destino

Selecione para ativar o roteamento de origem para um cliente Oracle8i ou Oracle8.

Esta opção é necessária quando o acesso ao destino requer mais de uma interrupção de endereço. Geralmente, esse recurso é usado para ativar recursos do Oracle Connection Manager.

Este parâmetro não é necessário para um cliente Oracle9i.

Usar somente o primeiro endereço

Selecione para desativar os recursos de roteamento de origem, failover de tempo de conexão e balanceamento de carga do cliente.

Usar Opções Compatíveis com Clientes do Net8 8.0

Selecione se estiver configurando um cliente cuja release seja anterior à 8.0. Você só poderá selecionar as seguintes opções:

-Tente acessar cada endereço, em ordem, até ser bem-sucedido

Usar cada endereço em ordem até atingir o destino

Desmarque esta opção se você estiver configurando um cliente Oracle9i ou Oracle8i.

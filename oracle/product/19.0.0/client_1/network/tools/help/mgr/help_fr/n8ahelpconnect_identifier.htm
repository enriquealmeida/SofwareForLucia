Identificateur de connexion

Nom de service réseau ou nom réel du service de base de données correspondant à un descripteur de connexion. Les clients utilisent l'identificateur de connexion dans la chaîne de connexion :

CONNECT NomUtilisateur/MotDePasse@identificateur_connexion

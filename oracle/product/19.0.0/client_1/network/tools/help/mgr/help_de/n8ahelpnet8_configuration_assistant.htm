Oracle Net-Konfigurationsassistent

Ein Tool, das nach der Installation verwendet wird und mit dem grundlegende Netzwerkkomponenten nach der Installation konfiguriert werden, einschließlich:

  • Listener-Namen und Protokolladressen

  • Benennungsmethoden, mit denen der Client Connect Identifier auflöst

  • Net Service-Namen in einer Datei tnsnames.ora

  • Directory-Server-Verwendung

Der Oracle Net-Konfigurationsassistent wird automatisch nach der Installation der Software ausgeführt, wie in Ihrem Oracle Installation Guide beschrieben. Er kann auf dem Client oder dem Datenbank-Server verwendet werden.

Er kann auch im Standalone-Modus ausgeführt werden, um die Verwendung von Benennungsmethoden, Listener, Net Service-Namen in der Datei tnsnames.ora und Verwendung des Directory Servers zu konfigurieren.

So führen Sie den Oracle Net-Konfigurationsassistenten im Standalone-Modus aus:

  • Bei UNIX führen Sie netmgr in $ORACLE_HOME/bin aus.

  • Bei Windows-Betriebssystemen wählen Sie Start > Programme > Oracle - HOME_NAME > Configuration and Migration Tools > Net-Konfigurationsassistent.

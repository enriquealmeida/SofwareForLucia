Willkommen bei Oracle Net Manager

Willkommen bei Oracle Net Manager. Oracle Net Manager kombiniert Konfigurationsmöglichkeiten mit Kontrolle über Komponenten, um eine integrierte Umgebung zur Konfiguration und Verwaltung von Oracle Net bereitzustellen.

Wer verwendet Oracle Net Manager?

Jeder Netzwerkadministrator, der für den Aufbau von Clientverbindungen zu Diensten verantwortlich ist. Mit Oracle Net Manager können Sie sich auf die Konfiguration bestimmter Bereiche des Netzwerkes konzentrieren.

Verwenden Sie Oracle Net Manager das erste Mal?

Wenn Sie noch nie mit Oracle Net Manager gearbeitet haben oder noch nie mit dem Networking allgemein konfrontiert waren, sollten Sie sich mit diesem Hilfesystem beschäftigen, um mehr über den Oracle Net Manager und die Konfiguration des Netzwerks zu erfahren. Außerdem wird auf Oracle10i Net Services Administrator's Guide verwiesen.

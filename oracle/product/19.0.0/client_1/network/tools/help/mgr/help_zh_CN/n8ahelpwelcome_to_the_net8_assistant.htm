欢迎使用 Oracle Net Manager

欢迎使用 Oracle Net Manager。Oracle Net Manager 将组件控制与配置能力相结合，以提供配置和管理 Oracle Net 的集成环境。

谁使用 Oracle Net Manager？

负责建立对服务的客户机连接的任何网络管理员。Oracle Net Manager 使您能够集中配置网络的特定区域。

是否初次使用 Oracle Net Manager？

如果您初次使用 Oracle Net Manager 或对网络比较陌生，则可能需要浏览本“帮助”系统，以便更多地了解 Oracle Net Manager 和配置网络的知识。还应当阅读 Oracle10i Net Services Administrator's Guide。

可以从以下几个地方开始了解帮助系统：

  • Oracle Net 概览

  • Oracle 网络连接概览

  • Oracle Net 配置概览

  • Oracle Net Manager 主要功能

  • Oracle Net Manager 环境概览

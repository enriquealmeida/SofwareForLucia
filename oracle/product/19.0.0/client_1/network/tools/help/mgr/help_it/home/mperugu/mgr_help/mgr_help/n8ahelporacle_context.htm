Contesto Oracle

Un nome distinto relativo (RDN) di cn=OracleContext in una struttura DIT (Directory Information Tree) all'interno di un contesto di denominazione o una voce di directory non pubblicata. Il contesto Oracle contiene le voci utilizzate con il software Oracle, ad esempio i servizi del database e i nomi di servizi di rete che vengono impiegati con la denominazione delle directory. Una directory può contenere uno o più contesti Oracle.

Oracle Internet Directory crea automaticamente un contesto Oracle nella radice della struttura DIT. Questo contesto Oracle radice dispone del nome distinto (DN) (dn:cn=OracleContext).

Quando si avvia Oracle Net Manager, quest'ultimo utilizza il contesto Oracle predefinito configurato dall'Assistente alla configurazione di Oracle Net. È possibile modificare questo contesto Oracle in un uno differente. Per la maggior parte delle distribuzioni, sarà necessario creare contesti Oracle aggiuntivi.

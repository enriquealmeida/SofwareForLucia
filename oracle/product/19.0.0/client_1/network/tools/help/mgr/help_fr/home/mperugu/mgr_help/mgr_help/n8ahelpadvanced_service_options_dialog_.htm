Boîte de dialogue Options de service avancées

La boîte de dialogue Options de service avancées vous permet de configurer des paramètres CONNECT_DATA avancés.

Paramètres de service supplémentaires

Elément Description
Nom de l'instance Saisissez le nom de l'instance de base de données à laquelle accéder. Il s'agit du nom constituant la valeur du paramètre INSTANCE_NAME dans le fichier de paramètres d'initialisation.
Unité de données de session Entrez la taille de l'unité de données de session (SDU) pour modifier les caractéristiques de performances des paquets envoyés sur le réseau. Définissez la taille de l'unité SDU comme un multiple de la taille de trame de transport normale. Bien que la taille de trame Ethernet normale soit 1500, la partie utilisable est 1460. Lorsque le cryptage n'est pas utilisé, un octet est réservé. L'algorithme doit être un multiple de 1460, plus 1 lorsque le cryptage n'est pas utilisé, ou un multiple de 1460 pour Ethernet.
Utilisation pour les services hétérogènes Choisissez cette option pour qu'un serveur Oracle accède à un système autre qu'Oracle via les services hétérogènes.

Paramètres BDR Oracle

Elément Description
Paramètres d'Oracle Rdb Indiquez le nom de fichier d'une base de données Oracle Rdb.
Type de service Indiquez le type de service à utiliser avec une base de données Oracle Rdb.
Nom de base de données global Entrez l'identificateur de la base de données relationnelle Oracle.

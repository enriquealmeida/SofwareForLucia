cman.ora 檔案

指定內送要求及本機和遠端命令，以及「Oracle 連線管理程式」參數之協定位址的組態檔。Oracle Net Manager 不支援此檔案的組態。必須手動設定。

cman.ora 檔案必須位於下列其中一個位置：

  1. TNS_ADMIN 環境變數指定的目錄。如果 TNS_ADMIN 環境變數不是定義為 Windows NT 的變數，它可能就在登錄中。

  2. 節點的全域組態目錄。如果是 Sun Solaris，此目錄為 /var/opt/oracle。Windows NT 則沒有中央目錄。

  3. UNIX 的 $ORACLE_HOME/network/admin 目錄，或 Windows 作業系統的 ORACLE_HOME\network\admin 目錄。

如需詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Configuring and Administering Oracle Connection Manager 章節。

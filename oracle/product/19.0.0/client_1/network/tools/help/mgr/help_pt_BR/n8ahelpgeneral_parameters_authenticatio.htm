Parâmetros Gerais: Autenticação

A guia Autenticação permite configurar a segurança para um listener. Dessa forma, operações privilegiadas, como salvar alterações de configuração ou interromper o listener, emitidas pelo utilitário Listener Control, exigirão uma senha. O Oracle Net Manager permite a definição de uma senha criptografada.

Não Exigir uma Senha para as Operações de Listener

Escolha para desativar a autenticação para tarefas do administrador.

Exigir uma Senha para as Operações de Listener

Escolha para ativar a autenticação para determinadas tarefas do administrador. Você poderá então informar a senha necessária. A senha default é "oracle".

Observação: O Oracle Net permite a configuração de uma senha por listener.

외부 이름 지정 방법 구성

프로파일의 NIS(네트워크 정보 서비스)를 포함하여 외부 이름 지정 방법에 필요한 클라이언트 매개변수를 구성할 수 있습니다.

외부 이름 지정 방법 매개변수를 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창의 목록에서 이름 지정을 선택합니다.

  3. 외부 탭을 선택합니다.

  4. 사용하고 있는 외부 이름 지정 방법에 해당하는 필드에 값을 입력합니다.

  5. 파일 > 네트워크 구성 저장을 선택합니다.

Configurazione delle opzioni avanzate relative ai dati di connessione

Oltre al nome di servizio o al SID (ID di sistema) Oracle del database, è possibile configurare la sezione CONNECT_DATA di un descrittore di connessione con le opzioni riportate di seguito.

  • Nome istanza, per connettersi a un'istanza specifica del database.

  • Oracle Home del database, per avviare il database dalla rete.

  • Unità dati sessione (SDU), per specificare le dimensioni del buffer.

  • Server dedicato, per connettersi a un processo server dedicato.

  • Servizi eterogenei, per accedere a un sistema non Oracle.

  • Impostazioni Oracle Rdb, per accedere a un database Oracle Rdb.

Per configurare le opzioni di connessione avanzate per un nome di servizio di rete, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Directory > Locale o Denominazione dei servizi.

  2. Selezionare un nome di servizio di rete o un nome di servizio di database. Nel riquadro di destra sono visualizzati il servizio di destinazione corrente e la lista degli indirizzi.

  3. Nella casella Identificazione servizio scegliere Avanzate. Viene visualizzata la finestra di dialogo Opzioni di servizio avanzate.

  4. Immettere le informazioni nei campi oppure selezionare le opzioni appropriate, quindi fare clic su OK.

  5. Nel riquadro di destra, scegliere Applica.

  6. Se si apportano queste modifiche alla cartella Locale, scegliere File > Salva configurazione di rete.

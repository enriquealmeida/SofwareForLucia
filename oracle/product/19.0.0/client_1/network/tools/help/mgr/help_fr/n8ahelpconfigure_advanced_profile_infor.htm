Configuration des informations de profil avancées

Le profil peut être configuré à l'aide des fonctions avancées suivantes :

  • Valeur de temporisation de connexion

  • Identificateur unique de client dans le fichier journal

  • Gestionnaires de signaux, qui nettoient les processus défunts

  • Interruptions hors bande

Pour configurer les informations de profil avancées :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Général.

  3. Cliquez sur l'onglet Avancé.

  4. Entrez les valeurs d'option que vous désirez définir.

  5. Choisissez Fichier > Enregistrer la configuration réseau.

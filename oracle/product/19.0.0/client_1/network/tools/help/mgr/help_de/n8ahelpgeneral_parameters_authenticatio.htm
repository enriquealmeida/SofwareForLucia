Allgemeine Parameter: Berechtigungsprüfung

Mit dem Register Berechtigungsprüfung können Sie die Sicherheit für einen Listener konfigurieren. Auf diese Weise erfordern bevorrechtigte Vorgänge, wie z.B. das Speichern von Konfigurationsänderungen oder das Stoppen des Listeners, die aus dem Utility Listener Control ausgegeben werden, ein Kennwort. Mit Oracle Net Manager können Sie ein verschlüsseltes Kennwort festlegen.

Kennwort für Listener-Vorgänge nicht erforderlich

Wählen Sie diese Option, um die Berechtigungsprüfung für Administrator-Aufgaben zu deaktivieren.

Kennwort für Listener-Vorgänge erforderlich

Wählen Sie diese Option, um die Berechtigungsprüfung für bestimmte Administrator-Aufgaben zu aktivieren. Danach können Sie das erforderliche Kennwort eingeben. Das Standardkennwort ist "oracle".

Hinweis: Oracle Net ermöglicht die Konfiguration von einem Kennwort für jeden Listener.

Verwandte Themen

Kennwort-Berechtigungsprüfung für den Listener konfigurieren

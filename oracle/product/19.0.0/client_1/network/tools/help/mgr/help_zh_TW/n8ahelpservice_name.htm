服務名稱

資料庫的邏輯表示法，這是對從屬端呈現資料庫的方法。資料庫可以呈現成多種服務，而服務可以實行為多個資料庫執行處理。服務名稱是一個全域資料庫名稱字串，亦即安裝或建立資料庫時，所輸入的名稱 (由資料庫名稱與網域名稱所組成)。如果您不確定全域資料庫名稱為何，可以從初始化參數檔的 SERVICE_NAMES 參數之相關結合值取得。

從屬端用來連線 Oracle9i 或 Oracle8i 資料庫的連線描述區，必須使用 SERVICE_NAME (沒有 S) 參數來設定。

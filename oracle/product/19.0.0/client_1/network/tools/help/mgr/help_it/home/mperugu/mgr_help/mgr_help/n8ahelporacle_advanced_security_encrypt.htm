Oracle Advanced Security: Cifratura

La scheda Cifratura consente di configurare i parametri di cifratura per i metodi di autenticazione selezionati.

Nota: utilizzare la scheda SSL per configurare le suite di cifratura per SSL (Secure Sockets Layer).

Cifratura

Nell'apposita lista, selezionare il computer che si sta configurando: client o server.

Tipo di cifratura

Selezionare un valore per specificare il comportamento del client o del server durante la negoziazione della cifratura e dell'integrità. Di seguito sono riportati i quattro tipi di cifratura disponibili.

Elemento Descrizione
accettato Il servizio sarà attivo se dall'altra parte della connessione viene specificato "necessario" o "richiesto" ed è disponibile un algoritmo compatibile; in caso contrario, il servizio sarà inattivo.
rifiutato Il servizio non deve essere attivo. Se dall'altra parte è specificato "obbligatorio", la connessione non viene stabilita.
richiesto Il servizio sarà attivo se dall'altra parte della connessione viene specificato "accettato", "necessario" o "richiesto" ed è disponibile un algoritmo compatibile; in caso contrario, il servizio sarà inattivo.
necessario Il servizio deve essere attivo. Se dall'altra parte viene specificato "rifiutato" o se non è disponibile un algoritmo compatibile, la connessione non viene stabilita.

Base di cifratura

Nel campo Base di cifratura immettere una stringa qualsiasi composta da un massimo di 256 caratteri. La base di cifratura viene utilizzata per generare chiavi crittografiche. È necessaria se la cifratura o l'integrità sono abilitate.

Nota: se si sceglie di utilizzare caratteri speciali, come ad esempio una virgola [,] o una parentesi tonda [)] come parte del parametro Base di cifratura, la Oracle consiglia di racchiudere il valore tra apici.

Lista Metodi disponibili/Metodi selezionati

Visualizza i metodi di autenticazione che è possibile selezionare e i metodi di cifratura selezionati. L'ordine dei metodi nella lista determina l'ordine di utilizzo nella negoziazione.

Utilizzare il pulsante freccia sinistra (<) e il pulsante freccia destra (>) per spostare l'algoritmo di cifratura selezionato da una lista all'altra. Oracle Advanced Security supporta gli algoritmi di cifratura riportati di seguito.

Elemento Descrizione
AES256 Chiave AES a 256 bit
RC4_256 Chiave RC4 a 256 bit
AES192 Chiave AES a 192 bit
3DES168 Triple DES con un'opzione a tre chiavi (a 168 bit)
AES128 Chiave AES a 128 bit
RC4_128 Chiave RC4 a 128 bit
3DES112 Triple DES con un'opzione a due chiavi (a 112 bit)
DES Chiave DES a 56 bit
RC4_40 Chiave RC4 a 40 bit
DES40 Chiave DES a 40 bit

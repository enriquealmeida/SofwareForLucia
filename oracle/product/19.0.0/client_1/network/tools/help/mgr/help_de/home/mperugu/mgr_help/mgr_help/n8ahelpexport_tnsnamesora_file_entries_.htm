Einträge in der Datei tnsnames.ora in Directory-Server exportieren

Net Service-Namen, die schon in einer Datei tnsnames.ora konfiguriert sind, können in einen Directory-Server exportiert werden. Der Exportierenvorgang erfolgt Domain für Domain.

Durch das Exportieren einer Datei tnsnames.ora wird der Net Service Name in dem momentan ausgewählten Oracle Context in dem Verzeichnis erstellt. Wenn Sie den Oracle Context ändern müssen, wählen Sie Befehl> Directory-Server> Oracle Context ändern.

So exportieren Sie Net Service Names, die in einer Datei tnsnames.ora gespeichert sind, in einen Directory Server:

  1. Wählen Sie Befehl> Directory > Service-Namen exportieren. Daraufhin wird der Assistent zur Directory-Migration gestartet.

  2. Klicken Sie auf Weiter.

    Wenn mehrere Domainn erkannt wurden, wird das Register "Domain auswählen" angezeigt. Fahren Sie mit Schritt 3 fort.

    Wenn die Net Service-Namen nicht domainqualifiziert sind, wird das Register "Net Service-Namen auswählen" angezeigt. Gehen Sie direkt zu Schritt 4.

  3. Wählen Sie die Netzwerk-Domain, deren Net Service-Namen exportiert werden sollen, und klicken Sie dann auf Weiter. Daraufhin wird die Seite "Net Service-Namen auswählen" angezeigt.

  4. Wählen Sie die zu exportierenden Net Service-Namen aus der Liste, und klicken Sie dann auf Weiter. Daraufhin wird das Register "Zielkontext auswählen" angezeigt.

    Führen Sie die folgenden Schritte im Register "Zielkontext auswählen" aus:

    a. Wählen Sie aus der Liste Verzeichnisbenennungskontext den Verzeichnisbenennungskontext, der den Oracle Context enthält. Der Verzeichnisbenennungskontext ist Bestandteil eines Verzeichnisunterbaums, der einen oder mehrere Oracle Contexts enthält, die Oracle-Einträge enthalten.

    b. Wählen Sie aus der Liste Oracle Context den Oracle Context, in den Sie die ausgewählten Net Service-Namen exportieren möchten.

    c. Klicken Sie auf Weiter. Daraufhin wird das Register "Aktualisierung von Directory-Server" mit dem Status des Exportvorgangs angezeigt.

  5. Klicken Sie auf Fertig, um den Directory-Server Migrations-Assistenten zu schließen.

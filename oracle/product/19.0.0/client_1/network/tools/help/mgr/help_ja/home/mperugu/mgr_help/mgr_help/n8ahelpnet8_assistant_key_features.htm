Oracle Net Managerの主な機能

Oracle Net Managerは、Oracleネットワーキング環境を構成および管理するためのツールです。次のOracle Net機能およびコンポーネントは、Oracle Net Managerで構成および管理できます。

サービス・ネーミング

tnsnames.oraファイル、ディレクトリ・サーバー、およびOracle Names Server内のデータベース・サービスについて、ネットワーク記述を作成または変更できます。ネットワーク記述は接続記述子と呼ばれ、クライアントがデータベース接続時に接続文字列に使用する接続識別子にマップされます。

リスナー

リスナー、つまり、データベース・サービスに対するクライアント接続リクエストをサーバーが送受信するプロセスを作成または変更できます。

プロファイル

プロファイル、つまり、クライアントからOracleネットワークへの接続方法を決定するパラメータの集合を作成または変更できます。ネーミング・メソッド、ロギング、トレース、外部ネーミング・パラメータおよびOracle Advanced Securityのクライアント・パラメータを使用できます。

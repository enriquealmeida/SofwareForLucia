일반 매개변수: 일반 사항

일반 사항 탭을 사용하여 리스너에 대한 일반 매개변수를 구성할 수 있습니다.

리스너 이름

리스너 이름을 표시합니다.

시작 대기 시간

리스너가 리스너 제어 유틸리티 START 명령에 응답하기 전에 대기하는 시간을 초 단위로 지정합니다.

주: 이 설정은 사용되지 않으므로 이후 버전에서는 지원되지 않을 것입니다. 리스너를 실행하는 데 이 매개변수가 필요한 경우 오라클 고객 지원 센터에 문의하십시오.

종료 시 구성 저장

리스너 제어 유틸리티의 SET 명령으로 변경 사항을 영구히 유지하도록 선택합니다.

SNMP 접속 정보

Oracle Enterprise Manager에서 사용하기 위해 snmp.ora 파일에 기록할 텍스트를 입력합니다. 이 정보는 모든 SNMP 접속을 사용하여 전송되고 문자열 종류나 개수에 상관 없으며 SNMP 접속을 생성하는 프로세스에 영향을 주지 않습니다. 이 정보는 일반적으로 클라이언트 사용자나 관리자의 접속 정보로 구성됩니다.

ADMIN_RESTRICTIONS_listener_name

리스너의 런타임 관리를 제한합니다.

기본 설정은 off입니다. ADMIN_RESTRICTIONS_listener_name=on을 설정하면 listener.ora에 있는 매개변수의 런타임 수정이 비활성화됩니다. 즉, 해당 매개변수를 수정하는 SET 명령이 리스너에서 거부됩니다. ADMIN_RESTRICTIONS_listener_name 자체를 포함하여 listener.ora에서 임의 매개변수를 변경하려는 경우에는 리스너를 명시적으로 정지하고 다시 시작할 필요 없이 listener.ora 파일을 수동으로 수정하고 RELOAD 명령을 사용하여 해당 매개변수를 재로드하여 새 변경 사항을 적용할 수 있습니다.

리스너의 비밀번호 보안에 대한 추가 정보는 Oracle Database Net Services Administrator's Guide를 참조하십시오.

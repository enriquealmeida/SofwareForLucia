設定外部程序的連線描述區

外部程序是從其他程式呼叫、但以不同語言撰寫而成的程序。呼叫一或多個執行特殊目的處理所需之 C 語言常式的 PL/SQL 程式即為一個例子。

若要 Oracle 資料庫連線至外部程序，資料庫伺服器必須設定網路服務名稱才可以存取外部程序。

注意：「Oracle Net 組態輔助程式」會在安裝期間自動建立外部程序的網路服務名稱。

設定外部程序的資料庫伺服器：

  1. 確定監聽器有外部程序的設定，如設定監聽器的外部程序的說明。

  2. 在資料庫的 tnsnames.ora 檔案中建立網路服務名稱。確定連線描述區資訊符合監聽器設定的資訊。

  3. 選擇檔案 > 儲存網路組態。

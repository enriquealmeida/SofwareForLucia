Oracle Context

이름 지정 컨텍스트 또는 등록되지 않은 디렉토리 항목 아래의 DIT(디렉토리 정보 트리)에 있는 cn=OracleContext라는 상대적 식별 이름입니다. Oracle Context는 데이터베이스 서비스 및 디렉토리 이름 지정에서 사용되는 네트 서비스 이름과 같은 오라클 소프트웨어에서 사용되는 항목을 포함합니다. 하나의 디렉토리에 하나 이상의 Oracle Context가 존재할 수 있습니다.

Oracle Internet Directory는 DIT 구조의 루트에 Oracle Context를 자동으로 생성합니다. 이 루트 Oracle Context에는 ((dn:cn=OracleContext))라는 DN(식별 이름)이 있습니다.

Oracle Net Manager는 시작할 때 Oracle Net Configuration Assistant에 의해 구성된 기본 Oracle Context를 사용합니다. 이 Oracle Context를 다른 컨텍스트로 변경할 수 있습니다. 대부분의 배치에서는 추가 Oracle Context를 생성해야 합니다.

Fichier ldap.ora

Fichier créé par l'assistant Configuration Oracle Net et contenant les informations suivantes sur la configuration d'annuaire :

  • Type d'annuaire

  • Emplacement de l'annuaire

  • Contexte Oracle par défaut que le client ou le serveur utilisera pour consulter ou configurer des identificateurs de connexion pour les connexions aux services de base de données

Le fichier ldap.ora doit résider dans l'un des emplacements suivants :

  1. Le répertoire indiqué par la variable d'environnement TNS_ADMIN . Si la variable d'environnement TNS_ADMIN n'est pas définie en tant que variable dans Windows NT, il se peut qu'elle soit dans la base de registres.

  2. Le répertoire $ORACLE_HOME/network/admin dans UNIX ou le répertoire ORACLE_HOME\network\admin dans les systèmes d'exploitation Windows.

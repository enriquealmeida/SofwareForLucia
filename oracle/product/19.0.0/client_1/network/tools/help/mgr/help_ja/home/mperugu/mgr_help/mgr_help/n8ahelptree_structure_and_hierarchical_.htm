ツリー構造および階層フォルダ

Oracle Net Managerでは、ネットワーク・オブジェクトがツリー形式で表示され、そこに含まれるオブジェクトがフォルダ階層に編成されます。ナビゲータ・ペインを使用すると、各フォルダ内のオブジェクトを表示、変更、追加および削除できます。

フォルダを展開すると、オブジェクトとフォルダのネストしたリストが表示されます。オブジェクトを選択すると、そのオブジェクトに関する情報がOracle Net Managerの右ペインに表示されます。

ディレクトリ

ディレクトリ・サーバー内で構成されている、接続識別子と呼ばれる単純名が表示されます。

注意:「ディレクトリ」フォルダは、ディレクトリへのアクセスが構成されている場合にのみ表示されます。ディレクトリ・アクセス構成とその他の要件の詳細は、「Oracle Net Managerのディレクトリ・ネーミング要件」を参照してください。

ローカル

ローカル・コンフィギュレーション・ファイル内で構成されているネットワーキング要素が表示されます。

  • tnsnames.oraファイル内のネット・サービス名

  • listener.oraファイル内のリスナー

  • sqlnet.oraファイル内のプロファイル

Oracle Names Server

Oracle Names Serverを表示します。

Caixa de Diálogo Opções Avançadas de Serviço

A caixa de diálogo Opções Avançadas de Serviço permite que você configure parâmetros CONNECT_DATA avançados.

Definições de Serviço Adicionais

Elemento Descrição
Nome da Instância Informe os nomes da instância do banco de dados a ser acessada. O nome da instância pode ser obtido no valor do parâmetro INSTANCE_NAME do arquivo de parâmetros de inicialização.
SDU (Session Data Unit) Informe o tamanho da unidade de dados da sessão para alterar as características de desempenho relacionadas aos pacotes enviados pela rede. Defina o tamanho da SDU como um múltiplo do tamanho de quadro de transporte normal. O tamanho normal do quadro Ethernet é 1500, mas a parte utilizável é 1460. Quando a criptografia não está sendo utilizada, um byte é reservado. O algoritmo a ser usado é um múltiplo de 1460 mais 1 quando a criptografia não é utilizada, ou um múltiplo de 1460 quando é para Ethernet.
Usar para Serviços Heterogêneos Escolha a opção para que um servidor Oracle acesse um sistema não Oracle através de serviços heterogêneos.

Definições do Oracle Rdb

Elemento Descrição
Definições do Oracle Rdb Especifique o nome de arquivo de um banco de dados Oracle Rdb.
Tipo de Serviço Especifique o tipo de serviço que deve ser usado para um banco de dados Oracle Rdb.
Nome do Banco de Dados Global Informe um identificador para o banco de dados Oracle Rdb.

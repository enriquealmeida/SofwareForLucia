Controlador de Serviço

Um processo que atua como um ponto de conexão do listener ao servidor de banco de dados. Um handler de serviço pode ser um dispatcher de servidor compartilhado ou servidor dedicado ou um broker do Pool de Conexões Residente do Banco de Dados. Durante uma conexão, um descritor de conexão é passado ao listener. É o trabalho do job do listener determinar que tipo de handler de serviço melhor atenderá as solicitações de clientes.

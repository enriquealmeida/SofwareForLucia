Funzioni chiave di Oracle Net Manager

Oracle Net Manager è uno strumento per la configurazione e la gestione dell'ambiente di rete Oracle. Di seguito sono elencati i componenti e le funzioni di Oracle Net che è possibile configurare e gestire con Oracle Net Manager.

Denominazione dei servizi

È possibile creare o modificare le descrizioni di rete per i servizi di database in un file tnsnames.ora, in un server delle directory o in un server Oracle Names. Le descrizioni di rete, denominate descrittori di connessione, vengono mappate agli identificativi di connessione utilizzati dai client nelle stringhe di connessione durante una connessione a un database.

Listener

È possibile creare o modificare un listener, un processo sul server che riceve e risponde alle richieste di connessione dei client per un servizio del database.

Profilo

È possibile creare o modificare un profilo, vale a dire una raccolta di parametri che determinano la modalità di connessione client alla rete Oracle. È possibile impostare i parametri client per i metodi di denominazione, le funzioni di log e traccia, i parametri di denominazione esterna e l'opzione Oracle Advanced Security.

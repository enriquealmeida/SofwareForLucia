Ruta de Solicitudes de Conexión

Los clientes se pueden configurar de manera que las solicitudes de conexión se dirijan siempre a un proceso de servidor dedicado o a Oracle Connection Manager.

Para dirigir solicitudes de conexión:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione General de la lista del panel de la derecha.

  3. Seleccione el separador Direccionamiento.

  4. Seleccione el modo de distribución de las solicitudes de conexión que desee.

  5. Seleccione Archivo > Guardar Configuración de Red.

ツールバー

ツールバーはOracle Net Managerの左側に配置され、フォルダ階層のネットワーク・オブジェクトに対応するアイコンが表示されます。ツールバー・ボタンは、ペイン内で表示または選択されているオブジェクトに応じて使用可能になります。ツールバー・アイコンにカーソルを合せると、そのアイコンの機能の説明が表示されます。

ツールバーには、次のアイコンが含まれています。

アイコン 説明
追加 該当のコンピュータ用に新規のネット・サービス名、リスナー、およびOracle Names Serverを作成します。
削除 サービス名オブジェクトとリスナー・オブジェクトを削除します。
接続の検証 選択したサービス名オブジェクトからデータベース・サービスへの接続性を検証します。接続性テスト中には、tnsnames.oraファイルまたはディレクトリ・サーバーに格納されている接続記述子情報を使用して、データベース・サービスへの接続が確立されます。
ヘルプ オンライン・ヘルプを表示します。

注意: 適切なタイプのオブジェクトを選択している場合にのみ使用可能なコマンドがあります。たとえば、リスナーを作成する場合は、最初に「リスナー」フォルダを選択する必要があります。

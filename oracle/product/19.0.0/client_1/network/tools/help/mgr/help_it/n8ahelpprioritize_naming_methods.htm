Assegnazione di priorità ai metodi di denominazione

Dopo avere configurato i metodi di denominazione, è necessario assegnare le priorità ai metodi all'interno del profilo. Per risolvere un nome di servizio di rete o un nome di servizio di database in un descrittore di connessione, viene utilizzato il primo metodo di denominazione presente nella lista. Se questo non funziona viene utilizzato il secondo e così via.

Per specificare l'ordine dei metodi di denominazione:

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Denominazione.

  3. Selezionare la scheda Metodi.

  4. Selezionare i metodi di denominazione dalla lista Metodi disponibili e fare clic sul pulsante freccia destra. Il metodo di denominazione selezionato si sposta nella lista Metodi selezionati.

  5. Ordinare i metodi di denominazione a seconda del metodo che Oracle Net deve utilizzare per primo per risolvere i nomi di servizi di rete o dei servizi di database. Selezionare il metodo di denominazione nella lista Metodi selezionati, quindi scegliere Avanza o Retrocedi per spostare la selezione in alto o in basso nella lista.

  6. Scegliere File > Salva configurazione di rete.

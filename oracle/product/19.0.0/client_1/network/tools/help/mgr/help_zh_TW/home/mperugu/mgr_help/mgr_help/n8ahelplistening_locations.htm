監聽位置

監聽器會設定成監聽特定網路協定位址上之資料庫的內送的連線要求。您可以使用監聽位置頁籤來設定一或多個等候從屬端連線的監聽位置位址。

協定

請從清單中選取位址的協定。根據選取之協定的不同，請在適當欄位輸入協定參數資訊。

「顯示進階/隱藏進階」按鈕

請使用「顯示進階/隱藏進階」來切換顯示或隱藏指定監聽器的 I/O 緩衝區空間限制的設定值。

TCP/IP、TCP/IP 與 SSL 和 SDP 協定支援這些設定值。

  • 總傳送緩衝區大小

指定階段作業之傳送作業的緩衝區空間 (位元組)。

因為資料庫伺服器會將資料寫入從屬端，所以通常在伺服器端設定此設定值就夠了。如果資料庫伺服器接收大量的要求，請同時設定總接收緩衝區大小設定值。

  • 總接收緩衝區大小

指定階段作業之接收作業的緩衝區空間 (位元組)。

靜態設定此位址專供 JServer 連線使用

請選擇使用 General Inter-Orb Protocol (GIOP) 端點來設定 listener.ora 檔案，用於 Oracle JServer 8.1 版連線。

新增位址

選擇此選項可建立新的協定位址。新的位址頁籤就會顯示，您可以在此處選取協定並在欄位中輸入協定資訊。

移除位址

選擇此選項可刪除選取的位址頁籤及其位址資訊。

如需有關設定 I/O 緩衝區空間的詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Optimizing Performance 章節。

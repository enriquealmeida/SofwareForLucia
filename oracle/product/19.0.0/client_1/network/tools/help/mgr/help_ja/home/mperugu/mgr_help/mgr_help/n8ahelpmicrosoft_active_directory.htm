Microsoft Active Directory

Windows 2000 Serverに組み込まれているLDAP準拠のディレクトリ・サービス。ネットワーク上のオブジェクトに関する情報が格納され、この情報をユーザーおよびネットワーク管理者が使用できるようにします。また、Active Directoryは、単一ログオン・プロセスを使用してネットワーク上のリソースへのユーザー・アクセスを提供します。

Active Directoryは、クライアントがアクセスできるサービス情報を格納するためのディレクトリ・ネーミング・メソッドとして構成できます。

Menu Comando

O menu Comando permite que você execute as seguintes operações:

Diretório

Este submenu contém comandos para a execução de ações em um servidor de diretórios:

Elemento Descrição
Alterar Contexto Atual Escolha essa opção para selecionar um Oracle Context diferente do atual. O Oracle Context é uma subárvore do diretório que armazena entradas Oracle, incluindo serviços de banco de dados e nomes de serviço de rede a serem usados com a nomeação de diretório. Quando um novo Oracle Context é selecionado, objetos são exibidos na pasta Configuração do Oracle Net > Diretório > Nomeação de Serviço do navegador para o novo Oracle Context.
Definir Autenticação Escolha para definir credenciais de autenticação para o servidor de diretórios.
Exportar Nomes de Serviço de Rede Escolha para iniciar o Assistente de Migração de Servidor de Diretórios. O assistente exporta nomes de serviço de rede armazenados em um arquivo tnsnames.ora para um servidor de diretórios.

Observação: O menu Diretório só será exibido se o acesso ao diretório tiver sido configurado. Para obter mais informações sobre a configuração de acesso a diretório e outros requisitos, consulte Requisitos de Nomeação de Diretório para o Oracle Net Manager.

Serviço de Teste

Escolha a opção de·conectar-se a um serviço de banco de dados usando informações do descritor de conexão armazenadas em um arquivo tnsnames.ora ou em um servidor de diretórios.

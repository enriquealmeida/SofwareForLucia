Cuadro de Diálogo Opciones de Servicio Avanzadas

El recuadro de diálogo Opciones de Servicio Avanzadas permite configurar parámetros avanzados de CONNECT_DATA.

Configuración Adicional de Servicio

Elemento Descripción
Nombre de la Instancia Introduzca los nombres de la instancia de base de datos a la que desea acceder. Puede obtener el nombre de la instancia en el parámetro INSTANCE_NAME del archivo de parámetros de inicialización.
Unidad de Datos de la Sesión (SDU) Introduzca el tamaño de la unidad de datos de la sesión para modificar las características de rendimiento relacionadas con los paquetes que se envían por la red. El tamaño de SDU debe ser un múltiplo del tamaño normal del bloque de transporte. El tamaño normal del bloque para Ethernet es 1500, pero la parte utilizable es 1460. Si no se utiliza cifrado, se reserva un byte. El algoritmo que se utiliza es un múltiplo de 1460 más 1 si no se utiliza cifrado o un múltiplo de 1460 si es para Ethernet.
Usar para Servicios Heterogéneos Seleccione un servidor Oracle para acceder a un sistema que no sea de Oracle mediante servicios heterogéneos.

Configuración de Oracle Rdb

Elemento Descripción
Valores de Oracle Rdb Especifique el nombre de archivo de una base de datos Oracle Rdb.
Tipo de Servicio Especifique el tipo de servicio que desea utilizar con una base de datos Oracle Rdb.
Nombre de la Base de Datos Global Introduzca un identificador para la base de datos Oracle Rdb.

검색

Oracle Names Server의 위치를 찾는 프로세스입니다. 검색하는 동안 첫번째 Oracle Names Server가 검색되면 검색하는 동안 첫번째 Oracle Names Server가 검색되면 이 Oracle Names Server는 관리 영역에 있는 다른 모든 Oracle Names Server를 핑합니다. Oracle Names Server 목록이 생성되고 UNIX의 경우 sdns.ora 또는 Windows의 경우 sdns.ora에 저장됩니다. 이 목록은 응답 시간 순서에 따라 정렬됩니다.

찾기는 다음 순서로 첫번째 Oracle Names Server를 검색합니다.

  1. sqlnet.ora 파일에서 선호되는 Oracle Names Server

  2. 잘 알려진 Oracle Names Server

  3. 1575 포트에서 TCP/IP로 구성된 로컬 Oracle Names Server

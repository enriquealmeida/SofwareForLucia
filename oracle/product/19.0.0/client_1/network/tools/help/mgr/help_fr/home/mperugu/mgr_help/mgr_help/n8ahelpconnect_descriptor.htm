Descripteur de connexion

Description spécialement formatée de la destination d'une connexion réseau. Un descripteur de connexion contient des informations relatives au service de destination et à la route réseau.

Le service de destination est identifié par l'utilisation de son nom de service pour les bases de données Oracle9i et Oracle8i ou de son Identificateur système (SID) Oracle pour les bases de données Oracle8 ou Oracle7.

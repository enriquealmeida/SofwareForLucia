Configurando Informações de Perfil Avançadas

O perfil pode ser configurado com os seguintes recursos avançados:

  • Valor de Timeout de Conexão

  • Um identificador cliente exclusivo no arquivo de log.

  • Handlers de sinal, que remove processos desativados.

  • Quebras fora da faixa

Para configurar as informações de perfil avançadas, faça o seguinte:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

  2. Na lista do painel direito, selecione Geral.

  3. Selecione a guia Avançado.

  4. Informe os valores para as opções a serem definidas.

  5. Escolha Arquivo > Salvar Configuração de Rede.

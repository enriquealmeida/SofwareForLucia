Instanz

Die Kombination aus System Global Area (SGA) und Oracle-Hintergrundprozessen. Wenn eine Datenbank auf einem Datenbank-Server (ungeachtet des Rechnertyps) hochgefahren wird, weist Oracle einen Speicherbereich mit der Bezeichnung SGA zu und startet einen oder mehrere Oracle-Prozesse. Der Speicher und die Prozesse einer Instanz verwalten die Daten der verknüpften Datenbank effizient und stehen den Datenbankbenutzern zur Verfügung. Sie können sich bei jeder Instanz anmelden, um auf Informationen in einer Parallel Server-Datenbank zuzugreifen.

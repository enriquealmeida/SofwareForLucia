Externe Benennungsmethoden konfigurieren

Sie können erforderliche Clientparameter für externe Benennungsmethoden, einschließlich Network Information Service (NIS) im Profil konfigurieren.

So konfigurieren Sie Parameter für externe Benennungsmethoden:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal > Profil im Navigator-Fensterbereich ein.

  2. Wählen Sie Namensgebung aus der Liste im rechten Fensterbereich.

  3. Wählen Sie das Register Extern.

  4. Geben Sie einen Wert in das entsprechende Feld für die verwendete externe Benennungsmethode ein.

  5. Wählen Sie Datei >Netzwerkkonfiguration speichern.

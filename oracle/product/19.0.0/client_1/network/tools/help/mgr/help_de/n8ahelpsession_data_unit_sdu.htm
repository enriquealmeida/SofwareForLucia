Sessiondateneinheit (SDU)

Ein Puffer, in dem Oracle Net Daten speichert, bevor diese über das Netzwerk übertragen werden. Oracle Net sendet die Daten in dem Puffer entweder auf Anforderung oder wenn der Puffer voll ist. Wenn große Datenmengen abgerufen werden und die Paketgröße konsistent dieselbe ist, kann das Abrufen der Daten beschleunigt werden, wenn die Standardgröße von SDU angepasst wird.

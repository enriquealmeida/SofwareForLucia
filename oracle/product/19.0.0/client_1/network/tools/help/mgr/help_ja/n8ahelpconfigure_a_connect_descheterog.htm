異機種間サービスの接続記述子の構成

異機種間サービスを使用すると、OracleサーバーからOracle以外のシステムにアクセスできます。Oracle以外のシステムへの接続を実行するために、Oracleサーバーはゲートウェイ上のリスナーを介してエージェント・プロセスを開始します。OracleデータベースをOracle以外のシステムに接続する場合は、ネット・サービス名を指定してサーバーを構成します。

異機種間サービス用にサーバーを構成する手順は、次のとおりです。

  1. 「リスナーの異機種間サービスの構成」の説明に従って、リスナーが異機種間サービス用に構成されていることを確認します。

  2. tnsnames.oraファイルに、Oracleデータベース・サーバーからOracle以外のシステムへの接続に使用できるネット・サービス名を作成します。接続記述子情報が、リスナーに構成された情報と一致することを確認します。

  3. 「ローカル」→「サービス・ネーミング」フォルダで、ネット・サービス名を選択します。

  4. 右ペインの「サービスの識別」ボックスで「詳細」を選択します。「詳細サービス・オプション」ダイアログ・ボックスが表示されます。

  5. 「異機種間サービスを使用」を選択し、「OK」を選択します。

  6. 右ペインで「適用」を選択します。

  7. 「ファイル」→「ネットワーク構成の保存」を選択します。

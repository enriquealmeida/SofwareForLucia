Requisiti di denominazione delle directory per Oracle Net Manager

Oracle Net Manager consente di creare le informazioni di instradamento di rete per un servizio di database, nonché i nomi di servizi di rete in un server delle directory.

Oracle Net Manager può scrivere in una directory solo se:

  • la directory è configurata con uno schema Oracle;

  • il computer sul quale è in esecuzione Oracle Net Manager è configurato con un contesto Oracle predefinito per consentire l'accesso e la scrittura delle voci.

Queste impostazioni di configurazione sono definite con l'Assistente alla configurazione di Oracle Net, secondo quanto descritto in Panoramica sulla configurazione della denominazione delle directory. Oracle Net Manager non supporta questa funzionalità.

Se l'accesso alle directory non è configurato, la cartella Configurazione di Oracle Net > Directory nel riquadro del Navigator e il menu Comando > Directory non sono disponibili.

Balanceamento de Carga do Cliente

Balanceamento de carga, no qual um cliente pode escolher aleatoriamente entre os listeners para suas solicitações de conexão se mais de um listener atender a um único banco de dados. Esse caráter aleatório permite que todos os listeners compartilhem a responsabilidade de atender às solicitações de conexão recebidas.

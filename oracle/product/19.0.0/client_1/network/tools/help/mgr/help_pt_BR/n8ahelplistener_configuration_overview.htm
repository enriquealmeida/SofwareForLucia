Visão Geral da Configuração de Listener

As seções a seguir são abrangidas neste tópico.

Configuração Durante a Instalação

Durante a instalação da Enterprise Edition ou da Standard Edition no servidor de banco de dados, o Assistente de Configuração do Oracle Net automaticamente configura um listener com o nome LISTENER que tem um endereço de protocolo TCP/IP do listener para o banco de dados Oracle. Durante uma instalação Personalizada, o Assistente de Configuração do Oracle Net solicita que você configure um nome de listener e um endereço de protocolo à sua escolha:

Adicionalmente, um endereço de protocolo IPC do listener para chamadas de procedimento externo é configurado automaticamente, independentemente do tipo de instalação.

O Assistente de Configuração do Oracle Net também configura automaticamente informações de serviço para os procedimentos externos no arquivo listener.ora.

Configuração com o Oracle Net Manager.

O Oracle Net Manager permite:

  • Criar um listener

  • Configurar informações de serviço para o Oracle8 ou bancos de dados anteriores

  • Configurar informações para serviços de banco de dados não Oracle, inclusive procedimentos externos e Serviços Heterogêneos

  • Ativar recursos de rastreamento e de log

  • Ativar a autenticação de senha para o utilitário Listener Control

Para acessar a configuração do listener:

No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

Para obter mais informações, consulte o capítulo Configuring and Administering the Listener no Oracle10i Net Administrator's Guide.

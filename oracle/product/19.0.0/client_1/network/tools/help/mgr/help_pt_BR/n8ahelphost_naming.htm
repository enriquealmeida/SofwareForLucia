Nomeação do Host

Um método de nomeação que permite aos usuários estabelecerem conexão com um servidor de banco de dados Oracle utilizando um apelido de host em um ambiente TCP/IP. Os nomes de host são mapeados para o nome de banco de dados global do servidor em um serviço de resolução de nomes existente, como o DNS (Domain Name System), o NIS (Network Information Service) ou um conjunto de arquivos /etc/hosts mantido de forma centralizada.

Para obter informações sobre como configurar o método de nomeação do host, consulte o capítulo Configuring Naming Methods no Oracle9i Net Services Administrator's Guide.

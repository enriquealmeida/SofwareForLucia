Creación de un Listener

Debe existir un listener en el servidor para que la base de datos pueda recibir conexiones.

Nota: Durante el proceso de instalación se crea un listener llamado LISTENER.

Para crear un listener:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Listeners.

  2. Haga clic en el signo más (+) en la barra de herramientas o seleccione Editar > Crear. Aparece el recuadro de diálogo Seleccionar Nombre del Listener.

  3. Acepte el nombre del listener que aparece en el campo Nombre del Listener o edite el campo con otro nombre. Por defecto, el nombre del primer listener es LISTENER . Los listeners siguientes adoptan por defecto el nombre LISTENERn. El nombre del listener debe ser único en cada nodo.

  4. Haga clic en Aceptar.

  5. Realice los pasos del 4 al 6 en Configuración de Direcciones de Listener.

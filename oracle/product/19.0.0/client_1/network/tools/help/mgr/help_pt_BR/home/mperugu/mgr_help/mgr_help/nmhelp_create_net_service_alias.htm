Caixa de Diálogo Criando Alias de Serviço de Rede

A caixa de diálogo Criar Apelido de Serviço de Rede permite que você crie apelidos de serviço de rede em um servidor de diretório.

Um apelido de serviço de rede é um nome alternativo para um nome do serviço de rede ou um serviço de banco de dados. Uma entrada de apelido de serviço de rede não tem informações de descritor de conexão. Em vez disso, ela só faz referência à localização do objeto para o qual é um apelido. Quando um cliente solicita uma pesquisa de diretório para um apelido de serviço de rede, o diretório determina que a entrada seja um apelido de serviço de rede e conclui a pesquisa como se fosse a entrada referenciada. Por exemplo, um apelido de db1alias pode ser criado para um serviço de banco de dados de db1. Quando db1alias é utilizado para estabelecer conexão com um serviço de banco de dados, como em CONNECT scott/tiger@db1alias , ele resolve e utiliza as informações do descritor de conexão para db1.

Há dois usos principais para apelidos de serviço de rede:

  • Use um apelido de serviço de rede como uma forma de clientes referenciarem um serviço de banco de dados ou um nome do serviço de rede por outro nome.

  • Utilize um apelido de serviço de rede em um Oracle Context para um serviço de banco de dados ou um nome do serviço de rede em outro Oracle Context. Isso permite que um serviço de banco de dados ou nome do serviço de rede seja definido uma vez no servidor de diretório, mas seja referenciado por clientes que utilizam outros Oracle Contexts.

Apelido de Serviço de Rede

Informe o nome do apelido de serviço de rede.

Nome do Serviço de Rede ou Serviço de Banco de Dados

Use a caixa Nome do Serviço de Rede ou Serviço de Banco de Dados para especificar informações sobre o serviço de banco de dados ou o nome do serviço de rede:

Oracle Context: Especifique o Oracle Context do serviço de banco de dados ou do nome do serviço de rede selecionando na lista ou informando no campo.

Nome: Especifique o DN ("distinguished name", nome distinto) do serviço de banco de dados ou nome do serviço de rede selecionando na lista ou informando no campo.

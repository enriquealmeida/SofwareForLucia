Bem-vindo ao Oracle Net Manager

Bem-vindo ao Oracle Net Manager. O Oracle Net Manager combina recursos de configuração com o controle de componentes, a fim de proporcionar um ambiente integrado para a configuração e o gerenciamento do Oracle Net.

Quem utiliza o Oracle Net Manager?

Qualquer administrador de rede que seja responsável por estabelecer conexões clientes com serviços. O Oracle Net Manager permite que você se concentre na configuração de áreas específicas da rede.

Novato no Oracle Net Manager?

Se você é novato no uso do Oracle Net Manager ou de redes em geral, talvez queira percorrer este sistema de ajuda para aprender mais sobre o Oracle Net Manager e a configuração da rede. Revise também o Oracle10i Net Services Administrator's Guide.

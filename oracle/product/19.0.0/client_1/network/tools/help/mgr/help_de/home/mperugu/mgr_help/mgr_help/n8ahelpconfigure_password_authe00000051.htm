Kennwort-Authentifizierung für den Listener konfigurieren

Der Listener kann mit der Sicherheitsfunktion konfiguriert werden. Auf diese Weise erfordern bevorrechtigte Vorgänge, wie z.B. das Speichern von Konfigurationsänderungen oder das Stoppen des Listeners, die aus dem Utility Listener Control verwendet werden, ein Kennwort. Mit Oracle Net Manager können Sie ein verschlüsseltes Kennwort festlegen.

So legen Sie ein Kennwort für den Listener fest:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal> Listener im Navigator-Fensterbereich ein.

  2. Wählen Sie einen Listener.

  3. Wählen Sie Allgemeine Parameter aus der Liste im rechten Fensterbereich.

  4. Wählen Sie das Register Authentifzierung.

  5. Klicken Sie auf Kennwort für Listener-Vorgänge erforderlich.

  6. Geben Sie das Kennwort in das Feld Kennwort ein, und bestätigen Sie es in dem Feld Kennwort bestätigen.

  7. Wählen Sie Datei >Netzwerkkonfiguration speichern.

Finestra di dialogo Crea alias di servizio di rete

La finestra di dialogo Crea alias di servizio di rete consente di creare alias di servizi di rete in un server delle directory.

Un alias di servizio di rete rappresenta un nome alternativo per un nome di servizio di rete o un servizio di database. Una voce alias del servizio di rete non dispone di informazioni del descrittore di connessione. Fa invece riferimento alla posizione dell'oggetto per il quale è stato creato. Quando un client richiede una ricerca all'interno di una directory di un alias del servizio di rete, la directory determina che la voce è un alias del servizio di rete e completa la ricerca come se si trattasse della voce di riferimento. Quando un client richiede una ricerca all'interno di una directory di un alias del servizio di rete, la directory determina che la voce è un alias del servizio di rete e completa la ricerca come se si trattasse della voce di riferimento. Ad esempio, un alias di servizio di rete db1alias può essere creato per un servizio di database db1. Quando si utilizza db1alias per connettersi a un servizio di database, come in CONNECT scott/tiger@db1alias, verrà effettuata la risoluzione e verranno utilizzate le informazioni del descrittore di connessione per db1.

Vi sono due utilizzi principali degli alias di servizio di rete.

  • Utilizzare un alias di servizio di rete per consentire ai client di fare riferimento a un servizio di database o a un nome di servizio di rete mediante un altro nome.

  • Utilizzare un alias del servizio di rete in un contesto Oracle per un servizio di database o un nome di servizio di rete in un contesto Oracle diverso. Ciò consente di definire una sola volta un servizio di database o un nome di servizio di rete nel server delle directory; tuttavia, a tale nome faranno riferimento i client che utilizzano altri contesti Oracle.

Alias di servizio di rete

Immettere il nome dell'alias di servizio di rete.

Nome servizio di rete o Servizio di database

Utilizzare la casella Nome servizio di rete o Servizio di database per immettere le informazioni sul servizio di database o sul nome di servizio di rete.

Contesto Oracle: specificare il contesto Oracle del servizio di database o del nome di servizio di rete selezionandone uno dalla lista o immettendone uno nel campo.

Nome: specificare il nome distinto (DN) del servizio di database o del nome di servizio di rete selezionandone uno dalla lista o immettendone uno nel campo.

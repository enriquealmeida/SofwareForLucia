Définition de la priorité des méthodes de résolution de noms

Lorsque les méthodes de résolution de noms sont configurées, leur priorité doit être définie dans le profil. La première méthode de la liste est utilisée en premier pour convertir un nom de service réseau ou un nom de base de données en descripteur de connexion. Si le nom ne peut pas être converti, la deuxième méthode de la liste est utilisée.

Pour définir l'ordre des méthodes de résolution de noms :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Affectation de noms.

  3. Sélectionnez l'onglet Méthodes.

  4. Sélectionnez les méthodes de résolution de noms dans la liste Méthodes disponibles, puis choisissez le bouton "flèche vers la droite". La méthode sélectionnée passe dans la liste Méthodes sélectionnées.

  5. Classez les méthodes de résolution de noms dans l'ordre où Oracle Net devra les essayer pour résoudre les noms de service réseau ou les noms de service de base de données. Sélectionnez une entrée dans la liste Méthodes sélectionnées, puis choisissez le bouton Promouvoir ou Rétrograder pour déplacer la sélection vers le haut ou vers le bas de la liste.

  6. Choisissez Fichier > Enregistrer la configuration réseau.

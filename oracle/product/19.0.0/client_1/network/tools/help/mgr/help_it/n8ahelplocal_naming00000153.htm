Denominazione locale

Un metodo di denominazione che risolve un nome di servizio di rete, memorizzato nel file tnsnames.ora di un client, in un descrittore di connessione. La denominazione locale è più adatta alle reti distribuite semplici con un numero ridotto di servizi che non cambiano spesso.

設定監聽器的密碼認證

您可以設定監聽器的安全性。這樣的話，從監聽器控制公用程式使用儲存組態變更或停止監聽器等授權的作業就需要密碼。您可以使用 Oracle Net Manager 來設定加密密碼。

設定監聽器的密碼：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

  2. 選取監聽器。

  3. 從右窗格的清單中，選取一般參數。

  4. 選取認證頁籤。

  5. 選擇執行監聽器作業需要密碼。

  6. 在密碼欄位輸入密碼，然後在確認密碼欄位重新輸入密碼。

  7. 選擇檔案 > 儲存網路組態。

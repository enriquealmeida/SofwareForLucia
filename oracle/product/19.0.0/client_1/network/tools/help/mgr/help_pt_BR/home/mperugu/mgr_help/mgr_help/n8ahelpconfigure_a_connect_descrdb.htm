Configurando um Descritor de Conexão para um Banco de Dados Oracle Rdb

Um banco de dados Oracle Rdb é um banco de dados para plataformas Digital de 64 bits. Os servidores Oracle Rdb funcionam exatamente como bancos de dados Oracle em relação aos clientes. Um banco de dados Oracle Rdb tem seu próprio listener. A configuração de cliente consiste na criação de umnome do serviço de rede com parâmetros específicos do Oracle Rdb para acessar o banco de dados.

Para configurar o cliente o banco de dados Oracle Rdb:

  1. Crie um nome de serviço de rede, conforme descrito em Criar um Nome de Serviço de Rede.

  2. Na caixa Identificação de Serviço do painel direito, escolha Avançado. A caixa de diálogo Opções Avançadas de Serviço será exibida.

  3. Informe o nome do arquivo de um banco de dados no campo Banco de Dados Rdb.

  4. Opcionalmente, informe o nome de banco de dados global no campo Nome de Banco de Dados Global e, se necessário, digite o tipo de serviço no campo Tipo de Serviço.

  5. Na caixa de Diálogo Opções Avançadas de Serviços, clique em OK.

  6. No painel direito, escolha Aplicar.

  7. Escolha Arquivo > Salvar Configuração de Rede.

Para obter mais informações, consulte a documentação do Oracle Rdb.

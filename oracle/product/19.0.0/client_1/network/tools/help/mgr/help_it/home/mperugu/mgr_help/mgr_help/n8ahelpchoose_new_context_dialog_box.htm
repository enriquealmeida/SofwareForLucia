Finestra di dialogo Scegliere un nuovo contesto

La finestra di dialogo Scegliere un nuovo contesto Oracle consente di impostare un altro contesto Oracle, che consiste in un contenitore cn=OracleContext in una struttura DIT (Directory Information Tree) contenente le voci da utilizzare con il software Oracle. Queste voci possono includere i nomi di servizi di rete e i nomi dei servizi di database per le connessioni ai servizi di database. Per impostazione predefinita, Oracle Net Manager utilizza il contesto Oracle configurato dall'Assistente alla configurazione di Oracle Net.

Contesto di denominazione delle directory

Nella lista selezionare una voce di directory contenente il contesto Oracle che si desidera visualizzare.

Contesto Oracle

Selezionare un contesto Oracle dall'apposito elenco. Il contesto Oracle contiene le voci della struttura delle directory per le quali si intende aggiungere o modificare le informazioni di rete.

La cartella Directory > Denominazione dei servizi nel riquadro del Navigator viene aggiornata con gli oggetti di rete del contesto Oracle selezionato.

Configuration d'Oracle Advanced Security

L'assistant Oracle Net Manager permet de configurer Oracle Advanced Security. Chaque onglet d'Oracle Advanced Security sert à configurer un ensemble distinct de paramètres. Oracle Advanced Security permet le cryptage et le contrôle de l'intégrité des données, l'authentification améliorée et la connexion unique. Oracle Advanced Security offre également une gestion utilisateur centralisée des serveurs d'annuaire compatibles LDAP et de la connexion unique fondée sur la certification ; cette fonctionnalité repose sur la couche SSL (Secure Sockets Layer).

Pour configurer Oracle Advanced Security :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Oracle Advanced Security.

  3. Les onglets affichés sont les suivants :

    • Authentification

    • Autres paramètres

    • Intégrité

    • Encryption

    • SSL

  4. Apportez les modifications nécessaires.

  5. Choisissez Fichier > Enregistrer la configuration réseau.

Les modifications sont enregistrées dans le fichier sqlnet.ora.

Remarque : Si Oracle Advanced Security n'apparaît pas dans la liste, c'est que cette fonction n'a pas été installée.

Pour plus d'informations, reportez-vous au Guide de l'administrateur Oracle Advanced Security.

Detección

El proceso de localización de servidores de Oracle Names. Durante la detección, se localiza un Oracle Names Server. Una vez detectado el primero, el servidor de Oracle Names detectado hace ping en todos los demás servidores de Oracle Names de la región administrativa. Se crea una lista de servidores de Oracle Names y se guarda en .sdns.ora en UNIX y en sdns.ora en Windows. Esta lista se ordena por orden de respuesta.

El proceso de detección busca el primer Oracle Names Server en el siguiente orden:

  1. Un Oracle Names Server preferido en el archivo sqlnet.ora

  2. Un Oracle Names Server conocido

  3. Un Oracle Names Server local configurado con TCP/IP en el puerto 1575

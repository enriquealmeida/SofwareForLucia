不正アクセスおよびユーザー処理の監査に対するバナーの構成

不正アクセスおよびユーザー処理の想定される監査に対してユーザーに警告するためのバナーを作成および構成する必要があります。データベースにログインしている場合、クライアント・アプリケーションに対して通知を使用できます。

これらのバナーを構成して表示するには、バナー情報を含むテキスト・ファイルを指し示すように、データベース・サーバー側で次のsqlnet.oraパラメータを設定します。

  • SEC_USER_UNAUTHORIZED_ACCESS_BANNER。例: SEC_USER_UNAUTHORIZED_ACCESS_BANNER = /opt/Oracle/11g/dbs/unauthaccess.txt

  • SEC_USER_AUDIT_ACTION_BANNER。例: SEC_USER_AUDIT_ACTION_BANNER = /opt/Oracle/11g/dbs/auditactions.txt

デフォルトでは、これらのパラメータは設定されません。

これらのパラメータを設定したら、これらのバナーを取得してエンドユーザーに表示するように、Oracle Call Interfaceアプリケーションで適切なOCI APIを使用する必要があります。

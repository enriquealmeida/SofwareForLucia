Criando um Apelido do Serviço de Rede

Observações:

  • Somente os usuários membros do grupo OracleNetAdmins ou OracleContextAdmins podem criar entradas de apelidos de serviço de rede em um diretório.

  • Os apelidos de serviços de rede não são suportados com o uso do Microsoft Active Directory.

  • Você pode exportar objetos de apelido existentes de um servidor do Oracle Names. Consulte o capítulo Exporting Naming Data to a Directory Server no Oracle10i Net Services Administrator's Guide para obter mais informações.

Você pode criar apelidos de serviço de rede para a nomeação de objetos de diretório --nomes de serviço de rede o u de serviços de banco de dados -- em um servidor de diretórios. Um apelido de serviço de rede armazena o nome do objeto a que ele se refere.

Quando um cliente solicita uma pesquisa de diretório para um apelido de serviço de rede, o diretório determina que a entrada seja um apelido de serviço de rede e conclui a pesquisa como se fosse a entrada referenciada. Por exemplo, um alias de serviço de rede salesalias pode ser criado para um nome de serviço de rede sales. Quando salesalias for usado para estabelecer conexão com um serviço de banco de dados, como em CONNECT scott/tiger@salesalias, ele na verdade será resolvido para informações do descritor de conexão e usará essas informações de sales.

Há dois usos principais para apelidos de nome do serviço de rede:

  • Use um apelido de nome do serviço de rede como uma forma de clientes referenciarem um serviço de banco de dados ou um nome do serviço de rede por outro nome.

  • Utilize um apelido de serviço de rede em um Oracle Context para um nome de serviço de banco de dados ou um nome do serviço de rede em outro Oracle Context. Isso permite que o nome de serviço de banco de dados ou o nome do serviço de rede seja definido uma vez no servidor de diretórios, mas seja referenciado por clientes que utilizam outros Oracle Contexts.

Para criar um apelido de nome do serviço de rede:

  1. No painel do navegador, expanda Configuração do Oracle Net >Diretório > Nomeação de Serviço > Apelidos.

  2. Escolha o sinal de adição (+) na barra de ferramentas ou escolha Editar > Criar na barra de menus. A caixa de diálogo Criar Apelido de Serviço de Rede é exibida.

  3. No campo Apelido de Serviço de Rede, informe um nome para o apelido do serviço de rede.

  4. Na caixa Serviço de Rede ou Serviço de Banco de Dados, especifique estas informações sobre o nome do serviço de rede ou o serviço de banco de dados:

    • Oracle Context: Especifique o Oracle Context do serviço de banco de dados ou do nome do serviço de rede selecionando na lista ou informando no campo.

    • Nome: Especifique o DN ("distinguished name", nome distinto) do serviço de banco de dados ou nome do serviço de rede selecionando na lista ou informando no campo.

  5. Selecione Criar.

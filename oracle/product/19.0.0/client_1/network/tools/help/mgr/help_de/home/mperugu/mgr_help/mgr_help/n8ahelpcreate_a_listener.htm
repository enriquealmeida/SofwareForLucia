Listener erstellen

Ein Listener muss auf dem Server vorhanden sein, damit die Datenbank Verbindungen empfangen kann.

Hinweis: Während der Installation wird ein Standard-Listener mit der Bezeichnung LISTENER erstellt.

So erstellen Sie einen Listener:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal> Listener im Navigator-Fensterbereich ein.

  2. Klicken Sie auf das Pluszeichen (+) in der Symbolleiste, oder wählen Sie Bearbeiten > Erstellen. Daraufhin wird das Dialogfeld Listener-Name auswählen angezeigt.

  3. Akzeptieren Sie den Listener-Namen, der in dem Feld Listener-Name angegeben wird, oder geben Sie einen anderen Namen ein. Standardmäßig lautet der Listener-Name für den ersten Listener LISTENER. Nachfolgende Listener-Namen sind standardmäßig LISTENERn . Der Listener-Name muss auf einem bestimmten Knoten eindeutig sein.

  4. Wählen Sie OK.

  5. Führen Sie die Schritte 4-6 in Listener-Adressen konfigurieren aus.

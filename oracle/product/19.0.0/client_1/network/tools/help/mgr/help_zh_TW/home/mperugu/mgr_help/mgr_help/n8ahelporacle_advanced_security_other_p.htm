Oracle Advanced Security：其他參數

您可以使用其他參數頁籤來設定其他的認證服務參數。

從認證服務清單選取認證方法。根據選取之認證方法的不同，請在適當欄位輸入資訊。

KERBEROS(v5)

元素 描述
服務 輸入任何字串，指定 Kerberos 服務名稱。
證明資料快取檔案 輸入任何有效路徑名稱，指定 Kerberos 證明資料快取路徑名稱。
組態檔 輸入任何有效路徑名稱，指定 Kerberos 組態檔。
範圍轉譯檔案 輸入任何有效路徑名稱，指定 Kerberos 主機名稱給範圍轉譯檔。
金鑰表格 輸入任何有效路徑名稱，指定 Kerberos 秘密金鑰檔。
時脈過期 輸入任何正整數，指定當傳送與接收證明資料時，可接受的秒差。

CYBERSAFE

GSSAPI 服務：指定 CYBERSAFE 服務主要項目。請以正確的格式輸入任何主要項目字串。

NTS

不需要其他參數。

RADIUS

元素 描述
主機名稱 輸入主要 RADIUS 伺服器的名稱。
連接埠號碼 輸入主要 RADIUS 伺服器的連接埠號碼。
逾時 (秒) 輸入 Oracle 伺服器應該等候主要 RADIUS 伺服器的回應時間 (秒)。
重試次數 輸入 Oracle 伺服器應該重新傳送訊息至主要 RADIUS 伺服器的次數。
機密檔案 輸入秘密金鑰檔的路徑名稱。
傳送帳號 輸入 on 啟用帳號或輸入 off 停用帳號。
挑戰回應 輸入 on 啟用查問回應，或輸入 off 停用查問回應。
預設的關鍵字 輸入要求 RADIUS 伺服器之查問的關鍵字。
介面類別名稱 輸入您建立用於處理 Oracle 從屬端與 RADIUS 伺服器之間查問回應對話的類別名稱。

如需詳細資訊，請參閱 Oracle Advanced Security Administrator's Guide。

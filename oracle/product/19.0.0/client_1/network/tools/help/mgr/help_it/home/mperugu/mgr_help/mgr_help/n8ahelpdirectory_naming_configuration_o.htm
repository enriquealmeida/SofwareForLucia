Panoramica sulla configurazione della denominazione delle directory

In questo argomento viene descritta nei dettagli la configurazione della denominazione delle directory.

Configurazione del server durante l'installazione

Durante un'installazione personalizzata del server, l'Assistente alla configurazione di Oracle Net richiede all'utente di:

  • selezionare il tipo di server delle directory;

  • identificare la posizione della directory;

  • selezionare o creare un contesto Oracle dal quale il server può cercare, creare e modificare gli identificativi di connessione;

  • configurare la denominazione delle directory come metodo di denominazione preferito.

Se l'uso della directory è configurato correttamente, l'Assistente alla configurazione del database tenta di connettersi alla directory e di registrare una voce del servizio di database. Questa voce è configurata con un descrittore di connessione contenente le informazioni di rete relative a:

  • indirizzo del listener;

  • nome di servizio di database.

Configurazione del client durante l'installazione

Durante tutte le installazioni client, l'Assistente alla configurazione di Oracle Net richiede di configurare le seguenti impostazioni di uso delle directory:

  • specificare il tipo di directory;

  • identificare la posizione della directory;

  • selezionare un contesto Oracle predefinito dal quale il client può cercare gli identificativi di connessione;

  • impostare la denominazione delle directory come metodo di denominazione preferito.

Configurazione con l'Assistente alla configurazione di Oracle Net dopo l'installazione

L'Assistente alla configurazione di Oracle Net può essere eseguito anche in modalità standalone per:

  • configurare il client o il server, secondo quanto descritto in precedenza;

  • creare lo schema Oracle;

  • creare un contesto Oracle.

Per eseguire l'Assistente alla configurazione di Oracle Net in modalità standalone, effettuare le operazioni riportate di seguito.

  • In ambiente UNIX, eseguire netca in $ORACLE_HOME/bin.

  • In Windows NT, scegliere Avvio > Programmi > Oracle - NOME_HOME > Amministrazione di rete > Assistente alla configurazione di Oracle Net.

Configurazione con Oracle Net Manager

Oracle Net Manager consente di:

  • modificare il contesto Oracle;

  • modificare le credenziali di autenticazione;

  • creare i nomi di servizi di rete;

  • creare gli alias dei servizi di rete;

  • configurare più instradamenti per un listener utilizzando più indirizzi di protocollo;

  • configurare le informazioni CONNECT_DATA avanzate per un servizio di database o un nome di servizio di rete;

  • eseguire la migrazione delle voci dei nomi di servizi di rete da un file tnsnames.ora a un server delle directory.

Per accedere alla configurazione della denominazione delle directory, scegliere Directory - Denominazione dei servizi dal riquadro del Navigator.

Per ulteriori informazioni, vedere la Guida in linea dell'Assistente alla configurazione di Oracle Net.

Denominazione host

Un metodo di denominazione che consente agli utenti di connettersi a un database server Oracle mediante un alias del nome host in un ambiente TCP/IP. I nomi host vengono mappati al nome del database globale del server in un servizio di risoluzione nomi esistente, quale DNS (Domain Name System), NIS (Network Information Service) o un set di file /etc/hosts gestito centralmente.

Per informazioni sulla configurazione del metodo di denominazione host, vedere il capitolo Configuring Naming Methods del manuale Oracle10i Net Services Administrator's Guide.

Creazione di indirizzi di protocollo aggiuntivi

È possibile accedere a un servizio di database da più instradamenti di rete o indirizzi di protocollo.

Per aggiungere un indirizzo di protocollo di rete a un servizio di database o a un nome di servizio di rete esistente, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Directory o Locale > Denominazione dei servizi.

  2. Selezionare un nome di servizio di rete o un nome di servizio di database. Nel riquadro di destra sono visualizzati il servizio di destinazione corrente e la lista indirizzi.

  3. Nella casella Configurazione indirizzi, fare clic sul segno più (+). Viene visualizzata una nuova scheda Indirizzo.

  4. Selezionare un protocollo, quindi immettere i parametri appropriati per il protocollo selezionato.

  5. Con i pulsanti freccia destra e sinistra, ordinare gli indirizzi in base alla posizione loro assegnata nella lista indirizzi. A meno che non siano configurate le opzioni per più indirizzi, viene contattato soltanto il primo indirizzo nella lista. Per impostare un ordine diverso, vedere Configurazione delle opzioni per più indirizzi.

  6. Nel riquadro di destra, scegliere Applica.

  7. Se si apportano queste modifiche alla cartella Locale, scegliere File > Salva configurazione di rete.

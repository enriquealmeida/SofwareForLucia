Service Name

Eine logische Darstellung einer Datenbank, d.h. die Darstellung der Datenbank gegenüber den Clients. Eine Datenbank kann in Form mehrerer Dienste dargestellt werden. Ein Dienst kann wiederum in Form von mehreren Datenbankinstanzen implementiert werden. Ein Service Name ist eine Zeichenfolge, bei der es sich um den globalen Datenbanknamen handelt, d.h. ein Name, der aus dem Datenbanknamen und Domainnamen besteht, die während der Installation oder beim Erstellen der Datenbank eingegeben wurden. Wenn Sie den globalen Datenbanknamen nicht genau kennen, können Sie ihn aus den kombinierten Werten des Parameters SERVICE_NAMES in der Initialisierungsparameterdatei ermitteln.

Verbindungsdeskriptoren, die von Clients zur Verbindung mit einer Oracle9i- oder Oracle8i-Datenbank verwendet werden, müssen mit dem Parameter SERVICE_NAME (ohne S) konfiguriert werden.

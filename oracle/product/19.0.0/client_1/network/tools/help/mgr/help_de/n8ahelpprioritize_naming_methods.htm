Priorität von Benennungsmethoden festlegen

Nachdem Benennungsmethoden konfiguriert wurden, muss ihre Priorität in dem Profil festgelegt werden. Die Benennungsmethode am Anfang der Liste wird als erste zur Auflösung eines Net Service-Namens oder Datenbankdienstnamens in einen Verbindungsdeskriptor verwendet. Wenn diese Methode den Namen nicht auflösen kann, wird die zweite Methode in der Liste verwendet.

So geben Sie die Reihenfolge von Benennungsmethoden an:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal > Profil im Navigator-Fensterbereich ein.

  2. Wählen Sie Namensgebung aus der Liste im rechten Fensterbereich.

  3. Wählen Sie das Register Methoden.

  4. Wählen Sie Benennungsmethoden aus der Liste Verfügbare Methoden, und klicken Sie auf den Pfeil nach rechts. Die ausgewählte Benennungsmethode wird in die Liste Ausgewählte Methoden verschoben.

  5. Ordnen Sie die Benennungsmethoden entsprechend der Methode an, die Oracle Net als Erstes bei der Auflösung von Net Service-Namen oder Datenbankdienstnamen verwenden soll. Wählen Sie die Benennungsmethode in der Liste Ausgewählte Methoden, und klicken Sie auf Vorziehen oder Zurücksetzen, um die Auswahl in der Liste nach oben oder unten zu verschieben.

  6. Wählen Sie Datei >Netzwerkkonfiguration speichern.

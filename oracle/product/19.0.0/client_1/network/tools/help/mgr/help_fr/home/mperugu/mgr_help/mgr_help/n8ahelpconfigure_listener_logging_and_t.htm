Configuration de la journalisation et de la trace du processus d'écoute

Vous pouvez configurer le processus d'écoute avec les paramètres de journalisation et de trace.

Les informations de journalisation sont généralement stockées dans le fichier listener.log , situé dans $ORACLE_HOME/network/log sous UNIX, et dans ORACLE_HOME\network\log sous Windows.

Les informations de trace sont stockées par défaut dans le fichier listener.trc disponible dans $ORACLE_HOME/network/trace sous UNIX et dans ORACLE_HOME\network\trace sous Windows.

Pour activer la journalisation ou la trace :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Processus d'écoute.

  2. Sélectionnez un processus d'écoute.

  3. Dans la liste du panneau de droite, sélectionnez Paramètres généraux.

  4. Sélectionnez l'onglet Journalisation et trace.

  5. Ajustez les paramètres de journalisation et de trace si nécessaire.

  6. Choisissez Fichier > Enregistrer la configuration réseau.

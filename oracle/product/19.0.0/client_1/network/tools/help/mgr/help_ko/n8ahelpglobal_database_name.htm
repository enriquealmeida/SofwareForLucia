전역 데이터베이스 이름

다른 데이터베이스와 구분되는 데이터베이스의 고유한 전체 이름입니다. 전역 데이터베이스 이름은 "database_name.database_domain" 형식입니다(예: sales.us.example.com).

데이터베이스 이름 부분인 sales는 데이터베이스를 지칭하는 간단한 이름입니다. 데이터베이스 도메인 부분인 us.example.com 은 해당 데이터베이스가 위치한 데이터베이스 도메인을 지정하며 전역 데이터베이스 이름을 고유하게 식별합니다. Oracle은 가능한 경우 데이터베이스 도메인에 네트워크 도메인이 반영되도록 지정할 것을 권장합니다.

전역 데이터베이스 이름은 초기화 매개변수 파일의 SERVICE_NAMES 매개변수에 지정된 데이터베이스의 기본 서비스 이름입니다.

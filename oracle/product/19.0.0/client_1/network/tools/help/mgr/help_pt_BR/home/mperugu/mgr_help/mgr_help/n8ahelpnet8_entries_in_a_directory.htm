Entradas do Oracle Net em um Diretório

As entradas de um diretório são organizadas em uma estrutura hierárquica em árvore com base em seus nomes distintos (DNs). Essa árvore de entradas de diretório é chamada deDIT (árvore de informações de diretório). Cada entrada contém um ou mais atributos que a descrevem.

As entradas do Oracle Net em um DIT podem ser:

  • Um serviço de banco de dados criado durante a criação do banco de dados

  • Nome do serviço de rede

  • Apelido do serviço de rede

Todas as entradas do Oracle Net são apresentadas sob uma subárvore do Oracle Context que armazena entradas Oracle. Essa subárvore é criada em um local especificado pelo usuário na DIT durante uma instalação Personalizada do servidor. Os clientes são configurados com a utilização de diretório e de um Oracle Context default a partir do qual é possível pesquisar entradas.

Cada entrada contém um ou mais atributos que a descrevem. Cada atributo dispõe de um tipo e um valor. As entradas do diretório Oracle Net têm estes atributos:

  • Nome comum

  • Versão da entrada

  • Descritor de conexão

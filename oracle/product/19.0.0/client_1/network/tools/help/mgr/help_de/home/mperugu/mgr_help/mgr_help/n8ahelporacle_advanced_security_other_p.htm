Oracle Advanced Security: Andere Parameter

Im Register Andere Parameter können Sie zusätzliche Parameter für Berechtigungsprüfungsdienste konfigurieren.

Wählen Sie die Berechtigungsprüfungsmethode aus der Liste Berechtigungsprüfungsdienst. Je nach ausgewählter Berechtigungsprüfungsmethode geben Sie die entsprechenden Informationen in die Felder ein.

KERBEROS(V5)

Element Beschreibung
Dienst Geben Sie eine Zeichenfolge zur Angabe des Kerberos-Servicenamens ein.
Cache-Datei für ID-Daten Geben Sie einen gültigen Pfadnamen zur Angabe des Pfadnamens für den Cache mit Kerberos-ID-Daten ein.
Konfigurationsdatei Geben Sie einen gültigen Pfadnamen zur Angabe der Kerberos-Konfigurationsdatei ein.
Realm-Übersetzungsdatei Geben Sie einen gültigen Pfadnamen zur Angabe des Kerberos-Hostnamens für die Realm-Übersetzungsdatei ein.
Schlüsseltabelle Geben Sie einen gültigen Pfadnamen zur Angabe der Datei mit dem Kerberos-Secret Key ein.
Clockskew Geben Sie eine positive Ganzzahl zur Angabe der annehmbaren Differenz bei der Anzahl von Sekunden zwischen dem Senden und Empfangen von ID-Daten ein.

CYBERSAFE

GSSAPI-Dienst:Geben Sie eine CYBERSAFE-Dienst-Identität an. Geben Sie eine richtig formatierte Zeichenfolge für die Dienst-Identität ein.

NTS

Keine zusätzlichen Parameter erforderlich.

RADIUS

Element Beschreibung
Hostname Geben Sie den Namen des primären RADIUS-Servers an.
Portnummer Geben Sie die Portnummer des primären RADIUS-Servers ein.
Zeitüberschreitung (Sekunden) Geben Sie die Anzahl von Sekunden ein, die der Oracle Datenbank-Server auf eine Antwort von dem primären RADIUS-Server warten soll.
Anzahl Wiederholungsversuche Geben Sie an, wie oft der Oracle Datenbank-Server erneut Meldungen an den primären RADIUS-Server senden soll.
Secret-Datei Geben Sie den Pfadnamen der Secret Key-Datei an.
Accounting senden Geben Sie ON ein, um das Accounting zu aktivieren bzw. zu deaktivieren.
Challenge-Antwort Geben Sie ON ein, um die Challenge-Antwort zu aktivieren bzw. OFF, um sie zu deaktivieren.
Standardschlüsselwort Geben Sie das Schlüsselwort für die Anforderung eines Challenge aus dem RADIUS-Server ein.
Interface Class Name Geben Sie den Namen der Klasse ein, die Sie erstellt haben, um den Challenge-Antwort-Dialog zwischen dem Oracle-Client und dem RADIUS-Server zu bearbeiten.

Weitere Informationen finden Sie im Oracle Advanced Security Administrator's Guide.

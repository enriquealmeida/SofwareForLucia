Configurando Várias Opções de Endereço

Quando um serviço de banco de dados está acessível para váriosendereços de protocolo de listener, é importante especificar a ordem na qual os endereços de protocolo serão usados. Por exemplo, os endereços de protocolo podem ser escolhidos aleatoriamente ou sequencialmente.

Quando vários endereços de protocolo são configurados para um nome do serviço de rede ou serviço de banco de dados, as seguintes opções podem ser configuradas:

  • Failover de tempo de conexão para escolher o primeiro endereço de protocolo e passar para o próximo endereço de protocolo se o primeiro falhar

  • Balanceamento de carga do cliente para escolher um endereço de protocolo aleatoriamente

  • Roteamento de origem para usar todos os endereços de protocolo em sequência

Para configurar várias opções de endereço:

  1. No painel do navegador, expanda Configuração do Oracle Net > Diretório > Local ou Nomeação de Serviço.

  2. Selecione um nome do serviço de rede ou um serviço de banco de dados. O painel direito exibirá o serviço de destino atual e uma lista de endereços.

  3. Na caixa Configuração de Endereço, escolha Avançado. A caixa de diálogo Opções da Lista de Endereços será exibida.

  4. Desmarque Usar Opções Compatíveis com Clientes do Net 8.0 se esta configuração se referir a um cliente Oracle9i ou Oracle8i.

  5. Selecione a opção apropriada para a lista de endereços de protocolo e, em seguida, escolha OK.

  6. No painel direito, escolha Aplicar.

  7. Se você estiver fazendo essas alterações na pasta Local, escolha Arquivo > Salvar Configuração de Rede.

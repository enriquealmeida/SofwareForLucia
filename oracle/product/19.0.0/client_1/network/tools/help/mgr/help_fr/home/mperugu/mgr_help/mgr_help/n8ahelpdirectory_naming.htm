Résolution de noms de répertoire

Méthode de résolution de noms convertissant un service de base de données ou un nom de service réseau en descripteur de connexion, stocké sur un serveur d'annuaire centralisé.

Un annuaire permet de gérer de façon centralisée les services de base de données et les noms de service réseau, ce qui réduit la nécessité d'ajouter ou de déplacer des services. Bien que les noms de service réseau puissent être configurés sous forme d'alias de service, l'annuaire peut faire directement référence à un service de base de données, sans utiliser de nom de service réseau. Le service de base de données est ajouté automatiquement en tant qu'entrée à l'annuaire lors de l'installation.

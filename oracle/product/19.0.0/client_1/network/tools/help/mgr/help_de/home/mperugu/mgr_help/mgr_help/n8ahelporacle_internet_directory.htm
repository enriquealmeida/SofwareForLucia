Oracle Internet Directory

Ein Directory Service, der als Anwendung in der Oracle-Datenbank implementiert ist. Dieser Dienst ermöglicht das Abrufen von Informationen über verteilte Benutzer und Netzwerkressourcen. Er kombiniert Lightweight Directory Access Protocol (LDAP) Version 3, das offene Internet-Standardprotokoll für den Verzeichniszugriff mit der hohen Leistung, Skalierbarkeit, Robustheit und Verfügbarkeit der Oracle-Datenbank.

Weitere Informationen finden Sie im Oracle Internet Directory Administrator's Guide.

一般参数：一般信息

一般信息选项卡使您能够配置监听程序的一般参数。

监听程序名称

显示监听程序的名称。

启动等待时间

指定在响应监听程序控制实用程序 START 命令之前监听程序等待的时间 (秒)。

注：此设置已过时，在以后的发行版中不再支持。如果运行监听程序时需要此参数，请通知 Oracle 技术支持。

在关闭时保存配置

选择此选项以永久保存由监听程序控制实用程序 SET 命令所作的任何更改。

SNMP 联系信息

输入任意文本，此文本将写入snmp.ora文件供 Oracle Enterprise Manager 使用。此信息可通过任何 SNMP 连接发送，并且可以是任意字符串。它不会影响创建 SNMP 连接的进程。此信息一般由客户机的用户或管理员的联系信息组成。

ADMIN_RESTRICTIONS_listener_name

限制监听程序的运行时管理。

默认设置为 off。设置 ADMIN_RESTRICTIONS_listener_name=on 将禁止对 listener.ora 中的参数进行运行时修改。即监听程序将拒绝接受变更其参数的 SET 命令。要更改 listener.ora 中的任何参数 (包括 ADMIN_RESTRICTIONS_listener_name 自身)，可手动修改 listener.ora 文件并重新加载其参数 (使用 RELOAD 命令) 以使新更改生效而无需显式停止并重新启动监听程序。

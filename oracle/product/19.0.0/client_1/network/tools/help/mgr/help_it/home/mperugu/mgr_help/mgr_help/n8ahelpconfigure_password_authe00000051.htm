Configurazione dell'autenticazione tramite password per il listener

Il listener può essere protetto mediante la configurazione. Le operazioni privilegiate eseguite dalla utility Controllo del listener, come il salvataggio delle modifiche apportate alla configurazione o l'arresto del listener, richiederanno pertanto l'immissione di una password. Oracle Net Manager consente di impostare una password cifrata.

Per impostare una password per il listener, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

  2. Selezionare un listener.

  3. Dalla lista presente nel riquadro di destra, selezionare Parametri generali.

  4. Selezionare la scheda Autenticazione.

  5. Scegliere Richiedi una password per le operazioni del listener.

  6. Immettere la password nel campo Password, quindi immetterla nuovamente nel campo Conferma password.

  7. Scegliere File > Salva configurazione di rete.

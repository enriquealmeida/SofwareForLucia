「ファイル」メニュー

「ファイル」メニューでは、開く、回復および保存操作ができます。

ネットワーク構成を開く

ローカル・コンフィギュレーション・ファイルを開きます。

ネットワーク構成の保存

現行の構成設定をデフォルト位置に保存します。

別名保存

現行の構成設定をデフォルト以外の位置に保存します。

保存済の構成に戻す

現行の構成設定を保存しない場合に選択します。かわりに、前回保存した構成を取り出します。

終了

Oracle Net Managerを閉じます。

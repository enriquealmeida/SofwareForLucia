Barra degli strumenti

La barra degli strumenti, visibile sul lato sinistro di Oracle Net Manager, contiene le icone che corrispondono agli oggetti di rete presenti nelle gerarchie delle cartelle. I pulsanti della barra degli strumenti sono abilitati a seconda degli oggetti correntemente visualizzati o selezionati in un riquadro. È possibile spostare il cursore del mouse su un'icona della barra degli strumenti per visualizzare una descrizione della funzione corrispondente.

La barra degli strumenti contiene le seguenti icone:

Icona Descrizione
Aggiungi Fare clic su questa icona per creare nuovi nomi dei servizi di rete, i listener e un server Oracle Names per questo computer.
Elimina Fare clic su questa icona per eliminare gli oggetti relativi alla denominazione dei servizi e ai listener.
Verifica connessione Fare clic su questa icona per verificare la connettività di un oggetto di denominazione dei servizi selezionato rispetto a un servizio di database. Durante un test di connettività viene stabilita una connessione a un servizio di database utilizzando le informazioni del descrittore di connessione memorizzate in un file tnsnames.ora o in un server delle directory .
? Fare clic su questa icona per visualizzare la Guida in linea.

Nota: alcuni comandi sono disponibili soltanto se è selezionato un determinato tipo di oggetto. Per creare un listener, ad esempio, è necessario selezionare prima la cartella Listener .

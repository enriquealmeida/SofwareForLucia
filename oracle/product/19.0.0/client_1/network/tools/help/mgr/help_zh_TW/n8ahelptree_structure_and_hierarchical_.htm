樹狀結構和階層資料夾

Oracle Net Manager 會顯示網路物件以及它們包含的物件 (以資料夾的階層加以組織) 的圖形樹狀結構視觀表。您可以使用瀏覽窗格來檢視、修改、新增以及刪除每一個資料夾中的物件。

展開資料夾時，您會看到物件與資料夾的巢狀清單。選取物件後，物件的相關資訊會顯示在 Oracle Net Manager 的右窗格中。

目錄

顯示目錄伺服器底下所設定，稱為連線 ID 的簡單名稱。

注意：如果已經設定目錄的存取方式，「目錄」資料夾就會顯示。如需目錄存取組態及其他需求的詳細資訊，請參閱 Oracle Net Manager 的目錄命名需求。

區域

顯示設定在區域組態檔案中的網路元素：

  • tnsnames.ora 檔案中的網路服務名稱

  • listener.ora 檔案中的監聽器

  • sqlnet.ora 檔案中的設定檔

Oracle Names 伺服器

顯示 Oracle Names 伺服器。

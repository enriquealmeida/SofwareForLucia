Verbindungsdeskriptor für Verbindungen zu Oracle Connection Manager konfigurieren

Um Clients über Oracle Connection Manager an den Datenbankserver weiterzuleiten, konfigurieren Sie die Datei tnsnames.oraDatei oder den Directory-Server mit einem Verbindungsdeskriptor, der die Protokolladresse von Oracle Connection Manager angibt. Mit dieser Adresse kann sich der Client bei dem Rechner von Oracle Connection Manager anmelden.

So konfigurieren Sie eine Protokolladresse für Oracle Connection Manager:

  1. Blenden Sie im Navigator-Fensterbereich Oracle Net-Konfiguration > Verzeichnis oder Lokal >Dienstebenennung ein.

  2. Klicken Sie auf das Pluszeichen (+) in der Symbolleiste, oder wählen Sie Bearbeiten > Erstellen aus der Menüleiste. Daraufhin wird die Willkommen-Seite des Net Service-Name-Assistenten angezeigt.

  3. Geben Sie einen beliebigen Namen in das Feld Net Service-Name ein.

  4. Klicken Sie auf Weiter. Daraufhin wird die Seite "Protokoll" angezeigt.

  5. Wählen Sie das Protokoll, auf das Oracle Connection Manager gemäß Konfiguration horcht. Standardmäßig ist dieses Protokoll TCP/IP.

  6. Klicken Sie auf Weiter. Daraufhin wird die Seite "Protokolleinstellungen" angezeigt.

  7. Geben Sie die entsprechenden Protokollparameter für das ausgewählte Protokoll in die entsprechenden Felder ein. Wenn Sie TCP/IP verwenden, ist der Standardport 1630.

  8. Klicken Sie auf Weiter. Daraufhin wird die Seite "Service" angezeigt.

  9. Wenn der Zielservice Oracle8i oder höher ist, wählen Sie Oracle8i oder höher und geben einen Servicenamen in das Feld Servicename ein. Wenn der Zieldienst Oracle8 oder Oracle7 ist, wählen Sie Oracle8 oder früher und geben einen Oracle System Identifier in das Feld Datenbank-SID ein.

  10. Klicken Sie auf Weiter. Daraufhin wird die Seite "Service" angezeigt.

  11. Klicken Sie auf Fertig, um die Konfiguration zu speichern und den Net Service-Name-Assistenten zu schließen.

  12. Wählen Sie Datei >Netzwerkkonfiguration speichern.

  13. Der neue Net Service-Name wird zu dem Ordner " Dienstebenennung" hinzugefügt.

Weitere Informationen über die vollständige Aktivierung der Oracle Connection Manager-Features finden Sie im Kapitel Configuring and Administering Oracle Connection Manager im Oracle10i Net Administrator's Guide.

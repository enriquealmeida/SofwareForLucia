Sistema de Nombres de Dominio (DNS)

Sistema de nomenclatura de computadoras y servicios de red organizado en jerarquías de dominios. El DNS se utiliza en las redes TCP/IP para encontrar computadoras mediante nombres fáciles de recordar. El DNS asigna un nombre fácil a una dirección IP, que entiende la computadora.

En la Red de Oracle, DNS traduce el nombre de host utilizado en las direcciones TCP/IP a una dirección IP.

Configurazione di Oracle Advanced Security

È possibile utilizzare Oracle Net Manager per configurare Oracle Advanced Security. Ogni scheda di Oracle Advanced Security consente di configurare un diverso gruppo di parametri. Oracle Advanced Security attiva le seguenti funzionalità: cifratura e verifica dell'integrità, autenticazione avanzata e Single Sign-On. Oracle Advanced Security supporta inoltre la gestione centralizzata degli utenti sui server delle directory conformi al protocollo LDAP, nonché la funzionalità Single Sign-On basata sui certificati. Questa funzionalità utilizza il protocollo SSL (Secure Sockets Layer).

Per configurare Oracle Advanced Security, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Oracle Advanced Security.

  3. Vengono visualizzati i seguenti pannelli della scheda:

    • Autenticazione

    • Altri parametri

    • Integrità

    • Cifratura

    • SSL

  4. Apportare le modifiche desiderate alla configurazione.

  5. Scegliere File > Salva configurazione di rete.

Le modifiche vengono salvate nel file sqlnet.ora.

Nota: se Oracle Advanced Security non è visualizzato nella lista, non è stato installato.

Per ulteriori informazioni, vedere il manuale Oracle Advanced Security Administrator's Guide.

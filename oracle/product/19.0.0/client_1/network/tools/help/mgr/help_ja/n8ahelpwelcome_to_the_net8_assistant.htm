Oracle Net Managerへようこそ

Oracle Net Managerへようこそ。Oracle Net Managerは、構成機能をコンポーネント制御機能と組み合せて、Oracle Netの構成および管理のための統合環境を提供します。

Oracle Net Managerの使用者

クライアントからサービスへの接続の確立を担当するネットワーク管理者です。Oracle Net Managerでは、ネットワークの特定領域の構成に焦点を絞ることができます。

初めてOracle Net Managerを使用する場合

Oracle Net Managerやネットワーキング全般の経験が少ない場合は、ヘルプ・システムを参照してOracle Net Managerおよびネットワーク構成に関する知識を得ることができます。Oracle10i Net Services管理者ガイドも参照してください。

詳細接続データ・オプションの構成

データベースのサービス名やOracleシステム識別子(SID)の他に、必要に応じて次のオプションを使用して接続記述子のCONNECT_DATA部分を構成できます。

  • インスタンス名。特定のデータベース・インスタンスへの接続に使用します。

  • ネットワークの外からデータベースを開始するためのデータベースのOracleホーム。

  • セッション・データ・ユニット・サイズ。バッファ・サイズを指定します。

  • 専用サーバー。専用サーバー・プロセスの接続に使用します。

  • 異機種間サービス。Oracle以外のシステムへのアクセスに使用します。

  • Oracle RDBデータベース(Oracle Rdb database)設定。Oracle RDBデータベースへのアクセスに使用します。

ネット・サービス名の詳細接続オプションを構成する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ディレクトリ」→「ローカル」または「サービス・ネーミング」を展開します。

  2. ネット・サービス名またはデータベース・サービスを選択します。右ペインに現行の接続先サービスおよびアドレス・リストが表示されます。

  3. 「サービスの識別」ボックスで「詳細」を選択します。「詳細サービス・オプション」ダイアログ・ボックスが表示されます。

  4. 必要に応じて、各フィールドに値を入力するかオプションを選択して、「OK」を選択します。

  5. 右ペインで「適用」を選択します。

  6. この変更を「ローカル」フォルダに対して行う場合は、「ファイル」→「ネットワーク構成の保存」を選択します。

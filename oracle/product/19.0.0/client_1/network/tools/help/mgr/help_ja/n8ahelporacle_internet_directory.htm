Oracle Internet Directory

Oracleデータベース上にアプリケーションとして実装されるディレクトリ・サービス。分散したユーザーおよびネットワーク・リソースに関する情報を検索できます。Oracle Internet Directoryによって、高度なパフォーマンス、拡張性および耐久性を持つオープンなインターネット標準のディレクトリ・アクセス・プロトコルであるLightweight Directory Access Protocol (LDAP)バージョン3が、Oracleデータベースの可用性と組み合されます。

詳細は、Oracle Internet Directory管理者ガイドを参照してください。

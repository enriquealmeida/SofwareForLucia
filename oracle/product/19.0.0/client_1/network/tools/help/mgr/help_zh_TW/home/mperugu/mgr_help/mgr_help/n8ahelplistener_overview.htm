監聽器簡介

服務會透過監聽器的協助來協調階段作業 (伺服器接收從屬端應用程式連線要求的處理作業)。監聽器會設定為「監聽」資料庫服務或非資料庫服務的協定位址。

您可以針對資料庫服務或非資料庫服務，在監聽器組態檔 listener.ora 中設定協定位址。Oracle9i 或 Oracle8i 資料庫服務會自動將其資訊 (例如服務名稱、執行處理名稱以及載入資訊) 註冊至監聽器。此功能 (稱為服務註冊) 不需要在 listener.ora 檔案中設定服務組態。其他服務 (包括 Oracle8 與 Oracle7 資料庫服務) 以及 Oracle Enterprise Manager 管理工具，則需要在listener.ora檔案中設定服務組態。

設定監聽器之後，您就可以使用監聽器控制公用程式來管理監聽器。

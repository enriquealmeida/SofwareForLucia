Rejeté

N'activez pas le service de sécurité, même si l'autre machine le demande.

Ma machine indique que le service de sécurité n'est pas autorisé. Si l'autre machine indique requis, la connexion sera interrompue avec le message d'erreur ORA-12650. Si l'autre machine est paramétrée à demandé, accepté ou rejeté, la connexion se poursuivra sans erreur et sans que le service de sécurité soit activé.

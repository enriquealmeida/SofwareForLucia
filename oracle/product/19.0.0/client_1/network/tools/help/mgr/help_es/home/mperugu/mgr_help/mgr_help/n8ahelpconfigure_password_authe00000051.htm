Configuración de Autenticación de Contraseña para el Listener

El listener se puede configurar con un sistema de seguridad. De este modo, será necesaria la contraseña para operaciones con privilegios, como guardar los cambios de configuración o detener el listener, que se realicen desde la Utilidad de Control de Listener. El Gestor de Red de Oracle permite definir una contraseña cifrada.

Para definir una contraseña para el listener:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Listeners.

  2. Seleccione un listener.

  3. Seleccione Parámetros Generales de la lista del panel de la derecha.

  4. Seleccione el separador Autenticación.

  5. Haga clic en Pedir una Contraseña para las Operaciones del Listener.

  6. Introduzca la contraseña en el campo Contraseña y vuelva a introducirla en el campo Confirmar Contraseña.

  7. Seleccione Archivo > Guardar Configuración de Red.

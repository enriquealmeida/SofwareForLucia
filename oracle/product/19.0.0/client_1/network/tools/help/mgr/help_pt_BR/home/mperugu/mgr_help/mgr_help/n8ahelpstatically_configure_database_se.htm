Configurar Informações sobre Serviço de Banco de Dados Estaticamente

Para que o listener aceite solicitações clientes para um banco de dados Oracle8 ou Oracle7, o arquivo listener.ora deverá ser configurado com informações sobre o banco de dados.

Se o banco de dados for um banco de dados Oracle9i ou Oracle8i, não serão necessárias informações estáticas sobre serviço no arquivo listener.ora. As informações sobre serviço são registradas no listener automaticamente através do registro de serviço. No entanto, se você estiver usando o Oracle Enterprise Manager, serão necessárias informações estáticas para um banco de dados Oracle8i.

Para configurar informações sobre o serviço de banco de dados para um listener, faça o seguinte:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

  2. Selecione um listener.

  3. Na lista do painel direito, selecione Serviços de Banco de Dados.

  4. Escolha Adicionar Banco de Dados. Uma nova guia Banco de Dados será exibida.

  5. Informe o nome do banco de dados global no campo Nome do Banco de Dados Global, a localização do diretório Oracle home no campoDiretório Oracle e SID (Identificador de Sistema Oracle) do banco de dados no campo SID.

  6. Escolha Arquivo > Salvar Configuração de Rede.

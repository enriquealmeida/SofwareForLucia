Services de base de données

L'onglet Services de base de données vous permet d'indiquer les services de base de données pour lesquels le processus d'écoute doit recevoir des demandes de connexion. Le processus d'écoute a besoin de ces informations pour connecter un client à la base de données.

Les instances Oracle9iet Oracle8 i utilisent l'inscription de service dynamique pour signaler au processus d'écoute leurs services de base de données. Les bases de données Oracle7 et Oracle8 version 8.0 et les outils d'Oracle Enterprise Manager nécessitent la configuration du service statique dans le fichier listener.ora.

Choisissez Ajouter une base de données pour ajouter des informations sur le service de base de données.

Nom global de base de données

Entrez le nom de la base de données et le nom de domaine de la base de données, tels qu'ils figurent dans le paramètre SERVICE_NAMES ou dans les paramètres DB_NAME et DB_DOMAIN du fichier de paramètres d'initialisation.

Répertoire d'origine Oracle Home

Indiquez l'emplacement du répertoire d'origine Oracle Home de la base de données.

Remarque : Windows NT connaît déjà le répertoire d'origine Oracle Home. C'est pourquoi cette option n'est pas nécessaire pour la connexion à un service sur Windows NT.

SID

Indiquez l'identificateur système (SID) Oracle de l'instance.

Bouton Ajouter une base de données

Choisissez cette option pour configurer des informations de service pour une autre base de données. Le nouvel onglet Base de données vous permet de fournir des informations de service pour une autre base de données.

Bouton Supprimer une base de données

Choisissez cette option pour supprimer l'onglet Base de données sélectionné et ses informations de service.

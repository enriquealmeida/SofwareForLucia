Parâmetros de Protocolo

O listener, o servidor Oracle Names e o Oracle Connection Manager são identificados por um endereço de protocolo. Os clientes que desejarem usar esses serviços também deverão ser configurados com o mesmo endereço. A tabela a seguir descreve os parâmetros utilizados pelos protocolos Oracle suportados:

Protocolo Parâmetro Descrição
IPC PROTOCOL Especifique ipc como o valor.
IPC KEY Especifique um nome exclusivo para o serviço. A Oracle Corporation recomenda o uso do nome ou doOracle System Identifier (SID) do serviço.
Named Pipes PROTOCOL Especifique nmp como o valor.
Named Pipes SERVER Especifique o nome do computador do servidor Oracle.
Named Pipes PIPE Especifique o nome do pipe que deverá estabelecer conexão com o servidor de banco de dados. Esse nome pode ser qualquer nome arbitrário.
TCP/IP e TCP/IP com SSL PROTOCOL Especifique tcp como o valor para TCP/IP, ou especifique tcps como o valor para TCP/IP com SSL.
TCP/IP e TCP/IP com SSL HOST Especifique o nome do host ou o endereço IP do computador.
TCP/IP e TCP/IP com SSL PORT Especifique o número da porta de listening.

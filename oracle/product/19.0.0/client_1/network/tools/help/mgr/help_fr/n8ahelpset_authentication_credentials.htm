Définition des éléments d'authentification

Définissez des éléments d'authentification auprès de l'annuaire de manière à effectuer des opérations d'écriture sur un serveur d'annuaire.

Pour définir des éléments d'authentification :

  1. Choisissez Commande > Annuaire > Définir l'authentification. La boîte de dialogue Authentification du serveur d'annuaire apparaît.

  2. Entrez un nom et un mot de passe utilisateur autorisé.

  3. Cliquez sur OK.

Microsoft Active Directory

Un servizio di directory conforme al protocollo LDAP incluso in Windows 2000 Server. Memorizza le informazioni sugli oggetti presenti nella rete e le mette a disposizione degli utenti e degli amministratori. Active Directory consente inoltre agli utenti di accedere alle risorse di rete tramite un unico processo di collegamento.

È possibile configurare Active Directory come metodo di denominazione delle directory per memorizzare le informazioni sul servizio alle quali possono accedere i client.

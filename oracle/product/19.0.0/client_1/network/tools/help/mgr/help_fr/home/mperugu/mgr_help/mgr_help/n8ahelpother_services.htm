Autres services

L'onglet Autres services vous permet d'indiquer les services autres que les services de base de données pour lesquels le processus d'écoute doit recevoir des demandes de connexion. Il peut s'agir de procédures externes ou de services hétérogènes.

Nom de service global

Entrez le nom de service et le nom de domaine du service.

Nom du programme

Entrez le nom du programme exécutable. Par exemple, extproc.exe.

Argument zéro du programme

Entrez le premier argument interne, souvent utilisé comme nom de programme alternatif. Par exemple : argv0

Arguments du programme

Entrez les arguments de ligne de commande devant être transmis lors du démarrage du service. Par exemple, -mode=5.

Environnement

Entrez toutes les variables d'environnement à définir avant le démarrage du service. Entrez les variables d'environnement sous la forme suivante :

variable_environnement=valeurs

Par exemple, user=scott, start=immediate.

Notez que les variables sont séparées par des virgules.

SID

Entrez le nom du programme ou l'identificateur système (SID) Oracle pour le service.

Répertoire d'origine Oracle Home

Entrez l'emplacement de l'exécutable.

Bouton Ajouter un service

Choisissez ce bouton pour configurer des informations d'un service autre qu'un service de base de données. Un nouvel onglet Service vous permet de fournir des informations pour un autre service.

Bouton Supprimer un service

Choisissez ce bouton pour supprimer l'onglet Service sélectionné et les informations de service.

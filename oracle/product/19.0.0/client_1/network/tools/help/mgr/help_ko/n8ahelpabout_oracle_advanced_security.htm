Oracle Advanced Security 정보

Oracle Advanced Security는 Oracle 환경을 위한 포괄적인 보안 기능을 추가합니다. 이 보안 기능은 엔터프라이즈 네트워크를 보호하고 보안을 유지하면서 인터넷으로 그 영역을 확대하는 한편, 네트워크 비밀번호화와 인증 솔루션, Single Sign-On 서비스, 보안 프로토콜 등의 통합 보안을 제공합니다. Oracle Advanced Security는 산업 표준을 통합하고 Oracle 네트워크 및 다른 네트워크에 탁월한 보안 기능을 제공합니다.

Verbindungsdeskriptor für externe Prozeduren konfigurieren

Externe Prozeduren sind Prozeduren, die aus einem anderen Programm aufgerufen werden, jedoch in einer anderen Sprache geschrieben sind. Ein Beispiel hierfür wäre ein PL/SQL-Programm, das eine oder mehrere C-Routinen aufruft, die zur Ausführung besonderer Verarbeitungen erforderlich sind.

Damit sich die Oracle-Datenbank bei einer externen Prozedur anmelden kann, muss der Datenbank-Server mit einem Net Service-Namen konfiguriert sein, um auf externe Prozeduren zuzugreifen.

Hinweis: Der Oracle Net-Konfigurationsassistent erstellt während der Installation automatisch einen Net Service-Name für externe Prozeduren.

So konfigurieren Sie einen Datenbank-Server für externe Prozeduren:

  1. Stellen Sie sicher, dass der Listener für externe Prozeduren konfiguriert ist, wie unter Externe Prozeduren für den Listener konfigurieren beschrieben.

  2. Erstellen Sie einen Net Service-Namen in der Datei tnsnames.ora in der Datenbank. Stellen Sie sicher, dass die Verbindungsdeskriptor-Informationen mit den Informationen übereinstimmen, die für den Listener konfiguriert sind.

  3. Wählen Sie Datei >Netzwerkkonfiguration speichern.

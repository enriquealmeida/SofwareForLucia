“选择新上下文”对话框

“选择新上下文”对话框使您能够设置另一个 Oracle Context，后者是目录信息树 (DIT) 中名为 cn=OracleContext 的容器，其中包含用于 Oracle 软件的条目。这些条目可以包含网络服务名和连接到数据库服务的数据库服务名。默认情况下，Oracle Net Manager 使用由 Oracle Net Configuration Assistant 配置的 Oracle Context。

目录命名上下文

从列表中，选择包含您要查看的 Oracle Context 的目录条目。

Oracle Context

从此列表中选择一个 Oracle Context。您选择的 Oracle Context 应包含目录树中您打算修改或添加其网络信息的条目。

“导航器”窗格中的目录 > 服务命名文件夹随所选的 Oracle Context 的网络对象进行更新。

Prueba de Conectividad

Puede comprobar la conectividad del objeto de nomenclatura de servicio seleccionado a un servicio de base de datos mediante una prueba de conectividad. Durante una prueba de conectividad, se realiza una conexión a un servicio de base de datos con la información del descriptor de conexión.

Para verificar la información de conexión de red:

  1. Asegúrese de que el listener y la base de datos se están ejecutando.

  2. En el panel del navegador, amplíe Configuración de Red de Oracle > Directorio o Local > Nomenclatura de Servicios.

  3. Seleccione un nombre de servicio de red o servicio de base de datos existente.

  4. Seleccione el símbolo de base de datos de la barra de herramientas o seleccione Comando > Servicio de Prueba. Aparece el recuadro de diálogo Prueba de Conexión que se actualizará conforme se generan los resultados de la prueba. Si la prueba se realiza correctamente, aparece el siguiente mensaje:

La prueba de conexión se ha realizado correctamente.

Para obtener información sobre el inicio de los componentes, consulte el capítulo sobre el establecimiento de una conexión y comprobación de la red en Oracle10i Net Administrator'sGuide (Guía del Administrador de Red de Oracle10i).

Parámetros Generales: Registro y Rastreo

El separador Registro y Rastreo permite activar el registro y rastreo para el listener. El Gestor de Red de Oracle permite especificar el nombre y la ubicación en que se escribirán los archivos log del listener.

Registro Desactivado

Seleccione esta opción para desactivar la función de registro en el listener.

Registro Activado

Seleccione esta opción para activar la función de registro en el listener. Una vez activada, puede especificar el lugar en que se escribirá la información de registro.

Archivo Log

Introduzca el nombre de la ruta de acceso del directorio y el nombre del archivo en el que se escribirá la información de log desde el listener. Tenga en cuenta que la extensión del nombre de este archivo siempre es .log. El nombre por defecto es listener.log. El directorio por defecto es $ORACLE_HOME/network/log en UNIX y ORACLE_HOME \network\log en plataformas de Windows.

Rastreo Desactivado

Seleccione esta opción para desactivar la función de rastreo en el listener.

Rastreo Activado

Seleccione esta opción para activar la función de rastreo en el listener. Una vez activada, puede especificar el nivel de rastreo que desea y el lugar en que se escribirá la información de rastreo, cambiando el archivo de rastreo por defecto.

Nivel de Rastreo

En la lista, seleccione el nivel de rastreo:

Off: El rastreo está desactivado. Éste es el valor por defecto.

User: Se activa un nivel de rastreo adecuado para los usuarios. El rastreo identifica las condiciones de error inducidas por los usuarios.

Administrator: Se activa un nivel de rastreo adecuado para los administradores de base de datos. El rastreo identifica los problemas específicos de la instalación.

Support: Se activa un nivel de rastreo adecuado para soporte al cliente.

Los archivos de rastreo pueden llegar a ser muy grandes. Oracle Corporation recomienda mantener el rastreo desactivado cuando no se vayan a solucionar problemas de red.

Archivo de Rastreo

Introduzca el nombre de la ruta de acceso del directorio y el nombre del archivo en el que se escribirá la información de rastreo desde el listener. Tenga en cuenta que la extensión de este archivo siempre es .trc. El nombre por defecto es listener.trc. El directorio por defecto es $ORACLE_HOME/network/trace en UNIX y ORACLE_HOME\network\trace en plataformas de Windows.

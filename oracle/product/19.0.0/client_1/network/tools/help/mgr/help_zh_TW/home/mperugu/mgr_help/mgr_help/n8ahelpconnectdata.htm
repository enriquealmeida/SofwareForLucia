CONNECT_DATA

連線描述區中定義目的地資料庫服務名稱或 Oracle 系統 ID (SID) 的部分。在下面的範例中，SERVICE_NAME 指定的是名為 sales.us.example.com 的資料庫服務：

(DESCRIPTION=

(ADDRESS=(PROTOCOL=tcp)(HOST=sales-pc)(PORT=1521)

(CONNECT_DATA=(SERVICE_NAME=sales.us.example.com)))

CONNECT_DATA 可以設定進階連線選項。

Dirección IP

Se utiliza para identificar el nodo de una red. Cada una de las computadoras de la red tiene asignada una dirección IP única, formada por el identificador de red y un identificador de host único. Esta dirección se representa normalmente con una notación decimal de puntos, con el valor decimal de cada octeto separado por un punto, por ejemplo, 144.45.9.22.

Solicitado

Ative o serviço de segurança se o outro lado o permitir.

Meu lado da conexão especifica que o serviço de segurança é recomendado, mas não é obrigatório. O serviço de segurança será ativado se o outro lado especificar aceito, solicitado ou obrigatório. Deverá haver um algoritmo correspondente disponível do outro lado, caso contrário o serviço não será ativado. Se o outro lado especificar obrigatório e não houver algoritmo correspondente, a conexão falhará.

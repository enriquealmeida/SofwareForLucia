Oracle Advanced Security: 기타 매개변수

기타 매개변수 탭을 사용하여 추가 인증 서비스 매개변수를 구성할 수 있습니다.

인증 서비스 목록에서 인증 방법을 선택합니다. 선택한 인증 방법에 따라 적합한 정보를 필드에 입력합니다.

KERBEROS(v5)

요소 설명
서비스 임의의 문자열을 입력하여 Kerberos 서비스 이름을 지정합니다.
인증서 캐시 파일 적합한 임의의 경로명을 입력하여 Kerberos 인증서 캐시 경로명을 지정합니다.
구성 파일 적합한 임의의 경로명을 입력하여 Kerberos 구성 파일을 지정합니다.
Realm 번역 파일 적합한 임의의 경로명을 입력하여 Realm 번역 파일에 대한 Kerberos 호스트 이름을 지정합니다.
키 테이블 적합한 임의의 경로명을 입력하여 Kerberos 보안 키 파일을 지정합니다.
허용 시간 차 임의의 양의 정수를 입력하여 인증서 전송 및 수신 간의 허용 시간 차(초)를 지정합니다.

CYBERSAFE

GSSAPI 서비스: CYBERSAFE 서비스 주체를 지정합니다. 정확한 형식으로 임의의 서비스 주체 문자열을 입력합니다.

NTS

추가 매개변수가 필요하지 않습니다.

RADIUS

요소 설명
호스트 이름 기본 RADIUS 서버의 이름을 입력합니다.
포트 번호 기본 RADIUS 서버의 포트 번호를 입력합니다.
시간 초과(초) Oracle 데이터베이스 서버가 기본 RADIUS 서버의 응답을 대기해야 하는 시간을 초로 입력합니다.
재시도 횟수 Oracle 데이터베이스 서버가 기본 RADIUS 서버에 메시지를 재전송해야 하는 횟수를 입력합니다.
보안 파일 보안 키 파일의 경로명을 입력합니다.
계정 보내기 ON을 입력하여 계정을 활성화하거나 [OFF]를 입력하여 사용 안함으로 설정합니다.
경쟁 응답 ON을 입력하여 경쟁 응답을 설정하거나 [OFF]를 입력하여 경쟁 응답을 해제합니다.
기본 키워드 RADIUS 서버에서 경쟁을 요청하기 위한 키워드를 입력합니다.
인터페이스 클래스 이름 Oracle 클라이언트와 RADIUS 서버 간의 경쟁 응답 컨버세션을 처리하기 위해 생성한 클래스 이름을 입력합니다.

자세한 내용은 Oracle Advanced Security Administrator's Guide를 참조하십시오.

Lokale Benennung

Eine Benennungsmethode, bei der ein Net Service-Name, der in der Datei tnsnames.ora eines Clients gespeichert ist, in einen Verbindungsdeskriptor aufgelöst wird. Die lokale Benennung eignet sich besonders für einfache verteilte Netzwerke mit einer kleinen Anzahl von Diensten, die sich nicht häufig ändern.

Datei sqlnet.ora

Eine Konfigurationsdatei für den Client oder Server, die Folgendes angibt:

  • Client-Domain, die an nicht vollständige Servicenamen oder Net Service-Namen angehängt werden soll

  • Reihenfolge der Benennungsmethoden, die der Client bei der Auflösung eines Namens verwenden soll

  • Logging- und Tracing-Funktionen

  • Route der Verbindungen

  • Bevorzugte Oracle Names Server

  • Externe Benennungs-Parameter

  • Oracle Advanced Security-Parameter

Die Datei sqlnet.ora muss in einem der folgenden Verzeichnisse gespeichert sein:

  1. Das Verzeichnis, das mit der Umgebungsvariablen TNS_ADMIN angegeben wird. Wenn die Umgebungsvariable TNS_ADMIN nicht als Variable in Windows NT gespeichert ist, kann sie in der Registrierung gespeichert sein.

  2. Das Verzeichnis $ORACLE_HOME/network/admin bei UNIX oder das Verzeichnis ORACLE_HOME\network\admin bei Windows-Betriebssystemen.

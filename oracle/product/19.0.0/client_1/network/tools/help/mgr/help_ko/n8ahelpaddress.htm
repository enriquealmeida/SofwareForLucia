주소

네트워크 객체의 네트워크 프로토콜 주소입니다.

접속에 성공하면 클라이언트와 요청 수신기(리스너, Oracle Names Server 또는 Oracle Connection Manager)가 동일한 프로토콜 주소로 구성됩니다. 클라이언트는 이 주소를 사용하여 특정 네트워크 객체 위치에 접속 요청을 보내고 수신자는 이 주소에서 보내진 요청을 "수신"합니다.

다음 주소는 리스너의 수신 주소를 정의합니다.

ADDRESS=

(PROTOCOL=tcp)

(HOST=sales-server)

(PORT=1521))

주소의 프로토콜 매개변수는 선택한 프로토콜에 따라 다릅니다.

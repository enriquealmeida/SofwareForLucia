Cualificador de Enlace

Cualificador agregado a un enlace de base de datos global para proporcionar valores alternativos para las credenciales de nombre de usuario y de contraseña de la base de datos. Por ejemplo, el cualificador de enlace emp se puede agregar al enlace de base de datos global sales.us.example.com. Un ejemplo de conexión a una base de datos remota con el enlace de base de datos global y el cualificador de enlace sería el siguiente:

CONNECT scott/tiger@sales.example.com@emp

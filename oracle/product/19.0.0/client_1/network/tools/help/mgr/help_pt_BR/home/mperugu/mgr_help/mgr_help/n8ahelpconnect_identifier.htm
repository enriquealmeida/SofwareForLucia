Identificador de Conexão

Um nome de serviço de rede ou o nome real do serviço que realiza o mapeamento para um descritor de conexão. Os clientes utilizam o identificador de conexão em suas strings de conexão, por exemplo:

CONNECT nome do usuário/senha@identificador_conexão

選擇新的相關資訊環境對話方塊

您可以利用「選擇新的相關資訊環境」對話方塊來設定另一個 Oracle Context，此 Oracle Context 是名為 cn=OracleContext 的容器，內含與 Oracle 軟體搭配使用之項目的目錄資訊樹狀結構 (DIT)。這些項目可包括網路服務名稱以及用來連線資料庫服務的資料庫服務名稱。依照預設，Oracle Net Manager 會使用「Oracle Net 組態輔助程式」設定的 Oracle Context。

目錄命名相關資訊環境

從清單中選取包含您要檢視之 Oracle Context 的目錄項目。

Oracle Context

從清單中選取一個 Oracle Context。您選取的 Oracle Context 應包含您要修改或新增網路資訊的目錄項目。

瀏覽窗格中的目錄 > 服務命名資料夾，會隨選取之 Oracle Context 的網路物件一起更新。

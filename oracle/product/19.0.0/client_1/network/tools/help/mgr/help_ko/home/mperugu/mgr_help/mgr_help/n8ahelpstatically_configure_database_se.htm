정적으로 데이터베이스 서비스 정보 구성

Oracle8 또는 Oracle7 데이터베이스에 대한 클라이언트 요청을 리스너가 받아들이도록 하려면 listener.ora 파일이 데이터베이스에 대한 정보로 구성되어야 합니다.

데이터베이스가 Oracle9i 또는 Oracle8i 데이터베이스인 경우 정적 서비스 정보가 listener.ora 파일에 필요하지 않습니다. 서비스 정보는 서비스 등록을 통해 리스너에 자동 등록됩니다. 그러나 Oracle Enterprise Manager를 사용하고 있는 경우 Oracle8i 데이터베이스에 대한 정적 정보가 필요합니다.

리스너에 대해 데이터베이스 정보를 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬> 리스너를 확장합니다.

  2. 리스너를 선택합니다.

  3. 오른쪽 창의 목록에서 데이터베이스 서비스를 선택합니다.

  4. 데이터베이스 추가를 선택합니다. 새 데이터베이스 탭이 나타납니다.

  5. 전역 데이터베이스 이름은 전역 데이터베이스 이름 필드에, Oracle 홈 디렉토리의 위치는 Oracle 홈 디렉토리 필드에, 데이터베이스의 Oracle SID(시스템 식별자)는 SID 필드에 각각 입력합니다.

  6. 파일 > 네트워크 구성 저장을 선택합니다.

Test connettività

È possibile verificare la connettività di un dato oggetto di denominazione dei servizi rispetto a un servizio di database. Nel corso del test viene stabilita una connessione al servizio di database utilizzando le informazioni del descrittore di connessione.

Per verificare le informazioni sulla connettività, procedere come segue.

  1. Assicurarsi che il listener e il database siano in esecuzione.

  2. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Directory o Locale > Denominazione dei servizi.

  3. Scegliere il nome di servizio di rete o del servizio di database esistente.

  4. Scegliere il simbolo del database nella barra degli strumenti oppure scegliere Comando > Esegui test del servizio. Viene visualizzata la finestra di dialogo Test di connessione, che viene aggiornata man mano che vengono generati i risultati. Se l'esito del test è positivo, viene visualizzato il seguente messaggio:

Test della connessione completato.

Per informazioni sull'avvio dei componenti, vedere il capitolo Establishing a Connection and Testing the Network del manuale Oracle10i Net Administrator'sGuide.

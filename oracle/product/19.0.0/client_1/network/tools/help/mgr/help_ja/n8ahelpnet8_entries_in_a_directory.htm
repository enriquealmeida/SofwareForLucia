ディレクトリ内のOracle Netエントリ

ディレクトリ内のエントリは、その識別名(DN)に基づいて階層ツリーに似た構造に配列されます。このディレクトリ・エントリ・ツリーは、ディレクトリ情報ツリー(DIT)と呼ばれます。各エントリには、そのエントリを記述する1つ以上の属性が含まれています。

DIT内のOracle Netエントリには、次のものがあります。

  • データベース作成中に作成されたデータベース・サービス

  • ネット・サービス名

  • ネット・サービス別名

すべてのOracle Netエントリは、Oracleエントリを格納するOracleコンテキストのサブツリーに表示されます。このサブツリーは、サーバー上での「カスタム」インストール中にDIT内でユーザーが指定した位置に作成されます。クライアントは、ディレクトリ使用と、エントリの参照元となるデフォルトのOracleコンテキストで構成されます。

各エントリには、そのエントリを記述する1つ以上の属性が含まれています。各属性は、タイプおよび値を1つずつ持っています。Oracle Netディレクトリ・エントリには、次の属性があります。

  • 一般名

  • エントリのバージョン

  • 接続記述子

Parâmetros Gerais: Geral

A guia Geral permite configurar parâmetros gerais para o listener.

Nome do Listener

Exibe o nome do listener.

Tempo de Espera para Inicialização

Especifique o tempo, em segundos, que o listener deverá aguardar antes de responder a um comando START do utilitário Controle de Listener.

Observação: Esta configuração está obsoleta e não será mais suportada nas próximas releases. Se você precisar desse parâmetro para executar o listener, notifique o Suporte Técnico Oracle.

Salvar Configuração no Encerramento

Escolha para tornar permanentes quaisquer alterações feitas pelo comando SET do utilitário Listener Control.

Informações sobre o Contato SNMP

Informe um texto a ser gravado no arquivo snmp.ora para uso com o Oracle Enterprise Manager. Essas informações serão enviadas a uma conexão SNMP e podem ser uma string qualquer. Isso não afeta o processo de estabelecimento de uma conexão SNMP. Em geral, essas informações consistem em informações de contato para o usuário ou administrador do cliente.

ADMIN_RESTRICTIONS_listener_name

Restringe a administração de run-time do listener.

A definição padrão é off. A definição de ADMIN_RESTRICTIONS_listener_name=on desativa a modificação em run-time dos parâmetros no listener.ora. Ou seja, o listener se recusa a aceitar os comandos SET que alterem seus parâmetros. Para alterar qualquer um dos parâmetros no listener.ora, incluindo o próprio ADMIN_RESTRICTIONS_listener_name, modifique o arquivo listener.ora manualmente e recarregue seus parâmetros (usando o comando RELOAD) para que as novas alterações entrem em vigor, sem que o listener seja interrompido e reinicializado explicitamente.

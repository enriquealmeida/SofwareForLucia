KERBEROS(V5)

KERBEROS (v5)認証方式の場合は、次のパラメータを設定する必要があります。

要素 説明
サービス 認証サービス、KERBEROSの名前を入力します。
資格証明キャッシュ・ファイル Kerberos資格証明キャッシュ・ファイルの完全パス。デフォルトは、オペレーティング・システムに依存します。
構成ファイル Kerberos構成ファイルの完全パス。このファイルには、デフォルトのキー配布センター(KDC)のレルムが含まれ、ホストがKDCホストにマップされています。
レルム変換ファイル Kerberosレルム変換ファイルの完全パス名。このファイルでは、ホスト名またはドメイン名がレルムにマップされています。デフォルトは、オペレーティング・システムに依存します。
キー表 Kerberosプリンシパル/シークレット・キー・マッピング・ファイルの完全パス名。この情報は、Oracleサーバーがキーを抽出しクライアントから受信する認証情報を複合化するために使用されます。デフォルトは、オペレーティング・システムに依存します。
時間誤差 Kerberos資格証明期限切れ後の経過秒数。デフォルトは、300秒です

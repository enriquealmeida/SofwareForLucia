Domain Name System (DNS)

Ein System für das Benennen von Rechnern und Netzwerkdiensten, die in einer Hierarchie mit Domains angeordnet sind. DNS wird in TCP/IP-Netzwerken verwendet, um Rechner mithilfe von benutzerfreundlichen Namen zu suchen. DNS löst einen benutzerfreundlichen Namen in eine IP-Adresse auf, die von Rechnern verstanden wird.

In Oracle Net übersetzt DNS den Hostnamen, der in TCP/IP-Adressen verwendet wird, in eine IP-Adresse.

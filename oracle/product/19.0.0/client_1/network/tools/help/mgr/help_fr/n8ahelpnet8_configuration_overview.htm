Présentation de la configuration d'Oracle Net Services

Oracle Net Manager vous permet de configurer de nombreux éléments du réseau, dont les méthodes de résolution de noms, les profils serveur et client et les processus d'écoute.

Méthodes de résolution de noms

Vous pouvez définir des noms simples pour identifier l'emplacement d'un service, tel qu'une base de données. Ces noms simples, appelés identificateurs de connexion, sont associés à des descripteurs de connexion qui contiennent l'emplacement du réseau et l'identification du service.

Oracle Net Manager vous permet de configurer différentes façons de résoudre les identificateurs de connexion en descripteurs de connexion via des méthodes de résolution de noms.

Profils serveur et client

Vous pouvez définir des préférences pour l'activation et la configuration de fonctionnalités Oracle Net sur les clients et sur le serveur au moyen des profils. Les profils vous permettent de définir l'ordre dans lequel utiliser les différentes méthodes de résolution de noms, d'activer des fonctions d'Oracle Advanced Security, d'activer la journalisation et la trace, et de router des connexions.

Configuration d'un processus d'écoute sur le serveur

Vous pouvez configurer le processus d'écoute pour la réception des connexions client. Le processus d'écoute doit être configuré avec :

  • une ou plusieurs adresses de protocole d'écoute

  • des informations de service sur le service de destination

Les adresses de protocole sont configurées dans le fichier de configuration du processus d'écoute, listener.ora. Des informations sur le service peuvent ou non être configurées dans le fichier listener.ora :

  • Une base de données Oracle8i ou supérieure inscrit automatiquement auprès du processus d'écoute certaines informations, telles que les noms de service et d'instance, les gestionnaires de services et les données de chargement. Cette fonctionnalité, appelée inscription de service, n'exige aucune configuration dans le fichier listener.ora.

  • D'autres services, dont la base de données Oracle7 ou Oracle8, nécessitent une configuration de service dans le fichier listener.ora .

Pour plus d'informations sur Oracle Names, reportez-vous à la présentation d'Oracle Names.

Pour plus d'informations, reportez-vous au chapitre Configuring Naming Methods dans le Guide de l'administrateur Oracle10i Net.

CONNECT_DATA

Teil des Verbindungsdeskriptors, der den Service-Namen oder Oracle System Identifier (SID) der Zieldatenbank definiert. Im folgenden Beispiel definiert SERVICE_NAME einen Datenbankservice mit der Bezeichnung sales.us.example.com:

(DESCRIPTION=

(ADDRESS=(PROTOCOL=tcp)(HOST=sales-pc)(PORT=1521)

(CONNECT_DATA=(SERVICE_NAME=sales.us.example.com)))

CONNECT_DATA kann mit erweiterten Verbindungsinformationen konfiguriert werden.

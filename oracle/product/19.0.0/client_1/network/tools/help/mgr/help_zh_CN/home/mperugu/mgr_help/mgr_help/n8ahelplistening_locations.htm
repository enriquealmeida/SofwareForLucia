监听位置

配置监听程序以在特定的网络协议地址上监听某个数据库的传入连接请求。监听位置选项卡使您能够配置一个或多个监听位置地址，监听程序将在这些地址上等待客户机的连接。

协议

从列表中,选择地址协议。根据所选协议的不同，在域中输入相应的协议参数信息。

“显示高级设置/隐藏高级设置”按钮

“显示高级设置/隐藏高级设置”按钮可进行切换，以显示或隐藏为监听程序指定 I/O 缓冲区空间限制的设置。

TCP/IP、使用 SSL 的 TCP/IP 以及 SDP 协议都支持这些设置。

  • 发送缓冲区大小总计

以字节为单位，指定会话的发送操作的缓冲区空间。

因为数据库服务器向客户机写入数据，所以一般在服务器端进行设置比较合适。如果数据库服务器要接收大量请求，则还要设置接收缓冲区大小总计的值。

  • 接收缓冲区大小总计

以字节为单位，指定会话的接收操作的缓冲区空间。

使该地址专用于 JServer 连接

选择使用用于 Oracle JServer 发行版 8.1 连接的通用 ORB 间协议 (GIOP) 端点来配置 listener.ora 文件。

添加地址

选择此选项以创建新的协议地址。将显示一个新的地址选项卡，在选项卡中可以选择协议，并在域中输入协议信息。

删除地址

选择此选项以删除所选的地址选项卡及其地址信息。

有关配置 I/O 缓冲区空间的详细信息，请参阅 Oracle10i Net Administrator's Guide 中的 Optimizing Performance 一章。

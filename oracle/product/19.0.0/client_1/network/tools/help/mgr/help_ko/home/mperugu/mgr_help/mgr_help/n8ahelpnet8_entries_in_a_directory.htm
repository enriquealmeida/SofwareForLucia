디렉토리의 Oracle Net 항목

디렉토리의 항목은 해당 DN(식별 이름)에 준해 계층 트리 구조로 배열됩니다. 이 디렉토리 항목 트리를 DIT(디렉토리 정보 트리)라고 합니다. 각 항목은 해당 항목을 설명하는 하나 이상의 속성을 포함합니다.

DIT의 Oracle Net 항목은 다음이 될 수 있습니다.

  • 데이터베이스를 생성하는 동안 생성된 데이터베이스 서비스

  • 네트 서비스 이름

  • 네트 서비스 별칭

모든 Oracle Net 항목은 Oracle 항목을 저장하는 Oracle Context 하위 트리에 나타납니다. 이 하위 트리는 서버에서 사용자 정의 설치 중 DIT의 사용자 정의 위치에 생성됩니다. 클라이언트는 디렉토리 및 항목을 조회할 수 있는 기본 Oracle Context를 사용하여 구성됩니다.

각 항목은 해당 항목을 설명하는 하나 이상의 속성을 포함합니다. 각 속성은 유형과 값을 가지고 있습니다. Oracle Net 디렉토리 항목은 다음 속성을 가집니다.

  • 일반 이름

  • 항목 버전

  • 접속 기술자

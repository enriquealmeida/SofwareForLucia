Configurazione di un descrittore di connessione per un database Oracle Rdb

Il database Oracle Rdb è un database per le piattaforme Digital a 64 bit. I client visualizzano i server Oracle Rdb nello stesso modo in cui visualizzano i database Oracle. Un database Oracle Rdb dispone di un proprio listener. La configurazione del client consiste nella creazione di un nome di servizio di rete contenente parametri specifici di Oracle Rdb per l'accesso al database.

Per configurare il client per un database Oracle Rdb:

  1. Creare un nome di servizio di rete, secondo quanto descritto in Creazione di un nome di servizio di rete.

  2. Nella casella Identificazione servizio presente nel riquadro di destra, scegliere Avanzate. Viene visualizzata la finestra di dialogo Opzioni di servizio avanzate.

  3. Immettere il nome del file di un database Oracle Rdb nel campo Database RDB.

  4. Facoltativamente, immettere il nome del database globale nel campo Nome di database globale e, se necessario, il tipo di servizio nel campo Tipo di servizio.

  5. Nella finestra di dialogo Opzioni di servizio avanzate, scegliere OK.

  6. Nel riquadro di destra, scegliere Applica.

  7. Scegliere File > Salva configurazione di rete.

Per ulteriori informazioni, vedere la documentazione di Oracle Rdb.

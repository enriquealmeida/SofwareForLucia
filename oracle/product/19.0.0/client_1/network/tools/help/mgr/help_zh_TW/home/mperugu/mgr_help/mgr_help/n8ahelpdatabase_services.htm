資料庫服務

您可以使用資料庫服務頁籤來指定監聽器必須接收連線要求的資料庫服務。監聽器必須有這項資訊，才能將從屬端連線至資料庫。

Oracle9i 與 Oracle8i 執行處理均使用動態服務註冊，來通知監聽器其資料庫服務的相關資訊。Oracle7 與 Oracle8 版本 8.0 資料庫以及 Oracle Enterprise Manager 工具則需要在listener.ora檔案中設定靜態服務組態。

選擇新增資料庫，即可新增資料庫服務資訊。

全域資料庫名稱

根據初始化參數檔中提供的 SERVICE_NAMES 參數值或 DB_NAME 與 DB_DOMAIN 參數值，輸入資料庫名稱和資料庫網域名稱。

Oracle 本位目錄

輸入資料庫的 Oracle 本位目錄位置。

注意：Windows NT 已經可以識別 Oracle 本位目錄。因此連線至 Windows NT 上的服務時不需要此欄位。

SID

輸入執行處理的 Oracle 系統 ID (SID)。

新增資料庫按鈕

選擇此按鈕可設定另一個資料庫的服務資訊。新的資料庫頁籤就會顯示，讓您輸入另一個資料庫的服務資訊。

移除資料庫按鈕

選擇此按鈕可刪除選取的資料庫頁籤及其服務資訊。

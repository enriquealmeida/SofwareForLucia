Cuadro de Diálogo Autenticación del Servidor de Directorios

El recuadro de diálogo Autenticación del Servidor de Directorios permite definir credenciales de autenticación para el Contexto de Oracle seleccionado en el servidor de directorios.

El Gestor de Red de Oracle debe estar autenticado con el servidor de directorios para realizar cualquier operación en este servidor, como mostrar, agregar o modificar entradas de Red de Oracle en el directorio.

Usuario

Introduzca un nombre de usuario autorizado que tenga acceso al directorio.

Si desea agregar o modificar entradas, introduzca un usuario administrador de directorios al que se le hayan proporcionado privilegios para agregar y modificar bajo la forma de un nombre distinguido (DN). Por ejemplo, c=US, o=EXAMPLE, ou=admin, cn=scott es el DN de un administrador llamado Scott.

Contraseña

Introduzca la contraseña para el usuario.

Nota: Si sólo desea ver objetos y el directorio se ha configurado con privilegios "anónimos", deje los campos Usuario y Contraseña en blanco.

Por defecto, el Gestor de Red de Oracle intenta realizar la autenticación con el directorio utilizando uno de los siguientes métodos:

  • Autenticación Nativa: Se utiliza el nombre de usuario y la contraseña del sistema operativo Windows NT.

  • Autenticación Completa: Se utiliza un certificado SSL (Secure Sockets Layer).

Numéros de port recommandés

Pour les adresses de protocole TCP/IP et TCP/IP avec SSL, Oracle conseille d'utiliser les numéros de port suivants :

Port Description
1521 Processus d'écoute par défaut pour les connexions client au processus d'écoute. Dans les versions futures, ce numéro de port deviendra probablement le numéro de port officiel 2483 pour TCP/IP et 2484 pour TCP/IP avec SSL.
1575 Port d'écoute par défaut agréé pour les connexions client à un serveur Oracle Names
1630 Port d'écoute par défaut et agréé pour les connexions client à Oracle Connection Manager
1830 Port d'écoute par défaut et agréé pour les commandes administratives destinées à Oracle Connection Manager.

Pour plus d'informations, reportez-vous à l'annexe Integrating Authentication Devices Using RADIUS dans le Guide de l'administrateur Oracle Advanced Security

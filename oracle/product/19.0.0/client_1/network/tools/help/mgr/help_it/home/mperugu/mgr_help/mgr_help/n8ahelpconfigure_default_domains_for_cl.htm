Configurazione dei domini predefiniti per i client

Negli ambienti in cui il client esegue frequenti richieste di nomi a un dominio specifico, è opportuno impostare un dominio predefinito. Dopo essere stato impostato nel profilo, il dominio predefinito viene automaticamente aggiunto ai nomi di servizi di rete o ai nomi dei servizi non qualificati. Ad esempio, se il dominio predefinito è impostato su us.example.com, sales viene cercato come sales.us.example.com nella stringa di connessione riportata di seguito:

CONNECT scott/tiger@sales

Per specificare un dominio predefinito, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Denominazione.

  3. Selezionare la scheda Metodi.

  4. Immettere il dominio nel campo Dominio predefinito.

  5. Scegliere File > Salva configurazione di rete.

Oracle Connection Manager

Ruta por la que se puede enviar una solicitud de conexión del cliente al próximo salto o directamente al servidor de la base de datos. Los clientes que dirigen sus solicitudes de conexión mediante Oracle Connection Manager pueden utilizar las funciones de control de acceso, multiplexación y conversión de protocolo configuradas en Oracle Connection Manager.

Oracle Connection proporciona direccionamientos de conexión para los entornos donde:

  • Un gran número de usuarios necesita acceso a un único servicio mediante un solo protocolo.

  • Hay diferentes protocolos instalados en el cliente y el servidor, produciendo un fallo de conexión normal.

  • Sea necesario que el cliente y el servidor tengan instalado el mismo protocolo para realizar las conexiones típicas de la Red de Oracle.

  • Sea necesario el control del acceso de clientes a servidores designados en un entorno TCP/IP.

Para obtener información sobre Oracle Connection Manager, consulte el capítulo sobre los retos de las redes en la era de Internet, en Oracle9i Net Services Administrator's Guide (Guía del Administrador de Servicios de Red de Oracle9i).

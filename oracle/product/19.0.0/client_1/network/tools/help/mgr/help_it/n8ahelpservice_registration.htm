Registrazione del servizio

Una funzione che consente alle istanze e ai dispatcher di registrare automaticamente le informazioni presso un listener. Per questo motivo, non è necessario configurare il file listener.ora con queste informazioni statiche.

Durante l'avvio del database, la registrazione del servizio fornisce al listener le seguenti informazioni:

  • i nomi dei servizi per ogni istanza del database in esecuzione;

  • i nomi delle istanze del database;

  • gli handler dei servizi (dispatcher e server dedicati) disponibili per ciascuna istanza;

  • informazioni su dispatcher, istanza e caricamento del nodo.

In questo modo il listener può indirizzare una richiesta del client in modo adeguato.

Le informazioni sul caricamento consentono al listener di determinare qual è il dispatcher ideale per gestire una richiesta di connessione del client. Se tutti i dispatcher sono bloccati, il listener può generare dinamicamente un server dedicato per la connessione.

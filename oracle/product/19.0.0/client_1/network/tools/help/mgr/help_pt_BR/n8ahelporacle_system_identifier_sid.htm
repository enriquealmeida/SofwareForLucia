Oracle System Identifier (SID)

Um nome que identifica uma instância específica de um banco de dados anterior ao Oracle8i. Para qualquer banco de dados, existe pelo menos uma instância fazendo referência ao banco de dados.

Nos bancos de dados anteriores ao Oracle8i, o SID era usado para identificar o banco de dados. O SID era incluído na parte do descritor de conexão em um arquivo tnsnames.ora e na definição do listener no arquivo listener.ora. Os descritores de conexão usados pelos clientes para estabelecer conexão com um banco de dados Oracle8 ou Oracle7 devem ser configurados com o parâmetro SID.

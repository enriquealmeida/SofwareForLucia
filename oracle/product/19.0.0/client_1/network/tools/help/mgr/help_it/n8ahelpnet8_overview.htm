Panoramica su Oracle Net Services

Oracle Net Services fornisce soluzioni di connettività a livello aziendale in ambienti informatizzati eterogenei e distribuiti. Oracle Net Services semplifica la complessità della configurazione e della gestione della rete, ottimizzando le prestazioni e migliorando le funzionalità diagnostiche inerenti ai problemi della rete.

Oracle Net, un componente di Oracle Net Services, consente di stabilire una sessione di rete dall'applicazione client a un database server Oracle. Una volta stabilita la connessione, Oracle Net funge da tramite nel flusso di dati tra l'applicazione client e il server del database. Oracle Net stabilisce e mantiene la connessione e consente lo scambio di messaggi tra l'applicazione client e il server del database. Oracle Net è in grado di eseguire queste attività perché è installato su ogni computer della rete.

Le sessioni di rete vengono stabilite con l'ausilio di un listener. Il listener è un processo separato che risiede sul database server. Il listener riceve le richieste di connessione client in entrata e gestisce il traffico di queste richieste in uscita, verso il server.

Il listener media la richiesta del client, trasferendola al server. Ogni volta che un client richiede una sessione di rete con un server, un listener riceve la richiesta effettiva. Se le informazioni del client corrispondono a quelle del listener, quest'ultimo assegna una connessione al server.

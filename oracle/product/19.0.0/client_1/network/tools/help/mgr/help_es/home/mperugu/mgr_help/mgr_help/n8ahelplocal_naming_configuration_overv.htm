Visión General de la Configuración de la Nomenclatura Local

En este tema, se incluyen las siguientes secciones.

Instalación Administrador o Tiempo de Ejecución

Si no se ha configurado el uso de directorios, el Asistente de Configuración de Red de Oracle le pedirá que configure nombres de servicio de red en el archivo tnsnames.ora para una instalación Administrador o Tiempo de Ejecución del cliente.

Instalación Personalizada

Si se ha seleccionado la nomenclatura local como método de nomenclatura durante la instalación Personalizada del cliente o del servidor, el Asistente de Configuración de Red de Oracle le pedirá que configure nombres de servicio de red en el archivo tnsnames.ora .

Configuración con el Gestor de Red de Oracle

El Gestor de Red de Oracle permite configurar nombres de servicio de red, así como:

  • Crear nombres de servicio de red para servicios de base de datos que no sean de Oracle.

  • Configurar varias rutas a un listener mediante diferentes direcciones de protocolo.

  • Configurar información avanzada de CONNECT_DATA para un nombre de servicio de base de datos o de red.

Para acceder a la configuración de nomenclatura local:

En el panel del navegador, seleccione la carpeta Local > Nomenclatura de Servicios para crear nombres de servicio de red o modificar la información del descriptor de conexión.

Para obtener más información, consulte el capítulo sobre configuración de métodos de nomenclatura, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

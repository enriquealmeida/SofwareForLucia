Configurazione di un descrittore per le connessioni a Oracle Connection Manager

Per instradare i client al database server mediante Oracle Connection Manager, configurare il file tnsnames.ora o il server delle directory con un descrittore di connessione che specifichi l'indirizzo di protocollo di Oracle Connection Manager. Tale indirizzo consente al client di connettersi al computer Oracle Connection Manager.

Per configurare un indirizzo di protocollo per Oracle Connection Manager, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Directory o Locale > Denominazione dei servizi.

  2. Fare clic sul segno più (+) nella barra degli strumenti oppure scegliere Modifica > Crea nella barra dei menu. Viene visualizzata la pagina Benvenuti della Creazione guidata dei nomi di servizi di rete.

  3. Immettere un nome nel campo Nome servizio di rete.

  4. Scegliere Successivo. Viene visualizzata la pagina Protocollo.

  5. Selezionare il protocollo di ascolto per il quale è configurato Oracle Connection Manager. Per impostazione predefinita, è selezionato il protocollo TCP/IP.

  6. Scegliere Successivo. Viene visualizzata la pagina Impostazioni protocollo.

  7. Immettere i parametri di protocollo appropriati per il protocollo scelto negli appositi campi. Se si utilizza il protocollo TCP/IP, la porta predefinita è 1630.

  8. Scegliere Successivo. Viene visualizzata la pagina Servizio.

  9. Se il servizio di destinazione è Oracle8i o successivo, selezionare Oracle8i o successivo e immettere un nome di servizio nel campo Nome servizio. Se il servizio di destinazione è Oracle8 o Oracle 7, selezionare Oracle8 o precedente e immettere un SID (ID di sistema) Oracle nel campo SID database.

  10. Scegliere Successivo. Viene visualizzata la pagina Servizio.

  11. Fare clic su Fine per salvare la configurazione e uscire dalla Creazione guidata dei nomi di servizi di rete.

  12. Scegliere File > Salva configurazione di rete.

  13. Il nuovo nome di servizio di rete viene aggiunto alla cartella Denominazione dei servizi.

Per ulteriori informazioni sull'abilitazione completa delle funzioni di Oracle Connection Manager, vedere il capitolo Configuring and Administering Oracle Connection Manager del manuale Oracle10i Net Administrator's Guide.

Oracle Advanced Security：SSL

您可以使用 SSL 頁籤來修改「安全通訊端層 (SSL)」設定值。SSL 是一種用於保護網路通訊安全的產業標準協定。SSL 提供認證、加密以及資料完整性。使用 SSL 可確保任何從屬端與任何伺服器之間的安全通訊。特別是，您可以使用 SSL 認證一或多個 Oracle 伺服器的任何從屬端或伺服器，或認證任何從屬端的 Oracle 伺服器。

設定 SSL

從清單選取以指定從屬端或伺服器的設定值。

伺服器所需設定的設定值與從屬端的設定值類似。只多了一個參數：「需要從屬端認證」核取方塊。

Cipher Suite 組態

預設已經安裝了多種 SSL Cipher Suite。如果您手動新增一或多個 Cipher Suite，將會覆寫這些預設 Cipher Suite。

元素 描述
「新增」按鈕 選擇此按鈕可以呼叫「選取要啟用的 Cipher Suite」對話方塊。請在對話方塊中選取一個 Suite，然後選擇「確定」。Cipher Suite 就會新增至清單方塊。注意：所有 Oracle Advanced Security 加密演算法以及金鑰長度均可供美國本土和國際使用。
「移除」按鈕 選擇此按鈕可以移除選取的 Cipher Suite。
「升高層次」按鈕 選擇此按鈕可以將選取的 Cipher Suite 移至清單中較高的層次。
「降低層次」按鈕 選擇此按鈕可以將選取的 Cipher Suite 移至清單中較低的層次。

撤銷檢查 (僅適用於伺服器)

指定撤銷憑證的檢查。請選取下列其中一個值：

  • 無：選取此選項可關閉憑證撤銷檢查。

  • 必要：選取此選項可以在有憑證時，執行憑證撤銷。如果憑證被撤銷而且找不到適當的「憑證撤銷清單 (CRL)」，就會拒絕 SSL 連線。如果找不到適當的 CRL 來確定憑證的撤銷狀態，而且憑證未被撤銷，就會接受 SSL 連線。

  • 要求：選取此選項可以在有 CRL 時，執行憑證撤銷。如果憑證被撤銷，就會拒絕 SSL。如果找不到適當的 CRL 來判斷憑證的撤銷狀態，而且憑證未被撤銷，就會接受 SSL 連線。

需要 SSL 版本 (選擇性)

從清單選取 SSL 的版本。從屬端與伺服器必須使用相容的 SSL 版本。您可以選取 SSL v3.0 或選擇可以使用任何現有或以後的 SSL 版本。

需要從屬端認證 (僅適用於伺服器)

依照預設會選取此核取方塊。如果您不需要從屬端認證，請取消選取此核取方塊。

符合伺服器 X.509 名稱 (僅適用於從屬端)

從清單中選取是否檢查伺服器的辨別名稱 (DN) 與其服務名稱是否相符。如果您強制相符驗證，SSL 會確定憑證是源自伺服器。如果您選取不強制相符驗證，SSL 會執行檢查，但無論是否相符都允許連線。未強制相符會讓伺服器有假造其 ID 的機會。請選取以下其中一個值：

  • 是：選取此選項可以檢查伺服器 DN。如果 DN 符合服務名稱，連線便會成功。如果 DN 不符合服務名稱，連線會成功，但會在sqlnet.log檔案中記錄錯誤。

  • 否：選取此選項便不會檢查伺服器 DN。略過此檢查會讓伺服器假造其 ID。

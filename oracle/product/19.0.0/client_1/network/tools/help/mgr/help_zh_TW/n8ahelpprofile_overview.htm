設定檔簡介

設定檔會指定啟用與設定從屬端或伺服器上的 Oracle Net 功能偏好設定。您可以使用設定檔執行下列動作：

  • 指定要附加至不完整名稱的從屬端網域

  • 排定命名方法的優先順序

  • 啟用記錄日誌與追蹤功能

  • 透過特定處理作業來遞送連線

  • 設定外部命名的參數

  • 設定 Oracle Advanced Security

若要存取設定檔，請從瀏覽窗格，展開「Oracle Net 組態 > 區域 > 設定檔」。

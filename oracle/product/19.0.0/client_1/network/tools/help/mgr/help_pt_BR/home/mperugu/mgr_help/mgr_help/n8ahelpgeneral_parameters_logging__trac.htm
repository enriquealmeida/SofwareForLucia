Parâmetros Gerais: Log e Rastreamento

A guia Log & Rastreamento permite a ativação de recursos de log e rastreamento para o listener. O Oracle Net Manager permite que você especifique o nome e a localização em que os arquivos de log do listener serão gravados.

Log Desativado

Selecione para desativar o log no listener.

Log Ativado

Selecione para ativar o log no listener. Uma vez que essa opção esteja ativada, você poderá especificar onde as informações de log serão gravadas.

Arquivo de Log

Informe o caminho de diretório e o nome do arquivo em que as informações de log do listener serão gravadas. Observe que a extensão de nome para esse arquivo sempre será .log. O nome padrão é listener.log. O diretório padrão é $ORACLE_HOME/network/log em plataformas UNIX e ORACLE_HOME \network\log em plataformas Windows.

Rastreamento Desativado

Selecione para desativar o rastreamento no listener.

Rastreamento Ativado

Selecione para ativar o rastreamento no listener. Uma vez que essa opção esteja ativada, você poderá especificar o nível de rastreamento desejado e onde as informações de rastreamento serão gravadas alterando o arquivo de rastreamento default.

Nível de Rastreamento

A partir da lista, selecione o nível de rastreamento:

Desativado-O rastreamento não está ativado. DESATIVADO é a definição default.

Usuário--O rastreamento é definido em um nível apropriado para os usuários. Os rastreamentos identificam condições de erro induzidas pelo usuário.

Administrador--O rastreamento é definido como um nível apropriado para os administradores de banco de dados. Os rastreamentos identificam problemas específicos de instalação.

Suporte--O rastreamento é definido como um nível adequado para suporte ao cliente.

Os arquivos de log podem se tornar muito grandes. A Oracle Corporation recomenda que você mantenha o recurso de rastreamento desativado quando não estiver diagnosticando problemas de rede.

Arquivo de Rastreamento

Informe o caminho de diretório e o nome do arquivo em que as informações de rastreamento do listener serão gravadas. Observe que a extensão de nome para esse arquivo sempre será .trc. O nome default é listener.trc. O diretório padrão é $ORACLE_HOME/network/trace em plataformas UNIX e ORACLE_HOME\network\trace em plataformas Windows.

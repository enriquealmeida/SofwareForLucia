Configurazione di un descrittore di connessione per Heterogeneous Services

Heterogeneous Services consente di accedere a sistemi non Oracle dal server Oracle. Per stabilire una connessione a un sistema non Oracle, il server Oracle avvia un processo agente mediante il listener sul gateway. Per consentire la connessione di un database Oracle a un sistema non Oracle, è necessario configurare il server con un nome di servizio di rete.

Per configurare il server per Heterogeneous Services:

  1. Assicurarsi che il listener sia configurato per Heterogeneous Services, secondo quanto descritto in Configurazione di Heterogeneous Services per il listener.

  2. Creare un nome di servizio di rete nel file tnsnames.ora per poterlo utilizzare per le connessioni dal database server Oracle a un sistema non Oracle. Assicurarsi che le informazioni del descrittore di connessione corrispondano a quelle configurate per il listener.

  3. Selezionare il nome di servizio di rete nella cartella Locale > Denominazione dei servizi.

  4. Nella casella Identificazione servizio presente nel riquadro di destra, scegliere Avanzate. Viene visualizzata la finestra di dialogo Opzioni di servizio avanzate.

  5. Scegliere Utilizza per Heterogeneous Services, quindi scegliere OK.

  6. Nel riquadro di destra, scegliere Applica.

  7. Scegliere File > Salva configurazione di rete.

Microsoft Active Directory

Windows 2000 Server에 포함된 LDAP 준수 디렉토리 서비스로 NDS는 네트워크 상의 객체에 대한 정보를 저장하여 사용자와 네트워크 관리자가 사용할 수 있도록 합니다. 또한 Microsoft Active Directory는 단일 로그온 프로세스를 사용하여 네트워크의 자원에 대한 사용자 액세스를 제공합니다.

Microsoft Active Directory는 클라이언트가 액세스할 수 있는 서비스 정보를 저장하는 디렉토리 이름 지정 방법으로 구성할 수 있습니다.

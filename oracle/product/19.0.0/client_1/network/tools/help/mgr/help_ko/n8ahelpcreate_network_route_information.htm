데이터베이스 서비스에 대한 네트워크 경로 지정 정보 수정

주: OracleNetAdmins 또는 OracleContextAdmins 그룹의 구성원인 사용자만 디렉토리에서 데이터베이스 서비스에 대한 네트워크 정보를 수정할 수 있습니다.

데이터베이스를 디렉토리에 등록하면 Database Configuration Assistant는 디렉토리에 데이터베이스 서비스 항목을 생성합니다. 기본적으로 이 항목에는 프로토콜 주소를 통한 리스너의 위치를 포함하여 해당하는 네트워크 경로 정보가 포함됩니다. 네트워크 경로 정보는 제거되었을 경우 재생성할 수도 있고 기존의 정보를 수정할 수도 있습니다.

데이터베이스 서비스에 대한 네트워크 경로 정보를 생성 또는 수정하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 디렉토리 > 서비스 이름 지정을 확장합니다.

  2. 데이터베이스 서비스를 선택합니다. 오른쪽 창에 현재 대상 서비스 이름이 표시됩니다.

  3. 주소 구성 상자에서 플러스 기호(+)를 선택합니다. 새 주소 탭이 나타납니다.

  4. 프로토콜을 선택하고 선택한 프로토콜에 해당하는 프로토콜 매개변수 정보를 입력합니다.

  5. 오른쪽 창에서 적용을 선택합니다.

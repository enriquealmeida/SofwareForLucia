Erkennung

Mit Erkennung wird die Suche nach Oracle Names Servern bezeichnet. Während der Erkennung wird ein Oracle Names Server gefunden. Nachdem der erste Oracle Names Server gefunden wurde, überprüft der gefundene Oracle Names Server alle anderen Oracle Names Server in der Admin-Region. Danach wird eine Liste mit Oracle Names Servern erstellt und in .sdns.ora unter UNIX und sdns.ora unter Windows-Plattformen gespeichert. Diese Liste wird in der Reihenfolge der Antwortzeit sortiert.

Bei der Erkennung wird nach dem ersten Oracle Names Server in folgender Reihenfolge gesucht:

  1. Ein bevorzugter Oracle Names Server in der Datei sqlnet.ora

  2. Ein bekannter Oracle Names Server

  3. Ein lokaler Oracle Names Server, der mit TCP/IP auf Port 1575 konfiguriert ist

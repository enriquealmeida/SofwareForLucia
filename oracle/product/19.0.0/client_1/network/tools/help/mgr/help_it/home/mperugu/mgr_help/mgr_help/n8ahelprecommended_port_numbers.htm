Numeri di porta consigliati

Oracle Corporation consiglia di utilizzare i numeri di porta per gli indirizzi di protocollo TCP/IP e TCP/IP con SSL riportati di seguito:

Porta Descrizione
1521 Porta di ascolto predefinita per le connessioni client al listener. Nelle release future questo numero di porta potrebbe essere modificato nel numero di porta ufficialmente registrato 2483 per TCP/IP e 2484 per TCP/IP con SSL.
1575 Porta di ascolto predefinita e ufficialmente registrata per le connessioni client a un server Oracle Names.
1630 Porta di ascolto predefinita e ufficialmente registrata per le connessioni client a Oracle Connection Manager.
1830 Porta di ascolto predefinita e ufficialmente registrata per comandi amministrativi a Oracle Connection Manager.

Per ulteriori informazioni, vedere l'appendice Integrating Authentication Devices Using RADIUS del manuale Oracle Advanced Security Administrator's Guide.

Présentation de la configuration de résolution de noms d'annuaire

Cette rubrique aborde les détails de configuration de résolution de noms d'annuaire.

Configuration du serveur pendant l'installation

Au cours d'une installation de serveur personnalisée, l'assistant Configuration Oracle Net vous invite à effectuer différentes opérations :

  • Sélectionnez le type de serveur d'annuaire.

  • Indiquez l'emplacement du répertoire.

  • Sélectionnez ou créez un contexte Oracle à partir duquel le serveur peut consulter, créer et modifier les identificateurs de connexion.

  • Configurez la résolution de noms d'annuaire comme méthode de résolution de noms à utiliser de préférence.

Si l'utilisation de l'annuaire est bien configurée, l'assistant Configuration de base de données tente de se connecter à l'annuaire et d'y inscrire une entrée de service de base de données. Cette entrée est configurée via un descripteur de connexion et contient des informations sur :

  • l'adresse du processus d'écoute

  • le nom du service de base de données

Configuration de client pendant l'installation

A chaque installation de client, l'assistant Configuration Oracle Net vous demande de configurer différents paramètres d'utilisation de l'annuaire :

  • Indiquez le type de répertoire

  • Indiquez l'emplacement du répertoire.

  • Sélectionnez un contexte Oracle par défaut à partir duquel le client pourra consulter les identificateurs de connexion.

  • Résolution de noms d'annuaire comme méthode de résolution de noms à utiliser de préférence.

Configuration à l'aide de l'assistant Configuration Oracle Net après l'installation

L'assistant Configuration Oracle Net peut également s'exécuter en mode autonome pour :

  • Configurer le client ou le serveur (comme décrit précédemment).

  • Créer le schéma Oracle.

  • Créer un contexte Oracle.

Pour lancer l'assistant Configuration Oracle Net en mode autonome :

  • Sous UNIX, exécutez netca dans $ORACLE_HOME/bin.

  • Sous Windows NT, choisissez Démarrer > Programmes > Oracle - HOME_NAME > Administration réseau > Assistant Configuration Oracle Net.

Configuration à l'aide d'Oracle Net Manager

Oracle Net Manager vous permet d'effectuer les opérations suivantes :

  • Changer le contexte Oracle

  • Changer les éléments d'authentification.

  • Créer des noms de service réseau

  • Créer des alias de service réseau

  • Configurer plusieurs chemins d'accès à un processus d'écoute par des adresses de protocole multiples.

  • Configurer des informations CONNECT_DATA avancées pour un nom de service réseau ou un service de base de données.

  • Migrer des entrées de nom de service réseau d'un fichier tnsnames.ora vers un serveur d'annuaire.

Pour accéder à la configuration de la résolution de noms d'annuaire, dans le panneau du navigateur, choisissez Résolution de noms d'annuaire.

Pour plus d'informations, reportez-vous à l'aide de l'assistant Configuration Oracle Net.

Esportazione delle voci del file tnsnames.ora in un server delle directory

I nomi di servizi odi rete già configurati in un file tnsnames.ora possono essere esportati in un server delle directory. L'esportazione viene eseguita per un solo dominio alla volta.

L'esportazione di un file tnsnames.ora implica la creazione di un nome di servizio di rete nel contesto Oracle correntemente selezionato nella directory. Se è necessario modificare il contesto Oracle, scegliere Comando > Server delle directory > Modifica contesto Oracle.

Per esportare i nomi di servizi di rete memorizzati in un file tnsnames.ora in un server delle directory, effettuare le operazioni riportate di seguito.

  1. Scegliere Comando > Directory > Esporta nomi di servizi. Viene avviata la Migrazione guidata del server delle directory.

  2. Scegliere Successivo.

    Se sono stati rilevati più domini, viene visualizzata la pagina Seleziona dominio. Continuare con il passo 3.

    Se i nomi di servizi di rete non sono qualificati con un dominio, viene visualizzata la pagina Seleziona nomi di servizi di rete. Andare al passo 4.

  3. Selezionare il dominio di rete cui appartengono i nomi di servizi di rete da esportare e scegliere Successivo. Viene visualizzata la pagina Seleziona nomi di servizi di rete.

  4. Dalla lista selezionare i nomi di servizi di rete da esportare, quindi scegliere Successivo. Viene visualizzata la pagina Seleziona contesto di destinazione.

    Nella pagina Seleziona contesto di destinazione, effettuare le operazioni riportate di seguito.

    a. Dalla lista Contesto di denominazione delle directory, selezionare il contesto di denominazione delle directory che contiene il contesto Oracle. Il contesto di denominazione delle directory fa parte di una struttura secondaria di directory che comprende uno o più contesti Oracle, contenenti voci Oracle.

    b. Dalla lista Contesto Oracle, selezionare il contesto Oracle in cui si desidera esportare i nomi di servizi di rete selezionati.

    c. Scegliere Successivo. Viene visualizzata la pagina Aggiornamento del server delle directory, con lo stato dell'operazione di esportazione.

  5. Scegliere Fine per uscire dalla Migrazione guidata del server delle directory.

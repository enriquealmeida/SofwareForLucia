Nom de service réseau

Nom simple de service converti en descripteur de connexion. Les utilisateurs lancent une demande de connexion en transmettant un nom utilisateur et un mot de passe accompagnés d'un nom de service réseau dans une chaîne de connexion pour le service auquel ils souhaitent se connecter :

CONNECT NomUtilisateur/MotDePasse@nom_service_réseau

En fonction de vos besoins, les noms de service réseau peuvent être stockés à différents emplacements :

  • Fichier de configuration local (fichier tnsnames.ora) sur chaque client

  • Serveur d'annuaire

  • Serveur Oracle Names

  • Service de résolution de noms externes, y compris NIS (Network Information Service)

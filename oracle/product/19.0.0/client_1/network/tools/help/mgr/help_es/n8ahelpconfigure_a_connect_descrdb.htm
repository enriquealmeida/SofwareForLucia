Configuración de un Descriptor de Conexión para Bases de Datos Oracle Rdb

Base de datos para una plataforma de 64 bits de Digital. Los servidores de Oracle Rdb se muestran de la misma forma a los clientes que una base de datos Oracle. Las bases de datos Oracle Rdb tienen sus propios listeners. La configuración del cliente consiste en la creación de un nombre de servicio de red con parámetros Rdb concretos de Oracle para acceder a la base de datos.

Para configurar el cliente para una base de datos Rdb de Oracle:

  1. Cree un nombre de servicio de red, tal y como se describe en Creación de un Nombre del Servicio de Red.

  2. En el recuadro Identificación de Servicios del panel de la derecha, seleccione Avanzado. Aparece el recuadro de diálogo Opciones de Servicio Avanzadas.

  3. Introduzca el nombre de archivo de una base de datos Oracle Rdb en el campo Base de Datos Rdb.

  4. También puede introducir el nombre de la base de datos global en el campo Nombre de la Base de Datos Global y, si es necesario, el tipo de servicio en el campo Tipo de Servicio.

  5. En el recuadro de diálogo Opciones de Servicio Avanzadas, haga clic en Aceptar.

  6. Haga clic en Aplicar en el panel de la derecha.

  7. Seleccione Archivo > Guardar Configuración de Red.

Para obtener más información, consulte la documentación de Oracle Rdb.

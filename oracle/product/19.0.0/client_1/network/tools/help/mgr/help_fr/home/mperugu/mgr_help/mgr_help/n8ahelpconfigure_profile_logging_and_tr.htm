Configuration de la journalisation et de la trace des profils

Le client peut être configuré pour la journalisation et la trace.

En général, les informations de journalisation sont stockées dans le fichier sqlnet.log disponible dans $ORACLE_HOME/network/log sous UNIX et dans ORACLE_HOME\network\log sous Windows.

Les informations de trace sont stockées par défaut dans sqlnet.trc pour le client et dans svr_pid.trc pour le serveur, situés dans $ORACLE_HOME/network/trace sous UNIX, et dans ORACLE_HOME\network\trace sous Windows.

Pour activer la journalisation ou la trace :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Général.

  3. Sélectionnez l'onglet Journalisation ouSuivi.

  4. Indiquez les paramètres de journalisation ou de trace.

  5. Choisissez Fichier > Enregistrer la configuration réseau.

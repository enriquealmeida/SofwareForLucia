接続性のテスト

接続テストを実行することで、選択したサービス名オブジェクトの、データベース・サービスに対する接続を検証できます。接続テスト中には、構成済の接続記述子情報を使用してデータベース・サービスへの接続が確立されます。

ネットワーク接続情報を検証する手順は、次のとおりです。

  1. リスナーとデータベースが稼働していることを確認します。

  2. ナビゲータ・ペインで、「Oracle Netの構成」→「ディレクトリ」または「ローカル」→「サービス・ネーミング」を展開します。

  3. 既存のネット・サービス名またはデータベース・サービスを選択します。

  4. ツールバーからデータベース記号を選択するか、「ツール」→「サービスのテスト」を選択します。「接続テスト」ダイアログ・ボックスが表示され、テスト結果が生成されると更新されます。テストに成功すると、次のメッセージが表示されます。

接続テストに成功しました。

コンポーネントの起動の詳細は、Oracle10i Net管理者ガイドの「接続の確立とネットワークのテスト」の章を参照してください。

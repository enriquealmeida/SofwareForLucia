Menü Befehl

Mit dem Menü Befehl können Sie folgende Vorgänge ausführen:

Verzeichnis

Dieses Untermenü enthält Befehle zur Ausführung von Aktionen auf einem Directory-Server:

Element Beschreibung
Aktuellen Kontext ändern Mit diesem Befehl können Sie einen anderen Oracle Context auswählen. Oracle Context ist ein Unterbaum in dem Verzeichnis, in dem Oracle-Einträge gespeichert werden, einschließlich Datenbankdiensten und Net Service-Namen zur Benutzung mit der Verzeichnisbenennung. Wenn ein neuer Oracle Context ausgewählt ist, werden Objekte im Ordner Oracle Net-Konfiguration > Verzeichnis > Dienstebenennung des Navigators für den neuen Oracle Context angezeigt.
Berechtigungsprüfung festlegen Mit diesem Befehl können Sie die Berechtigungsprüfungs-ID-Daten auf den Directory-Server festlegen.
Net Service-Namen exportieren Wählen Sie diesen Befehl, um den Directory Server Migrations-Assistenten zu starten. Dieser Assistent exportiert Net Service-Namen, die in einer Datei tnsnames.ora gespeichert sind, in einen Directory-Server.

Hinweis: Das Menü Verzeichnis wird nur angezeigt, wenn der Zugriff auf das Verzeichnis konfiguriert wurde. Weitere Informationen über die Konfiguration des Verzeichniszugriffs und andere Anforderungen finden Sie unter Anforderungen an die Verzeichnisbenennung bei Oracle Net Manager.

Dienst testen

Wählen Sie diesen Befehl, um sich bei einem Datenbankdienst mit den Verbindungsdeskriptor-Informationen anzumelden, die in einer Datei tnsnames.ora oder einem Directory-Server gespeichert sind.

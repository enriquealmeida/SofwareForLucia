Source-Routing

Gibt an, dass alle Netzwerk-Routes verwendet werden. Source Routing wird typischerweise in Umgebungen verwendet, bei denen eine große Anzahl von Verbindungen auf denselben Dienst zugreifen müssen. Oracle Net bietet ein Routing über Oracle Connection Manager an.

Informationen zu Oracle Connection Manager finden Sie im Kapitel Networking Challenges in the Internet Age im Oracle10i Net Administrator's Guide.

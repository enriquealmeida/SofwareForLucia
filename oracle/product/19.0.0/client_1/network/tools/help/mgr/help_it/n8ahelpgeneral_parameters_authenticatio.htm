Parametri generali: Autenticazione

La scheda Autenticazione consente di configurare la sicurezza per un listener. Le operazioni privilegiate, come il salvataggio delle modifiche apportate alla configurazione o l'arresto del listener, che vengono eseguite dalla utility Controllo del listener richiederanno pertanto l'immissione di una password. Oracle Net Manager consente di impostare una password cifrata.

Non richiedere una password per le operazioni del listener

Scegliere questa opzione per disabilitare l'autenticazione per le attività amministrative.

Richiedi una password per le operazioni del listener

Scegliere questa opzione per abilitare l'autenticazione per alcune attività amministrative. È quindi possibile immettere la password richiesta. La password predefinita è "oracle".

Nota: Oracle Net consente di configurare una sola password per ogni listener.

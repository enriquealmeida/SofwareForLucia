Dedizierter Server

Ein Server, der einen dedizierten Server-Prozess für jeden Benutzerprozess erfordert. Für jeden Client ist ein Server-Prozess vorhanden. Oracle Net sendet die Adresse eines bestehenden Server-Prozesses an den Client zurück. Der Client sendet dann die Anmeldeanforderung erneut an die angegebene Server-Adresse. Dieser Modus steht im Gegensatz zum Shared Server-Modus.

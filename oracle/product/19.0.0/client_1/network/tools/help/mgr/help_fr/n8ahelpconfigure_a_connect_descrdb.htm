Configuration d'un descripteur de connexion pour une base de données Oracle Rdb

Une base de données Oracle RdB est une base de données pour plate-forme Digital sur 64 bits. Pour les clients, les serveurs Oracle Rdb ressemblent à la base de données Oracle. Une base de données Oracle Rdb a son propre processus d'écoute. La configuration d'un client consiste à créer un nom de service réseau comportant des paramètres propres aux bases de données Oracle pour accéder à la base de données.

Pour configurer le client pour une base de données Oracle Rdb :

  1. Créez un nom de service réseau, comme décrit dans Création d'un nom de service réseau.

  2. Dans la case Identification de service du panneau de droite, choisissez Options avancées. La boîte de dialogue Options de service avancées apparaît.

  3. Indiquez le nom de fichier d'une base de données Oracle Rdb dans le champ Base de données Rdb.

  4. Vous pouvez aussi indiquer le nom global de base de données dans le champ Nom global de base de données et, si nécessaire, le type de service dans le champ Type de service.

  5. Dans la boîte de dialogue Options de service avancées, cliquez sur OK.

  6. Dans le panneau de droite, choisissez Appliquer.

  7. Choisissez Fichier > Enregistrer la configuration réseau.

Pour plus d'informations, reportez-vous à la documentation sur Oracle Rdb.

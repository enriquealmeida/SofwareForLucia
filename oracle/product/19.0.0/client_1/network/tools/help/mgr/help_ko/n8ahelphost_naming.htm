호스트 이름 지정

사용자가 TCP/IP 환경에서 호스트명 별칭을 사용하여 Oracle 데이터베이스 서버에 접속할 수 있게 하는 이름 지정 방법입니다. 호스트 이름은 DNS(Domain Name System), NIS(Network Information Service), 중앙 관리 /etc/hosts 파일 집합과 같은 기존 이름 변환 서비스의 서버 전역 데이터베이스 이름에 매핑됩니다.

호스트 이름 지정 방법 구성에 대한 자세한 내용은 Oracle10i Net Services Administrator's Guide의 Configuring Naming Methods 장을 참조하십시오.

Caixa de Diálogo Escolher Novo Contexto

A caixa de diálogo Escolher Novo Contexto permite a definição de outro Oracle Context, que é um container chamado cn=OracleContext em uma DIT (Directory Information Tree) com entradas para uso com o software Oracle. Essas entradas podem incluir nomes de serviço de rede e nomes de serviço de banco de dados para conexões com serviços de banco de dados. Por default, o Oracle Net Manager utiliza o Oracle Context configurado pelo Oracle Net Configuration Assistant.

Contexto de Nomeação de Diretório

Na lista, selecione uma entrada que contenha o Oracle Context que deseja ver.

Oracle Context

Na lista, selecione um Oracle Context. O Oracle Context selecionado deverá conter as entradas na árvore de diretórios cujas informações de rede você pretende modificar ou adicionar.

A pasta Diretório > Nomeação de Serviço do painel de navegação é atualizada com os objetos de rede do Oracle Context selecionado.

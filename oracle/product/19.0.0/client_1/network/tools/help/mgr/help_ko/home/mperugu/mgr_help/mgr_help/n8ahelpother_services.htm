기타 서비스

[기타 서비스] 탭을 사용하여 리스너가 데이터베이스 서비스 이외의 서비스에 대한 접속 요청을 수신하도록 지정할 수 있습니다. 데이터베이스 서비스 이외의 서비스에는 외부 프로시저 또는 이기종 서비스가 포함될 수 있습니다.

전역 서비스 이름

서비스의 서비스 이름과 도메인 이름을 입력합니다.

프로그램 이름

실행 가능한 프로그램 이름 (예: extproc.exe)

프로그램 영 인수

다른 프로그램 이름으로 자주 사용되는 첫번째 내부 인수 (예: argv0)

프로그램 인수

서비스 시작 시 전달되어야 하는 명령행 인수 (예: -mode=5)

환경

서비스 시작 전에 설정되어야 하는 환경 변수를 입력합니다. 환경 변수는 다음 형식으로 지정됩니다.

environment_variable=values

예: user=scott, start=immediate

변수는 쉼표로 구분합니다.

SID

서비스에 대한 프로그램 이름 또는 Oracle SID(시스템 식별자)를 입력합니다.

Oracle 홈 디렉토리

실행 가능한 프로그램 위치를 입력합니다.

서비스 추가 단추

데이터베이스 서비스 이외의 서비스 정보를 구성합니다. 다른 서비스에 대한 정보를 입력할 수 있는 새 서비스 탭이 나타납니다.

서비스 제거 단추

선택된 서비스 탭과 해당 서비스 정보를 삭제합니다.

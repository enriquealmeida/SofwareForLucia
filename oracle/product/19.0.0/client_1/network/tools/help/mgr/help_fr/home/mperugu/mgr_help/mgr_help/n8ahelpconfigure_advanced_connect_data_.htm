Configuration des options de connexion avancées

Outre le nom de service ou le SID Oracle de la base de données, vous pouvez éventuellement configurer la section CONNECT_DATA d'un descripteur de connexion avec les options suivantes :

  • Nom de l'instance pour se connecter à une instance de base de données spécifique

  • Répertoire d'origine Oracle Home de la base de données pour démarrer la base de données du réseau

  • Taille de l'unité de données de session (SDU) pour indiquer la taille du tampon

  • Serveur dédié pour connecter un processus de serveur dédié

  • Services hétérogènes pour accéder à un système autre qu'Oracle

  • Paramètres d'Oracle Rdb pour accéder à une base Oracle Rdb.

Pour configurer les options de connexion avancées d'un nom de service réseau :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Annuaire > Local ou Résolution de noms de service.

  2. Sélectionnez un nom de service réseau ou un service de base de données. Le panneau de droite affiche le service de destination et la liste d'adresses courants.

  3. Dans la case Identification de service, choisissez Options avancées. La boîte de dialogue Options de service avancées apparaît.

  4. Remplissez les champs ou sélectionnez les options appropriées, puis cliquez sur OK.

  5. Dans le panneau de droite, choisissez Appliquer.

  6. Si vous effectuez ces modifications dans le dossier Local, choisissez Fichier > Enregistrer la configuration réseau.

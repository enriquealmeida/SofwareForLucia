Números de Porta Recomendados

A Oracle Corporation recomenda os seguintes números de porta para endereços de protocolo TCP/IP e TCP/IP com SSL:

Porta Descrição
1521 Porta do listener default para conexões clientes com o listener. Em releases futuras, esse número de porta poderá se alterar para o número de porta oficialmente registrado de 2483 para TCP/IP e 2484 para TCP/IP com SSL.
1575 Porta do listener default e oficialmente registrada para conexões clientes a um servidor Oracle Names.
1630 Porta do listener default e oficialmente registrada para conexões de clientes ao Oracle Connection Manager.
1830 Porta de listening default e oficialmente registrada para comandos administrativos do Oracle Connection Manager.

Para obter mais informações, consulte o apêndice Integrating Authentication Devices Using RADIUS do Oracle Advanced Security Administrator's Guide

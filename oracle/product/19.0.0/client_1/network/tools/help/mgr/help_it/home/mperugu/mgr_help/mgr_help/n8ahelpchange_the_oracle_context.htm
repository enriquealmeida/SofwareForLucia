modificare il contesto Oracle;

Quando si avvia Oracle Net Manager, gli oggetti visualizzati nella cartella Directory > Denominazione dei servizi sono le voci dei servizi Oracle contenute nel contesto Oracle predefinito, che viene impostato durante la configurazione dell'uso delle directory con l'Assistente alla configurazione di Oracle Net. È possibile creare e modificare le voci relative a un altro contesto Oracle per il quale si dispone dell'autenticazione.

Per modificare il contesto Oracle corrente, effettuare le operazioni riportate di seguito.

  1. Scegliere Comando > Directory > Modifica contesto corrente. Viene visualizzata la finestra di dialogo Scegliere un nuovo contesto.

  2. Nella lista Contesto di denominazione delle directory, selezionare la voce di directory contenente il contesto Oracle che si desidera visualizzare.

  3. Nella lista Contesto Oracle selezionare il contesto Oracle desiderato, quindi fare clic su OK. Se le modifiche apportate non sono state salvate nel contesto Oracle corrente, viene visualizzata una finestra di dialogo Salva in cui viene richiesto di eseguire il salvataggio.

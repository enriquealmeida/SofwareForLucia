Configuration des services hétérogènes pour le processus d'écoute

Les services hétérogènes vous permettent d'accéder à un système non-Oracle à partir d'un serveur Oracle. Pour établir une connexion à un système non-Oracle, le serveur Oracle démarre un processus d'agent via le processus d'écoute sur la passerelle. La configuration d'un service hétérogène pour le processus d'écoute implique la création d'informations concernant le service pour l'agent hétérogène.

Pour configurer un service hétérogène pour le processus d'écoute :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Processus d'écoute.

  2. Sélectionnez un processus d'écoute.

  3. Dans la liste du panneau de droite, sélectionnez Autres services.

  4. Choisissez Ajouter un service. Un nouvel onglet Service apparaît.

  5. Indiquez l'exécutable d'agent dans le champ Nom du programme, l'identificateur système (SID) Oracle de l'agent dans le champ SID et le répertoire d'origine Oracle Home de l'exécutable dans le champ Répertoire Oracle Home.

  6. Assurez-vous que le serveur est configuré avec un descripteur de connexion pour accéder au service hétérogène, comme décrit dans Configuration d'un descripteur de connexion pour des procédures externes.

  7. Choisissez Fichier > Enregistrer la configuration réseau.

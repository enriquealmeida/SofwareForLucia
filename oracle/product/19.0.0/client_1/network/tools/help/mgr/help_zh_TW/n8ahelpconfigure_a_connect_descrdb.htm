設定 Oracle Rdb 資料庫的連線描述區

Oracle Rdb 資料庫是 Digital 64 位元平台使用的資料庫。Oracle Rdb 伺服器會以 Oracle 資料庫相同的方式出現在從屬端。Oracle Rdb 資料庫有自己的監聽器。從屬端組態包含使用 Oracle Rdb 特定參數建立，用以存取資料庫的網路服務名稱。

設定 Oracle Rdb 資料庫的從屬端：

  1. 建立網路服務名稱，請參閱建立網路服務名稱。

  2. 在右窗格的服務識別方塊中，選擇進階。進階服務選項對話方塊就會顯示。

  3. 在 Rdb 資料庫欄位輸入 Oracle Rdb 資料庫的檔案名稱。

  4. 或者，在全域資料庫名稱欄位輸入全域資料庫名稱，並視需要在服務類型欄位輸入服務的類型。

  5. 在「進階服務選項」對話方塊中，選擇確定。

  6. 在右窗格中，選擇套用。

  7. 選擇檔案 > 儲存網路組態。

如需詳細資訊，請參閱 Oracle Rdb 文件。

监听程序配置概览

本主题包含以下章节。

安装期间的配置

在数据库服务器上安装企业版或标准版期间，Oracle Net Configuration Assistant 将配置名为LISTENER的监听程序，该程序中含有 Oracle 数据库的 TCP/IP 监听协议地址。在进行定制安装期间，Oracle Net Configuration Assistant 将提示您为自己选取的监听程序配置监听程序名称和协议地址。

另外，无论采用何种安装类型，外部过程调用的监听 IPC 协议地址都是自动配置的。

Oracle Net Configuration Assistant 还自动在 listener.ora 文件中配置外部过程的服务信息。

具有 Oracle Net Manager 的配置

Oracle Net Manager 使您能够完成以下操作：

  • 创建监听程序

  • 配置 Oracle8 或以前数据库的服务信息

  • 配置非 Oracle 数据库服务的服务信息，包括外部过程和异构服务

  • 启用跟踪和事件记录功能

  • 为监听程序控制实用程序启用口令验证

要访问监听程序配置，请执行以下操作：

在导航器窗格中，展开 Oracle Net 配置 > 本地 > 监听程序。

有关详细信息，请参阅 Oracle10i Net Administrator's Guide 中的 Configuring and Administering the Listener 一章。

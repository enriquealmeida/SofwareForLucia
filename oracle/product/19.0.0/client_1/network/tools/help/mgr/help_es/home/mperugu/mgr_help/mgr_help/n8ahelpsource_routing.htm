Direccionamiento de Origen

Especifica que todas las rutas de red se utilizan. El direccionamiento de origen se suele utilizar en entornos donde un gran número de conexiones deben acceder al mismo servicio. La Red de Oracle ofrece direccionamiento mediante Oracle Connection Manager.

Para obtener información sobre Oracle Connection Manager, consulte el capítulo sobre los retos de las redes en la era de Internet, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

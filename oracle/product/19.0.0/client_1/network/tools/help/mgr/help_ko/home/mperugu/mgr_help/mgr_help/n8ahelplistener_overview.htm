리스너 개요

서비스는 클라이언트 응용 프로그램 대신 접속 요청을 수신하는 서버 프로세스인 리스너의 도움을 받아 세션을 조정합니다. 리스너는 데이터베이스 서비스나 기타 서비스에 대해 프로토콜 주소를 "수신"하도록 구성됩니다.

데이터베이스 서비스나 기타 서비스에 대한 프로토콜 주소는 리스너 구성 파일인 listener.ora 파일에서 구성됩니다. Oracle9i 또는 Oracle8i 데이터베이스 서비스는 서비스 이름, 인스턴스 이름 및 로드 정보와 같은 해당 정보를 리스너에 자동으로 등록합니다. 이 서비스 등록 기능의 경우 listener.ora 파일에서 서비스 구성을 정의할 필요가 없습니다. Oracle8 및 Oracle7 데이터베이스 서비스를 포함하는 기타 서비스와 Oracle Enterprise Manager 관리 도구의 경우 listener.ora 파일의 서비스 구성이 있어야 합니다.

일단 리스너가 구성되면 리스너 제어 유틸리티로 관리할 수 있습니다.

권장 포트 번호

Oracle Corporation은 TCP/IP와 SSL을 사용하는 TCP/IP 프로토콜 주소에 다음 포트 번호를 권장합니다.

포트 설명
1521 클라이언트 리스너에 대한 클라이언트 접속의 기본 수신 포트입니다. 이후 릴리스에서 이 포트 번호는 공식적으로 등록된 포트 번호(TCP/IP의 경우 2483, SSL을 사용하는 TCP/IP의 경우 2484)로 변경될 수 있습니다.
1575 클라이언트가 Oracle Names Server에 접속할 때 사용하는 공식적으로 등록된 기본 수신 포트입니다.
1630 클라이언트가 Oracle Connection Manager에 접속할 때 사용하는 공식적으로 등록된 기본 수신 포트입니다.
1830 Oracle Connection Manager에 대한 관리 명령의 공식적으로 등록된 기본 수신 포트입니다.

자세한 내용은 Oracle Advanced Security Administrator's Guide의 부록 Integrating Authentication Devices Using RADIUS를 참조하십시오.

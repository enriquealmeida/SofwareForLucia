Présentation de la configuration de la méthode de résolution de noms locaux

Les sections suivantes sont couvertes dans cette rubrique.

Installation administrateur ou runtime

Si l'utilisation de l'annuaire n'a pas été configurée, l'assistant Configuration Oracle Net vous invite à configurer les noms de service réseau dans le fichier tnsnames.ora, pour une installation client administrateur ou runtime.

Installation personnalisée

Si la résolution de noms locaux est sélectionnée comme méthode de résolution de noms pendant une installation personnalisée client ou serveur, l'assistant Oracle Net Configuration vous demande de configurer les noms de service réseau dans le fichier tnsnames.ora .

Configuration à l'aide d'Oracle Net Manager

Oracle Net Manager vous permet de configurer des noms de service réseau et de :

  • Créer des noms de service réseau pour des services de base de données autres qu'Oracle.

  • Configurer plusieurs chemins d'accès à un processus d'écoute par des adresses de protocole multiples.

  • Configurer des informations CONNECT_DATA avancées pour un nom de service réseau ou un service de base de données.

Pour accéder à la configuration de la méthode de résolution de noms locaux :

Dans le panneau du navigateur, choisissez Local > Résolution de noms de service pour créer des noms de service réseau ou modifier des informations sur les descripteurs de connexion.

Pour plus d'informations, reportez-vous au chapitre Configuring Naming Methods dans le Guide de l'administrateur Oracle10i Net.

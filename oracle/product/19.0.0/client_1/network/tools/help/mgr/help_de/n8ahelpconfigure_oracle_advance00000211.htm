Oracle Advanced Security konfigurieren

Verwenden Sie Oracle Net Manager, um Oracle Advanced Security zu konfigurieren. In jedem Register von Oracle Advanced Security können Sie eine separate Gruppe von Parametern konfigurieren. Oracle Advanced Security aktiviert die Datenverschlüsselung und Integritätsprüfung, die erweiterte Authentifizierung und Single Sign-On. Oracle Advanced Security ermöglicht außerdem die zentralisierte Benutzerverwaltung bei LDAP-konformen Directory-Servern sowie die zertifikatbasierte Einzelanmeldung. Diese Funktionalität beruht auf Secure Sockets Layer (SSL).

So konfigurieren Sie Oracle Advanced Security:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal > Profil im Navigator-Fensterbereich ein.

  2. Wählen Sie Oracle Advanced Security aus der Liste im rechten Fensterbereich.

  3. Daraufhin werden die folgenden Register angezeigt:

    • Berechtigungsprüfung

    • Andere Parameter

    • Integrität

    • Encryption

    • SSL

  4. Nehmen Sie die Änderungen an der Konfiguration vor.

  5. Wählen Sie Datei >Netzwerkkonfiguration speichern.

Ihre Änderungen werden in der Datei sqlnet.ora gespeichert.

Hinweis: Wenn Oracle Advanced Security nicht in der Liste angezeigt wird, wurde diese Funktion nicht installiert.

Weitere Informationen finden Sie im Oracle Advanced Security Administrator's Guide.

DNS (Domain Name System)

Un sistema per la denominazione di computer e servizi di rete organizzato in una gerarchia di domini. Il DNS viene utilizzato nelle reti TCP/IP per individuare i computer tramite nomi semplici. Tale sistema risolve i nomi semplici in un indirizzo IP che i computer sono in grado di riconoscere.

In Oracle Net, il sistema DNS converte il nome host utilizzato negli indirizzi TCP/IP in un indirizzo IP.

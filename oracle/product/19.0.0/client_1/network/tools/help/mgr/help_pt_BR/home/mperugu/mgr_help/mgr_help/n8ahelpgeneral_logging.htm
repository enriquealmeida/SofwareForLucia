Geral: Log

A guia Log permite que você defina parâmetros de log para este computador. Se este computador for um cliente, use os campos Informações sobre o Cliente. Se for um servidor, use os campos de Informações sobre o Servidor.

Diretório de Log

Informe o caminho de diretório em que os arquivos de log serão gravados. O default é $ORACLE_HOME/network/log nas plataformas UNIX e em ORACLE_HOME\network\log nas plataformas Windows.

Arquivo de Log

Informe o nome do arquivo de log no cliente. O nome default é sqlnet.log.

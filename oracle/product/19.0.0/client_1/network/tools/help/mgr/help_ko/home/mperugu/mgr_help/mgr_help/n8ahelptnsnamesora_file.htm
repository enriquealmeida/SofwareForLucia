tnsnames.ora 파일

접속 기술자에 매핑되는 네트 서비스 이름이 포함된 구성 파일로

tnsnames.ora 파일은 다음 위치 중 하나에 있어야 합니다.

  1. TNS_ADMIN 환경 변수에 의해 지정된 디렉토리입니다. TNS_ADMIN 환경 변수가 Windows NT에서 변수로 정의되지 않은 경우 레지스트리에 있을 수 있습니다.

  2. 노드의 전역 구성 디렉토리입니다. Sun Solaris의 경우 이 디렉토리는 /var/opt/oracle입니다. Windows NT에는 중앙 디렉토리가 없습니다.

  3. UNIX의 $ORACLE_HOME/network/admin 디렉토리 또는 Windows 운영 체제의 ORACLE_HOME\network\admin 디렉토리입니다.

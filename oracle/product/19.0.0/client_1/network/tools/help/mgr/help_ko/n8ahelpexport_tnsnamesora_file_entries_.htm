디렉토리 서버에 tnsnames.ora 파일 항목 익스포트

tnsnames.ora 파일에 이미 구성되어 있는 네트 서비스 이름을 디렉토리 서버로 익스포트할 수 있습니다. 익스포트는 한 번에 하나의 도메인에 대해 수행됩니다.

tnsnames.ora 파일을 익스포트하면 해당 디렉토리의 현재 선택된 Oracle Context에 네트 서비스 이름이 생성됩니다. Oracle Context를 변경해야 할 경우 명령 > 디렉토리 서버 > Oracle Context 변경을 선택합니다.

tnsnames.ora 파일에 저장된 네트 서비스 이름을 디렉토리 서버로 익스포트하려면 다음과 같이 하십시오.

  1. 명령 > 디렉토리 > 서비스 이름 익스포트를 선택합니다. 디렉토리 이전 마법사가 시작됩니다.

  2. 다음을 선택합니다.

    여러 도메인이 존재하면 [도메인 선택] 페이지가 나타납니다. 단계 3으로 계속합니다.

    네트 서비스 이름이 도메인으로 한정되어 있지 않으면 [네트 서비스 이름 선택] 페이지가 나타납니다. 단계 4로 건너뜁니다.

  3. 네트 서비스 이름을 익스포트할 네트워크 도메인을 선택한 후 다음을 누릅니다. [네트 서비스 이름 선택] 페이지가 나타납니다.

  4. 네트 서비스 이름을 익스포트할 네트워크 도메인을 선택한 후 다음을 누릅니다. [대상 컨텍스트 선택] 페이지가 나타납니다.

    대상 컨텍스트 선택 페이지에서 다음 작업을 수행합니다.

    a. 디렉토리 이름 지정 컨텍스트 목록에서 Oracle Context가 포함된 디렉토리 이름 지정 컨텍스트를 선택합니다. 디렉토리 이름 지정 컨텍스트는 Oracle 항목을 포함하는 하나 이상의 Oracle Context가 포함된 디렉토리 하위 트리의 일부입니다.

    b. Oracle Context 목록에서 선택한 네트 서비스 이름을 익스포트할 Oracle Context를 선택합니다.

    c. 다음을 선택합니다. 익스포트 작업 상태를 보여주는 [디렉토리 서버 업데이트] 페이지가 나타납니다.

  5. 완료를 선택하여 디렉토리 서버 이전 마법사를 종료합니다.

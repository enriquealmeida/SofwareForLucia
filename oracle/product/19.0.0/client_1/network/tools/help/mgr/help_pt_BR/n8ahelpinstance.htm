Instância

A combinação da Área Global do Sistema (SGA) e processos Oracle em segundo plano. Quando um banco de dados é iniciado em um servidor de banco de dados (independente do tipo de computador), o Oracle aloca uma área de memória denominada SGA e inicia um ou mais processos Oracle. A memória e os processos de uma instância gerenciam de maneira eficiente os dados do banco de dados associado e servem aos usuários do banco de dados. Você pode se conectar a qualquer instância para acessar informações contidas em um banco de dados de servidor paralelo.

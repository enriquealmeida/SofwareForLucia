Oracle Advanced Securityについて

Oracle Advanced Securityは、Oracle環境に対するセキュリテイ機能の包括的なパッケージを追加します。このセキュリティ機能パッケージにより、企業ネットワークが保護され、安全にインターネットへ拡張できます。ネットワークの暗号化と認証のソリューション、シングル・サインオン・サービスおよびセキュリティ・プロトコルを統合する単一ソースを提供します。Oracle Advanced Securityは業界標準を組み入れ、Oracleネットワークの内外で最高のセキュリティを実現します。

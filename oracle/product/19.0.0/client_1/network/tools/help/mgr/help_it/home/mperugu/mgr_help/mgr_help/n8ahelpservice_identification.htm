Identificazione servizio

Questo foglio delle proprietà consente di visualizzare o modificare le informazioni di rete per un nome di servizio di rete o un servizio di database selezionato.

Identificazione servizio

Utilizzare la casella Identificazione servizio per visualizzare o modificare le informazioni CONNECT_DATA.

Elemento Descrizione
Nome servizio Immettere un nome di servizio se si sta configurando il client per la connessione a un servizio di database Oracle8 i o successivo. Nota: per un servizio di database memorizzato in un server directory, questo campo visualizza solo il valore corrente. Non sono consentite modifiche.
SID Visualizza il SID (ID di sistema) Oracle per un database Oracle8.
Tipo di connessione Utilizzare la lista per specificare se si desidera che il nome di servizio di rete utilizzi un server dedicato, un server condiviso o un altro componente qualsiasi con cui è stato configurato il database. Se nel file dei parametri di inizializzazione è stato configurato un server condiviso, selezionare Server dedicato per indurre il listener ad avviare un server dedicato per ignorare la configurazione del server condiviso. Se nel file dei parametri di inizializzazione è stato configurato un server condiviso e si desidera garantire che per questa connessione venga sempre utilizzato un server condiviso, selezionare Server condiviso. Se sul server è abilitato il connection pooling residente del database, è possibile selezionare Server in pool per ottenere una connessione dal pool.
Utilizza identificazione compatibile con Oracle8 Release 8.0 Selezionare questa opzione se si sta configurando il client per la connessione a un servizio di database di una release precedente a Oracle8i. Deselezionare questa opzione se la configurazione riguarda un servizio di database di Oracle9i o Oracle8i.
Avanzate Scegliere questo pulsante per configurare i parametri avanzati CONNECT_DATA.

Configurazione indirizzo

Utilizzare la casella Indirizzo per modificare, aggiungere o eliminare un indirizzo di protocollo del listener.

Se è già stato configurato un indirizzo, nei campi sono visibili le informazioni sul protocollo corrispondenti.

Elemento Descrizione
Protocollo Selezionare nell'apposita lista il protocollo per l'indirizzo del listener. A seconda del protocollo selezionato, immettere le informazioni appropriate nei campi.
Pulsante Mostra avanzate/Nascondi avanzate Utilizzare il pulsante Mostra avanzate/Nascondi avanzate per visualizzare o nascondere le impostazioni che consentono di specificare il limite di spazio del buffer di I/O per l'indirizzo.

Queste impostazioni sono supportate dai protocolli TCP/IP, TCP/IP con SSL e SDP.

  • Dimensione totale del buffer di invio: specificare, in byte, lo spazio del buffer per le operazioni di invio delle sessioni.

  • Dimensione totale del buffer di ricezione: specificare, in byte, lo spazio del buffer per le operazioni di ricezione delle sessioni.

Nota: la specifica di queste impostazioni ha la priorità su tali impostazioni in un file sqlnet.ora sul lato client.
Pulsante + Scegliere questo pulsante per creare un nuovo indirizzo di protocollo. Viene visualizzata una nuova scheda Indirizzo, nella quale è possibile selezionare il protocollo e immettere le informazioni opportune nei campi. È necessario che il protocollo scelto sia installato sia sul client che sul server (dove risiede il listener).
Pulsante X Scegliere questo pulsante per rimuovere la scheda Indirizzo selezionata e le informazioni sull'indirizzo in essa contenute.
Pulsante < Scegliere questo pulsante per spostare l'indirizzo di protocollo in alto nella lista.
Pulsante > Scegliere questo pulsante per spostare l'indirizzo di protocollo in basso nella lista.
Pulsante Avanzate Fare clic su questo pulsante per configurare le opzioni relative a più indirizzi, ad esempio failover della fase di connessione e bilanciamento del carico del client.

Pulsante Applica

Scegliere questo pulsante per applicare le impostazioni.

Pulsante Ripristina

Scegliere questo pulsante per ripristinare le impostazioni originali.

Per ulteriori informazioni sulla configurazione dello spazio del buffer di I/O, vedere il capitolo Optimizing Performance del manuale Oracle10i Net Administrator's Guide.

Microsoft Active Directory

Ein LDAP-konformer Directory Service im Windows 2000 Server. In diesem Service werden Informationen über Objekte in dem Netzwerk gespeichert und werden diese Informationen Benutzern und Netzwerkadministratoren bereitgestellt. Außerdem ermöglicht Active Directory Benutzern den Zugriff auf Ressourcen in dem Netzwerk mit einem einzigen Anmeldeprozess.

Active Directory kann als Verzeichnisbenennungs-Methode konfiguriert werden, um Dienstinformationen zu speichern, auf die Clients zugreifen können.

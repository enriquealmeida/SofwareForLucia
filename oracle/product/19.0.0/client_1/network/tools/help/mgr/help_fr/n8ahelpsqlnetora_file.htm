Fichier sqlnet.ora

Fichier de configuration du client ou du serveur qui indique :

  • le domaine client à ajouter aux noms de service ou aux noms de service réseau non qualifiés

  • l'ordre des méthodes de résolution de noms que le client doit utiliser lors de la résolution d'un nom

  • les fonctions de journalisation et de trace à utiliser

  • Chemin des connexions

  • Serveurs Preferred Oracle Names

  • les paramètres de résolution de noms externes

  • les paramètresOracle Advanced Security

Le fichier sqlnet.ora doit résider dans l'un des emplacements suivants :

  1. Le répertoire indiqué par la variable d'environnement TNS_ADMIN. Si la variable d'environnement TNS_ADMIN n'est pas définie en tant que variable dans Windows NT, il se peut qu'elle soit dans la base de registres.

  2. Le répertoire $ORACLE_HOME/network/admin dans UNIX ou le répertoire ORACLE_HOME\network\admin dans les systèmes d'exploitation Windows.

Bienvenue dans Oracle Net Manager

Bienvenue dans Oracle Net Manager. Oracle Net Manager allie des fonctions de configuration et de gestion des composants pour offrir un environnement intégré de configuration et de gestion d'Oracle Net.

Qui utilise Oracle Net Manager ?

Tout administrateur de réseau responsable de l'établissement des connexions entre clients et services. Avec Oracle Net Manager, vous pourrez vous consacrer à la configuration de certaines parties spécifiques du réseau.

Oracle Net Manager est nouveau pour vous ?

Si vous ne connaissez pas encore Oracle Net Manager ou les réseaux en général, vous pouvez naviguer dans l'aide afin d'en savoir plus sur Oracle Net Manager et la configuration du réseau. Vous devez également vous reporter au Guide de l'administrateur Oracle10i Net Services.

Vous pourriez commencer par les rubriques suivantes :

  • Présentation de la connectivité d'Oracle Net

  • Présentation de la configuration d'Oracle Net

  • Principales fonctions d'Oracle Net Manager

  • Présentation de l'environnement d'Oracle Net Manager

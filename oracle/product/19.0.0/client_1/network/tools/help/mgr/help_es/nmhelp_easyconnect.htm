Nomenclatura de Conexión Sencilla

Método de nomenclatura que permite a los clientes conectarse al servidor de base de datos sin ninguna configuración. Los clientes utilizan una dirección TCP/IP sencilla que consta de un nombre de host y un nombre de puerto y de servicio opcional:

CONNECT usuario/contraseña@host[:puerto][/nombre_servicio]

Para obtener información sobre el uso del método de nomenclatura de conexión sencilla, consulte el capítulo sobre configuración de métodos de nomenclatura, en Oracle10i Net Services Administrator's Guide (Guía del Administrador de Servicios de Red de Oracle10i).

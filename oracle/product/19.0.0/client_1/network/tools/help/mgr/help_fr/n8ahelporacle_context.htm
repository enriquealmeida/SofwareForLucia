Contexte Oracle

Nom distinctif relatif de cn=OracleContext dans une arborescence d'informations d'annuaire (DIT) située sous un contexte d'appellation ou une entrée d'annuaire non publiée. Un contexte Oracle contient les entrées à utiliser avec le logiciel Oracle, par exemple les services de base de données et les noms de service réseau à utiliser pour la résolution de noms d'annuaire. Un annuaire peut inclure un ou plusieurs contextes Oracle.

Oracle Internet Directory crée automatiquement un contexte Oracle à la racine de la structure de l'arborescence d'informations d'annuaire (DIT). Ce contexte comporte un nom distinctif (DN) (dn:cn=OracleContext).

Lorsqu'Oracle Net Manager démarre, il utilise le contexte Oracle par défaut configuré par l'assistant Configuration Oracle Net. Vous pouvez demander à utiliser un autre contexte Oracle. Pour la plupart des déploiements, vous devrez créer des contextes Oracle supplémentaires.

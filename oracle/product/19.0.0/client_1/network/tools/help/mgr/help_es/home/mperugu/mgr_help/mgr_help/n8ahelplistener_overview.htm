Visión General del Listener

Los servicios coordinan sus sesiones con ayuda de un listener, proceso del servidor que recibe las solicitudes de conexión procedentes de una aplicación cliente. Los listeners se configuran para recibir direcciones de protocolo de un servicio de base de datos o de otro tipo.

Las direcciones de protocolo se configuran en el archivo de configuración del listener, listener.ora, para un servicio de base de datos o de otro tipo. Un servicio de base de datos Oracle9i u Oracle8i registra automáticamente la información con el listener, como el nombre del servicio, los nombres de instancia y la información de carga. Esta función, denominada registro del servicio, no necesita configuración de servicios en el archivo listener.ora. Otros servicios, entre los que se incluyen servicios de base de datos Oracle8 y Oracle7, así como las herramientas de gestión de Oracle Enterprise Manager, se deben configurar en el archivo listener.ora .

Una vez configurado, el listener se puede gestionar con la utilidad de control del listener.

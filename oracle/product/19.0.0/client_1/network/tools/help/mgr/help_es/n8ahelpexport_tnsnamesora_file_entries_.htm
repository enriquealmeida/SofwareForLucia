Exportación de Entradas del Archivo tnsnames.ora al Servidor de Directorios

Los nombres de servicio de red configurados en el archivo tnsnames.ora se pueden exportar a un servidor de directorios. La exportación se realiza en un dominio cada vez.

Al exportar un archivo tnsnames.ora se crea un nombre de servicio de red en el Contexto de Oracle seleccionado en el directorio. Si necesita cambiar de contexto, seleccione Comando > Servidor de Directorios > Cambiar Oracle Context.

Para exportar a un servidor de directorios nombres de servicio de red almacenados en un archivo tnsnames.ora :

  1. Seleccione Comando > Directorio > Exportar Nombres de Servicio. Se inicia el Asistente de Migración del Servidor de Directorios.

  2. Haga clic en Siguiente.

    Si se detectan varios dominios, aparece la página Seleccionar Dominio. Continúe con el paso 3.

    Si los nombres de servicio de red no es un dominio cualificado, aparece la página Seleccionar Nombres de Servicio de Red. Vaya al Paso 4.

  3. Seleccione el dominio de red cuyos nombres de servicio de red desea exportar y haga clic en Siguiente. Aparece la página Seleccionar Nombres de Servicio de Red.

  4. Seleccione los nombres de los servicios de red que desea exportar de la lista y haga clic en Siguiente. Aparece la página Seleccionar Contexto de Destino.

    En esta página, realice las siguientes operaciones:

    a. De la lista Contexto de Nomenclatura de Directorios, seleccione el contexto de nomenclatura de directorios que contiene el Oracle Context. El contexto de nomenclatura de directorios forma parte de un árbol secundario del directorio que contiene uno o más Contextos con entradas de Oracle.

    b. De la lista Oracle Context, seleccione el Oracle Context al que desea exportar los nombres de servicio de red seleccionados.

    c. Haga clic en Siguiente. Aparece la página Actualización del Servidor de Directorios con el estado de la operación de exportación.

  5. Haga clic en Terminar para cerrar el Asistente de Migración del Servidor de Directorios.

記錄類型

儲存在 Oracle Names 伺服器的網路物件類型。Oracle Names 伺服器支援以下的記錄類型：

- A.SMD：網路位址，包括資料庫服務位址和網路服務名稱

- CNAME.SMD：任何網路物件的別名

- DL.RDBMS.OMD：全域資料庫連結

- DLCR.RDBMS.OMD：連結限定字元

- NS.SMD：Oracle Names 伺服器位址

- V1ADD.NPO.OMD：SQL*Net Version 1 連線字串

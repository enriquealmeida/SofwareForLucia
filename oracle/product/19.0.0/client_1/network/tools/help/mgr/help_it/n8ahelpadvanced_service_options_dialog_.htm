Finestra di dialogo Opzioni di servizio avanzate

La finestra di dialogo Opzioni di servizio avanzate consente di configurare i parametri CONNECT_DATA avanzati.

Impostazioni di servizio aggiuntive

Elemento Descrizione
Nome istanza Immettere i nomi dell'istanza di database a cui accedere. Il nome dell'istanza può essere ottenuto dal valore del parametro INSTANCE_NAME nel file dei parametri di inizializzazione.
Unità dati sessione (SDU) Immettere la dimensione dell'unità dati sessione (SDU) per modificare le caratteristiche delle prestazioni relative ai pacchetti inviati sulla rete. Impostare la dimensione dell'unità SDU come multiplo della dimensione normale dei frame di trasporto. La dimensione normale per i frame Ethernet è 1500, ma la porzione utilizzabile è 1460. Quando non si utilizza la cifratura, un byte è riservato. L'algoritmo da utilizzare è un multiplo di 1460 più 1 quando non si utilizza la cifratura o un multiplo di 1460 per Ethernet.
Utilizza per Heterogeneous Services Selezionare questa opzione per accedere da un server Oracle a un sistema non Oracle tramite i servizi eterogenei.

Impostazioni Oracle Rdb

Elemento Descrizione
Impostazioni Oracle Rdb Specificare il nome file di un database Oracle Rdb.
Tipo di servizio Immettere il tipo di servizio da utilizzare per un database Oracle Rdb.
Nome di database globale Immettere un identificativo per il database Oracle Rbd.

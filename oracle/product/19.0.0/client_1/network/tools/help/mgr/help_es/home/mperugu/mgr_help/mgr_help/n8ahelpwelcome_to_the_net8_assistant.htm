Bienvenido al Gestor de Red de Oracle

Bienvenido al Gestor de Red de Oracle. El Gestor de Red de Oracle combina funciones de configuración con control de componentes para proporcionar un entorno integrado para la configuración y gestión de la Red de Oracle.

Usuarios del Gestor de Red de Oracle

Puede utilizarlo cualquier administrador de redes responsable de establecer conexiones del cliente a servicios. El Gestor de Red de Oracle permite centrarse en la configuración de áreas específicas de la red.

Nivel Inicial del Gestor de Red de Oracle

Si no está familiarizado con el Gestor de Red de Oracle o las redes en general, puede que le interese examinar este sistema de ayuda para obtener información sobre el Gestor de Red de Oracle y la configuración de la red. También debe consultar Oracle10i Net Services Administrator's Guide (Guía del Administrador de Servicios de Red de Oracle10i).

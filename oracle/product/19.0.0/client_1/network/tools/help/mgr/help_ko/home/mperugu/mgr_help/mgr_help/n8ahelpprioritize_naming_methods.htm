이름 지정 방법 우선 순위 지정

이름 지정 방법을 구성한 후에는 프로파일에서 우선 순위를 지정해야 합니다. 목록에서 제일 처음에 있는 이름 지정 방법이 첫번째로 사용되어 네트 서비스 이름이나 데이터베이스 서비스 이름을 접속 기술자로 이름 분석합니다. 첫번째 방법으로 이름을 분석할 수 없으면 목록의 두번째 방법을 사용합니다.

이름 지정 방법의 순서를 지정하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창의 목록에서 이름 지정을 선택합니다.

  3. 방법 탭을 선택합니다.

  4. 사용 가능한 방법 목록에서 이름 지정 방법을 선택한 다음 오른쪽 화살표 단추를 선택합니다. 선택한 이름 지정 방법이 선택된 방법 목록으로 이동합니다.

  5. 네트 서비스 이름 또는 데이터베이스 서비스 이름을 분석할 때 Oracle Net이 처음으로 사용하게 하고자 하는 방법에 따라 이름 지정 방법의 순서를 지정합니다. 선택된 방법 목록에서 이름 지정 방법을 선택하고 올림이나 내림을 눌러 목록에서 선택 내용을 위 또는 아래로 이동합니다.

  6. 파일 > 네트워크 구성 저장을 선택합니다.

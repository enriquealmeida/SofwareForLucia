Configuration de l'authentification des mots de passe pour le processus d'écoute

Vous pouvez configurer la sécurité du processus d'écoute. Les opérations privilégiées, comme l'enregistrement des modifications de configuration ou l'arrêt du processus d'écoute, effectuées à l'aide de l'utilitaire USNRCTL (Listener Control) exigeront alors un mot de passe. Oracle Net Manager vous permet de définir un mot de passe crypté.

Pour définir un mot de passe pour le processus d'écoute :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Processus d'écoute.

  2. Sélectionnez un processus d'écoute.

  3. Dans la liste du panneau de droite, sélectionnez Paramètres généraux.

  4. Sélectionnez l'onglet Authentification.

  5. Choisissez Exiger un mot de passe pour les opérations du processus d'écoute.

  6. Entrez le mot de passe d'abord dans le champ Mot de passe, puis dans le champ Confirmer le mot de passe.

  7. Choisissez Fichier > Enregistrer la configuration réseau.

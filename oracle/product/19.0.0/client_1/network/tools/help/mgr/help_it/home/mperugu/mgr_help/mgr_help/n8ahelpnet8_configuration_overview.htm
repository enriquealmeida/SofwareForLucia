Panoramica sulla configurazione di Oracle Net Services

Oracle Net Manager consente di configurare molteplici elementi della rete, compresi i metodi di denominazione, i profili client e server e i listener.

Metodi di denominazione

Definire nomi semplici per identificare la posizione di un servizio, ad esempio un database. Questi nomi semplici, definiti identificativi di connessione, vengono mappati ai descrittori di connessione, che contengono la posizione nella rete e l'identificazione del servizio.

Con Oracle Net Manager è possibile configurare sistemi differenti per la risoluzione degli identificativi di connessione nei descrittori di connessione mediante i metodi di denominazione.

Profili client e server

È possibile impostare le preferenze di abilitazione e configurazione delle funzioni di Oracle Net sui client e sui server mediante i profili. I profili consentono infatti di impostare le preferenze che determinano la priorità dei metodi di denominazione, abilitare le funzioni di Oracle Advanced Security, abilitare il log e la funzione di trace o instradare le connessioni.

Configurazione del listener sul server

È possibile configurare il listener per ricevere le connessioni client. Il listener dovrebbe essere configurato con i seguenti elementi:

  • uno o più indirizzi di protocollo di ascolto;

  • informazioni sul servizio di destinazione.

Gli indirizzi di protocollo vengono configurati nel file di configurazione del listener, listener.ora. Le informazioni sul servizio non sono configurate necessariamente in tale file:

  • Un database Oracle8i o successivo registra automaticamente alcune informazioni presso il listener, ad esempio i nomi dei servizi, i nomi delle istanze, gli handler del servizio e le informazioni sul caricamento. Questa funzione, denominata registrazione del servizio, non richiede la configurazione nel file listener.ora.

  • Gli altri servizi, incluso un database Oracle7 o Oracle8, richiedono la configurazione nel file listener.ora .

Per ulteriori informazioni su Oracle Names, vedere Panoramica di Oracle Names.

Per ulteriori informazioni, vedere il capitolo Configuring Naming Methods del manuale Oracle10i Net Administrator's Guide.

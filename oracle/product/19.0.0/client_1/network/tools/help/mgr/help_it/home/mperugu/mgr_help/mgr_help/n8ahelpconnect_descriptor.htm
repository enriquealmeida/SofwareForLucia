Descrittore di connessione

Una descrizione formattata in maniera speciale e riguardante la destinazione di una connessione di rete. Un descrittore di connessione contiene le informazioni sul servizio di destinazione e sull'instradamento di rete.

Il servizio di destinazione è identificato dal nome di servizio per i database Oracle9i o Oracle8i o dal SID (ID di sistema) Oracle per i database Oracle8 o Oracle7.

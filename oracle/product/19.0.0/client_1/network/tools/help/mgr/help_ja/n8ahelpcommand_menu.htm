「コマンド」メニュー

「コマンド」メニューでは、次の操作を実行できます。

ディレクトリ

このサブメニューには、ディレクトリ・サーバーへのアクションを実行する次のコマンドが含まれています。

要素 説明
カレント・コンテキストの変更 現行のものとは別のOracleコンテキストを使用する場合に選択します。Oracleコンテキストは、ディレクトリのネーミングに使用されるデータベース・サービス名やネット・サービス名などのOracleエントリが格納されるディレクトリのサブツリーです。新しいOracleコンテキストを選択すると、ナビゲータの「Oracle Netの構成」→「ディレクトリ」→「サービス・ネーミング」フォルダに、そのOracleコンテキストの情報が表示されます。
認証の設定 ディレクトリ・サーバーへの認証資格証明を設定する場合に選択します。
ネット・サービス名のエクスポート 「ディレクトリ・サーバー移行ウィザード」を起動する場合に選択します。このウィザードにより、tnsnames.oraファイルに格納されているネット・サービス名がディレクトリ・サーバーにエクスポートされます。

注意:「ディレクトリ」メニューは、ディレクトリへのアクセスが構成されている場合にのみ表示されます。ディレクトリ・アクセス構成とその他の要件の詳細は、「Oracle Net Managerのディレクトリ・ネーミング要件」を参照してください。

サービスのテスト

tnsnames.oraファイルまたはディレクトリ・サーバーに格納されている接続記述子情報を使用して、データベース・サービスに接続します。

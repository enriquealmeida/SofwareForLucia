Server condiviso

Un server configurato per consentire la condivisione di un numero esiguo di processi server da parte di molti processi utente, in modo da aumentare il numero di utenti supportati. In una configurazione con server condivisi, molti processi utente si connettono a un dispatcher. Il dispatcher indirizza le richieste di sessioni di rete multiple in entrata verso una coda comune. Un processo server condiviso inattivo da uno shared pool di processi server sceglie una richiesta dalla coda. Questo significa che un piccolo pool di processi server può servire un numero considerevole di client. È il contrario di server dedicato.

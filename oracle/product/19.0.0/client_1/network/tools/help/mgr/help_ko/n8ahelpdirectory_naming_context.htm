이름 지정 컨텍스트

이 절에서는 다음 항목을 다룹니다.

설치 시 서버 구성

하나의 디렉토리 서버에 모두 상주하는 하위 트리입니다.이름 지정 컨텍스트는 연속 하위 트리입니다. 즉, 이름 지정 컨텍스트는 하위 트리의 최상위로 사용되는 항목에서 시작해야 하며 하위 항목 또는 종속 이름 지정 컨텍스트에 대한 참조로 아래쪽으로 확장되어야 합니다. 이 컨텍스트의 범위는 단일 항목에서부터 전체 DIT(디렉토리 정보 트리)까지 될 수 있습니다.

Oracle Context는 이름 지정 컨텍스트에서 생성될 수 있습니다.

Configurando Serviços Heterogêneos para o Listener

Os Serviços Heterogêneos permitem que você acesse sistemas não Oracle de um servidor Oracle. Para iniciar uma conexão com um sistema não Oracle, o servidor Oracle inicia um processo agente por intermédio do listener do gateway. A configuração de um Serviço Heterogêneo para o listener envolve a criação de informações de serviço para o agente Heterogêneo.

Para configurar um Serviço Heterogêneo para o listener:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

  2. Selecione um listener.

  3. Na lista do painel direito, selecione Outros Serviços.

  4. Escolha Adicionar Serviço. Uma nova guia Serviço será exibida.

  5. Informe o executável do agente no campo Nome do Programa, o SID (Identificador de Sistema Oracle) do agente no campo SID e o Oracle home do executável no campo Diretório Oracle Home.

  6. Certifique-se de que o servidor está configurado com um descritor de conexão para acessar o Serviço Heterogêneo, conforme descrito em Configurar um Descritor de Conexão para Procedimentos Externos.

  7. Escolha Arquivo > Salvar Configuração de Rede.

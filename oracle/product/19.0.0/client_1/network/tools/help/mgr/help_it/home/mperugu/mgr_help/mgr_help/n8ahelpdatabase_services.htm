Servizi di database

La scheda Servizi di database consente di specificare i servizi di database per i quali il listener deve ricevere le richieste di connessione. Il listener necessita di tali informazioni per garantire la connessione dei client al database.

Le istanze di Oracle9i e Oracle8i utilizzano la registrazione dinamica del servizio per comunicare al listener i loro servizi di database. I database Oracle7 e Oracle8 release 8.0 e gli strumenti Oracle Enterprise Manager richiedono la configurazione stati del servizio nel file listener.ora.

Scegliere Aggiungi database per aggiungere le informazioni sul servizio di database.

Nome database globale

Immettere il nome del database e del relativo dominio forniti dal valore del parametro SERVICE_NAMES o dai valori dei parametri DB_NAME e DB_DOMAIN nel file dei parametri di inizializzazione.

Directory Oracle Home

Immettere la directory Oracle Home del database.

Nota: Windows NT conosce già la posizione della directory Oracle Home. Pertanto è possibile lasciare vuoto questo campo se si tratta di una connessione a un servizio su Windows NT.

SID

Immettere il SID (ID di sistema) Oracle per l'istanza.

Pulsante Aggiungi database

Scegliere questo pulsante per configurare le informazioni sul servizio per un altro database. Viene visualizzata un'altra scheda Database in cui è possibile immettere le informazioni sul servizio per un altro database.

Pulsante Rimuovi database

Scegliere questo pulsante per eliminare la scheda Database selezionata e le relative informazioni sul servizio.

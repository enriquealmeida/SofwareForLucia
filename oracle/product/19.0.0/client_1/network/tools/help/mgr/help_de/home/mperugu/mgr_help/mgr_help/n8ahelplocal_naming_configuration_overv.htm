Überblick über die Konfiguration der lokalen Benennung

In diesem Hilfethema werden die folgenden Abschnitte aufgeführt:

Administrator- oder Laufzeitinstallation

Wenn die Directory-Verwendung nicht konfiguriert wurde, fordert der Oracle Net-Konfigurationsassistent Sie auf, Net Service-Namen in der Datei tnsnames.ora für eine Administrator- oder Laufzeit-Client-Installation zu konfigurieren.

Benutzerdefinierte Installation

Wenn die lokale Benennung als Benennungsmethode während einer Benutzerdefinierten Client- oder Server-Installation ausgewählt wurde, fordert der Oracle Net-Konfigurationsassistent Sie auf, Net Service-Namen in der Datei tnsnames.ora zu konfigurieren.

Konfiguration mit Oracle Net Manager

Mit Oracle Net Manager können Sie Net Service-Namen konfigurieren und:

  • Net Service-Namen für fremde Datenbankdienste erstellen.

  • mehrere Routes zu einem Listener über mehrere Protokolladressen konfigurieren.

  • erweiterte CONNECT_DATA-Informationen für einen Datenbankdienst oder einen Net Service-Namen konfigurieren.

So greifen Sie auf die lokale Benennungskonfiguration zu:

Wählen Sie aus dem Navigator-Fensterbereich den Ordner Lokal > Dienstebenennung, um Net Service-Namen zu erstellen oder Verbindungs-Deskriptor-Informationen zu ändern.

Weitere Informationen finden Sie im Kapitel Configuring Naming Methods im Oracle10i Net Administrator's Guide.

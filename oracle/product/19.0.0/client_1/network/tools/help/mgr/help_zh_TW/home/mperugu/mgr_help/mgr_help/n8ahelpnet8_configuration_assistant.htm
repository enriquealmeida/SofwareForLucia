Oracle Net 組態輔助程式

一種可以在安裝之後設定基本元件的後續安裝工具，包括：

  • 監聽器名稱和協定位址

  • 從屬端用來解析連線 ID 的命名方法

  • tnsnames.ora 檔案中的網路服務名稱

  • 目錄伺服器用途

「Oracle Net 組態輔助程式」會自動在軟體安裝之後執行，如 Oracle 安裝手冊所述。它可以用於從屬端或資料庫伺服器。

「Oracle Net 組態輔助程式」也可以在獨立模式下執行，設定命名方法用途、監聽器、tnsnames.ora檔案中的網路服務名稱，以及目錄伺服器用途。

在獨立模式下啟動「Oracle Net 組態輔助程式」：

  • 如果是在 UNIX，請執行 $ORACLE_HOME/bin 中的 netmgr。

  • 如果是在 Windows 作業系統，請選擇開始 > 程式集 > Oracle - HOME_NAME > 組態與移轉工具 > Net 組態輔助程式。

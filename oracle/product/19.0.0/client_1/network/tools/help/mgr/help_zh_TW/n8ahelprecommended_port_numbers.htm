建議的連接埠號碼

Oracle Corporations 建議讓 TCP/IP 和 TCP/IP 與 SSL 協定位址使用以下連接埠號碼：

連接埠 描述
1521 監聽器之從屬端連線所使用的預設監聽連接埠。在較新的版本中，此連接埠號碼可能會變更成正式註冊連接埠號碼，例如 TCP/IP 使用 2483，TCP/IP 與 SSL 使用 2484。
1575 Oracle Names 伺服器之從屬端連線所使用的預設及正式註冊監聽連接埠。
1630 「Oracle 連線管理程式」之從屬端連線所使用的預設及正式註冊監聽連接埠。
1830 「Oracle 連線管理程式」之管理命令所使用的預設及正式註冊監聽連接埠。

Über Oracle Advanced Security

Mit Oracle Advanced Security wird eine umfassende Suite mit Sicherheitsfunktionen zu Ihrer Oracle-Umgebung hinzugefügt. Diese Suite mit Sicherheitsfunktionen schützt Unternehmensnetzwerke und erweitert Unternehmensnetzwerke auf sichere Weise im Internet. Diese Funktion bietet eine einzige Integrationsquelle mit Netzwerkverschlüsselungs- und Berechtigungsprüfungsmethoden, Single Sign-on-Diensten und Sicherheitsprotokollen. Oracle Advanced Security integriert Industriestandards und verleiht Oracle-Netzwerken eine unvergleichliche Sicherheit.

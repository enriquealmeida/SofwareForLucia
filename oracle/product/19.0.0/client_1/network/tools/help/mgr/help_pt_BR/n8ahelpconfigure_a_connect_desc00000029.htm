Configurando um Descritor de Conexão para Procedimentos Externos

Procedimentos externos são aqueles chamados de outro programa, mas gravados em outro idioma. Um exemplo seria um programa PL/SQL chamar uma ou mais rotinas C necessárias à execução de processamento com finalidades especiais.

Para que o banco de dados Oracle se conecte a um procedimento externo, o servidor de banco de dados deverá ser configurado com um nome do serviço de rede para acessar procedimentos externos.

Observação: O Assistente de Configuração do Oracle Net cria automaticamente um nome do serviço de rede para procedimentos externos durante a instalação.

Para configurar um servidor de banco de dados para procedimentos externos:

  1. Verifique se o listener está configurado para procedimentos externos, conforme descrito em Configurar Procedimentos Externos para o Listener.

  2. Crie um nome do serviço de rede no arquivo tnsnames.ora do banco de dados. Certifique-se de que as informações sobre descritor de conexão correspondam às informações configuradas para o listener.

  3. Escolha Arquivo > Salvar Configuração de Rede.

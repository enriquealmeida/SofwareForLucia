リスナーの作成

リスナーは、データベースが接続を受信できるように、サーバー上に存在する必要があります。

注意: インストール・プロセス中に、LISTENERという名前のデフォルト・リスナーが作成されます。

リスナーを作成する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「リスナー」を展開します。

  2. ツールバーのプラス(+)ボタンを選択するか、「編集」→「作成」を選択します。「リスナー名の選択」ダイアログ・ボックスが表示されます。

  3. 「リスナー名」フィールドに表示されるリスナー名を受け入れるか、別の名前を入力します。最初のリスナーに対するデフォルトのリスナー名はLISTENERです。後続のリスナーのデフォルト名はLISTENERnになります。リスナー名はノード内で一意にする必要があります。

  4. 「OK」を選択します。

  5. 「リスナー・アドレスの構成」の手順4から6に従って操作します。

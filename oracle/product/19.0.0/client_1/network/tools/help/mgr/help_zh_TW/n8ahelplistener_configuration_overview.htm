監聽器組態簡介

本主題涵蓋下列段落：

安裝期間的組態

在資料庫伺服器安裝企業版或標準版時，「Oracle Net 組態輔助程式」會設定一個稱為LISTENER的監聽器，它有一個 Oracle 資料庫的 TCP/IP 監聽協定位址。在自訂安裝時，「Oracle Net 組態輔助程式」會提示您設定選取的監聽器名稱以及協定位址。

另外，無論安裝類型為何，「Oracle Net 組態輔助程式」都會自動設定外部程序呼叫的監聽 IPC 協定位址。

「Oracle Net 組態輔助程式」也會自動為 listener.ora 檔案中的外部程序設定服務資訊。

使用 Oracle Net Manager 進行設定

您可以使用 Oracle Net Manager 來執行下列動作：

  • 建立監聽器

  • 設定 Oracle8 或先前資料庫的服務資訊

  • 設定非 Oracle 資料庫服務的服務資訊，包括外部程序與異質服務

  • 啟用追蹤與記錄日誌功能

  • 輸入監聽器控制公用程式的密碼認證

存取監聽器組態：

在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

如需詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Configuring and Administering the Listener 章節。

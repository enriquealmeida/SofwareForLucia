設定監聽器記錄日誌和追蹤

您可以使用記錄日誌和追蹤參數來設定監聽器。

日誌資訊通常儲存在$ORACLE_HOME/network/log(UNIX 平台) 及ORACLE_HOME\network\log(Windows 平台) 的listener.log檔中。

追蹤資訊預設儲存在listener.trc檔案中，此檔案位於 UNIX 平台的$ORACLE_HOME/network/trace和 Windows 平台的ORACLE_HOME\network\trace中。

啟用記錄日誌或追蹤：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

  2. 選取監聽器。

  3. 從右窗格的清單中，選取一般參數。

  4. 選取記錄日誌與追蹤頁籤。

  5. 根據需要調整記錄日誌和追蹤設定值。

  6. 選擇檔案 > 儲存網路組態。

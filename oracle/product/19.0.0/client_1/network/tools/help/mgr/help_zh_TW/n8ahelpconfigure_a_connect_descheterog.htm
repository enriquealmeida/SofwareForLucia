設定異質服務的連線描述區

您可以使用「異質服務」從 Oracle 伺服器存取非 Oracle 系統。起始連線至非 Oracle 系統時，Oracle 伺服器會透過閘道上的監聽器啟動代理程式處理作業。若要 Oracle 資料庫連線至非 Oracle 系統，請設定伺服器的網路服務名稱。

設定異質服務的伺服器：

  1. 確定監聽器設定有「異質服務」，請參閱設定監聽器的異質服務。

  2. 在 tnsnames.ora 檔案中建立網路服務名稱，讓 Oracle 資料庫伺服器可連線至非 Oracle 系統。確定連線描述區資訊符合監聽器設定的資訊。

  3. 在區域 > 服務命名資料夾選取網路服務名稱。

  4. 在右窗格的服務識別方塊中，選擇進階。進階服務選項對話方塊就會顯示。

  5. 選擇用於異質服務，然後選擇確定。

  6. 在右窗格中，選擇套用。

  7. 選擇檔案 > 儲存網路組態。

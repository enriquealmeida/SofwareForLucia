Cuadro de Diálogo Opciones de la Lista de Direcciones

El recuadro de diálogo Opciones de la Lista de Direcciones permite especificar el orden y la forma en que las direcciones de protocolo de descriptores de conexión se utilizan en las conexiones.

Pruebe cada dirección, en orden, hasta que funcione una de ellas

Seleccione esta opción para activar una operación de failover de tiempo de conexión para un cliente Oracle9i u Oracle8i. Si no se puede acceder a la primera dirección, la operación de failover permite al cliente utilizar la siguiente dirección de protocolo del listener. Seleccione esta opción junto con la opción Usar Opciones Compatibles con Net8 8.0 Clients para desactivar el direccionamiento de origen en clientes de la versión 8.0 o de versiones anteriores.

Pruebe cada dirección, al azar, hasta que funcione una de ellas

Seleccione para activar el equilibro de carga del cliente y la redirección automática en caso de fallo del tiempo de conexión para los clientes de Oracle9i u Oracle8i.

Pruebe una dirección seleccionada al azar

Seleccione para activar el equilibrio de carga del cliente para los clientes de Oracle9i u Oracle8i.

Utilice cada dirección en orden hasta alcanzar el destino

Seleccione para activar el direccionamiento de origen de un cliente de Oracle8i u Oracle8.

Esta opción es necesaria cuando para alcanzar el destino se necesita más de una parada de dirección. Esta función se suele utilizar para activar las funciones de Oracle Connection Manager.

Este parámetro no es necesario para los clientes de Oracle9i.

Utilice sólo la primera dirección

Seleccione para desactivar las funciones de direccionamiento de origen, redirección automática en caso de fallo del tiempo de conexión y equilibrio de carga del cliente.

Usar Opciones Compatibles con Clientes de Net8 8.0

Seleccione si va a configurar un cliente de la versión 8.0. Sólo podrá seleccionar entre las opciones siguientes:

-Pruebe cada dirección, en orden, hasta que funcione una de ellas

Utilice cada dirección en orden hasta alcanzar el destino

Desactive esta opción si va a configurar un cliente de Oracle9i u Oracle8i.

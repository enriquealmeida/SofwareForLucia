將 tnsnames.ora 檔案項目匯至目錄伺服器

您可以將 tnsnames.ora 檔案中已設定的網路服務名稱匯出至目錄伺服器。一次只可以為一個網域執行匯出。

匯出 tnsnames.ora 檔案時，會在目錄中目前選取的 Oracle Context 建立網路服務名稱。如果需要變更 Oracle Context，請選擇命令 > 目錄伺服器 > 變更 Oracle Context。

將tnsnames.ora檔案中儲存的網路服務名稱，匯至目錄伺服器：

  1. 選擇命令> 目錄 > 匯出服務名稱。「目錄移轉精靈」就會啟動。

  2. 選擇下一步。

    如果偵測到多個網域，「選取網域」頁面就會顯示。請繼續步驟 3。

    如果網路服務名稱不是有效的網域，「選取網路服務名稱」頁面就會顯示。請直接跳到步驟 4。

  3. 選取您想匯出其網路服務名稱的網路網域，然後選擇下一步。「選取網路服務名稱」頁面就會顯示。

  4. 從清單選取要匯出的網路服務名稱，然後選擇下一步。「選取目的地相關資訊環境」頁面就會顯示。

    在「選取目的地相關資訊環境」頁面，執行以下動作：

    a.從目錄命名相關資訊環境清單，選取包含 Oracle Context 的目錄命名相關資訊環境。目錄命名相關資訊環境是目錄子樹狀結構的一部分，包含一或多個內含 Oracle 項目的 Oracle Context。

    b.從 Oracle Context 清單，選取要匯入所選網路服務名稱的 Oracle Context。

    c.選擇下一步。「目錄伺服器更新」頁面就會顯示，內含匯出作業的狀態。

  5. 選取完成，結束「目錄伺服器移轉精靈」。

고급 접속 데이터 옵션 구성

데이터베이스의 서비스 이름 또는 Oracle SID(시스템 식별자) 외에도 접속 기술자의 CONNECT_DATA 섹션에서 필요에 따라 다음 옵션을 구성할 수 있습니다.

  • 특정 데이터베이스 인스턴스에 접속하기 위한 인스턴스 이름

  • 네트워크를 통해 데이터베이스를 시작하기 위한 데이터베이스의 Oracle 홈

  • 버퍼 크기를 지정하기 위한 세션 데이터 단위 크기

  • 전용 서버 프로세스에 접속하기 위한 전용 서버

  • 비Oracle 시스템에 액세스하기 위한 이기종 서비스

  • Oracle RDB 데이터베이스에 액세스하기 위한 Oracle Rdb

네트 서비스 이름에 대한 고급 접속 옵션을 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 디렉토리 > 로컬 또는 서비스 이름 지정을 확장합니다.

  2. 네트 서비스 이름 또는 데이터베이스 서비스를 선택합니다. 오른쪽 창에 현재 대상 서비스와 주소 목록이 표시됩니다.

  3. 서비스 ID 상자에서 고급을 선택합니다. 고급 서비스 옵션 대화상자가 나타납니다.

  4. 필드에 입력하거나 적절한 옵션을 선택한 다음 확인을 선택합니다.

  5. 오른쪽 창에서 적용을 선택합니다.

  6. 로컬 폴더를 이렇게 변경하는 경우 파일 > 네트워크 구성 저장을 선택합니다.

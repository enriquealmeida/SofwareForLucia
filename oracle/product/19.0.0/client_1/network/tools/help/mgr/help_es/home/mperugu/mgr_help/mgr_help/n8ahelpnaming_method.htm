Método de Nomenclatura

Método de resolución utilizado por una aplicación cliente para resolver un identificador de conexión en un descriptor de conexión al conectarse a un servicio de la base de datos. La Red de Oracle soporta cuatro métodos de nomenclatura:

  • Nomenclatura Local

  • Nomenclatura de Directorios

  • Nomenclatura de conexión sencilla (denominado anteriormente Nomenclatura del host)

  • Nomenclatura Externa

Nombre de la Base de Datos Global

Nombre completo de la base de datos que se identifica de forma única desde cualquier otra base de datos. El nombre de la base de datos global tiene el formato "basededatos_nombre.basededatos_dominio", por ejemplo, sales.us.example.com.

La parte del nombre de la base de datos, sales, es el nombre sencillo que desea asignar a la base de datos. La parte del dominio de la base de datos, us.example.com , especifica el dominio en el que se ubica la base de datos, con lo cual, la base de datos global tiene asignado un nombre único. Si es posible, Oracle recomienda que el dominio de la base de datos refleje el dominio de la red.

El nombre de la base de datos global es, por defecto, el nombre del servicio de base de datos, como se especifica en el parámetro SERVICE_NAMES del archivo de parámetros de inicialización.

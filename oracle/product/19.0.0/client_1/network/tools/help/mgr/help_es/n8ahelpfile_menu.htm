Menú Archivo

El menú Archivo permite abrir, restablecer, guardar y guardar como.

Abrir Configuración de Red

Abre archivos de configuración local.

Guardar Configuración de Red

Guarda la configuración actual en la ubicación por defecto.

Guardar como

Guarda la configuración actual en una ubicación que no es la ubicación por defecto.

Volver a la Configuración Guardada

Seleccione esta opción para no guardar la configuración actual. En su lugar, recupera la última configuración guardada.

Salir

Cierra el Gestor de Red de Oracle.

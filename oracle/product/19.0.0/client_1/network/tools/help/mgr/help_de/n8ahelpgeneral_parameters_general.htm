Allgemeine Parameter: Allgemein

Im Register Allgemein können Sie allgemeine Parameter für den Listener konfigurieren.

Listener-Name

Zeigt den Namen des Listeners an.

Wartezeit für Start

Geben Sie die Zeit in Sekunden an, die der Listener warten soll, bevor er auf einen START-Befehl des Listener Control-Utilitys antwortet.

Hinweis: Diese Einstellung ist veraltet und wird in einem zukünftigen Release nicht mehr unterstützt. Wenn dieser Parameter erforderlich ist, um den Listener auszuführen, unterrichten Sie Oracle Support Services.

Konfiguration beim Herunterfahren speichern

Wählen Sie diese Option, damit die mit dem Befehl SET des Utilitys Listener Control vorgenommenen Änderungen festgeschrieben werden.

SNMP-Kontaktinformationen

Geben Sie einen beliebigen Text ein, der in die Datei snmp.ora zur Benutzung mit Oracle Enterprise Manager geschrieben werden soll. Diese Informationen, bei denen es sich um eine beliebige Zeichenfolge handeln kann, werden mit der SNMP-Verbindung gesendet. Sie wirken sich nicht auf das Erstellen einer SNMP-Verbindung aus. Diese Informationen bestehen im Allgemeinen aus Kontaktinformationen für den Benutzer oder Administrator des Clients.

ADMIN_RESTRICTIONS_listener_name

Begrenzt die Laufzeitadministration des Listeners.

Die Standardeinstellung ist off. Durch Festlegung von ADMIN_RESTRICTIONS_listener_name=on wird die Laufzeitänderung von Parametern in listener.ora deaktiviert. Das heißt, der Listener akzeptiert keine SET-Befehle, die seine Parameter ändern. Um einen der Parameter in listener.ora zu ändern, einschließlich ADMIN_RESTRICTIONS_listener_name selbst, ändern Sie die Datei listener.ora manuell und laden die Parameter (mit dem Befehl RELOAD) neu, damit die Änderungen wirksam werden, ohne den Listener explizit zu stoppen und neu zu starten.

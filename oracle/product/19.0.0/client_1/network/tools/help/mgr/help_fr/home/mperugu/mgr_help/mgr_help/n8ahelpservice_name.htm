Nom de service

Représentation logique d'une base de données, ou façon dont celle-ci est présentée aux clients. Une base de données peut être présentée sous forme de services multiples, et un service peut être mis en oeuvre sous forme d'instances de base de données multiples. Le nom de service est une chaîne correspondant au nom global de base de données, c'est-à-dire un nom composé du nom de base de données et du nom de domaine, entrés lors de l'installation ou de la création de la base de données. Si vous n'êtes pas sûr du nom global de base de données, consultez les valeurs combinées du paramètre SERVICE_NAMES dans le fichier de paramètres d'initialisation.

Les descripteurs de connexion que les clients utilisent pour se connecter à une base de données Oracle9i ou Oracle8i doivent être configurés avec le paramètre SERVICE_NAME (sans S).

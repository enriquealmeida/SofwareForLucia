Datenbanklink

Ein Zeiger, der einen einseitigen Kommunikationspfad von einem Oracle Datenbank-Server zu einem anderen Datenbank-Server definiert. Der Link-Zeiger ist als Eintrag in einer Data Dictionary-Tabelle definiert. Um auf den Link zuzugreifen, müssen Sie bei der lokalen Datenbank angemeldet sein, die den Data Dictionary-Eintrag enthält.

Eine Datenbank-Link-Verbindung ist insofern einseitig, als ein Client, der bei der lokalen Datenbank A angemeldet ist, mit einem Link, der in Datenbank A gespeichert ist, auf Informationen in einer entfernten Datenbank B zugreifen kann, Benutzer, die bei Datenbank B angemeldet sind, jedoch nicht denselben Link für den Zugriff auf Daten in Datenbank A verwenden können. Wenn lokale Benutzer in Datenbank B auf Daten in Datenbank A zugreifen möchten, müssen sie einen Link definieren, der im Data Dictionary von Datenbank B gespeichert ist.

Die folgenden Typen von Datenbank-Links werden unterstützt:

  • Ein privater Datenbank-Link in einem bestimmten Schema einer Datenbank. Nur der Eigentümer eines privaten Datenbank-Links kann diesen verwenden.

  • Ein öffentlicher Datenbank-Link für die Datenbank. Alle Benutzer in der Datenbank können diesen verwenden.

Weitere Informationen über das Erstellen von privaten und öffentlichen Datenbank-Links finden Sie im Oracle10i Database Administrator's Guide.

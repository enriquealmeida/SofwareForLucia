一般：路由

您可以使用路由選擇頁籤隨時將連線要求從此電腦導向特定的處理作業。

一律使用專用伺服器

選擇此選項可強制監聽器為此從屬端的所有網路階段作業衍生專用伺服器。只要將(SERVER=dedicated)新增至連線描述區的CONNECT_DATA部分，即可達到此目的。如此一來，即使有設定共用伺服器，來自此從屬端的連線仍會使用專用伺服器。

使用此選項可讓連線變快，但伺服器速度卻會變慢。Oracle Corporation 建議您要在很短的時間內進行大量資料庫交易時，使用此選項。

注意：如果啟用此選項，它會取代服務識別特性表中的連線類型選項。

偏好的連線管理員路由選擇

選擇此選項可在可行時將從屬端傳送至 Oracle 連線管理程式。如果沒有可用的「Oracle 連線管理程式」位址，則連線會透過任何可用的監聽器位址來進行。

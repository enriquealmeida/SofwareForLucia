簡易連線命名

一種命名方法，可讓從屬端在不需設定任何組態的情況下連線資料庫伺服器。從屬端使用的是簡易 TCP/IP 位址，由主機名稱和選擇性的連接埠及服務名稱所組成：

CONNECTusername/password@host[:port][/service_name]

如需有關使用簡易連線命名方法的資訊，請參閱 Oracle10i Net Services Administrator's Guide 中的 Configuring Naming Methods 章節。

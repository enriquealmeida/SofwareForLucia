Résolution de noms Easy Connect

Méthode de résolution de noms permettant à des clients de se connecter à un serveur de base de données sans configuration. Les clients utilisent une adresse TCP/IP simple, composée d'un nom d'hôte et, éventuellement, d'un port et d'un nom de service :

CONNECT NomUtilisateur/MotDePasse@hôte[:port][/nom_service]

Pour obtenir des informations sur l'utilisation de la méthode de résolution de noms Easy Connect, reportez-vous au chapitre Configuring Naming Methods dans le Guide de l'administrateur Oracle10i Net Services.

Qualificador de Link

Um qualificador acrescentado a um link de banco de dados global para proporcionar definições alternativas para as credenciais de nome do usuário e senha. Por exemplo, um qualificador de link emp pode ser anexado a um link de banco de dados global sales.us.example.com. Uma conexão com um banco de dados remoto com o link de banco de dados global e o qualificador de link seria:

CONNECT scott/tiger@sales.example.com@emp

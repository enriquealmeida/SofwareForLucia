Adresse IP

Adresse servant à identifier un noeud sur un réseau. Une adresse IP unique est affectée à chaque ordinateur sur le réseau. Elle se compose d'un ID de réseau, ainsi que d'un ID d'hôte unique. Cette adresse est représentée généralement dans la notation décimale à point, c'est-à-dire que la valeur décimale de chaque octet est séparée par un point, par exemple 144.45.9.22.

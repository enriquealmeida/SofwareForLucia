Nome servizio

Una rappresentazione logica di un database, che corrisponde al modo in cui un database viene presentato ai client. Un database può essere presentato come più servizi e un servizio può essere implementato come più istanze del database. Il nome di servizio è una stringa che corrisponde al nome del database globale, ossia un nome formato dal nome del database e da quello del dominio, immesso durante l'installazione o la creazione del database. Se non si conosce il nome del database globale, è possibile ottenerlo dai valori combinati del parametro SERVICE_NAMES nel file dei parametri di inizializzazione.

I descrittori di connessione utilizzati dai client per connettersi a un database Oracle9i o Oracle8i devono essere configurati con il parametro SERVICE_NAME (senza la S).

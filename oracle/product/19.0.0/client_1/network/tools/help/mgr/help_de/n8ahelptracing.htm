Tracing

Eine Funktion, die detaillierte Informationen über einen Vorgang in eine Ausgabedatei schreibt. Die Trace-Funktion erzeugt eine detaillierte Sequenz von Anweisungen, die die Ereignisse eines Vorgangs während der Ausführung beschreiben. Administratoren verwenden die Trace-Funktion, um anormale Bedingungen zu diagnostizieren. Diese Funktion ist normalerweise nicht eingeschaltet. Durch das Tracing eines Vorgangs können Sie mehr Informationen über die internen Vorgänge einer Komponente erhalten als mit dem Logging.

Hinweis: Das Tracing belegt viel Speicherplatz auf der Festplatte und kann sich wesentlich auf die Leistung auswirken. Deshalb sollten Sie das Tracing nur aktivieren, wenn dies unbedingt erforderlich ist.

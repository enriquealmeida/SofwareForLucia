Überblick über Oracle Net Services-Konfiguration

Mit Oracle Net Manager können Sie viele Elemente des Netzwerks, einschließlich Benennungsmethoden, Client- und Server-Profilen und Listenern konfigurieren.

Benennungsmethoden

Definieren Sie einfache Namen zur Identifzierung der Adresse eines Dienstes, wie z.B. einer Datenbank. Diese einfachen Namen, die als Connect-Bezeichner bezeichnet werden, werden Verbindungsdeskriptoren zugeordnet, die die Netzwerkadresse und Identifikation des Dienstes enthalten.

Mit Oracle Net Manager können Sie verschiedene Möglichkeiten konfigurieren, mit denen Connect-Bezeichner über Benennungsmethoden in Verbindungsdeskriptoren aufgelöst werden.

Client- und Server-Profile

Konfigurieren Sie Voreinstellungen für die Aktivierung und Konfiguration von Oracle Net-Funktionen auf Clients und Server über Profile. Mit Profilen können Sie Voreinstellungen festlegen, mit denen die Prioritätsfolge von Benennungsmethoden festgelegt wird, die Funktionen von Oracle Advanced Security aktivieren, Logging und Tracing aktivieren oder Verbindungen weiterleiten.

Listener-Konfiguration auf dem Server

Konfigurieren Sie den Listener so, dass Clientverbindungen empfangen werden. Der Listener muss wie folgt konfiguriert werden:

  • Mit einer oder mehreren Listening-Protokolladressen

  • Mit Dienstinformationen über den Zieldienst

Protokolladressen werden in der Listener-Konfigurationsdatei, listener.ora, konfiguriert. Dienstinformationen können in der Datei listener.ora konfiguriert werden oder nicht:

  • Eine Oracle8i- oder höhere Datenbank registriert automatisch bestimmte Informationen mit dem Listener, wie z.B. Servicenamen, Instanznamen, Service Handler und Lastinformationen. Dieses Feature, das als Serviceregistrierung bezeichnet wird, erfordert keine Konfiguration in der Datei listener.ora.

  • Andere Dienste, einschließlich einer Oracle7- oder Oracle8-Datenbank, erfordern die Dienstkonfiguration der Datei listener.ora .

Weitere Informationen über Oracle Names finden Sie in "Überblick über Oracle Names"

Weitere Informationen finden Sie im Kapitel Configuring Naming Methods im Oracle10i Net Administrator's Guide.

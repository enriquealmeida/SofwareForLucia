Datei listener.ora

Eine Konfigurationsdatei für den Listener, mit der folgende Angaben gemacht werden:

  • Listener-Name

  • Protokolladressen, auf denen Verbindungsanforderungen akzeptiert werden

  • Dienste (Datenbank- und Nicht-Datenbank-Dienste), auf die gehört wird

Die Datei listener.ora muss in einem der folgenden Verzeichnisse gespeichert sein:

  1. Das Verzeichnis, das mit der Umgebungsvariablen TNS_ADMIN angegeben wird. Wenn die Umgebungsvariable TNS_ADMIN nicht als Variable in Windows NT gespeichert ist, kann sie in der Registrierung gespeichert sein.

  2. Das globale Konfigurationsverzeichnis des Knotens. Bei Sun Solaris ist dieses Verzeichnis /var/opt/oracle. Windows NT hat kein zentrales Verzeichnis.

  3. Das Verzeichnis $ORACLE_HOME/network/admin bei UNIX, oder das Verzeichnis ORACLE_HOME\network\admin bei Windows-Betriebssystemen.

Hinweis: Oracle9i-und Oracle8i-Datenbanken erfordern keine Identifikation des Datenbankdienstes wegen der Dienstregistrierung. Allerdings ist eine statische Dienstkonfiguration erforderlich, wenn Sie die Datenbank mit Oracle Enterprise Manager überwachen möchten.

Oracle Internet Directory

Um serviço de diretório implementado como uma aplicação no banco de dados Oracle. Permite a recuperação de informações sobre usuários dispersos e recursos de rede. Combina o LDAP (Lightweight Directory Access) Versão 3, o protocolo aberto padrão de acesso a diretório da Internet, com o alto desempenho, escalabilidade, robustez e disponibilidade do banco de dados Oracle.

Para obter mais informações, consulte Oracle Internet Directory Administrator's Guide.

리스너 로깅 및 추적 구성

로깅 및 추적 매개변수로 리스너를 구성할 수 있습니다.

로그 정보는 일반적으로 UNIX의 경우 $ORACLE_HOME/network/log , Windows 플랫폼의 경우 ORACLE_HOME\network\log 에 있는 listener.log 파일에 저장됩니다.

추적 정보는 기본적으로 UNIX의 경우 $ORACLE_HOME/network/trace , Windows 플랫폼의 경우 ORACLE_HOME\network\trace 에 있는 listener.trc 파일에 저장됩니다.

로깅 및 추적을 사용으로 설정하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬> 리스너를 확장합니다.

  2. 리스너를 선택합니다.

  3. 오른쪽 창에 있는 목록에서 일반 매개변수를 선택합니다.

  4. 로깅 및 추적 탭을 선택합니다.

  5. 필요에 따라 로깅과 추적 설정을 조정합니다.

  6. 파일 > 네트워크 구성 저장을 선택합니다.

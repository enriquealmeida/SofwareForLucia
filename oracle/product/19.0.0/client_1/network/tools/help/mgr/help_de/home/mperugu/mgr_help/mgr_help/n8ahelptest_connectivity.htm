Konnektivität testen

Sie können die Konnektivität eines ausgewählten Dienstebenennungsobjektes zu einem Datenbankdienst überprüfen, indem Sie einen Konnektivitätstest durchführen. Während eines Konnektivitätstests wird eine Verbindung zu dem Datenbankdienst mit den konfigurierten Verbindungsdeskriptor-Informationen hergestellt.

So prüfen Sie Informationen zur Netzwerkkonnektivität:

  1. Stellen Sie sicher, dass Listener und Datenbank hochgefahren sind.

  2. Blenden Sie im Navigator-Fensterbereich Oracle Net-Konfiguration > Verzeichnis oder Lokal >Dienstebenennung ein.

  3. Wählen Sie einen bestehenden Net Service-Namen oder Datenbankdienst.

  4. Klicken Sie auf das Datenbanksymbol in der Symbolleiste, oder wählen Sie Befehl > Dienst testen. Daraufhin wird das Dialogfeld "Verbindungstest" angezeigt und aktualisiert, während die Testergebnisse generiert werden. Bei einem erfolgreichen Test wird folgende Meldung ausgegeben:

Der Verbindungstest war erfolgreich.

Informationen zum Starten der Komponenten finden Sie im Kapitel Establishing a Connection and Testing the Network im Oracle10i Net Administrator'sGuide.

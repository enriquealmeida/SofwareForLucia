Servidor de Diretórios

Um servidor de diretório que é acessado com o LDAP (Lightweight Directory Access Protocol). Um diretório compatível com o LDAP pode fornecer armazenamento e recuperação centralizados de componentes de rede, preferências de políticas corporativas e de usuário, informações sobre autenticação de usuário e segurança, substituição de arquivos localizados no cliente e no servidor.

O Oracle Net suporta os seguintes serviços de diretório:

  • Oracle Internet Directory

  • Microsoft Active Directory

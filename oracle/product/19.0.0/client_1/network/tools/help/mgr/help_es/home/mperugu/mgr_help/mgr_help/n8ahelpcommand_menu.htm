Menú Comando

El menú Comando permite realizar las siguientes operaciones:

Directorio

Este submenú contiene comandos que realizan acciones en un servidor de directorios:

Elemento Descripción
Cambiar Contexto Actual Permite seleccionar otro Oracle Context que no sea el actual. Oracle Context es un árbol secundario del directorio que almacena las entradas de Oracle, incluidos servicios de bases de datos y nombres de servicios de red para su uso con la nomenclatura de directorios. Al seleccionar un nuevo Oracle Context, aparecen los objetos de la carpeta Configuración de Red de Oracle > Directorio > Nomenclatura de Servicios del navegador para el nuevo Oracle Context.
Definir Autenticación Permite definir credenciales de autenticación para el servidor de directorios.
Exportar Nombres de Servicio de Red Permite iniciar el Asistente de Migración del Servidor de Directorios. Este asistente exporta los nombres de servicio de red almacenados en un archivo tnsnames.ora a un servidor de directorios.

Nota: El menú Directorio sólo aparece si se ha configurado el acceso al directorio. Para obtener más información sobre la configuración del acceso a directorios y otros requisitos, consulte Requisitos de Nomenclatura de Directorios para Oracle Net Manager.

Probar Servicio

Puede conectarse a un servicio de base de datos mediante la información del descriptor de conexión almacenada en el archivo tnsnames.ora o a un servidor de directorios.

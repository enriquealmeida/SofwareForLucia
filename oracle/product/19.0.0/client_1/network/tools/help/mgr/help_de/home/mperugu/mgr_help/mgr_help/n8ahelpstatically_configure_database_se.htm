Datenbankdienstinformationen statisch konfigurieren

Damit der Listener Clientanforderungen für eine Oracle8- oder Oracle7-Datenbank akzeptiert, muss die Datei listener.ora mit Informationen über die Datenbank konfiguriert sein.

Wenn es sich bei der Datenbank um eine Oracle9i- oder Oracle8i-Datenbank handelt, sind keine statischen Dienstinformationen in der Datei listener.ora erforderlich. Dienstinformationen werden automatisch über die Dienstregistrierung mit dem Listener registriert. Wenn Sie jedoch Oracle Enterprise Manager verwenden, sind statische Informationen für eine Oracle8i-Datenbank erforderlich.

So konfigurieren Sie Datenbankdienstinformationen für einen Listener:

  1. Blenden Sie Oracle Net-Konfiguration > Lokal> Listener im Navigator-Fensterbereich ein.

  2. Wählen Sie einen Listener.

  3. Wählen Sie aus der Liste im rechten Fensterbereich Datenbankdienste.

  4. Klicken Sie auf Datenbank hinzufügen. Daraufhin wird ein neues Register Datenbank angezeigt.

  5. Geben Sie den globalen Datenbanknamen in das Feld Globaler Datenbankname ein, den Speicherort des Oracle-Standardverzeichnisses in das Feld Oracle-Standardverzeichnis und den Oracle System Identifier (SID) der Datenbank in das Feld SID ein.

  6. Wählen Sie Datei >Netzwerkkonfiguration speichern.

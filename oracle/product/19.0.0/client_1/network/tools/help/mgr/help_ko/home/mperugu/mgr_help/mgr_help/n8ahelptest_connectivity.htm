접속 테스트

접속 테스트를 수행하여 선택된 서비스 이름 지정 객체의 데이터베이스 서비스 접속을 검증할 수 있습니다. 구성된 접속 기술자 정보를 사용하여 접속 테스트를 수행할 때 데이터베이스 서비스에 대한 접속을 구현합니다.

네트워크 접속 정보를 확인하려면 다음과 같이 하십시오.

  1. 리스너와 데이터베이스가 실행 중인지 확인합니다.

  2. 네비게이터 창에서 Oracle Net 구성 > 디렉토리 또는 로컬 > 서비스 이름 지정을 확장합니다.

  3. 기존의 네트 서비스 이름이나 데이터베이스 서비스를 선택합니다.

  4. 도구 모음에서 데이터베이스 심볼을 선택하거나 명령 > 서비스 테스트를 선택합니다. 생성된 테스트 결과에 따라 [접속 테스트] 대화상자가 표시되고 업데이트됩니다. 테스트를 성공하면 다음 메시지가 나타납니다.

접속 테스트를 성공했습니다.

구성 요소 시작에 대한 자세한 내용은 Oracle10i Net Administrator'sGuide의 Establishing a Connection and Testing the Network 장을 참조하십시오.

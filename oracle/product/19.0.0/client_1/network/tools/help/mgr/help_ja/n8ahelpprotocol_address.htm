プロトコル・アドレス

ネットワーク・オブジェクトのネットワーク・アドレスを識別するアドレス。

接続の確立時に、クライアントとリクエストの受信者(リスナー、Oracle Names Server、Oracle Connection Managerなど)は、同じプロトコル・アドレスを使用して構成されます。クライアントはこのアドレスを使用して特定のネットワーク・オブジェクト位置に接続リクエストを送信し、受信者はこのアドレスに基づきリクエストをリスニングします。クライアントおよび接続受信者が、同じアドレスを構成すること、および同じプロトコルをインストールすることが重要です。

プロトコル・アドレスは、ADDRESSおよびADDRESS_LIST要素で構成されます。

Rifiutato

Non attivare il servizio di sicurezza anche se l'altro lato lo richiede.

La propria parte della connessione indica che il servizio di sicurezza non è consentito. Se l'altra parte specifica necessario, la connessione termina con un messaggio di errore ORA-12650. Se l'altra parte è impostata su richiesto, accettato orifiutato, la connessione continua senza errori e senza l'attivazione del servizio di sicurezza.

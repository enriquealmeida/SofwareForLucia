Modificación de Información de Ruta de Red para un Servicio de Base de Datos

Nota: Sólo los usuarios que son miembros de los grupos OracleNetAdmins u OracleContextAdmins pueden modificar información de red para un servicio de base de datos en un directorio.

Al terminar el registro de la base de datos en el directorio, el Asistente de Configuración de Bases de Datos crea una entrada de servicio de base de datos en el directorio. Por defecto, esta entrada contiene información de la ruta de red que incluye la ubicación del listener en una dirección de protocolo. Puede volver a crear esta información, si se ha eliminado, o modificar la ya existente.

Para crear o modificar información de ruta de red para un servicio de base de datos:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Directorio > Nomenclatura de Servicios.

  2. Seleccione el servicio de base de datos. El panel de la derecha muestra el nombre del servicio de destino actual.

  3. En el campo Configuración de Direcciones, haga clic en el signo más (+). Aparece un nuevo separador Dirección.

  4. Seleccione un protocolo e introduzca la información sobre el parámetro de protocolo correspondiente al protocolo seleccionado.

  5. Haga clic en Aplicar en el panel de la derecha.

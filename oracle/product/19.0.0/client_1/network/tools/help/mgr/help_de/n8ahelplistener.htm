Listener

Ein auf dem Server befindlicher Prozess, der eingehende Anforderungen für Clientverbindungen überwacht und den Datenverkehr zum Server verwaltet.

Wenn ein Client eine Netzwerksession mit einem Server anfordert, geht die tatsächliche Anforderung beim Listener ein. Wenn die Client-Informationen mit den Listener-Informationen übereinstimmen, gibt der Listener eine Verbindung zum Server frei.

Configuração do Servidor Durante a Instalação

Uma subárvore que reside totalmente em um servidor de diretório. É uma subárvore contígua, ou seja, ela deve começar em uma entrada que funciona como o ponto máximo da subárvore e que se estende para baixo até as entradas de folha ou referências a contextos de nomeação subordinados. O tamanho pode variar de uma única entrada até a DIT (Directory Information Tree – Árvore de Informações de Diretório) inteira.

Um Oracle Context pode ser criado sob um contexto de nomeação.

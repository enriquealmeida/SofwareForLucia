要求

接続先が許可する場合は、セキュリティ・サービスをオンにしてください。

セキュリティ・サービスが望ましいが必要(必須)ではないと指定されています。セキュリティ・サービスがアクティブになるのは、接続先で「適用」、「要求」または「必要」が指定されている場合です。接続先で使用可能な一致するアルゴリズムが存在しない場合、サービスはアクティブ化されません。接続先で「必要」が指定されており、一致するアルゴリズムが存在しない場合、接続は失敗します。

命令功能表

您可以利用命令功能表來執行下列作業：

目錄

這個子功能表包含在目錄伺服器執行動作的命令：

元素 描述
變更目前的相關資訊環境 選擇此選項可選取與目前的相關資訊環境不同的 Oracle Context。Oracle Context 是目錄中的子樹狀結構，用來儲存 Oracle 項目，這包括目錄命名所使用之資料庫服務與網路服務名稱。選取新的 Oracle Context 時，瀏覽器的 Oracle Net 組態 > 目錄 > 服務命名資料夾會顯示新的 Oracle Context 物件。
設定認證 選擇此選項可設定目錄伺服器的認證證明資料。
匯出網路服務名稱 選擇此選項可啟動「目錄伺服器移轉精靈」。這個精靈會將儲存在tnsnames.ora檔案的網路服務名稱匯出到目錄伺服器。

注意：目錄功能表只會在已經設定存取目錄時才會顯示。如需目錄存取組態及其他需求的詳細資訊，請參閱 Oracle Net Manager 的目錄命名需求。

測試服務

選擇此選項可使用儲存在 tnsnames.ora 檔案或目錄伺服器中的連線描述區資訊，連線至資料庫服務。

關於 Oracle Advanced Security

Oracle Advanced Security 為 Oracle 環境新增完整的安全性功能組件。此安全性功能組件可保護企業網路，使企業網路得以安全地延伸至網際網路。它整合了網路加密與驗證解決方案、Single Sign-On 服務以及安全協定。Oracle Advanced Security 也整合了產業標準，提供 Oracle 網路最強的安全性。

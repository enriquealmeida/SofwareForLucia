Testando a Conectividade

Você pode verificar a conectividade de um objeto de nomeação de serviço a um serviço de banco de dados executando um teste de conectividade. Durante um teste de conectividade, é estabelecida uma conexão com um serviço de banco de dados através da utilização de informações sobre odescritor de conexão configurado.

Para verificar as informações de conectividade de rede:

  1. Certifique-se de que o listener e o banco de dados estão sendo executados.

  2. No painel do navegador, expanda Configuração do Oracle Net > Diretório ou Local > Nomeação de Serviço.

  3. Escolha um nome do serviço de rede ou de serviço de banco de dados existente.

  4. Escolha o símbolo de banco de dados na barra de ferramentas, ou escolha Comando > Testar Serviço. O diálogo Teste de Conexão será exibido e atualizado à medida que os resultados do teste forem gerados. Um teste bem-sucedido resulta na seguinte mensagem:

Teste de conexão bem-sucedido.

Para obter informações sobre como iniciar os componentes, consulte o capítulo Establishing a Connection and Testing the Network, no Oracle 10i Net Administrator'sGuide.

Assistente alla configurazione di Oracle Net

Uno strumento che, dopo l'installazione, consente di configurare i seguenti componenti di base della rete:

  • nomi di listener e indirizzi di protocollo;

  • metodi di denominazione utilizzati dal client per risolvere gli identificativi di connessione;

  • nomi di servizi di rete in un file tnsnames.ora;

  • uso del server delle directory.

L'Assistente alla configurazione di Oracle Net viene avviato automaticamente dopo l'installazione del software, come descritto nella guida di installazione Oracle. Può essere utilizzato sul client o sul database server.

È inoltre possibile eseguire l'assistente in modalità standalone per configurare l'uso dei metodi di denominazione, il listener, i nomi di servizi di rete nel file tnsnames.ora e l'uso del server delle directory.

Per eseguire l'Assistente alla configurazione di Oracle Net in modalità standalone, procedere come segue:

  • In ambiente UNIX eseguire netmgr in $ORACLE_HOME/bin.

  • Nei sistemi operativi Windows, scegliere Start/Avvio > Programmi > Oracle - NOME_HOME > Strumenti di configurazione e migrazione > Assistente alla configurazione di Oracle Net.

Oracle Context

目錄資訊樹狀結構 (DIT) 中 cn=OracleContext 的相對辨別名稱，位於命名相關資訊環境或未發佈的目錄項目底下。Oracle Context 包含用於 Oracle 軟體的項目，例如用於目錄命名的資料庫服務與網路服務名稱。一個目錄中可以有一或多個 Oracle Context。

Oracle Internet Directory 會自動在 DIT 結構的根目錄建立 Oracle Context。此根 Oracle Context 有一個辨別名稱 (DN) 為 (dn:cn=OracleContext)。

當 Oracle Net Manager 啟動時，它會使用「Oracle Net 組態輔助程式」設定的預設 Oracle Context。您可以將 Oracle Context 變更成其他相關資訊環境。至於大部分的 Oracle 建置，您需要建立其他的 Oracle Context。

Qualificatif de lien

Qualificatif ajouté à la fin d'un lien global de base de données en vue de fournir des paramètres supplémentaires pour les informations d'identification et de connexion correspondant au nom utilisateur et au mot de passe de la base de données. Par exemple, le qualificatif de lien emp peut être ajouté à la fin du lien global de base de données sales.us.example.com. La connexion à une base de données distante avec le lien global de base de données et le qualificatif de lien serait comme suit :

CONNECT scott/tiger@sales.example.com@emp

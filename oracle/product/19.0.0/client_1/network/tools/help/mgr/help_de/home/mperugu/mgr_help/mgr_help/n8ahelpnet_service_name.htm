Net Service-Name

Ein einfacher Name für einen Dienst, der als Verbindungsdeskriptor aufgelöst wird. Benutzer lösen eine Verbindungsanforderung aus, indem sie einen Benutzernamen und ein Kennwort mit einem Net Service-Namen in einer Verbindungszeichenfolge für den Dienst übergeben, bei dem sie sich anmelden möchten:

CONNECT Benutzername/Kennwort@Net Service-Name

Net Service-Namen können entsprechend Ihrem Bedarf an verschiedenen Orten gespeichert werden. Beispiele:

  • Lokale Konfigurationsdatei, tnsnames.ora-Datei, auf jedem Client

  • Directory-Server

  • Oracle Names Server

  • Externer Benennungsdienst, einschließlich Network Information Service (NIS)

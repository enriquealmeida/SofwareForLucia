General: Registro

El separador Registro permite definir los parámetros de registro para esta computadora. Si se trata de una computadora cliente, utilice los campos Información del Cliente. Si es un servidor, utilice los campos Información del Servidor.

Directorio Log

Introduzca la ruta de acceso del directorio en el que se escribirán los archivos log. La ruta de acceso por defecto es $ORACLE_HOME/network/log en UNIX y ORACLE_HOME\network\log en plataformas Windows.

Archivo Log

Introduzca el nombre del archivo log en el cliente. El nombre por defecto es sqlnet.log.

推奨ポート番号

TCP/IPおよびSSL付きTCP/IPのプロトコル・アドレスには、次のポート番号を使用することをお薦めします。

ポート 説明
1521 リスナーへのクライアント接続に使用されるデフォルトのリスニング・ポート。将来のリリースでは、このポート番号が正規に登録されたポート番号(TCP/IPの場合は2483、SSL付きTCP/IPの場合は2484)に変更される可能性があります。
1575 Oracle Names Serverへのクライアント接続用に正規に登録されているデフォルトのリスニング・ポート。
1630 Oracle Connection Managerへのクライアント接続用に正規に登録されているデフォルトのリスニング・ポート。
1830 Oracle Connection Managerへの管理コマンド用に正規に登録されているデフォルトのリスニング・ポート。

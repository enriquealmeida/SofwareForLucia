Configuration de méthodes de résolution de noms externes

Vous pouvez configurer dans le profil les paramètres client requis pour les méthodes de résolution de noms externes, dont NIS (Network Information Service).

Pour configurer des paramètres de méthode de résolution de noms externes :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Profil.

  2. Dans la liste du panneau de droite, sélectionnez Affectation de noms.

  3. Sélectionnez l'onglet Externe.

  4. Indiquez une valeur dans le champ approprié pour la méthode de résolution de noms externes que vous utilisez.

  5. Choisissez Fichier > Enregistrer la configuration réseau.

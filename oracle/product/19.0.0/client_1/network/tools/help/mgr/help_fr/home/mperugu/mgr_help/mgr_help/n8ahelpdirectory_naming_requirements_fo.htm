Exigences relatives à la résolution de noms pour Oracle Net Manager

Oracle Net Manager vous permet de créer des informations sur les itinéraires réseau pour un service de base de données et de créer des noms de service réseau dans un serveur d'annuaire.

Pour qu'Oracle Net Manager puisse écrire dans un annuaire :

  • L'annuaire doit être configuré avec un schéma Oracle.

  • L'ordinateur exécutant Oracle Net Manager doit être configuré avec un contexte Oracle par défaut pour pouvoir écrire des entrées et y accéder.

L'assistant Oracle Net Configuration permet de définir ces paramètres, tel que décrit à la rubrique Présentation de la configuration de résolution de noms d'annuaire. Oracle Net Manager n'offre pas cette fonctionnalité.

Si l'accès à l'annuaire n'est pas configuré, le dossier Oracle Net Configuration > Annuaire du panneau du navigateur et le menu Commande > Annuaire ne seront pas disponibles.

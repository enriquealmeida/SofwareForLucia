로컬 이름 지정 구성 개요

이 절에서는 다음 항목을 다룹니다.

관리자 또는 런타임 설치

디렉토리 사용이 구성되어 있지 않은 경우 Oracle Net Configuration Assistant는 tnsnames.ora 파일에 관리자 또는 런타임 클라이언트 설치에 대한 네트 서비스 이름을 구성하라는 메시지를 표시합니다.

사용자 정의 설치

클라이언트나 서버를 사용자 정의 설치할 때 이름 지정 방법으로 로컬 이름 지정을 선택하는 경우 Oracle Net Configuration Assistant는 tnsnames.ora 파일에 네트 서비스 이름을 구성하라는 메시지를 표시합니다.

Oracle Net Manager로 구성

Oracle Net Configuration Assistant를 사용하여 네트 서비스 이름 및 다음 사항을 구성할 수 있습니다.

  • 비Oracle 데이터베이스 서비스에 대한 네트 서비스 이름 생성

  • 여러 프로토콜 주소를 통해 리스너에 대한 다중 경로 구성

  • 데이터베이스 서비스 또는 네트 서비스 이름에 대한 고급 CONNECT_DATA 정보 구성

로컬 이름 지정을 구성하려면 다음과 같이 하십시오.

네비게이터 창에서 로컬 > 서비스 이름 지정 폴더를 선택하여 네트 서비스 이름을 생성하거나 접속 기술자 정보를 수정합니다.

자세한 내용은 Oracle10i Net Administrator's Guide의 Configuring Naming Methods 장을 참조하십시오.

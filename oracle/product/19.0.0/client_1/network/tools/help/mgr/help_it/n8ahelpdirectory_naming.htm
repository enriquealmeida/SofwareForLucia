Denominazione delle directory

Un metodo di denominazione che risolve un servizio di database o un nome di servizio di rete in un descrittore di connessione, memorizzato in un server delle directory centrale.

Una directory consente di amministrare centralmente i servizi di database e i nomi di servizi di rete, semplificando il lavoro di aggiunta o trasferimento dei servizi. Sebbene sia possibile configurare i nomi di servizi di rete come alias di un servizio, la directory può fare riferimento direttamente a un servizio di database senza utilizzare un nome di servizio di rete. Il servizio di database viene aggiunto automaticamente come voce alla directory durante l'installazione.

이기종 서비스

Oracle 데이터베이스 서버에서 비Oracle 시스템에 액세스하기 위한 포괄적 기술을 제공하는 통합 구성 요소입니다. 이기종 서비스를 사용하여 다음 작업을 수행할 수 있습니다.

  • Oracle SQL을 사용하여 비Oracle 시스템에 저장된 데이터를 Oracle 서버에 저장된 데이터처럼 액세스할 수 있습니다.

  • Oracle 프로시저 호출을 사용하여 Oracle 분산 환경에서 비Oracle 시스템, 서비스, 응용 프로그래밍 인터페이스(API)에 액세스할 수 있습니다.

비Oracle 시스템에 대한 접속을 시작하기 위해 Oracle8i 데이터베이스는 게이트웨이의 리스너를 통해 에이전트 프로세스를 시작합니다. Oracle8i 데이터베이스를 에이전트 프로세스에 연결하려면 서버는 에이전트에 액세스할 수 있도록 네트 서비스 이름으로 구성되어야 하고 리스너는 에이전트에 대한 서비스 정보로 구성되어야 합니다.

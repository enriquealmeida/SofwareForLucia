监听程序概览

在服务器 (它代表客户机应用程序接收连接请求) 上的监听程序、进程的帮助下，服务可以协调其会话。配置监听程序，使其在数据库服务或非数据库服务的协议地址上进行“监听”。

协议地址是在数据库服务或非数据库服务的监听程序配置文件 listener.ora 中配置的。Oracle9i 或 Oracle8i 数据库服务将其信息自动注册到监听程序，如服务名、实例名及加载信息。这种名为服务注册的功能不需要在 listener.ora 文件中进行服务配置。其他服务 (包括 Oracle8 和 Oracle7 数据库服务以及 Oracle Enterprise Manager 管理工具) 需要在listener.ora文件中配置服务。

一旦配置了监听程序，就可以用监听程序控制实用程序管理它。

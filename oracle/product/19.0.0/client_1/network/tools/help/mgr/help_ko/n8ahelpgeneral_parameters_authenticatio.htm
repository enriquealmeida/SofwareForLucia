일반 매개변수: 인증

인증 탭을 사용하여 리스너에 대한 보안을 구성할 수 있습니다. 이렇게 리스너 제어 유틸리티에서 실행되는 구성 변경 사항 저장이나 리스너 정지 등 권한이 필요한 작업에는 비밀번호가 필요합니다. Oracle Net Manager를 사용하여 암호화된 비밀번호를 설정할 수 있습니다.

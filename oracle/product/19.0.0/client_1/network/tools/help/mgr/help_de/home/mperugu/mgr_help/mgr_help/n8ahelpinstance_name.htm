Instanzname

Oracle9i und Oracle8i-Datenbankinstanzen werden mit dem Parameter INSTANCE_NAME in der Initialisierungsparameterdatei identifiziert. INSTANCE_NAME entspricht dem Oracle System Identifier (SID) der Instanz. Clients können sich bei einer bestimmten Instanz anmelden, indem sie den Parameter INSTANCE_NAME im Verbindungsdeskriptor angeben.

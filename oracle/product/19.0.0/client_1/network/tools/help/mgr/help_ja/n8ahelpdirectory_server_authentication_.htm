「ディレクトリ・サーバーの認証」ダイアログ・ボックス

「ディレクトリ・サーバーの認証」ダイアログ・ボックスでは、ディレクトリ・サーバー内で選択したOracleコンテキストの認証資格証明を設定できます。

ディレクトリ内で、Oracle Netエントリの表示、追加または変更などのディレクトリ・サーバー操作を実行するには、Oracle Net Managerをディレクトリ・サーバーで認証する必要があります。

ユーザー

ディレクトリへのアクセス権を持つ許可ユーザーを入力します。

エントリを追加または変更する場合は、追加および変更権限が与えられているディレクトリ管理者ユーザーを識別名(DN)形式で入力します。たとえば、c=US,o=EXAMPLE,ou=admin,cn=scottは管理者ScottのDNです。

パスワード

ユーザーのパスワードを入力します。

注意: オブジェクトの表示のみが必要であり、ディレクトリに匿名権限が設定されている場合は、「ユーザー」および「パスワード」フィールドを空白にしてください。

デフォルトでは、Oracle Net Managerは次のいずれかの方法でディレクトリを認証します。

  • システム固有の認証: Windows NTオペレーティング・システムのユーザー名とパスワードが使用されます。

  • 厳密認証: Secure Sockets Layer (SSL)資格証明が使用されます。

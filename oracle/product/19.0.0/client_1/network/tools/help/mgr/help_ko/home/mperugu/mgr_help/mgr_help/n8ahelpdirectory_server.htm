디렉토리 서버

LDAP(Lightweight Directory Access Protocol)로 액세스하는 디렉토리 서버입니다. LDAP 준수 디렉토리는 데이터베이스 네트워크 구성 요소의 중앙 저장 및 검색, 사용자 및 기업 정책 환경설정, 사용자 인증 및 보안 정보 등을 제공하여 클라이언트측 및 서버측의 지역화된 파일을 교체합니다.

Oracle Net은 다음의 디렉토리 서비스를 지원합니다.

  • Oracle Internet Directory

  • Microsoft Active Directory

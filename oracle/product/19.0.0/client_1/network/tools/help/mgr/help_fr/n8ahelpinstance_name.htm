Nom de l'instance

Pour la base de données Oracle9i et Oracle8i, les instances sont identifiées par le paramètre INSTANCE_NAME dans le fichier de paramètres d'initialisation. Le paramètre INSTANCE_NAME correspond au SID Oracle de l'instance. Les clients peuvent se connecter à une instance spécifique en indiquant le paramètre INSTANCE_NAME dans le descripteur de connexion.

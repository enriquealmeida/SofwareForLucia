Configuración de Direcciones de Listener

El listener se debe configurar con una o más direcciones de protocolo. El listener recibe solicitudes de conexión de los clientes configurados con las mismas direcciones.

Para configurar una dirección de protocolo para el listener:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Listeners.

  2. Seleccione un listener.

  3. Seleccione Ubicaciones de Listeners de la lista situada a la derecha del panel.

  4. Haga clic en Agregar Dirección. Aparece un nuevo separador Dirección.

  5. Seleccione un protocolo e introduzca la información sobre el parámetro de protocolo correspondiente al protocolo seleccionado.

    Si se configura el listener para que reciba en TCP/IP para clientes de Red de Oracle, debe seleccionar 1521 o el puerto por defecto. Si no es así, deberá configurar el parámetro LOCAL_LISTENER del archivo de parámetros de inicialización y resolver el nombre del listener mediante un método de nomenclatura.

  6. Si la computadora tiene más de una dirección IP y desea que el listener esté configurado en todas las direcciones IP disponibles, configure TCP/IP o TCP/IP con SSL e introduzca el nombre del host de la computadora en el campo Host.

  7. Puede hacer clic en Mostrar Avanzadas para especificar los valores para concretar el espacio en buffer de E/S de las operaciones de envío y recepción de la sesión.

  8. Seleccione Archivo > Guardar Configuración de Red.

Para obtener más información sobre la configuración de un puerto que no sea por defecto, consulte el capítulo sobre configuración y administración del listener en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

Para obtener más información sobre la configuración del espacio del buffer de E/S, consulte el capítulo sobre optimización del rendimiento, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

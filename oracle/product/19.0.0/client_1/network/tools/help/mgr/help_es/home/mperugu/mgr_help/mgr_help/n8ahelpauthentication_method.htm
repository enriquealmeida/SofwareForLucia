Método de Autenticación

Método de seguridad que permite conseguir un alto nivel de confianza en la identidad de los usuarios, clientes y servidores de los entornos distribuidos. Además, los métodos de autenticación de red ofrecen la ventaja de un único proceso de inicio de sesión para los usuarios. Pueden estar soportados los siguientes métodos de autenticación en Oracle9i y Oracle8i, si está instalado Oracle Advanced Security:

  • CyberSafe

  • KERBEROS(v5)

  • RADIUS

  • SSL

  • Autenticación Nativa de Windows NT

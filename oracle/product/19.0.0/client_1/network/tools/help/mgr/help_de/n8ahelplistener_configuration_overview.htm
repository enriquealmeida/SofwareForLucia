Überblick über die Listener-Konfiguration

In diesem Hilfethema werden die folgenden Abschnitte aufgeführt:

Konfiguration während der Installation

Während der Installation einer Enterprise Edition oder Standard Edition auf dem Datenbank-Server konfiguriert der Oracle Net-Konfigurationsassistent einen Listener mit der Bezeichnung LISTENER , der eine TCP/IP-Listening-Protokolladresse für die Oracle-Datenbank enthält. Während einer Benutzerdefinierten Installation fordert der Oracle Net-Konfigurationsassistent Sie auf, einen Listener-Namen und eine Protokolladresse Ihrer Wahl zu konfigurieren.

Zusätzlich wird automatisch eine Listening-IPC-Protokolladresse für externe Prozeduraufrufe konfiguriert, unabhängig von der Installationsart.

Außerdem konfiguriert der Oracle Net-Konfigurationsassistent automatisch Dienstinformationen für die externen Prozeduren in der Datei listener.ora.

Konfiguration mit Oracle Net Manager

Mit Oracle Net Manager können Sie:

  • einen Listener erstellen.

  • Dienstinformationen für Oracle8 oder frühere Datenbanken konfigurieren

  • Dienstinformationen für fremde Dienste konfigurieren, einschließlich externen Prozeduren und heterogenen Diensten.

  • Tracing- und Logging-Funktionen aktivieren.

  • Kennwortberechtigungsprüfung für das Utility Listener Control aktivieren.

So greifen Sie auf die Listener-Konfiguration zu:

Blenden Sie Oracle Net-Konfiguration >Lokal> Listener im Navigator-Fensterbereich ein.

Weitere Informationen finden Sie im Kapitel Configuring and Administering the Listener im Oracle10i Net Administrator's Guide.

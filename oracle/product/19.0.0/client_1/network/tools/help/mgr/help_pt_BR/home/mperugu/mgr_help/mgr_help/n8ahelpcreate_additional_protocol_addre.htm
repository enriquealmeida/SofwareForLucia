Criando Endereços de Protocolo Adicionais

Um serviço de banco de dados pode ser acessado por mais de uma rota de rede, ou endereço de protocolo.

Para adicionar um endereço de protocolo de rede a um nome do serviço de rede existente, faça o seguinte:

  1. No painel do navegador, expanda Configuração do Oracle Net > Diretório ou Local > Nomeação de Serviço.

  2. Selecione um nome do serviço de rede ou um serviço de banco de dados. O painel direito exibirá o serviço de destino atual e uma lista de endereços.

  3. Na caixa Configuração de Endereço, escolha o sinal de adição (+). Uma nova guia Endereço será exibida.

  4. Selecione um protocolo e especifique informações apropriadas sobre parâmetros de protocolo para o protocolo selecionado.

  5. Ordene os endereços de acordo como quiser na lista usando os botões de seta para a esquerda e de seta para a direita. A menos que várias opções de rede sejam configuradas, apenas o primeiro endereço da lista será contactado. Para configurar outra ordem, consulte Configurar Várias Opções de Endereço.

  6. No painel direito, escolha Aplicar.

  7. Se você estiver fazendo essas alterações na pasta Local, escolha Arquivo > Salvar Configuração de Rede.

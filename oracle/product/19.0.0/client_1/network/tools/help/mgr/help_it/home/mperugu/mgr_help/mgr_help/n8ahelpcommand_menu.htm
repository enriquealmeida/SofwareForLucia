Menu Comando

Il menu Comando consente di eseguire le operazioni riportate di seguito.

Directory

Questo sottomenu contiene i comandi per eseguire operazioni su un server delle directory.

Elemento Descrizione
Modifica contesto corrente Scegliere questa opzione per selezionare un contesto Oracle diverso da quello corrente. Il contesto Oracle è una struttura secondaria della directory in cui sono memorizzate le voci Oracle, compresi i servizi del database e i nomi di servizi di rete da utilizzare con la denominazione delle directory. Quando si seleziona un nuovo contesto Oracle, nella cartella Configurazione di Oracle Net > Directory > Denominazione dei servizi del Navigator vengono visualizzati gli oggetti relativi al nuovo contesto Oracle.
Imposta autenticazione Scegliere questa opzione per impostare le credenziali di autenticazione per il server delle directory.
Esporta nomi di servizi di rete Scegliere di avviare la Migrazione guidata del server delle directory. Questa procedura guidata esporta i nomi di servizi di rete memorizzati in un file tnsnames.ora in un server delle directory.

Nota: il menu Directory viene visualizzato soltanto se è stato configurato l'accesso alle directory. Per ulteriori informazioni sulla configurazione dell'accesso alle directory e sugli altri requisiti, vedere Requisiti di denominazione delle directory per Oracle Net Manager.

Esegui test del servizio

Scegliere questa opzione per connettersi a un servizio di database utilizzando le informazioni relative al descrittore di connessione memorizzate in un file tnsnames.ora o in un server delle directory.

Oracle Net Manager

Benvenuti in Oracle Net Manager. Oracle Net Manager integra le funzioni di configurazione al controllo dei componenti allo scopo di fornire un ambiente integrato per la configurazione e la gestione di Oracle Net.

Utenti di Oracle Net Manager

Oracle Net Manager è uno strumento utilizzato dagli amministratori di rete per gestire le connessioni client ai servizi. Oracle Net Manager consente di studiare in dettaglio la configurazione di aree specifiche della rete.

Configurando o Oracle Advanced Security

Use o Oracle Net Manager para configurar a Segurança Avançada Oracle. Cada página da tab Segurança Avançada Oracle permite que você configure um conjunto de parâmetros separado. O Oracle Advanced Security permite a criptografia e a verificação de integridade de dados, a autenticação aperfeiçoada e o sign-on único. O Oracle Advanced Security também oferece o gerenciamento centralizado de usuários em servidores de diretórios compatíveis com LDAP e sign-on único·baseado em certificado; esta funcionalidade se baseia na SSL (Secure Sockets Layer).

Para configurar o Oracle Advanced Security:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

  2. Na lista do painel direito, selecione Oracle Advanced Security.

  3. Os seguintes painéis de guia são exibidos:

    • Autenticação

    • Outros Parâmetros

    • Integridade

    • Criptografia

    • SSL

  4. Faça alterações de configuração.

  5. Escolha Arquivo > Salvar Configuração de Rede.

As alterações serão salvas no arquivo sqlnet.ora.

Observação: Se o Oracle Advanced Security não for exibido na lista, isso significa que não foi instalado.

Para obter mais informações, consulte o Oracle Advanced Security Administrator's Guide.

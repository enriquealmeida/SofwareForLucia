Modification d'informations sur les itinéraires réseau pour un service de base de données

Remarque : seuls les utilisateurs faisant partie du groupe OracleNetAdmins ou OracleContextAdmins peuvent modifier les informations réseau d'un service de base de données dans un annuaire.

Une fois la base de données inscrite auprès de l'annuaire, l'assistant Configuration de base de données crée une entrée de service de base de données dans l'annuaire. Par défaut, cette entrée contient les informations de route réseau qui incluent l'emplacement du processus d'écoute via une adresse de protocole. Vous pouvez recréer ces données si elle ont été supprimées ou modifier les informations de route réseau existantes.

Pour créer ou modifier des informations sur des itinéraires réseau pour un service de base de données :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Annuaire > Résolution de noms de service.

  2. Sélectionnez le service de base de données. Le panneau de droite affiche le nom du service de destination courant.

  3. Dans la case Configuration de l'adresse, choisissez plus (+). Un nouvel onglet Adresse apparaît.

  4. Sélectionnez un protocole et indiquez les informations appropriées sur les paramètres de protocole du protocole sélectionné.

  5. Dans le panneau de droite, choisissez Appliquer.

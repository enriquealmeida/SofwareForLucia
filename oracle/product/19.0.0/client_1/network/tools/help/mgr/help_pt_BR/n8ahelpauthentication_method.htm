Método de Autenticação

Um método de segurança que permite que você tenha uma alta confiança na identidade dos usuários, clientes e servidores em ambientes distribuídos. Os métodos de autenticação de rede também podem oferecer o benefício do sign-on único para os usuários. Os seguintes métodos de autenticação são suportados no Oracle9i e no Oracle8i, dependendo do fato de o Oracle Advanced Security estar ou não instalado:

  • CyberSafe

  • KERBEROS(V5)

  • RADIUS

  • SSL

  • Autenticação nativa do Windows NT

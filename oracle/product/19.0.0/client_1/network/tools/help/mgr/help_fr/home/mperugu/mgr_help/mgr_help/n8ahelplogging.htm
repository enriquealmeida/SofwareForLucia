Journalisation

Fonction permettant d'écrire les erreurs, les activités de service et les statistiques dans un fichier journal. Le fichier journal fournit à un administrateur des informations complémentaires lorsque le message d'erreur affiché à l'écran ne permet pas de comprendre l'incident. Grâce à la pile d'erreurs, ce fichier présente l'état du logiciel à différentes couches.

トレース

操作に関する詳細情報を出力ファイルに書き込む機能。トレース機能では、操作のイベントを実行したとおりに記述する、一連の詳細ステートメントが作成されます。管理者は異常発生時の診断にトレース機能を使用するため、この機能は通常起動されていません。操作をトレースすると、コンポーネントの内部操作について、ロギングよりも詳細な情報を取得できます。

注意: トレース機能は大量のディスク領域を使用し、パフォーマンスに重大な影響を与えます。そのため、トレース機能は必要な場合にのみ使用可能にする必要があります。

Voci di Oracle Net in una directory

Le voci contenute in una directory sono organizzate in una struttura ad albero gerarchica in base al nome distinto (DN). Questa struttura di voci delle directory è denominata DIT (Directory Information Tree). Ogni voce contiene uno o più attributi che la descrivono.

Le voci di Oracle Net in una struttura DIT possono essere le seguenti:

  • un servizio di database creato insieme al database;

  • un nome di servizio di rete;

  • alias di servizio di rete.

Tutte le voci di Oracle Net vengono visualizzate in una struttura secondaria del contesto Oracle che memorizza le voci Oracle. Questa struttura secondaria viene creata in una posizione specificata dall'utente nel DIT nel corso di un'installazione personalizzata sul server. I client sono configurati utilizzando la directory e il contesto Oracle predefinito dal quale verranno ricercate le voci.

Ogni voce contiene uno o più attributi di descrizione. Ogni attributo ha un tipo e un valore. Le voci di directory Oracle Net hanno i seguenti attributi:

  • nome comune;

  • versione della voce;

  • descrittore di connessione.

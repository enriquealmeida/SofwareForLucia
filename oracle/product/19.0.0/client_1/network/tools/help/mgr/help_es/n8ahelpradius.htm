RADIUS

RADIUS (servicio de usuario de marcado de autenticación remota) es un protocolo de seguridad cliente/servidor conocido por activar la autenticación y el acceso remotos. Puede activar la red para utilizar cualquier método de autenticación que soporte el estándar RADIUS (incluidas las tarjetas de identificación y las tarjetas inteligentes); sólo tiene que instalar y configurar el método RADIUS. El método de autenticación RADIUS necesita que defina los siguientes parámetros:

Elemento Descripción
Nombre del Host Nombre de host del servidor RADIUS principal. Por defecto = localhost
Número de Puerto Número de puerto de recepción del servidor RADIUS principal. Por defecto = 1645
Timeout (segundos) Número de segundos que debe esperar Oracle Server para recibir la respuesta procedente del servidor RADIUS principal. Por defecto = 15 segundos
Número de Reintentos Número de veces que el servidor de la base de datos Oracle debe enviar los mensajes al servidor RADIUS principal.

Valor por defecto = 3
Archivo Secreto Ubicación del archivo de la clave secreta en el servidor de Oracle. Por razones de seguridad, Oracle recomienda cambiar este archivo a acceso sólo de raíz.
Enviar Contabilidad Las cuentas de RADIUS registran la información de acceso del servidor de la base de datos Oracle y almacenan esa información en el servidor de cuentas de RADIIUS. Puede utilizar esta función sólo si los dos servidores de autenticación de RADIUS la soportan. Valores = on u off.
Comprobar Respuesta El valor de este campo está activado si desea presentar al usuario una GUI de comprobación de respuesta.
Palabra Clave por Defecto Palabra clave que verifica la identidad de un usuario mediante un mecanismo de comprobación de respuesta. Por defecto = comprobar.
Nombre de Clase de Interfaz Nombre de la clase que gestiona la conversación de respuesta de comprobación entre el cliente de Oracle y el servidor RADIUS. Por defecto = DefaultRadiusInterface.

Para obtener más información, consulte el apéndice sobre integración de dispositivos de autenticación con RADIUS en Oracle Advanced Security Administrator's Guide (Guía del Administrador de Oracle Advanced Security).

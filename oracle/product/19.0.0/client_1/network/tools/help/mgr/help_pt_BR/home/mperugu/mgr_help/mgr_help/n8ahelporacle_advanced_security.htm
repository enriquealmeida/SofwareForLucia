Oracle Advanced Security

Um produto que oferece um conjunto abrangente de recursos de segurança para o ambiente Oracle. Esse conjunto de recursos de segurança protege redes empresariais e permite conexões seguras de redes corporativas com a Internet. Ele possibilita uma única fonte de integração com soluções de criptografia de rede e autenticação, serviços de sign-on único e protocolos de segurança. O Oracle Advanced Security integra padrões do mercado e proporciona uma segurança sem paralelo à rede Oracle e muito mais.

Para obter mais informações, consulte o Oracle Advanced Security Administrator's Guide.

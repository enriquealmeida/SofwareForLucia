KERBEROS(V5)

El método de autenticación KERBEROS(V5) requiere que se establezcan los siguientes parámetros:

Elemento Descripción
Servicio Escriba el nombre del servicio de autenticación: KERBEROS.
Archivo de Caché de Credenciales La ruta de acceso completa del archivo de caché de credenciales de Kerberos, por defecto, depende del sistema operativo.
Archivo de Configuración Ruta de acceso completa del archivo de configuración de Kerberos. Este archivo contiene el dominio del centro de distribución de claves (KDC) por defecto y asigna hosts a hosts de KDC, por defecto, depende del sistema operativo.
Archivo de Traducción de Dominios Ruta de acceso completa del archivo de traducción de dominios de Kerberos. Este archivo asigna un nombre de host o de dominio a un dominio, por defecto, depende del sistema operativo.
Tabla de Claves La ruta de acceso completa al archivo de asignación de claves principales/secretas de Kerberos. Esta ruta de acceso la utiliza el servidor de Oracle para extraer su clave y descifrar la información de autenticación entrante del cliente. Por defecto, depende del sistema operativo.
Diferencia en Segundos Número de segundos después de los cuales la credencial de Kerberos se considera anticuada. Por defecto = 300 segundos.

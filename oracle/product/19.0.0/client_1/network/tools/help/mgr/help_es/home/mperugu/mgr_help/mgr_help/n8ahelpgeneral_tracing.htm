General: Rastreo

El separador Rastreo permite definir los parámetros de rastreo para esta computadora. Si se trata de una computadora cliente, utilice los campos Información del Cliente. Si es un servidor, utilice los campos Información del Servidor.

El rastreo del cliente registra las operaciones realizadas por el cliente. El rastreo del servidor registra las operaciones realizadas por el servidor durante una conexión de red.

Nivel de Rastreo

En la lista, seleccione el nivel de rastreo:

OFF: El rastreo está desactivado. Éste es el valor por defecto.

USER: Se activa un nivel de rastreo adecuado para los usuarios. El rastreo identifica las condiciones de error inducidas por los usuarios.

ADMIN: Se activa un nivel de rastreo adecuado para los administradores de base de datos. El rastreo identifica los problemas específicos de la instalación.

SUPPORT: Se activa un nivel de rastreo adecuado para soporte al cliente.

Los archivos de rastreo pueden llegar a ser muy grandes. Oracle Corporation recomienda mantener el rastreo desactivado cuando no se vayan a solucionar problemas de red.

Directorio de Rastreo

Introduzca el directorio en el que se escribirán los archivos de rastreo. Aunque los archivos de rastreo del cliente y del servidor incluyen información sobre el cliente o el servidor respectivamente, ambos archivos se escribirán en la computadora local. El directorio por defecto está en $ORACLE_ HOME/network/trace en UNIX y ORACLE_HOME\network\trace en Windows.

Archivo de Rastreo

Introduzca el nombre del archivo de rastreo. Por defecto, se utiliza el nombre sqlnet.trc para el cliente y svr_pid..trc para el servidor.

Nombre de Archivo de Rastreo Único

Seleccione esta opción para crear nombres únicos para cada archivo de rastreo. Se agrega un identificador de proceso a cada archivo de rastreo, para que puedan coexistir varios archivos de rastreo. Desactive esta opción si no desea crear archivos de rastreo únicos y desea que toda la información de rastreo se almacene en el mismo archivo de rastreo.

協定參數

監聽器、Oracle Names 伺服器以及 Oracle 連線管理程式均由協定位址識別。想要使用這些服務的從屬端也必須設定相同的位址。下表格說明 Oracle 支援的協定使用的參數：

協定 參數 描述
IPC PROTOCOL 指定 ipc 當成值。
IPC KEY 指定服務的唯一名稱。Oracle Corporation 建議使用服務名稱或服務的 Oracle 系統 ID (SID)。
Named Pipes PROTOCOL 指定 nmp 當成值。
Named Pipes SERVER 指定 Oracle 伺服器電腦的名稱。
Named Pipes PIPE 指定要連線到資料庫伺服器的管路名稱。此名稱可為任意名稱。
TCP/IP 和 TCP/IP 與 SSL PROTOCOL 將 TCP/IP 的值指定為 tcp，或將 TCP/IP 與 SSL 的值指定為 tcps。
TCP/IP 和 TCP/IP 與 SSL HOST 指定電腦的主機名稱或 IP 位址。
TCP/IP 和 TCP/IP 與 SSL PORT 指定監聽連接埠號碼。

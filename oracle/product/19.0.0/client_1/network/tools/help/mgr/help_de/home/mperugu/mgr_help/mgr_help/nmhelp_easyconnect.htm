Easy Connect-Benennung

Eine Benennungsmethode, mit der Clients sich ohne weitere Konfiguration bei einem Datenbank-Server anmelden. Clients verwenden eine einfache TCP/IP-Adresse, die aus einem Hostnamen und einem optionalen Port- und Service-Namen besteht:

CONNECT username/password@host[:port][/service_name]

Weitere Informationen über die Easy Connect-Benennungsmethode finden Sie im Kapitel Configuring Naming Methods im Oracle10i Net Services Administrator's Guide.

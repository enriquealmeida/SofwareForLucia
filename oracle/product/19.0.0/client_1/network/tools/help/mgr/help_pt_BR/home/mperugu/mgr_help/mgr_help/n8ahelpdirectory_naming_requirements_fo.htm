Requisitos de Nomeação de Diretório para o Oracle Net Manager

O Oracle Net Manager permite criar informações sobre rota de rede para um serviço de banco de dados, além de criar nomes de serviços de rede, em um servidor de diretório

Para que o Oracle Net Manager efetue gravações em um diretório.

  • O diretório deve estar configurado com um esquema Oracle.

  • O computador que executa o Oracle Net Manager deve estar configurado com um Oracle Context default, para acessar e gravar entradas.

Essas configurações são definidas com o Assistente de Configuração do Oracle Net, conforme é descrito em Visão Geral da Configuração de Nomeação de Diretório. O Assistente do Oracle Net não oferece essa funcionalidade

Se o acesso a diretório não for configurado, a pasta Configuração do Oracle Net > Diretório no painel do navegador e o menu Comando > Diretório não estarão disponíveis.

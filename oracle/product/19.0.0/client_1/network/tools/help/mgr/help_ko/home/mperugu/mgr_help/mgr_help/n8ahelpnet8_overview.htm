Oracle Net Services 개요

Oracle Net Services는 이기종의 분산 컴퓨팅 환경에서 enterprise 접속 솔루션을 제공합니다. Oracle Net Services는 복잡한 네트워크 구성 및 관리를 쉽게 해주고 성능을 최대화하며 네트워크 진단 기능을 향상시킵니다.

Oracle Net은 Oracle Net Services의 구성 요소로 클라이언트 응용 프로그램에서 오라클 데이터베이스 서버로 네트워크 세션을 활성화합니다. 네트워크 세션이 설정되면 Oracle Net은 클라이언트 응용 프로그램 및 데이터베이스 서버에 대한 데이터 전달자로 작동합니다. 클라이언트 응용 프로그램과 데이터베이스 서버 간의 메시지 교환뿐 아니라 접속 설정 및 유지를 담당합니다. Oracle Net은 네트워크의 각 컴퓨터에 위치해 있기 때문에 이러한 작업을 수행할 수 있습니다.

네트워크 세션은 리스너를 사용하여 설정됩니다. 리스너는 데이터베이스 서버에 상주하는 별도의 프로세스로, 클라이언트 접속 요청을 수신하고 서버에 대한 이러한 접속 요청 통신량을 관리합니다.

리스너는 클라이언트 요청을 서버에 전달하는 중계자 역할을 합니다. 클라이언트가 서버에 네트워크 세션을 요청할 때마다 리스너는 실제 요청을 수신합니다. 클라이언트 정보가 리스너 정보와 일치하면 리스너는 서버 접속을 허용합니다.

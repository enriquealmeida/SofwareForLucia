Oracle Advanced Security: その他のパラメータ

「その他のパラメータ」タブでは、追加の認証サービス・パラメータを構成できます。

「認証サービス」リストから認証方式を選択します。選択した認証方式ごとに、各フィールドに適切な情報を入力します。

KERBEROS (v5)

要素 説明
サービス Kerberosのサービス名の文字列を入力します。
資格証明キャッシュ・ファイル 有効なKerberos資格証明キャッシュ・パス名を入力します。
構成ファイル 有効なKerberos構成ファイルのパス名を入力します。
レルム変換ファイル 有効なKerberosホスト名のパス名を入力します。
キー表 有効なKerberosシークレット・キー・ファイルのパス名を入力します。
時間誤差 送信した資格証明が戻ってくるまでの許容時間を、秒単位で入力します。

CYBERSAFE

GSSAPIサービス: CYBERSAFEサービス・プリンシパルを指定します。サービス・プリンシパル文字列を正しい書式で入力します。

NTS

追加パラメータを入力する必要はありません。

RADIUS

要素 説明
ホスト名 プライマリRADIUSサーバーの名前を入力します。
ポート番号 プライマリRADIUSサーバーのポート番号を入力します。
タイムアウト (秒) Oracleデータベース・サーバーがプライマリRADIUSサーバーからの応答を待つ時間を秒単位で入力します。
再試行回数 Oracleデータベース・サーバーがプライマリRADIUSサーバーにメッセージを再送する回数を入力します。
シークレット・ファイル シークレット・キー・ファイルのパス名を入力します。
アカウンティングの送信 アカウンティングを使用可能にするには「ON」と入力し、使用禁止にするには「OFF」と入力します。
チャレンジ・レスポンス チャレンジ・レスポンスを使用可能にするには「ON」と入力し、使用禁止にするには「OFF」と入力します。
デフォルト・キーワード RADIUSサーバーから要求/応答のキーワードを入力します。
インタフェース・クラス名 OracleクライアントとRADIUSサーバーの間のチャレンジ・レスポンス対話を処理するために作成したクラス名を入力します。

詳細は、Oracle Advanced Security管理者ガイドを参照してください。

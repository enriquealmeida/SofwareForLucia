인증 방식

분산 환경에서 사용자, 클라이언트 및 서버의 ID에 대한 신뢰도를 높일 수 있는 보안 방법입니다. 또한 네트워크 인증 방법은 사용자에게 Single Sign-On의 이점을 제공할 수도 있습니다. Oracle Advanced Security의 설치 여부에 따라 Oracle9i 및 Oracle8i에서는 다음 인증 방법을 지원합니다.

  • CyberSafe

  • KERBEROS(v5)

  • RADIUS

  • SSL

  • Windows NT 원시 인증

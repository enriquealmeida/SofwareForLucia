이기종 서비스에 대한 접속 기술자 구성

이기종 서비스를 사용하여 Oracle 서버에서 비Oracle 시스템에 액세스할 수 있습니다. 비Oracle 시스템에 대한 접속을 시작하기 위해 Oracle 서버는 게이트웨이의 리스너를 통해 에이전트 프로세스를 시작합니다. 오라클 데이터베이스에서 비Oracle 시스템에 접속하려면 네트 서비스 이름으로 서버를 구성합니다.

이기종 서비스에 대해 서버를 구성하려면 다음과 같이 하십시오.

  1. 리스너에 대한 이기종 서비스 구성에 설명된 대로 리스너가 이기종 서비스에 대해 구성되어 있는지 확인합니다.

  2. Oracle 데이터베이스 서버에서 비Oracle 시스템에 접속하는 데 사용할 수 있는 tnsnames.ora 파일에 네트 서비스 이름을 생성합니다. 접속 기술자 정보가 리스너에 대해 구성된 정보와 일치하는지 확인합니다.

  3. 로컬 > 서비스 이름 지정 폴더에서 네트 서비스 이름을 선택합니다.

  4. 오른쪽 창의 서비스 식별 상자에서 고급을 선택합니다. 고급 서비스 옵션 대화상자가 나타납니다.

  5. 이기종 서비스에 사용을 선택한 다음 확인을 선택합니다.

  6. 오른쪽 창에서 적용을 선택합니다.

  7. 파일 > 네트워크 구성 저장을 선택합니다.

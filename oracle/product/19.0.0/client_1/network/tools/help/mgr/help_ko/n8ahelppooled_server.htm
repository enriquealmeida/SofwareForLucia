데이터베이스 상주 접속 풀링

데이터베이스 상주 접속 풀링은 클라이언트 응용 프로그램 또는 프로세스에 "전용" 접속 풀링을 제공합니다. 전용 접속이란 전용 서버와 해당 데이터베이스 세션을 함께 통칭하는 것입니다. 중간 계층과 같은 여러 클라이언트 응용 프로그램이 데이터베이스에 영구 접속을 유지하거나 메모리와 같은 서버 리소스를 최적화해야 하는 경우에 데이터베이스 상주 접속 풀링을 사용합니다.

데이터베이스 상주 접속 풀링에서 클라이언트는 풀링 기능을 제공하는 접속 중개자라는 백그라운드 프로세스에 접속됩니다.

전용 서버 및 공유 서버와 대조됩니다.

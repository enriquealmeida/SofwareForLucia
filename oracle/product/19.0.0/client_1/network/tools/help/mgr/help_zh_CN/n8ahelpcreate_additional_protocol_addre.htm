创建其他协议地址

可以通过多个网络路由或协议地址访问数据库服务。

要将网络协议地址添加到现有网络服务名或数据库服务，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle Net 配置 > 目录/本地 > 服务命名。

  2. 选择网络服务名或数据库服务。右侧窗格显示当前目标服务和地址列表。

  3. 在地址配置框中，选取加号 (+)。将显示新的地址选项卡。

  4. 请选择一个协议并为所选协议输入相应的协议参数信息。

  5. 根据地址列表中地址应处的位置用左箭头和右箭头按钮对地址进行排序。除非配置了多个地址选项，否则只联系列表中的第一个地址。要配置不同的次序，请参阅配置多个地址选项。

  6. 在右侧窗格中，选择应用。

  7. 如果要对本地文件夹进行上述更改，请选择文件 > 保存网络配置。

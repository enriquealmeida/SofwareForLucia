SID (ID di sistema) Oracle

Un nome che identifica un'istanza specifica di un database in esecuzione di una release precedente a Oracle8i. Per ogni database esiste almeno un'istanza che vi fa riferimento.

L'identificazione dei database precedenti alla release Oracle8i avveniva tramite il SID. Il SID era incluso nella parte del descrittore di connessione in un file tnsnames.ora e nella definizione del listener nel file listener.ora. I descrittori di connessione utilizzati dai client per stabilire la connessione a un database Oracle8 o Oracle7 dovevano essere configurati con il parametro SID.

Rastreo

Utilidad que escribe información detallada sobre una operación en un archivo de salida. La utilidad de rastreo genera una secuencia detallada de sentencias que describen los eventos de una operación al ejecutarse. Los administradores utilizan el rastreo para diagnosticar una condición anormal; normalmente, no está activado. Al rastrear una operación puede obtener más información sobre las operaciones internas de un componente que la que se puede obtener mediante el registro.

Nota: El rastreo necesita una gran cantidad de espacio en disco y puede afectar seriamente al rendimiento. Así pues, debe realizar esta operación sólo si es necesario.

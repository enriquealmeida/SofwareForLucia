Oracle Advanced Security: Otros Parámetros

El separador Otros Parámetros permite configurar parámetros de servicios de autenticación adicionales.

En la lista Servicio de Autenticación, seleccione el método de autenticación. Según cuál sea el método que elija, introduzca la información correspondiente en los campos.

KERBEROS(v5)

Elemento Descripción
Servicio Introduzca una cadena para especificar el nombre del servicio Kerberos.
Archivo de Caché de Credenciales Introduzca un nombre de ruta de acceso válido para especificar el nombre de ruta de acceso de la caché de credenciales de Kerberos.
Archivo de Configuración Introduzca un nombre de ruta de acceso válido para especificar el archivo de configuración de Kerberos.
Archivo de Traducción de Dominios Introduzca un nombre de ruta de acceso válido para especificar el nombre de host de Kerberos para el archivo de traducción de dominios.
Tabla de Claves Introduzca un nombre de ruta de acceso válido para especificar el archivo de claves secretas de Kerberos.
Diferencia en Segundos Introduzca un número entero positivo para especificar la diferencia admisible en el número de segundos transcurridos entre el envío y la recepción de una credencial.

CYBERSAFE

Servicio GSSAPI: Especifique el servicio CYBERSAFE principal. Introduzca una cadena principal de servicio formateada correctamente.

NTS

No se necesitan parámetros adicionales.

RADIUS

Elemento Descripción
Nombre del Host Introduzca el nombre del servidor RADIUS principal.
Número de Puerto Introduzca el número de puerto del servidor RADIUS principal.
Timeout (segundos) Introduzca el número de segundos que debe esperar el servidor de la base de datos Oracle para recibir la respuesta procedente del servidor RADIUS principal.
Número de Reintentos Introduzca el número de veces que el servidor de la base de datos Oracle debe reenviar los mensajes al servidor RADIUS principal.
Archivo Secreto Introduzca el nombre de la ruta de acceso del archivo de claves secretas.
Enviar Contabilidad Introduzca on para activar el envío de la contabilidad u off para desactivarlo.
Comprobar Respuesta Introduzca on para activar la comprobación de respuestas u off para desactivarla.
Palabra Clave por Defecto Introduzca la palabra clave para solicitar una comprobación del servidor RADIUS.
Nombre de Clase de Interfaz Introduzca el nombre de la clase que haya creado para gestionar la conversación de comprobación de respuestas entre el cliente Oracle y el servidor RADIUS.

Para obtener más información, consulte Oracle Advanced Security Administrator's Guide (Guía del Administrador de Oracle Advanced Security).

データベース・サービス

「データベース・サービス」 タブでは、リスナーが接続リクエストを受信する必要のあるデータベース・サービスを指定できます。リスナーがクライアントをデータベースに接続するには、この情報が必要です。

Oracle9iおよびOracle8iのインスタンスは動的なサービス登録を使用し、リスナーにデータベース・サービスの情報を通知します。Oracle7およびOracle8リリース8.0データベース、およびOracle Enterprise Managerツールの場合は、listener.oraファイルに静的にサービスを構成する必要があります。

「データベースの追加」を選択して、データベース・サービス情報を追加します。

グローバル・データベース名

初期化パラメータ・ファイル内のSERVICE_NAMESパラメータ値またはDB_NAMEおよびDB_DOMAINパラメータ値の設定のとおり、データベース名とデータベースのドメイン名を入力します。

Oracleホーム・ディレクトリ

データベースのOracleホームの位置を入力します。

注意: Windows NTでは、Oracleホームはすでに設定されています。したがって、Windows NT上のサービスに接続する場合、このフィールドは不要です。

SID

インスタンスのOracleシステム識別子(SID)を入力します。

「データベースの追加」ボタン

別のデータベースのサービス情報を構成します。新しい「データベース」タブが表示され、別のデータベースの情報を入力できます。

「データベースの削除」ボタン

選択した「データベース」タブとそのサービス情報を削除します。

KERBEROS(V5)

La méthode d'authentification KERBEROS(V5) exige que vous définissiez les paramètres suivants :

Elément Description
Service Indiquez le nom du service d'authentification : KERBEROS.
Fichier cache des préférences Chemin complet d'accès au fichier cache des préférences : la valeur par défaut dépend du système d'exploitation.
Fichier de configuration Chemin complet d'accès au fichier de configuration Kerberos. Ce fichier contient le domaine KDC par défaut et établit la correspondance entre les hôtes et les hôtes KDC : la valeur par défaut dépend du système d'exploitation.
Fichier de conversion de domaine Chemin complet d'accès au fichier de conversion de domaine Kerberos. Ce fichier établit la correspondance entre un nom d'hôte, ou un nom de domaine, et un domaine : la valeur par défaut dépend du système d'exploitation.
Table des clés Chemin complet d'accès au fichier de mapping des clés principales/secrètes de Kerberos. Ce chemin est utilisé par le serveur Oracle pour extraire sa clé et décrypter les informations sur les authentifications entrantes fournies par le client. La valeur par défaut dépend du système d'exploitation.
Ecart d'horloge acceptable Nombre de secondes après lequel les préférences de Kerberos sont considérées comme périmées. Valeur par défaut = 300 secondes

Cómo Establecer Prioridades en los Métodos de Nomenclatura

Después de configurar los métodos de nomenclatura, debe establecer prioridades en el perfil. El método del principio de la lista se utiliza primero para resolver un nombre de servicio de red o de servicio de base de datos en un descriptor de conexión. Si no se puede resolver el nombre, se utiliza el segundo método de la lista.

Para especificar el orden de los métodos de nomenclatura:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione Nomenclatura de la lista del panel de la derecha.

  3. Seleccione el separador Métodos.

  4. Seleccione los métodos de nomenclatura de la lista Métodos Disponibles y haga clic en el botón de flecha a la derecha. El método de nomenclatura seleccionado pasa a la lista Métodos Seleccionados.

  5. Ordene los métodos de nomenclatura según el método que Red de Oracle vaya a probar primero para resolver los nombres de servicio de red o de base de datos. Seleccione el método de nomenclatura de la lista Métodos Seleccionados y haga clic en Aumentar o Reducir para que la selección ascienda o descienda en la lista.

  6. Seleccione Archivo > Guardar Configuración de Red.

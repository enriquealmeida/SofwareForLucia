Barra de ferramentas

A barra de ferramentas, localizada no lado esquerdo do Assistente do Oracle Net, contém ícones que correspondem aos objetos de rede nas hierarquias de pastas. Os botões da barra de ferramentas são ativados dependendo dos objetos exibidos ou selecionados em um painel. Move o cursor do mouse sobre um ícone da barra de ferramentas para exibir a descrição da função do ícone.

A barra de ferramentas contém os seguintes ícones:

Ícone Descrição
Adicionar Escolha para criar novos nomes de serviços de rede, listeners e um servidor Oracle Names para este computador.
Excluir Escolha para deletar objetos de listener e de nomeação de serviço.
Verificar Conexão Escolha para verificar a conectividade de um objeto de nomeação de serviço selecionado com um serviço de banco de dados. Durante um teste de conectividade, é estabelecida uma conexão com um serviço de banco de dados utilizando-se informações de descritor de conexão armazenadas em um arquivo tnsnames.oraou em um servidor de diretórios.
Ajuda Escolha para exibir a ajuda on-line.

Observação: Alguns dos comandos só estão disponíveis quando você seleciona o tipo de objeto apropriado. Por exemplo, para criar um listener, primeiro selecione a pasta Listeners.

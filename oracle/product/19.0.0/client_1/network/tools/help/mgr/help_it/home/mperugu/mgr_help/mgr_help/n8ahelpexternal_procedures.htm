Procedure esterne

Una procedura PL/SQL in esecuzione su un server Oracle può richiamare una funzione o una procedura esterna scritta nel linguaggio di programmazione C e memorizzata in una libreria condivisa. Affinché il database Oracle possa connettersi alle procedure esterne, è necessario che il server sia configurato con un nome di servizio di rete e il listener sia configurato con l'indirizzo di protocollo e le informazioni sul servizio.

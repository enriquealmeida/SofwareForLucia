KERBEROS(V5)

KERBEROS(V5) 認證方法會要求您設定以下參數：

元素 描述
服務 輸入認證服務的名稱：KERBEROS。
證明資料快取檔案 Kerberos 證明資料快取檔案的完整路徑：視作業系統而定，會有不同的預設值。
組態檔 Kerberos 組態檔的完整路徑。此檔案包含預設金鑰分送中心 (KDC) 的範圍，並對應主機至 KDC 主機：預設值與作業系統有關。
範圍轉譯檔案 Kerberos 範圍轉譯檔的完整路徑。此檔案會將主機名稱，或網域名稱，對應至範圍：視作業系統而定，會有不同的預設值。
金鑰表格 Kerberos 原則/秘密金鑰對應檔的完整路徑。Oracle 伺服器會使用此路徑來擷取其金鑰，並解密從屬端送來的認證資訊。視作業系統而定，會有不同的預設值。
時脈過期 Kerberos 證明資料在多少秒之後，被視為過期。預設值 = 300 秒。

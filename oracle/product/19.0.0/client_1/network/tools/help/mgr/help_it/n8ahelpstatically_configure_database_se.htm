Informazioni sulla configurazione statica dei servizi del database

Per consentire al listener di accettare le richieste client a un database Oracle8 o Oracle7, è necessario che il file listener.ora venga configurato con le informazioni relative a quel database.

Se si tratta di un database Oracle9i o Oracle8i, le informazioni statiche sul servizio non devono essere necessariamente presenti nel file listener.ora. Le informazioni statiche vengono registrate automaticamente presso il listener durante la registrazione del servizio. Se invece si utilizza Oracle Enterprise Manager, le informazioni statiche sono indispensabili per un database Oracle8i.

Per configurare le informazioni sul servizio del database per un listener, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

  2. Selezionare un listener.

  3. Dalla lista presente nel riquadro di destra, selezionare Servizi di database.

  4. Scegliere Aggiungi database. Viene visualizzata una nuova scheda Database.

  5. Immettere il nome del database globale nel campo Nome di database globale, la posizione della directory Oracle Home nel campo Directory Oracle Home e il SID (ID di sistema) Oracle del database nel campo SID.

  6. Scegliere File > Salva configurazione di rete.

其他服務

您可以使用其他服務頁籤，來指定監聽器要接收連線要求的非資料庫服務。非資料庫服務可以是外部程序或異質服務。

全域服務名稱

輸入服務的服務名稱和網域名稱。

程式名稱

輸入執行檔程式名稱。例如，extproc.exe。

程式引數零

輸入內部第一個引數，它通常當成替代的程式名稱使用。例如：argv0

程式引數

輸入啟動服務時應該傳送的命令行引數。例如，-mode=5。

環境

輸入啟動服務之前應該設定的任何環境變數。環境變數的指定格式如下：

environment_variable=values

例如 user=scott, start=immediate。

注意：變數以逗號分隔。

SID

輸入服務的程式名稱或 Oracle 系統 ID (SID)。

Oracle 本位目錄

輸入程式執行檔的位置。

新增服務按鈕

選擇此按鈕可設定非資料庫服務資訊。新的服務頁籤就會顯示，您可以在此輸入其他服務的資訊。

移除服務按鈕

選擇此按鈕可刪除選取的服務頁籤及其服務資訊。

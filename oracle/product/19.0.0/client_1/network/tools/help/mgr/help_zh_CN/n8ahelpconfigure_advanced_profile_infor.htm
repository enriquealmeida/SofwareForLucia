配置高级概要文件信息

概要文件可以配置为具有下列高级功能：

  • 连接超时值

  • 日志文件中唯一的客户机标识符

  • 清除死进程的信号处理程序

  • 段外中断

要配置高级概要文件信息，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle Net 配置 > 本地 > 概要文件。

  2. 从右侧窗格的列表中选择一般信息。

  3. 选择高级选项卡。

  4. 输入要设置的选项值。

  5. 选择文件 > 保存网络配置。

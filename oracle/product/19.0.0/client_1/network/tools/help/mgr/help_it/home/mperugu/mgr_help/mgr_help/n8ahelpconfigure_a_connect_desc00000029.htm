Configurazione di un descrittore di connessione per le procedure esterne

Le procedure esterne sono procedure richiamate da un altro programma, ma scritte in un linguaggio differente. Un esempio è un programma PL/SQL che richiama una o più routine C necessarie per eseguire elaborazioni con scopi speciali.

Affinché il database Oracle possa connettersi a una procedura esterna, il database server deve essere configurato con un nome di servizio di rete per l'accesso alle procedure esterne.

Nota: durante l'installazione, l'Assistente alla configurazione di Oracle Net crea automaticamente un nome di servizio di rete per le procedure esterne.

Per configurare un database server per le procedure esterne:

  1. Assicurarsi che il listener sia configurato per le procedure esterne, secondo quanto descritto in Configurazione delle procedure esterne per il listener.

  2. Creare un nome di servizio di rete nel file tnsnames.ora sul database. Assicurarsi che le informazioni del descrittore di connessione corrispondano a quelle configurate per il listener.

  3. Scegliere File > Salva configurazione di rete.

建立監聽器

伺服器上必須有監聽器，資料庫才可以接收連線。

注意：安裝處理作業期間會建立名為 LISTENER 的預設監聽器。

建立監聽器：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 監聽器。

  2. 從工具列選擇加號 (+)，或選擇編輯 > 建立。選擇監聽器名稱對話方塊就會顯示。

  3. 接受監聽器名稱欄位中提供的名稱，或以其他名稱編輯此欄位。依照預設，第一個監聽器的名稱為LISTENER。後續的監聽器名稱則預設為 LISTENERn。監聽器名稱必須在所有指定節點上都是唯一的。

  4. 選擇確定。

  5. 完成設定監聽器位址的步驟 4-6。

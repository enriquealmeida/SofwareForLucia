Configurazione dei metodi di autenticazione di Oracle Advanced Security

I metodi di autenticazione centralizzati e protetti consentono di conoscere con certezza l'identità degli utenti, dei client e dei server negli ambienti distribuiti. I metodi di autenticazione di rete offrono inoltre agli utenti i vantaggi della funzione Single Sign-On.

Utilizzare Oracle Net Manager per configurare i seguenti adattatori di autenticazione:

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • Autenticazione originaria di Windows NT

Nota: se il protocollo TCP/IP è stato configurato con il protocollo SSL (Secure Sockets Layer), quest'ultimo verrà configurato per impostazione predefinita. Qualsiasi metodo di autenticazione selezionato in questa scheda avrà la precedenza rispetto alle funzioni di autenticazione SSL. Per configurare le altre opzioni SSL, vedere la scheda SSL. Se è impostato il protocollo TCP/IP con il protocollo SSL, non occorre utilizzare altri metodi di autenticazione.

Per configurare i metodi di autenticazione, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Oracle Advanced Security.

  3. Selezionare la scheda Autenticazione.

  4. Dalla lista Metodi disponibili, selezionare il metodo di autenticazione che si desidera utilizzare e fare clic sul pulsante freccia destra (>).

  5. Ordinare i metodi di autenticazione in base al metodo desiderato. Selezionare il metodo di autenticazione nella lista Metodi selezionati e utilizzare il pulsante Avanza o Retrocedi per spostare la selezione all'interno della lista.

  6. Scegliere la scheda Altri parametri per configurare ulteriori parametri per i seguenti metodi di autenticazione:

    • Kerberos

    • CyberSAFE

    • RADIUS

  7. Scegliere File > Salva configurazione di rete.

排定命名方法的優先順序

設定命名方法之後，必須排定命名方法在設定檔中的優先順序。會先使用清單中最上方的命名方法將網路服務名稱或資料庫服務名稱解析成連線描述區。如果它無法解析名稱，就使用清單中的第二個方法。

指定命名方法的順序：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 設定檔。

  2. 從右窗格的清單中，選取命名。

  3. 選取方法頁籤。

  4. 從可用的方法清單選取命名方法，然後選取向右箭號按鈕。選取的命名方法會移至選取的方法清單。

  5. 根據 Oracle Net 解析網路服務名稱或資料庫服務名稱時要先嘗試的方法排序命名方法。從選取的命名方法清單中選取命名方法，然後選取升級或降級，在清單中上下移動選擇的項目。

  6. 選擇檔案 > 儲存網路組態。

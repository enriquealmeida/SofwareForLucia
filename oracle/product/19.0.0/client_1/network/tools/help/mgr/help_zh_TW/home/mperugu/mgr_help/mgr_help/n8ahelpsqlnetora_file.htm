sqlnet.ora 檔案

從屬端或伺服器的組態檔，其中指定：

  • 要附加至不完整服務名稱或網路服務名稱的從屬端網域

  • 解析名稱時從屬端應該使用的命名方法順序

  • 要使用的記錄日誌與追蹤功能

  • 連線路由

  • 偏好的 Oracle Names 伺服器

  • 外部命名參數

  • Oracle Advanced Security 參數

sqlnet.ora 檔案必須位於以下其中一個位置：

  1. TNS_ADMIN 環境變數指定的目錄。如果 TNS_ADMIN 環境變數不是定義為 Windows NT 的變數，它可能就在登錄中。

  2. UNIX 的 $ORACLE_HOME/network/admin 目錄，或 Windows 作業系統的 ORACLE_HOME\network\admin 目錄。

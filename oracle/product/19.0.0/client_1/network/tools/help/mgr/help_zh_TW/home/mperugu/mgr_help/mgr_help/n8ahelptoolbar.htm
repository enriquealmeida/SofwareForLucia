工具列

位在 Oracle Net Manager 左方的工具列，包含對應至資料夾階層的網路物件。工具列按鈕會依窗格中檢視或選取的物件啟用。將滑鼠指標移至工具列圖示上方，可以顯示圖示的功能描述。

工具列包含以下圖示：

圖示 描述
新增 選擇此圖示可建立此電腦的新網路服務名稱、監聽器以及 Oracle Names 伺服器。
刪除 選擇此圖示可刪除服務命名以及監聽器物件。
驗證連線 選擇此圖示可驗證所選服務命名物件與資料庫服務的連線。進行連線測試時，會使用儲存在 tnsnames.ora 檔案或目錄伺服器中的連線描述區資訊，來建立資料庫服務連線。
說明 選擇此圖示可顯示線上說明。

注意：部分命令僅在您選取適當的物件類型時，才可以使用。例如，如果想要建立監聽器，您必須先選取監聽器資料夾。

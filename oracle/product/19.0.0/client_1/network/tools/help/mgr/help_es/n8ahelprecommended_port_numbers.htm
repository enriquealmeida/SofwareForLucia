Números de Puerto Recomendados

Oracle Corporation recomienda los siguientes números de puerto para las direcciones de protocolo TCP/IP y TCP/IP con SSL:

Puerto Descripción
1521 Puerto de recepción por defecto para las conexiones de cliente al listener. En versiones futuras, este número de puerto puede cambiar al número de puerto registrado oficialmente de 2483 para TCP/IP y 2484 para TCP/IP con SSL.
1575 Puerto de recepción por defecto y registrado oficialmente para las conexiones cliente con Oracle Names Server.
1630 Puerto de recepción por defecto y registrado oficialmente para las conexiones cliente con Oracle Connection Manager.
1830 Puerto de recepción por defecto y registrado oficialmente para los comandos administrativos para Oracle Connection Manager.

Para obtener más información, consulte el apéndice sobre integración de dispositivos de autenticación con RADIUS, en Oracle Advanced Security Administrator's Guide (Guía del Administrador de Oracle Advanced Security).

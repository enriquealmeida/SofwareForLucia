为 Oracle Rdb 数据库配置连接描述符

Oracle Rdb 数据库是用于 Digital 的 64 位平台的数据库。Oracle Rdb 服务器对客户机的显示方式与 Oracle 数据库相同。Oracle Rdb 数据库有自己的监听程序。客户机配置包含使用 Oracle Rdb 特定参数创建一个网络服务名以访问数据库。

要配置 Oracle Rdb 数据库的客户机，请执行以下操作：

  1. 请创建网络服务名 (这在创建网络服务名中进行说明)。

  2. 在右侧窗格的服务标识框中，选择高级。将显示“高级服务选项”对话框。

  3. 在 Rdb 数据库字段中，输入 Oracle Rdb 数据库的文件名。

  4. 请随意在全局数据库名域中输入全局数据库名，并且如果需要的话，请在服务类型域输入服务类型。

  5. 在“高级服务选项”对话框中，选择确定。

  6. 在右侧窗格中，选择应用。

  7. 选择文件 > 保存网络配置。

有关详细信息，请参阅 Oracle Rdb 文档。

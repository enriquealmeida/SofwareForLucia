리스너 생성

데이터베이스가 접속을 수신하려면 서버에 리스너가 존재해야 합니다.

주: 설치 프로세스가 진행되는 동안 기본 리스너 LISTENER가 생성됩니다.

리스너를 생성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬> 리스너를 확장합니다.

  2. 도구 모음에서 플러스 기호(+)를 선택하거나 편집 > 생성을 선택합니다. 리스너 이름 선택 대화상자가 나타납니다.

  3. 리스너 이름 필드에 제공된 리스너 이름을 적용하거나 다른 이름으로 필드를 편집합니다. 첫번째 리스너의 기본 리스너 이름은 LISTENER 입니다. 다음 리스너 이름은 기본적으로 LISTENERn으로 설정됩니다. 리스너 이름은 지정된 모든 노드에서 고유해야 합니다.

  4. 확인을 선택합니다.

  5. 리스너 주소 구성에서 단계 4-6을 완료합니다.

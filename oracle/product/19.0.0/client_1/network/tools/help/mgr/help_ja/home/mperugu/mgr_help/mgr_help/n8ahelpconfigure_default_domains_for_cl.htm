クライアントのデフォルト・ドメインの構成

クライアントが特定のドメインからの名前を頻繁に要求する環境では、デフォルト・ドメインを設定できます。プロファイル内でデフォルト・ドメインを設定すると、未修飾のネット・サービス名またはサービス名に自動的に追加されます。たとえば、デフォルト・ドメインをus.example.comに設定すると、次の接続文字列内のsalesがsales.us.example.comとして検索されます。

CONNECT scott/tiger@sales

デフォルト・ドメインを指定する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「プロファイル」を展開します。

  2. 右ペインのリストから「ネーミング」を選択します。

  3. 「メソッド」タブを選択します。

  4. 「デフォルト・ドメイン」フィールドにドメインを入力します。

  5. 「ファイル」→「ネットワーク構成の保存」を選択します。

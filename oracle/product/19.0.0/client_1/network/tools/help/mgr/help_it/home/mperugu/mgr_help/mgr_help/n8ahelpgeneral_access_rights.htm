Generale: Diritti di accesso

La scheda Diritti di accesso consente di specificare quali client hanno accesso o meno al database.

Verificare i diritti di accesso dei client TCP/IP

Selezionare questa casella di controllo se si desidera specificare quali client possono accedere o meno al database. Deselezionare questa casella di controllo per disabilitare questa funzionalità. Se la casella di controllo non è selezionata, l'accesso è consentito a tutti i client.

Client esclusi dall'accesso

Immettere un nome host o un indirizzo IP per specificare a quali client non è consentito l'accesso al database. Utilizzare una virgola per separare le voci inserite nella stessa riga.

Client a cui è consentito l'accesso

Immettere un nome host o un indirizzo IP per specificare a quali client è consentito l'accesso al database. Utilizzare una virgola per separare le voci inserite nella stessa riga.

Nota: l'opzione Client a cui è consentito l'accesso ha la precedenza sull'opzione Client esclusi dall'accesso.

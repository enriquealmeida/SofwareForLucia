ldap.ora 파일

다음과 같은 디렉토리 구성 정보를 포함하는 Oracle Net Configuration Assistant에 의해 생성된 파일입니다.

  • 디렉토리 유형

  • 디렉토리 위치

  • 클라이언트 또는 서버가 데이터베이스 서비스 접속을 위한 접속 식별자를 찾거나 구성할 때 사용할 기본 Oracle Context입니다.

ldap.ora 파일은 다음 위치 중 하나에 있어야 합니다.

  1. TNS_ADMIN 환경 변수에 의해 지정된 디렉토리입니다. TNS_ADMIN 환경 변수가 Windows NT에서 변수로 정의되지 않은 경우 레지스트리에 있을 수 있습니다.

  2. UNIX의 $ORACLE_HOME/network/admin 디렉토리 또는 Windows 운영 체제의 ORACLE_HOME\network\admin 디렉토리입니다.

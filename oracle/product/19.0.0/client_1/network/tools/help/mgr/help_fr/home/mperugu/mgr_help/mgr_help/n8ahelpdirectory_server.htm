Serveur d'annuaire

Serveur d'annuaire auquel on accède avec le protocole LDAP (Lightweight Directory Access Protocol). Un annuaire compatible LDAP peut fournir un emplacement centralisé pour le stockage et la recherche des composants de réseau de base de données, des préférences en matière de règles d'entreprise et des préférences utilisateur, de l'authentification des utilisateurs et des informations de sécurité, ce qui permet de remplacer des fichiers côtés client et serveur.

Oracle Net prend en charge les services d'annuaire suivants :

  • Oracle Internet Directory

  • Microsoft Active Directory

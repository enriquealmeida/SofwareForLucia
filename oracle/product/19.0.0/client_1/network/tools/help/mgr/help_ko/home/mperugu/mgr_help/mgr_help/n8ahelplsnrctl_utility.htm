리스너 제어 유틸리티

리스너 상태 시작, 정지 및 얻기와 같은 다양한 리스너 기능을 제어하기 위해 Oracle Net Services와 함께 포함된 유틸리티입니다. 리스너 제어 유틸리티는 다음 방법으로 실행됩니다.

lsnrctl command [listener_name]

[listener_name] - listener.ora 파일에 정의된 리스너 이름입니다. 기본 리스너 이름 LISTENER를 사용하고 있으면 리스너를 식별할 필요가 없습니다.

자세한 내용은 Oracle10i Net Services Reference Guide의 Listener Control Utility 장을 참조하십시오.

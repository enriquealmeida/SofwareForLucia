Creación de Direcciones de Protocolo Adicionales

Puede acceder a un servicio de base de datos por varias rutas de red o dirección de protocolo.

Para agregar una dirección de protocolo de red a un nombre de servicio de red o servicio de base de datos existente:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Directorio o Local > Nomenclatura de Servicios.

  2. Seleccione un nombre de servicio de red o de base de datos. El panel de la derecha muestra el servicio de destino actual y una lista de direcciones.

  3. En el campo Configuración de Direcciones, haga clic en el signo más (+). Aparece un nuevo separador Dirección.

  4. Seleccione un protocolo e introduzca la información sobre el parámetro de protocolo correspondiente al protocolo seleccionado.

  5. Ordene las direcciones según la posición en la que deben estar en la lista de direcciones con los botones de flecha a la izquierda y a la derecha. A menos que haya opciones de varias direcciones configuradas, sólo se contacta con la primera dirección de la lista. Para configurar un orden diferente, consulte Configuración de Opciones de Varias Direcciones.

  6. Haga clic en Aplicar en el panel de la derecha.

  7. Si desea realizar estos cambios en la carpeta Local, seleccione Archivo > Guardar Configuración de Red.

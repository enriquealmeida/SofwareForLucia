명령 메뉴

[명령] 메뉴를 사용하여 다음 작업을 수행할 수 있습니다.

디렉토리

이 하위 메뉴는 디렉토리 서버에 대해 다음 작업을 수행하는 명령을 포함합니다.

요소 설명
현재 컨텍스트 변경 현재 컨텍스트가 아닌 다른 Oracle Context를 선택합니다. Oracle Context는 데이터베이스 서비스 및 디렉토리 이름 지정용 네트 서비스 이름 등의 Oracle 항목을 저장하는 디렉토리의 하위 트리입니다. 새 Oracle Context가 선택된 경우 네비게이터의 Oracle Net 구성 > 디렉토리 > 서비스 이름 지정 폴더의 객체가 새 Oracle Context에 대해 표시됩니다.
인증 설정 디렉토리 서버에 대한 인증서를 설정합니다.
네트 서비스 이름 익스포트 디렉토리 서버 이전 마법사를 실행합니다. 이 마법사는 tnsnames.ora 파일에 저장된 네트 서비스 이름을 디렉토리 서버로 익스포트합니다.

주: 디렉토리에 대한 액세스가 구성된 경우 디렉토리 메뉴만 표시됩니다. 디렉토리 액세스 구성 및 다른 요구 사항에 대한 자세한 내용은 Oracle Net Manager에 대한 디렉토리 이름 지정 요구 사항을 참조하십시오.

서비스 테스트

tnsnames.ora 파일 또는 디렉토리 서버에 저장된 접속 기술자 정보를 사용하여 데이터베이스 서비스에 접속하도록 선택합니다.

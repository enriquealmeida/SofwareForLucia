設定 Oracle Advanced Security

使用「Oracle Net 管理員」來設定 Oracle Advanced Security。您可以使用每一個 Oracle Advanced Security 頁籤來設定個別的參數集。Oracle Advanced Security 會啟用資料加密與完整性檢查、增強認證以及 Single Sign-On。Oracle Advanced Security 也提供 LDAP 相容目錄伺服器上的集中式使用者管理以及憑證式 Single Sign-On，這個功能是藉由 Secure Sockets Layer (SSL) 來完成。

設定 Oracle Advanced Security：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 設定檔。

  2. 從右方窗格中的清單選取 Oracle Advanced Security。

  3. 下列頁籤窗格就會顯示：

    • 認證

    • 其他參數

    • 完整性

    • 加密

    • SSL

  4. 變更組態。

  5. 選擇檔案 > 儲存網路組態。

您所做的變更便會儲存至 sqlnet.ora 檔案中。

注意：如果清單未顯示 Oracle Advanced Security，即表示未安裝。

如需詳細資訊，請參閱 Oracle Advanced Security Administrator's Guide。

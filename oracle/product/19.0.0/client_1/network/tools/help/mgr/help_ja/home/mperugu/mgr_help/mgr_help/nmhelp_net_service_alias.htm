ネット・サービス別名

ディレクトリ・サーバー内のディレクトリ・ネーミング・オブジェクトの別名。ディレクトリ・サーバーでは、定義済のネット・サービス名またはデータベース・サービスのネット・サービス別名を格納します。ネット・サービス別名のエントリには、接続記述子の情報はありません。かわりに、別名に対するオブジェクトの位置の参照のみを行います。クライアントがネット・サービス別名のディレクトリ検索を要求すると、ディレクトリは、エントリがネット・サービス別名であることを判断し、参照エントリであるかのように検索を完了します。

Paramètres généraux : Journalisation & trace

L'onglet Journalisation & trace vous permet d'activer la journalisation et la trace du processus d'écoute. Oracle Net Manager vous permet d'indiquer le nom et l'emplacement des fichiers de journalisation du processus d'écoute.

Journalisation désactivée

Sélectionnez cette option pour désactiver la journalisation sur le processus d'écoute.

Journalisation activée

Sélectionnez cette option pour activer la journalisation sur le processus d'écoute. Vous pouvez alors indiquer le nom et l'emplacement du fichier journal.

Fichier journal

Indiquez le chemin d'accès au répertoire et le nom du fichier dans lequel seront écrites les informations de journalisation du processus d'écoute. Notez que l'extension du fichier est toujours .log et que son nom par défaut est listener.log. Le répertoire par défaut est $ORACLE_HOME/network/log sous UNIX, et ORACLE_HOME\network\log sous Windows.

Trace désactivée

Sélectionnez cette option pour désactiver la trace sur le processus d'écoute.

Trace activée

Sélectionnez cette option pour activer la trace sur le processus d'écoute. Vous pouvez alors indiquer le niveau de trace voulu et le fichier de trace (autre que le fichier de trace par défaut).

Niveau de trace

Dans la liste, sélectionnez le niveau de trace :

Désactivé -- La trace est désactivée. Il s'agit du paramètre par défaut.

Utilisateur -- La trace est définie à un niveau approprié pour les utilisateurs. Trace pour identifier les erreurs imputables aux utilisateurs.

Administrateur -- La trace est définie à un niveau approprié pour les administrateurs de base de données. Trace pour identifier les problèmes d'installation.

Support technique -- La trace est définie à un niveau approprié au support technique.

Les fichiers de trace peuvent devenir très volumineux. Oracle vous recommande de n'activer cette fonction que lors du diagnostic de problèmes réseau.

Fichier de trace

Indiquez le répertoire et le nom du fichier d'écriture des informations de trace du processus d'écoute. Notez que l'extension du fichier est toujours .trc. Son nom par défaut est listener.trc. Le répertoire par défaut est $ORACLE_HOME/network/trace (sous UNIX) ou ORACLE_HOME\network\trace (sous Windows).

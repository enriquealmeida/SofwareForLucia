Oracleコンテキスト

ネーミング・コンテキストまたは非公開のディレクトリ・エントリの下に配置されるディレクトリ情報ツリー(DIT)に格納されるcn=OracleContextという相対識別名。Oracleコンテキストには、データベース・サービスなどのOracleソフトウェアで使用するエントリ、およびディレクトリ・ネーミングで使用するネット・サービス名が含まれています。ディレクトリには1つまたは複数のOracleコンテキストが存在します。

Oracle Internet Directoryは、DIT構造のルートにOracleコンテキストを自動的に作成します。このルートOracleコンテキストには、識別名(DN) (dn:cn=OracleContext)があります。

Oracle Net Managerの起動時には、Oracle Net Configuration Assistantによって構成されたデフォルトのOracleコンテキストを使用します。デフォルトのOracleコンテキストは変更可能です。ほとんどの場合、追加のOracleコンテキストを作成する必要があります。

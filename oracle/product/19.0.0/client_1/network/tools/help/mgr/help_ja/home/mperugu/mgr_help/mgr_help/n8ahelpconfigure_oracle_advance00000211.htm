Oracle Advanced Securityの構成

Oracle Net Managerを使用してOracle Advanced Securityを構成します。Oracle Advanced Securityの各タブ・ページでは、一連のパラメータを個別に構成できます。Oracle Advanced Securityを使用すると、データの暗号化と整合性のチェック、高度な認証およびシングル・サインオンが可能になります。LDAP準拠のディレクトリ・サーバーと証明書ベースのシングル・サインオンについてのユーザー一元管理も可能になりますが、この機能はSecure Sockets Layer (SSL)に依存しています。

Oracle Advanced Securityを構成する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「プロファイル」を展開します。

  2. 右ペインのリストから「Oracle Advanced Security」を選択します。

  3. 次の各タブ・パネルが表示されます。

    • 認証

    • その他のパラメータ

    • 整合性

    • 暗号化

    • SSL

  4. 構成を変更します。

  5. 「ファイル」→「ネットワーク構成の保存」を選択します。

変更は、sqlnet.oraファイルに保存されます。

注意: Oracle Advanced Securityがリストに表示されない場合は、Oracle Advanced Securityがインストールされていないことを示します。

詳細は、Oracle Advanced Security管理者ガイドを参照してください。

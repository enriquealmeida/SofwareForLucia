Configuración de un Descriptor de Conexión para Conectarse a Oracle Connection Manager

Para direccionar clientes al servidor de base de datos mediante Oracle Connection Manager, configure el archivo tnsnames.ora o el servidor de directorios con un descriptor de conexión que especifique la dirección de protocolo de Oracle Connection Manager. Esta dirección permite al cliente conectar a la computadora de Oracle Connection Manager.

Para configurar una dirección de protocolo para Oracle Connection Manager:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Directorio o Local > Nomenclatura de Servicios.

  2. Haga clic en el signo más (+) en la barra de herramientas o seleccione Editar > Crear de la barra de menús. Aparece la página de bienvenida del Asistente de Nombre de Servicio de Red.

  3. Introduzca cualquier nombre en el campo Nombre de Servicio de Red.

  4. Haga clic en el botón Siguiente. Aparece la página Protocolo.

  5. Seleccione el protocolo en el que Oracle Connection Manager ha configurado el listener. Por defecto, este protocolo es TCP/IP.

  6. Haga clic en el botón Siguiente. Aparece la página Valores del Protocolo.

  7. Introduzca los parámetros de protocolo correspondientes al protocolo seleccionado en los campos proporcionados. Si utiliza TCP/IP, el puerto por defecto es 1630.

  8. Haga clic en el botón Siguiente. Aparece la página Servicio.

  9. Si el servicio de destino es Oracle8i o posterior, haga clic en Oracle8i o posterior e introduzca un nombre de servicio en el campo Nombre de Servicio. Si el servicio de destino es Oracle8 u Oracle7, haga clic en Oracle8 o anterior e introduzca un identificador del sistema Oracle en el campo SID de Base de Datos.

  10. Haga clic en el botón Siguiente. Aparece la página Servicio.

  11. Haga clic en Terminar para guardar la configuración y salir del Asistente de Nombre de Servicio de Red.

  12. Seleccione Archivo > Guardar Configuración de Red.

  13. El nuevo nombre de servicio se ha agregado a la carpeta Nomenclatura de Servicios.

Para obtener más información sobre la activación completa de las funciones de Oracle Connection Manager, consulte el capítulo sobre configuración y administración de Oracle Connection Manager, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

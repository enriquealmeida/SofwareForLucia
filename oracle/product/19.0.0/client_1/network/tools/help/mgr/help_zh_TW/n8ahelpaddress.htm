位址

網路物件的網路協定位址。

進行連線時，從屬端及要求的接受者 (監聽器、Oracle Names 伺服器或 Oracle 連線管理程式) 會設定為相同的協定位址。從屬端使用此位址將連線要求傳送至特定的網路物件位置，而收件者則在此位址上「監聽」要求。

下列位址定義監聽器的監聽位址。

ADDRESS=

(PROTOCOL=tcp)

(HOST=sales-server)

(PORT=1521))

位址中的協定參數會根據選取的協定而有不同。

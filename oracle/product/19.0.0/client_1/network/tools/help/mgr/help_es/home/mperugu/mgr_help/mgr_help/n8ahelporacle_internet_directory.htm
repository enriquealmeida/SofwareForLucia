Oracle Internet Directory

Servicio de directorios implementado como aplicación en la base de datos Oracle. Permite la recuperación de información sobre usuarios dispersos y recursos de red. Combina el protocolo LDAP (Lightweight Directory Access Protocol) Versión 3, protocolo de acceso a directorios estándar abierto de Internet, con el alto rendimiento, escalabilidad, fuerza y disponibilidad de la base de datos Oracle.

Para obtener más información, consulte Oracle Internet Directory Administrator's Guide (Guía del Administrador de Oracle Internet Directory).

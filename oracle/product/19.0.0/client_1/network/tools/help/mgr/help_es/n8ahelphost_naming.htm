Nomenclatura de Host

Método de nomenclatura que permite a los usuarios conectarse a un servidor de bases de datos Oracle mediante un alias del nombre del host en un entorno TCP/IP. Los nombres de host se asignan al nombre de la base de datos global del servidor en un servicio de resolución de nombres existente, como el sistema de nombres de dominio (DNS), el Servicio de Información de Red (NIS) o un juego de archivos de /etc/hosts mantenidos centralmente.

Para obtener información sobre la configuración del método de nomenclatura del host, consulte el capítulo sobre configuración de métodos de nomenclatura, en Oracle10i Net Services Administrator's Guide (Guía del Administrador de Servicios de Red de Oracle10i).

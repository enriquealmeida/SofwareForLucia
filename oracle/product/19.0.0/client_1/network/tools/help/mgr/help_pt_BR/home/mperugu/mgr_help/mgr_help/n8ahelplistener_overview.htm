Visão Geral sobre Listener

Os serviços coordenam suas sessões com a ajuda de um listener, que consiste em um processo do servidor que recebe solicitações de conexão de uma aplicação cliente. Os listeners são configurados para "atender" em endereços de protocolo para um serviço que pode ou não ser de banco de dados.

Os endereços de protocolo são configurados no arquivo de configuração do listener, listener.ora, para um serviço que pode ser ou não de banco de dados. Um serviço de banco de dados Oracle9i ou Oracle8i registra automaticamente suas informações no listener, como seus nomes de serviço, nomes de instância e informações de carga. Esse recurso, denominado registro de serviço, não exige a configuração do serviço no arquivo listener.ora. Outros serviços, incluindo serviços de banco de dados Oracle8 e Oracle7, além de ferramentas de gerenciamento do Oracle Enterprise Manager, requerem a configuração de um serviço no arquivo listener.ora.

Uma vez que o listener esteja configurado, ele poderá ser gerenciado com o utilitário Listener Control.

Oracle Connection Manager

Um roteador através do qual uma solicitação de conexão de cliente pode ser enviada ao seu próximo hop ou diretamente ao servidor de banco de dados. Os clientes que efetuam o roteamento de suas solicitações de conexão através de um Oracle Connection Manager podem tirar proveito dos recursos de controle de acesso, multiplexação e conversão de protocolo configurados nesse Oracle Connection Manager.

O Oracle Connection oferece o roteamento de conexão para os ambientes em que:

  • Um grande número de usuários precisa acessar um único serviço através de um único protocolo.

  • Diferentes protocolos estão instalados no cliente e no servidor, provocando uma falha de conexão típica.

  • As conexões típicas do Oracle Net requerem que o cliente e o servidor tenham o mesmo protocolo instalado.

  • É necessário o controle do acesso de clientes aos servidores designados em um ambiente TCP/IP.

Para obter informações sobre o Oracle Connection Manager, consulte o capítulo Networking Challenges in Internet Age no Oracle9i Net Services Administrator's Guide.

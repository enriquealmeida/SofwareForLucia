Services hétérogènes

Composant intégré qui fournit la technologie générique permettant d'accéder à des systèmes autres qu'Oracle à partir du serveur de base de données Oracle. Les services hétérogènes vous permettent :

  • d'utiliser Oracle SQL pour accéder de manière transparente aux données stockées dans des systèmes autres qu'Oracle comme si les données résidaient dans un serveur Oracle

  • d'utiliser les appels de procédure Oracle pour accéder de manière transparente aux systèmes, services ou API (Application Programming Interface) à partir de votre environnement distribué Oracle.

Pour établir une connexion à un système autre qu'Oracle, la base de données Oracle8i lance un processus d'agent via le processus d'écoute sur la passerelle. Pour que la base de données Oracle8i se connecte à l'agent, il faut que le serveur soit configuré avec un nom de service réseau pour accéder à l'agent et que le processus d'écoute soit configuré avec des informations de service sur l'agent.

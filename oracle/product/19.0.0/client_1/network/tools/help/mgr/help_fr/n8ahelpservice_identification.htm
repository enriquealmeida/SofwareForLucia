Identification de service

Cette feuille de propriétés vous permet de visualiser ou de modifier les informations réseau d'un nom de service réseau ou d'un service de base de données sélectionné.

Identification de service

Utilisez la case Identification du service pour visualiser ou modifier les informations CONNECT_DATA.

Elément Description
Nom de service Saisissez un nom de service si vous configurez ce client pour qu'il se connecte à un service de base de données Oracle 8i ou version ultérieure. Remarque : pour un service de base de données stocké dans un serveur d'annuaire, ce champ affiche uniquement la valeur en cours. Les modifications ne sont pas autorisées.
SID Affiche l'identificateur système (SID) Oracle d'une base de données Oracle8.
Type de connexion Sélectionnez dans la liste l'option appropriée pour ce service réseau : serveur dédié, serveur partagé ou tout autre type de connexion configuré pour la base de données. Si un serveur partagé est configuré dans le fichier de paramètres d'initialisation, la sélection de Serveur dédié obligera le processus d'écoute à démarrer un serveur dédié pour passer outre à la configuration. Si un serveur partagé est configuré dans le fichier de paramètres d'initialisation et que vous voulez garantir que cette connexion utilisera bien un serveur partagé, sélectionnez Serveur partagé. Si le pool de connexions résidant en base de données est activé sur le serveur, vous pouvez sélectionner Serveur en pool pour obtenir une connexion du pool.
Utiliser l'identification compatible avec Oracle8 version 8.0 Sélectionnez cette option si vous configurez un client pour une connexion à un service de base de données antérieur à Oracle8i. Désactivez cette option si la configuration concerne un service de base de données Oracle9i ou Oracle8i.
Avancé Choisissez cette option pour configurer des paramètres CONNECT_DATA avancés.

Configuration de l'adresse

Utilisez la case Adresse pour modifier, ajouter ou supprimer une adresse de protocole de processus d'écoute.

Lorsqu'une adresse est déjà configurée, ses informations de protocole s'affichent dans les champs.

Elément Description
Protocole Dans la liste, sélectionnez le protocole pour l'adresse du processus d'écoute. En fonction du protocole choisi, saisissez les informations requises dans les champs appropriés.
Bouton Afficher paramètres avancés/Masquer paramètres avancés Utilisez le bouton Afficher paramètres avancés/Masquer paramètres avancés pour afficher ou masquer les paramètres permettant de définir l'espace tampon d'E/S maximum pour l'adresse.

Ces paramètres sont pris en charge par les protocoles TCP/IP, TCP/IP avec SSL et SDP.

  • Taille totale de tampon d'émission : indiquez (en octets) l'espace du tampon pour les opérations d'envoi de sessions.

  • Taille totale de tampon de réception : indiquez (en octets) l'espace du tampon pour les opérations de réception de sessions.

Remarque : le fait d'indiquer ces paramètres remplace ceux définis dans le fichier sqlnet.ora côté client.
Bouton + Cliquez sur ce bouton pour créer une adresse de protocole. Un nouvel onglet Adresse apparaît ; il vous permet de sélectionner le protocole et d'entrer les informations correspondantes dans les champs. Ce protocole doit être installé à la fois sur le client et le serveur (où figure le processus d'écoute).
Bouton X Cliquez sur ce bouton pour enlever l'onglet Adresse sélectionné et les informations correspondantes.
Bouton < Choisissez ce bouton pour faire monter l'adresse dans la liste.
Bouton > Choisissez ce bouton pour faire descendre l'adresse dans la liste.
Bouton Avancé Choisissez cette option pour configurer des options d'adresses multiples, comme la reprise sur délai de connexion et la répartition de la charge du client.

Bouton Appliquer

Choisissez ce bouton pour appliquer les modifications.

Bouton Rétablir

Choisissez ce bouton pour revenir aux valeurs d'origine des paramètres.

Pour plus d'informations sur la configuration de l'espace tampon d'E/S, reportez-vous au chapitre Optimizing Performance dans le Guide de l'administrateur Oracle10i Net.

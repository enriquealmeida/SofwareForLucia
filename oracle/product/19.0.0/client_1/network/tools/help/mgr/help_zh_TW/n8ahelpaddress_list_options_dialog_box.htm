位址清單選項對話方塊

您可以使用「位址清單選項」對話方塊來指定協定位址的順序，以及在連線中使用連線描述區中協定位址的方式。

依序嘗試每一個位址，直到成功為止

選取此選項即可啟用 Oracle9i 或 Oracle8i 從屬端的連線階段容錯移轉功能。如果無法連線第一個位址，容錯移轉會讓從屬端使用下一個監聽器協定位址。選取此選項及「使用與 Net8 8.0 從屬端相容的選項」選項，則可關閉 8.0 或之前版本從屬端的來源路由功能。

隨機嘗試每一個位址，直到成功為止

選取此選項可啟用 Oracle9i 或 Oracle8i 從屬端的從屬端負載平衡及連線階段容錯移轉功能。

嘗試隨機選取的位址

選取此選項可啟用 Oracle9i 或 Oracle8i 從屬端的從屬端載入平衡功能。

依序使用每一個位址，直到連線至目的地為止

選取此選項可啟用 Oracle8i 或 Oracle8 從屬端的來源路由。

當連線至目的地需要一個以上的位址時就需要此選項。此功能通常用於啟用「Oracle 連線管理程式」功能。

Oracle9i 從屬端不需要此參數。

只使用第一個位址

選取此選項可停用來源路由、連線階段容錯移轉以及從屬端負載平衡功能。

使用與 Net8 8.0 從屬端相容的選項

如果您要設定版本 8.0 從屬端請選取此選項。這時，您只可以選取下列選項：

-依序嘗試每一個位址，直到成功為止

- 依序使用每一個位址，直到連線至目的地為止

如果您要設定 Oracle9i 或 Oracle8i 從屬端，請清除此選項。

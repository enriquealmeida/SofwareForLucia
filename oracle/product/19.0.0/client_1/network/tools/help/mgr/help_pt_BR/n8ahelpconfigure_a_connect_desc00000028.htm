Configurando um Descritor de Conexão para Conexões com o Oracle Connection Manager

Para rotear clientes para o servidor de bancos de dados pelo Oracle Connection Manager, configure o arquivotnsnames.ora ou o servidor de diretórios com um descritor de conexão que especifique o endereço do protocolo do Oracle Connection Manager. Este endereço permite ao cliente conectar-se ao computador do Oracle Connection Manager.

Para configurar um endereço de protocolo para o Oracle Connection Manager:

  1. No painel do navegador, expanda Configuração do Oracle Net > Diretório ou Local > Nomeação de Serviço.

  2. Escolha o sinal de adição (+) na barra de ferramentas ou escolha Editar > Criar na barra de menus. A página de boas-vindas do Assistente de Nome do Serviço de Rede será exibida.

  3. Informe qualquer nome no campo Nome do Serviço de Rede.

  4. Escolha Próximo. A página Protocolo será exibida.

  5. Selecione o protocolo no qual o Oracle Connection Manager está configurado para atender. Por default, esse protocolo é o TCP/IP.

  6. Escolha Próximo. A página Definições de Protocolo é exibida.

  7. Nos campos fornecidos, especifique os parâmetros de protocolo apropriados para o protocolo escolhido. Se você estiver usando TCP/IP, a porta default é 1630.

  8. Escolha Próximo. A página Serviço será exibida.

  9. Se o serviço de destino for o Oracle8i ou versão mais recente, selecione Oracle8 i ou versão mais recente e informe um nome de serviço no campo Nome do Serviço. Se o serviço de destino for o Oracle7 ou o Oracle8, selecione Oracle8 ou Anterior e informe um Oracle System Identifier (Identificador do Sistema) no campo SID do Banco de Dados.

  10. Escolha Próximo. A página Serviço será exibida.

  11. Clique em Finalizar para salvar sua configuração e fechar o Assistente de Nome do Serviço de Rede.

  12. Escolha Arquivo > Salvar Configuração de Rede.

  13. O novo nome do serviço de rede é adicionado à pasta Nomeação de Serviço.

Para obter mais informações sobre a ativação total das funcionalidades do Oracle Connection Manager, consulte o Capítulo Configuring and Administering Oracle Connection Manager, no Oracle10i Net Administrator's Guide.

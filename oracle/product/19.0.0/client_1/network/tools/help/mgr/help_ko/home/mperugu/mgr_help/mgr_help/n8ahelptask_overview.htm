작업 개요

다음은 네트워크에 대해 Oracle Advanced Security를 구성하는 방법을 단계별로 설명한 목록입니다.

  1. 인증 방법을 선택합니다.

  2. 클라이언트/서버 조정 시 인증 방법이 사용되는 우선 순위를 지정합니다.

  3. 선택한 인증 방법에 대한 [기타 매개변수]를 구성합니다.

  4. 클라이언트측과 서버측 무결성(체크섬 레벨)을 구성합니다.

  5. 클라이언트측과 서버측 암호화를 구성합니다.

  6. 클라이언트측과 서버측 Secure Socket Layer(SSL) 통신 설정을 구성합니다.

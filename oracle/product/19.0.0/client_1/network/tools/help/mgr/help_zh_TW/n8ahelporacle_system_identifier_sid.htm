Oracle 系統 ID (SID)

這是一個名稱，用來識別執行中之 Oracle8i 之前版本的資料庫的特定執行處理。每個資料庫都至少會有一個參照資料庫的執行處理。

Oracle8i 之前版本的資料庫是使用 SID 來識別資料庫。SID 包含在 tnsnames.ora 檔案的連線描述區，以及 listener.ora 檔案的監聽器定義中。從屬端用來連線 Oracle8 或 Oracle7 資料庫的連線描述區中，應該設定 SID 參數。

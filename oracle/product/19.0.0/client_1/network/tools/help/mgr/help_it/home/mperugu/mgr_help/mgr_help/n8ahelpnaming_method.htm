Metodo di denominazione

Un metodo di risoluzione utilizzato da un'applicazione client per risolvere un identificativo di connessione in un descrittore di connessione durante i tentativi di connessione a un servizio di database. Oracle Net fornisce quattro metodi di denominazione:

  • Denominazione locale

  • Denominazione delle directory

  • Denominazione di connessione semplice (in precedenza denominazione host)

  • Denominazione esterna

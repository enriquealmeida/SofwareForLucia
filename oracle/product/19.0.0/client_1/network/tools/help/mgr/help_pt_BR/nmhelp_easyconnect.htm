Nomeação de Conexão Fácil

Um método de nomeação que permite que os clientes se conectem a um servidor de banco de dados sem qualquer configuração. Os clientes usam um endereço TCP/IP simples, que consiste em um nome de host e em porta e nome de serviço opcionais:

CONNECT nome do usuário/senha@host[:porta][/nome_do_serviço]

Para obter informações sobre como usar o método de nomeação de conexão fácil, consulte o capítulo Configuring Naming Methods no Oracle10i Net Services Administrator's Guide.

File cman.ora

Un file di configurazione in cui sono specificati gli indirizzi di protocollo per le richieste in entrata e i comandi locali e remoti, nonché i parametri di Oracle Connection Manager. Oracle Net Manager non supporta la configurazione di questo file. Deve essere configurato manualmente.

Il file cman.ora deve risiedere in una delle posizioni riportate di seguito.

  1. La directory specificata dalla variabile di ambiente TNS_ADMIN . Se la variabile di ambiente TNS_ADMIN non è definita come variabile in Windows NT, è possibile che si trovi nel registro.

  2. La directory di configurazione globale del nodo. Per Sun Solaris la directory è /var/opt/oracle. Windows NT non dispone di una directory centrale.

  3. La directory $ORACLE_HOME/network/admin nei sistemi operativi UNIX o la directory ORACLE_HOME\network\admin nei sistemi operativi Windows.

Per ulteriori informazioni, vedere il capitolo Configuring and Administering Oracle Connection Manager del manuale Oracle10i Net Administrator's Guide.

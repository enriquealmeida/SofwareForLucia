Oracle Net Configuration Assistant

설치 후 기본 네트워크 구성 요소를 구성하는 도구로 다음을 포함합니다.

  • 리스너 이름 및 프로토콜 주소

  • 클라이언트가 접속 식별자를 분석하기 위해 사용할 이름 지정 방법

  • tnsnames.ora 파일의 네트 서비스 이름

  • 디렉토리 서버 사용

Oracle Net Configuration Assistant는 Oracle 설치 설명서에 설명된 대로 소프트웨어 설치 후 자동으로 실행합니다. 클라이언트 또는 데이터베이스 서버에서 사용할 수 있습니다.

또한 독립형 모드로 실행하여 이름 지정 방법 사용, 리스너, tnsnames.ora 파일의 네트 서비스 이름 및 디렉토리 서버 사용을 구성할 수도 있습니다.

Oracle Net Configuration Assistant를 독립형 모드로 시작하려면 다음과 같이 하십시오.

  • UNIX의 경우 $ORACLE_HOME/bin에서 netmgr을 실행합니다.

  • Windows 운영 체제의 경우 시작 > 프로그램 > Oracle - HOME_NAME > 구성 및 이전 툴 > Net Configuration Assistant를 선택합니다.

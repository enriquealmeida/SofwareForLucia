Serviceregistrierung

Eine Funktion, mit der Instanzen und Dispatcher automatisch Informationen mit einem Listener registrieren. Da diese Informationen mit dem Listener registriert werden, muss die Datei listener.ora nicht mit diesen statischen Informationen konfiguriert werden.

Beim Hochfahren der Datenbank stellt die Dienstregistrierung dem Listener folgende Informationen zur Verfügung:

  • Servicenamen für jede laufende Instanz der Datenbank

  • Instanznamen der Datenbank

  • Service Handler (Dispatcher und dedizierte Server), die für jede Instanz verfügbar sind

  • Dispatcher-, Instanz- und Knotenlastinformationen

Auf diese Weise kann der Listener eine Clientanforderung korrekt bearbeiten.

Mit dieser Lastinformationen kann der Listener feststellen, welcher Dispatcher eine Clientverbindungsanforderung am Besten bearbeiten kann. Wenn alle Dispatcher blockiert sind, kann der Listener einen dedizierten Server für die Verbindung hochfahren.

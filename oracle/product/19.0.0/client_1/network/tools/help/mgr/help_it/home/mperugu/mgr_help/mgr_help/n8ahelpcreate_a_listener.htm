Creazione di un listener

Il database non può ricevere le connessioni se sul server non è presente un listener.

Nota: nel corso del processo di installazione viene creato un listener predefinito di nome LISTENER.

Per creare un listener, procedere nel modo riportato di seguito:

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

  2. Fare clic sul segno più (+) nella barra degli strumenti oppure scegliere Modifica > Crea. Viene visualizzata la finestra di dialogo Scegli nome listener.

  3. Accettare il nome del listener fornito nel campo Nome listener oppure immettere un altro nome. Per impostazione predefinita, il nome del primo listener è LISTENER. I listener successivi saranno denominati LISTENERn. I nomi dei listener devono essere univoci su qualsiasi nodo.

  4. Scegliere OK.

  5. Completare i passi da 4 a 6 in Configurazione degli indirizzi del listener.

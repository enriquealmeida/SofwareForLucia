將 Oracle Names 項目匯至目錄伺服器

此版本已不再支援 Oracle Names。使用 Oracle Names 9.2 版時，必須將 Oracle Names 伺服器中的資料庫與物件別名，匯出至目錄伺服器或間接匯出成 LDAP Data Interchange Format (LDIF) 檔案，此種檔案可以用於載入目錄伺服器。匯出資料庫物件時，僅會匯出位址記錄 (SMD)。位址記錄包含資料庫物件的連線描述區資訊。資料庫物件會以網路服務名稱的形式匯出。別名以網路服務別名的形式匯出。資料是從指定的網域匯出。

如需詳細資訊，請參閱 Oracle10i Net Administrator's Guide 中的 Configuring Naming Methods 章節。

如需有關從 Oracle Names 伺服器匯出資料的完整說明，請參閱版本 9.2 的 Oracle9i Net Services Administrator's Guide。

Solicitado

Active el servicio de seguridad si el otro lado lo permite.

Mi lado de la conexión especifica que desea el servicio de seguridad, pero no es necesario. El servicio de seguridad se activará si el otro lado especifica aceptado, solicitado o necesario. Debe haber un algoritmo coincidente disponible en el otro lado o no se activará el servicio. Si el otro lado especifica necesario y no hay ningún algoritmo coincidente, falla la conexión.

異質服務

一種提供從 Oracle 資料庫伺服器存取非 Oracle 系統的通用技術整合性元件。您可以使用「異質服務」來執行下列作業：

  • 使用 Oracle SQL，直接存取非 Oracle 系統中儲存的資料，就像資料是存放在 Oracle 伺服器中一般。

  • 使用 Oracle 程序呼叫，直接從您的 Oracle 分散式環境，存取非 Oracle 系統、服務或應用程式設計介面 (API)。

若要起始非 Oracle 系統的連線，Oracle8i 資料庫會透過閘道上的監聽器，啟動代理程式處理作業。要讓 Oracle8i 資料庫連線代理程式，伺服器必須設定網路服務名稱來存取代理程式，而且監聽器必須設定代理程式相關的服務資訊。

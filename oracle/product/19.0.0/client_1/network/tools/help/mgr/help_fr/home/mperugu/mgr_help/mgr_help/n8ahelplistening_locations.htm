Emplacements d'écoute

Les processus d'écoute sont configurés pour écouter les demandes de connexion entrantes destinées à une base de données sur une adresse de protocole réseau spécifique. L'onglet Emplacements d'écoute vous permet de configurer la ou les adresses d'écoute sur lesquelles le processus attendra les connexions client.

Protocole

Dans la liste, sélectionnez le protocole de l'adresse. Complétez ensuite les champs appropriés en fonction du paramètre de protocole choisi.

Bouton Afficher paramètres avancés/Masquer paramètres avancés

Utilisez le bouton Afficher paramètres avancés/Masquer paramètres avancés pour afficher ou masquer les paramètres permettant de définir l'espace tampon d'E/S maximum pour le processus d'écoute.

Ces paramètres sont pris en charge par les protocoles TCP/IP, TCP/IP avec SSL et SDP.

  • Taille totale de tampon d'émission

Indiquez, en octets, l'espace du tampon pour les opérations d'envoi de sessions.

Le serveur de base de données envoyant des données sur les clients, il est généralement approprié de définir ce paramètre du côté serveur. Si le serveur reçoit un volume important de demandes, définissez également le paramètre Taille totale de tampon de réception.

  • Taille totale de tampon de réception

Indiquez, en octets, l'espace du tampon pour les opérations de réception de sessions.

Réserver cette adresse pour les connexions JServer

Choisissez de configurer le fichier listener.ora avec des points d'extrémité GIOP (General Inter-Orb Protocol) pour les connexions Oracle JServer version 8.1.

Ajouter une adresse

Choisissez cette option pour créer une nouvelle adresse de protocole. Un nouvel onglet Adresse apparaît pour vous permettre de sélectionner le protocole et d'entrer les informations appropriées dans les champs.

Supprimer une adresse

Choisissez cette option pour supprimer l'onglet Adresse sélectionné et ses informations d'adresse.

Pour plus d'informations sur la configuration de l'espace tampon d'E/S, reportez-vous au chapitre Optimizing Performance dans le Guide de l'administrateur Oracle10i Net.

서비스 이름

데이터베이스가 클라이언트에게 제공하는 데이터의 논리적 표현입니다. 데이터베이스는 여러 서비스를 제공받을 수 있고 서비스는 여러 데이터베이스 인스턴스로 구현될 수 있습니다. 서비스 이름은 전역 데이터베이스 이름인 문자열입니다. 즉, 설치 또는 데이터베이스 생성 중에 입력된 데이터베이스 이름 및 도메인 이름으로 구성된 이름입니다. 전역 데이터베이스 이름이 무엇인지 잘 모르겠으면 초기화 매개변수 파일에서 SERVICE_NAMES 매개변수의 조합 값 중에서 찾아볼 수 있습니다.

클라이언트가 Oracle9i 또는 Oracle8i 데이터베이스에 접속하기 위해 사용하는 접속 기술자는 SERVICE_NAME(S 없이) 매개변수로 구성되어야 합니다.

服務註冊

一種可讓執行處理與分配器自動在監聽器註冊資訊的功能。由於此資訊是在監聽器註冊，因此不需在 listener.ora 檔案設定此靜態資訊。

啟動資料庫時，服務註冊會提供監聽器以下資訊：

  • 資料庫每一個執行中執行處理的服務名稱

  • 資料庫的執行處理名稱

  • 每一個執行處理可以使用的服務處理程式 (分配器與專用伺服器)

  • 分配器、執行處理以及節點載入資訊

以上資訊可以讓監聽器適當導向從屬端要求。

此負載資訊可以讓監聽器判斷哪個分配器最能處理從屬端連線要求。如果所有分配器被阻隔，監聽器可以衍生連線的專用伺服器。

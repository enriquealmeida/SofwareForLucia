追加のプロトコル・アドレスの作成

データベース・サービスには、複数のネットワーク・ルートで、プロトコル・アドレスからアクセスできます。

既存のネット・サービス名またはデータベース・サービスにネットワーク・プロトコル・アドレスを追加する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ディレクトリ」または「ローカル」→「サービス・ネーミング」を展開します。

  2. ネット・サービス名またはデータベース・サービスを選択します。右ペインに現行の接続先サービスおよびアドレス・リストが表示されます。

  3. 「アドレスの構成」ボックスでプラス(+)ボタンを選択します。新しい「アドレス」タブが追加されます。

  4. プロトコルを選択し、そのプロトコルに適切なプロトコル・パラメータ情報を入力します。

  5. 左右の矢印ボタンを使用して、アドレス・リスト内で各アドレスを並べ替えます。複数のアドレス・オプションが構成されている場合を除き、リストの最初のアドレスにのみ接続されます。異なる順序を構成する方法については、「複数のアドレス・オプションの構成」を参照してください。

  6. 右ペインで「適用」を選択します。

  7. この変更を「ローカル」フォルダに対して行う場合は、「ファイル」→「ネットワーク構成の保存」を選択します。

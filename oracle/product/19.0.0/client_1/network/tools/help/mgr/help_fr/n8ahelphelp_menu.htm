Menu Aide

Le menu Aide vous permet d'accéder à l'aide en ligne d'Oracle Net Manager.

Sommaire de l'aide

Choisissez cette option pour afficher la fenêtre Rubriques d'aide qui contient les rubriques et les entrées d'index du système d'aide d'Oracle Net Manager.

Rechercher

Choisissez cette option pour rechercher dans l'aide des rubriques d'aide avec mot-clé.

A propos d'Oracle Net Manager

Choisissez cette option pour afficher une boîte de dialogue contenant des informations sur Oracle Net Manager.

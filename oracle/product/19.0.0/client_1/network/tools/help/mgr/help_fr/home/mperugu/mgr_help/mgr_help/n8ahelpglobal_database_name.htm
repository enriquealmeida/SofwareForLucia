Nom de base de données global

Nom complet de la base de données, qui la distingue de toute autre base de données. Le format du nom de base de données global est le suivant : "nom_BdD.domaine_BdD", par exemple, sales.us.example.com.

La partie du nom correspondant à la base de données, sales , est un nom simple permettant de désigner cette base. La partie domaine, .us.example.com , indique le domaine dans lequel se trouve la base, rendant ainsi le nom global unique. Lorsque cela s'avère possible, Oracle recommande que le domaine de la base de données soit identique au domaine de réseau.

Le nom global de base de données est le nom de service par défaut de la base de données, tel qu'indiqué par le paramètre SERVICE_NAMES dans le fichier de paramètres d'initialisation.

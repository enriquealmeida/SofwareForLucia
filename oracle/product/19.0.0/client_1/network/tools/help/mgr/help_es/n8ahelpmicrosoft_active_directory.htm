Microsoft Active Directory

Servicio de directorios compatible con LDAP que se incluye en el Servidor de Windows 2000. Almacena información sobre objetos de la red y pone esta información a disposición de los usuarios y administradores de red. También proporciona a los usuarios acceso a los recursos de la red mediante un único proceso de conexión.

Se puede configurar como un método de nomenclatura de directorios para almacenar información de servicio a la que pueden acceder los clientes.

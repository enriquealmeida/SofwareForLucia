Datei cman.ora

Eine Konfigurationsdatei, die Protokolladressen für ankommende Anforderungen und lokale bzw. entfernte Befehle sowie Parameter für Oracle Connection Manager angibt. Oracle Net Manager unterstützt die Konfiguration dieser Datei nicht. Sie muss manuell konfiguriert werden.

Die Datei cman.ora muss in einem der folgenden Verzeichnisse gespeichert sein:

  1. Das Verzeichnis, das mit der Umgebungsvariablen TNS_ADMIN angegeben wird. Wenn die Umgebungsvariable TNS_ADMIN nicht als Variable in Windows NT gespeichert ist, kann sie in der Registrierung gespeichert sein.

  2. Das globale Konfigurationsverzeichnis des Knotens. Bei Sun Solaris ist dieses Verzeichnis /var/opt/oracle. Windows NT verfügt über kein zentrales Verzeichnis.

  3. Das Verzeichnis $ORACLE_HOME/network/admin bei UNIX oder das Verzeichnis ORACLE_HOME \network\admin bei Windows-Betriebssystemen.

Weitere Informationen finden Sie im Kapitel Configuring and Administering Oracle Connection Manager im Oracle10i Net Administrator's Guide.

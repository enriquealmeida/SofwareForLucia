異機種間サービス(Heterogeneous Services)

Oracleデータベース・サーバーからOracle以外のシステムにアクセスできるように、汎用テクノロジを提供する統合コンポーネント。異機種間サービスによって、次の操作が可能になります。

  • Oracle SQLを使用して、Oracle以外のシステムに格納されているデータに、Oracleサーバーに格納されている場合と同様に透過的にアクセスします。

  • Oracleプロシージャ・コールを使用して、Oracle分散環境からOracle以外のシステム、サービスまたはApplication Program Interface (API)に透過的にアクセスします。

Oracle以外のシステムへの接続を実行するために、Oracle8iデータベースはゲートウェイ上のリスナーを介してエージェント・プロセスを開始します。Oracle8iデータベースからエージェントに接続するには、ネット・サービス名を指定してエージェントにアクセスするようにサーバーを構成し、エージェントに関するサービス情報を指定してリスナーを構成する必要があります。

Details zu Net Service-Alias

In diesem Attributblatt können Sie die Details des ausgewählten Net Service-Alias anzeigen.

Net Service-Alias

Zeigen Sie den Namen des Net Service-Alias an, oder ändern Sie ihn.

Net Service-Name oder Datenbankdienst

Zeigen Sie die folgenden Informationen für den Datenbankdienst oder Net Service-Namen an, oder ändern Sie sie:

  • Oracle Context Zeigen Sie den Oracle Context des Datenbankdienstes oder Net Service-Namens an, oder ändern Sie ihn.

  • Name: Zeigen Sie den Distinguished Name (DN) des Datenbankdienstes oder Net Service-Namens an, oder ändern Sie ihn.

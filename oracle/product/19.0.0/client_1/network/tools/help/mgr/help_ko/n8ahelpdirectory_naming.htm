디렉토리 이름 지정

데이터베이스 서비스 또는 네트 서비스 이름을 중앙 디렉토리 서버에 저장된 접속 기술자로 변환하는 이름 지정 방법입니다.

디렉토리를 사용하면 데이터베이스 서비스 및 네트 서비스 이름을 중앙에서 관리할 수 있으므로 서비스 추가나 재배치에 관련된 작업을 줄일 수 있습니다. 서비스 이름을 별칭으로 설정하도록 네트 서비스 이름을 구성할 수 있지만 디렉토리는 네트 서비스 이름을 사용하지 않고 직접 데이터베이스 서비스를 참조할 수 있습니다. 데이터베이스 서비스는 설치 과정에서 디렉토리 항목에 자동으로 추가됩니다.

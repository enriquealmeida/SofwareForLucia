Fonction de trace

Fonction permettant d'écrire dans un fichier de sortie des informations détaillées sur une opération. Elle produit une séquence détaillée d'instructions décrivant les événements d'une opération au moment de leur exécution. Les opérateurs utilisent la fonction de trace pour diagnostiquer une anomalie ; en temps normal, cette fonction n'est pas activée. Le traçage d'une opération vous permet d'obtenir davantage d'informations sur les opérations internes d'un composant que la journalisation.

Remarque : Le traçage exige beaucoup d'espace disque et peut avoir un impact important sur les performances. Vous ne devrez donc l'activer qu'à bon escient.

設定加密

加密是指掩飾訊息使得別人無法閱讀，只有預期的收件者才可以閱讀的處理作業。

設定加密參數：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 設定檔。

  2. 從右方窗格中的清單選取 Oracle Advanced Security。

  3. 選取加密頁籤。

  4. 從加密清單選取從屬端或伺服器。

  5. 從加密類型清單選取一個值，以指定從屬端或伺服器在交涉加密和完整性時的行為。下面是四個可能的值：

    元素 描述
    接受 如果連線的另一端為「必要」或「要求」，並且也有可用的相容演算法，則服務會在作用中；否則，服務將會是非作用中。
    拒絕 服務必定不是作用中，如果另一端為「必要」，連線將會失敗。
    要求 如果連線的另一端為「接受」、「必要」或「要求」，並且也有可用的相容演算法，則服務會在作用中；否則，服務將會是非作用中。
    必要 服務必須為作用中，如果另一端為「拒絕」，或者沒有可用的相容演算法，連線將會失敗。

  6. 在「加密原始內建值」欄位輸入加密原始內建值：最多 256 字元的隨機字串。這個加密原始內建值是用來與您在下一步驟中選取的加密演算法搭配使用。

  7. 從「可用的方法」清單中，選取一個加密演算法，然後選擇右箭號 (<) 按鈕，即可將其移至「選取的方法」清單。「選取的方法」清單中方法的順序會決定偏好的交涉順序。Oracle Advanced Security 支援下列演算法：

    元素 描述
    RC4_256 256 位元金鑰大小
    3DES168 三重 DES 與三金鑰 (168 位元) 選項
    RC4_128 128 位元金鑰大小
    3DES112 三重 DES 與雙金鑰 (112 位元) 選項
    RC4_56 56 位元金鑰大小
    DES 標準 56 位元金鑰大小
    RC4_40 40 位元金鑰大小
    DES40 40 位元金鑰大小

  8. 選擇檔案 > 儲存網路組態。

注意：所有 Oracle Advanced Security 加密演算法和金鑰長度僅適用於美國國內。

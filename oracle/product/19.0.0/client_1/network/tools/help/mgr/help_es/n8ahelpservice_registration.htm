Registro de Servicio

Función mediante la que las instancias y los distribuidores registran información automáticamente con un listener. Ya que esta información se registra con el listener, no es necesario configurar el archivo listener.ora con esta información estática.

Durante el inicio de la base de datos, el registro del servicio proporciona al listener información sobre:

  • Nombres de servicio para cada instancia en ejecución de la base de datos

  • Nombres de instancias de la base de datos

  • Manejadores de servicios (distribuidores y servidores dedicados) disponibles para la instancia

  • Información de carga de nodos, instancias y distribuidores

Permite al listener dirigir una solicitud del cliente de la forma correcta.

Esta información de carga permite al listener determinar el distribuidor que mejor gestionará la solicitud de conexión de un cliente. Si todos los distribuidores están bloqueados, el listener puede originar un servidor dedicado para la conexión.

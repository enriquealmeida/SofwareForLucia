Boîte de dialogue Choisir un nouveau contexte

La boîte de dialogue Choisir un nouveau contexte vous permet de définir un autre contexte Oracle, c'est-à-dire un conteneur appelé cn=OracleContext dans une arborescence d'informations d'annuaire (DIT). Ce conteneur contient les entrées à utiliser avec le logiciel Oracle, par exemple les noms de service réseau et les noms de service de base de données à utiliser pour les connexions aux services de base de données. Par défaut, Oracle Net Manager utilise le contexte Oracle configuré par l'assistant Oracle Net Configuration.

Contexte de résolution de noms de répertoire

Dans la liste, sélectionnez une entrée d'annuaire contenant le contexte Oracle que vous voulez afficher.

contexte Oracle

Sélectionnez un contexte Oracle dans la liste. Le contexte que vous sélectionnez doit contenir les entrées dans l'arborescence dont vous voulez modifier les informations réseau où à laquelle vous voulez ajouter des informations réseau.

Le dossier Annuaire > Résolution de noms de service du panneau du navigateur est mis à jour en fonction des objets réseau du contexte Oracle sélectionné.

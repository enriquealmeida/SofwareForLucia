일반 사항: 경로 지정

경로 지정 탭을 사용하여 언제든지 해당 컴퓨터에서 특정 프로세스로 접속 요청을 지정할 수 있습니다.

항상 전용 서버 사용

리스너가 모든 네트워크 세션에 대한 전용 서버를 클라이언트에서 생성하도록 합니다. 접속 기술자의 CONNECT_DATA 부분에 (SERVER=dedicated) 를 추가함으로써 이 작업을 수행합니다. 이렇게 하면 공유 서버가 구성된 경우에도 이 클라이언트의 접속은 전용 서버를 사용합니다.

이 옵션을 사용하면 접속 속도는 빨라지지만 서버는 느려집니다. Oracle은 유휴 시간이 거의 없이 대용량의 데이터베이스 트랜잭션을 수행할 경우에만 이 옵션을 사용할 것을 권장합니다.

주: 이 옵션을 설정하는 경우 서비스 식별 속성 시트의 접속 유형 옵션은 무효화됩니다.

Connection Manager 경로 지정 선호

가능할 때 마다 Oracle Connection Manager에 클라이언트를 전송합니다. Oracle Connection Manager 주소를 사용할 수 없는 경우 사용 가능한 리스너 주소를 통해 접속 경로가 지정됩니다.

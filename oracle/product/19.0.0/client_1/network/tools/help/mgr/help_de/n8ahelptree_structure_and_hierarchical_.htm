Baumstruktur und hierarchische Ordner

Oracle Net Manager zeigt eine grafische Baumstruktur mit Netzwerkobjekten sowie die darin enthaltenen Objekte in einer Ordnerhierarchie an. Im Navigator-Fensterbereich können Sie in jedem Ordner Objekte anzeigen, ändern, hinzufügen und löschen.

Wenn Sie einen Ordner einblenden, sehen Sie eine verschachtelte Liste von Objekten und Ordnern. Wenn ein Objekt ausgewählt ist, werden Informationen über das Objekt im rechten Fensterbereich von Oracle Net Manager angezeigt.

Verzeichnis

Zeigt einfache Namen an, die als Connect-Bezeichner bezeichnet werden und in einem Directory Server konfiguriert sind.

Hinweis: Der Ordner Verzeichnis wird nur angezeigt, wenn der Zugriff zu dem Verzeichnis konfiguriert wurde. Weitere Informationen über die Konfiguration des Verzeichniszugriffs und andere Anforderungen finden Sie unter Anforderungen an die Verzeichnisbenennung bei Oracle Net Manager.

Lokal

Zeigt Netzwerkelemente an, die in lokalen Konfigurationsdateien konfiguriert sind:

  • Net Service-Namen in der Datei tnsnames.ora

  • Listener in der Datei listener.ora

  • Profile in der Datei sqlnet.ora

승인되지 않은 액세스에 대한 배너 구성 및 사용자 작업 감사

승인되지 않은 액세스에 대해 사용자에게 경고를 표시하기 위한 배너와 사용자 작업에 대한 가능한 감사를 생성 및 구성해야 합니다. 클라이언트 응용 프로그램이 데이터베이스에 로그인하면 알림이 표시됩니다.

이러한 배너가 표시되도록 구성하려면 데이터베이스 서버측에서 다음과 같은 sqlnet.ora 매개변수를 배너 정보가 포함된 텍스트 파일로 설정합니다.

  • SEC_USER_UNAUTHORIZED_ACCESS_BANNER. 예: SEC_USER_UNAUTHORIZED_ACCESS_BANNER = /opt/Oracle/11g/dbs/unauthaccess.txt

  • SEC_USER_AUDIT_ACTION_BANNER. 예: SEC_USER_AUDIT_ACTION_BANNER = /opt/Oracle/11g/dbs/auditactions.txt

기본적으로 이러한 매개변수는 설정되지 않습니다.

이러한 매개변수를 설정한 다음에는 Oracle Call Interface 응용 프로그램이 적합한 OCI API를 사용하여 이러한 배너를 검색하고 이를 일반 사용자에게 제공해야 합니다.

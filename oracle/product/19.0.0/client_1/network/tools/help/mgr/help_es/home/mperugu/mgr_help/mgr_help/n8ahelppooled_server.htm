Pool de Conexiones Residentes de la Base de Datos

El pool de conexiones residentes de la base de datos proporciona el pool de conexiones "dedicadas" en las aplicaciones o los procesos del cliente. Una conexión dedicada hace referencia a un servidor dedicado junto con su sesión de base de datos. Utilice el pool de conexiones residentes de la base de datos cuando varias aplicaciones del cliente (como las capas medias) deban mantener conexiones persistentes con la base de datos o cuando los recursos del servidor (como la memoria) deban optimizarse.

En el pool de conexiones residentes de la base de datos, los clientes se conectan a un proceso en segundo plano denominado Broker de Conexión que proporciona la función de pool.

Puede compararlo con el servidor dedicado y el servidor compartido.

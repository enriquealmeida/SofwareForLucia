Visão Geral de Tarefas

A seguir, apresentamos uma lista de etapas que descrevem como configurar o Oracle Advanced Security para a rede.

  1. Selecione os métodos de autenticação.

  2. Priorize a ordem em que os métodos de autenticação serão usados durante a negociação entre cliente/servidor.

  3. Configure Outros Parâmetros para cada um dos métodos de autenticação selecionados.

  4. Configure a integridade para o lado servidor e o lado cliente (nível de checksum).

  5. Configure a criptografia para o lado cliente e o servidor.

  6. Configure as definições de comunicações da SSL (Secure Socket Layer) no lado cliente e no lado servidor.

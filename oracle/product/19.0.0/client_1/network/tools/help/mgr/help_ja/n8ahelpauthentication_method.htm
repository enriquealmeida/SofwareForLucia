認証方式

分散環境でユーザー、クライアントおよびサーバーのアイデンティティの機密を保護できるセキュリティ方式。ネットワーク認証方式には、ユーザーがシングル・サインオンできるというメリットもあります。Oracle Advanced Securityをインストール済かどうかに関係なく、Oracle9iおよびOracle8iでは、次の認証方式がサポートされます。

  • CyberSafe

  • KERBEROS (v5)

  • RADIUS

  • SSL

  • Windows NTシステム固有の認証方式

Asistente de Configuración de Red de Oracle

Herramienta de postinstalación que configura componentes de red básicos, entre los que se incluyen:

  • Nombres del listener y direcciones de protocolo

  • Métodos de nomenclatura que utilizará el cliente para resolver identificadores de conexión

  • Nombres de servicio de red de un archivo tnsnames.ora

  • Uso del servidor de directorios

El Asistente de Configuración de Red de Oracle se ejecuta automáticamente después de la instalación del software, como se describe en la guía de instalación de Oracle. Se puede utilizar en el cliente o en el servidor de bases de datos.

También se puede ejecutar en modo autónomo para configurar el uso de los métodos de nomenclatura, el listener, los nombres del servicio de red en un archivo tnsnames.ora y el uso del servidor de directorios.

Para iniciar el Asistente de Configuración de Red de Oracle en modo autónomo:

  • En UNIX, ejecute netmgr en $ORACLE_HOME/bin.

  • En los sistemas operativos de Windows, haga clic en Inicio > Programas > Oracle - HOME_NAME > Herramientas de Configuración y Migración > Asistente de Configuración de Red de Oracle.

Configuración de los Dominios por Defecto para Clientes

En entornos donde el cliente a menudo solicita nombres de un dominio concreto, es conveniente definir un dominio por defecto. Cuando se define un dominio por defecto en el perfil, se agrega automáticamente a cualquier nombre de servicio de red no cualificado o nombre de servicio. Por ejemplo, si se define un dominio por defecto como us.example.com, sales en la siguiente cadena de conexión se busca como sales.us.example.com:

CONNECT scott/tiger@sales

Para especificar un dominio por defecto:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione Nomenclatura de la lista del panel de la derecha.

  3. Seleccione el separador Métodos.

  4. Introduzca el dominio en el campo Dominio por Defecto.

  5. Seleccione Archivo > Guardar Configuración de Red.

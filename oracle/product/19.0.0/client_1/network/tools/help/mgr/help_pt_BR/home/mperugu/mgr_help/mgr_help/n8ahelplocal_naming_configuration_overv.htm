Visão Geral da Configuração de Nomeação Local

As seções a seguir são abrangidas neste tópico.

Instalação do Administrador ou de Runtime

Se o acesso a diretório não tiver sido configurado, o Assistente de Configuração do Oracle Net solicitará que você configure nomes de serviço de rede no arquivo tnsnames.ora para uma instalação de administrador ou de cliente de runtime.

Instalação Personalizada

Se a nomeação local for selecionada como método de nomeação durante uma instalação Personalizada de cliente ou servidor, o Assistente de Configuração do Oracle Net solicitará que você configure nomes de serviços de rede no arquivo tnsnames.ora.

Configuração com o Oracle Net Manager.

O Oracle Net Manager permite que você configure nomes de serviço de rede, além de:

  • Criar nomes de serviço de rede para serviços de bancos de dados não Oracle

  • Configurar várias rotas para um listener através de vários endereços de protocolo

  • Configure informações avançadas sobre CONNECT_DATA para um nome de serviço de banco de dados ou de rede.

Para acessar a configuração de nomeação local:

No painel de navegação, escolha a pasta Local > Nomeação de Serviço para criar nomes de serviços de rede ou modificar informações sobre o descritor de conexão.

Para obter mais informações, consulte o capítulo Configuring Naming Methods no Oracle10i Net Administrator's Guide.

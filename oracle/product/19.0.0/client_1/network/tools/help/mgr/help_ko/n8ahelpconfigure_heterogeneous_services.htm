리스너에 대한 이기종 서비스 구성

이기종 서비스를 사용하여 Oracle 서버에서 비Oracle 시스템에 액세스할 수 있습니다. 비Oracle 시스템에 대한 접속을 초기화하기 위해 Oracle 서버는 게이트웨이의 리스너를 통해 에이전트 프로세스를 시작합니다. 리스너에 대한 이기종 서비스 구성은 이기종 에이전트에 대한 서비스 정보 생성을 포함합니다.

리스너에 대한 이기종 서비스를 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬> 리스너를 확장합니다.

  2. 리스너를 선택합니다.

  3. 오른쪽 창의 목록에서 기타 서비스를 선택합니다.

  4. 서비스 추가를 선택합니다. 새 서비스 탭이 나타납니다.

  5. 프로그램 이름 필드에 에이전트 실행 파일을 입력하고, SID 필드에 에이전트의 Oracle SID(시스템 식별자)를 입력하고, Oracle 홈 디렉토리 필드에 실행 가능한 Oracle 홈을 입력합니다.

  6. 외부 프로시저에 대한 접속 기술자 구성에 설명된 대로 서버가 이기종 서비스 액세스를 위한 접속 기술자로 구성되어 있는지 확인합니다.

  7. 파일 > 네트워크 구성 저장을 선택합니다.

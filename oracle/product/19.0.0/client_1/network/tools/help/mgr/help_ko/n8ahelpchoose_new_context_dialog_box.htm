새 컨텍스트 선택 대화상자

[새 컨텍스트 선택] 대화상자를 사용하여 Oracle 소프트웨어와 함께 사용하기 위한 항목을 포함하는 DIT(Directory Information Tree)에서 cn=OracleContext 이름의 컨테이너인 다른 Oracle Context를 설정할 수 있습니다. 이러한 항목은 데이터베이스 서비스 접속을 위한 네트 서비스 이름 및 데이터베이스 서비스 이름을 포함할 수 있습니다. 기본적으로 Oracle Net Manager에서는 Oracle Net Configuration Assistant로 구성된 Oracle Context를 사용합니다.

디렉토리 이름 지정 컨텍스트

목록에서 보고자 하는 Oracle Context를 포함하는 디렉토리 항목을 선택합니다.

Oracle Context

목록에서 Oracle Context를 선택합니다. 선택한 Oracle Context에는 네트워크 정보를 수정하거나 추가할 디렉토리 트리의 항목이 포함되어 있어야 합니다.

네비게이터 창의 디렉토리 > 서비스 이름 지정 폴더는 선택한 Oracle Context의 네트워크 객체를 사용하여 업데이트됩니다.

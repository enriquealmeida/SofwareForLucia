Heterogene Dienste

Eine integrierte Komponente, die eine allgemeine Technologie für den Zugriff auf fremde Systeme aus dem Oracle-Server bereitstellt. Mit heterogenen Diensten können Sie:

  • Oracle SQL verwenden, um transparent auf Daten in fremden Systemen zuzugreifen, so als wären die Daten in einem Oracle-Server gespeichert.

  • Oracle-Prozeduraufrufe verwenden, um transparent auf fremde Systeme, Dienste oder APIs aus Ihrer verteilen Oracle-Umgebung zuzugreifen.

Um eine Verbindung zu einem fremden System einzuleiten, fährt die Oracle8i-Datenbank einen Agent-Prozess über den Listener auf dem Gateway hoch. Damit sich die Oracle8i-Datenbank bei dem Agent anmelden kann, muss der Server mit einem Net Service-Namen konfiguriert sein, um auf den Agent zuzugreifen, und der Listener muss mit Dienstinformationen über den Agent konfiguriert sein.

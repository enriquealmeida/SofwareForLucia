Empfohlene Portnummern

Oracle Corporation empfiehlt die folgenden Portnummern für TCP/IP- und TCP/IP with SSL-Protokolladressen:

Port Beschreibung
1521 Standard-Listening-Port für Clientverbindungen zu dem Listener. In zukünftigen Releases kann sich diese Portnummer in die offiziell registrierte Portnummer 2483 für TCP/IP und 2484 für TCP/IP with SSL ändern.
1575 Standardmäßiger und offiziell registrierter Listening-Port für Clientverbindungen zu einem Oracle Names Server
1630 Standardmäßiger und offiziell registrierter Listening-Port für Clientverbindungen zu Oracle Connection Manager
1830 Standardmäßiger und offiziell registrierter Listening-Port für Admin-Befehle an Oracle Connection Manager

Weitere Informationen finden Sie im Anhang Integrating Authentication Devices Using RADIUS, im Oracle Advanced Security Administrator's Guide

Endereço do Protocolo

Um endereço que identifica o endereço de um objeto da rede.

Quando uma conexão é estabelecida, o cliente e o receptor da solicitação, como, por exemplo, o listenerservidor Oracle Names ou o Oracle Connection Manager são configurados com endereços de protocolo idênticos. O cliente utiliza esse endereço para enviar a solicitação de conexão à localização de um objeto de rede específico e o destinatário "atende" solicitações nesse endereço. É importante instalar os mesmos protocolos para o cliente e o destinatário da conexão, além de configurar os mesmos endereços.

O endereço de protocolo é composto por um elemento ADDRESS e um elemento ADDRESS_LIST.

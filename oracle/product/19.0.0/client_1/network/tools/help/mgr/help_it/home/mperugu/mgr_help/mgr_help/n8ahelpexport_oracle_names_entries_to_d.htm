Esportazione di voci Oracle Names in un server delle directory

Oracle Names non è più supportato in questa release. Se si utilizza una release 9.2 di Oracle Names, è necessario esportare gli oggetti di database e alias memorizzati in un server Oracle Names in un server delle directory oppure indirettamente in un file LDIF (Data Interchange Format) LDAP, che può essere quindi utilizzato per caricare il server delle directory. Quando si esportano gli oggetti di database, viene esportato solo il record degli indirizzi (A.SMD). Il record degli indirizzi contiene le informazioni relative al descrittore di connessione per l'oggetto di database. Gli oggetti di database sono esportati come nomi di servizi di rete. Gli alias sono esportati come alias di servizi di rete. I dati sono esportati da un dominio specificato.

Per ulteriori informazioni, vedere il capitolo Configuring Naming Methods del manuale Oracle10i Net Administrator's Guide.

Per istruzioni complete sull'esportazione dei dati da un server Oracle Names, vedere il manuale Oracle9i Net Services Administrator's Guide a partire dalla release 9.2.

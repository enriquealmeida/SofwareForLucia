「詳細サービス・オプション」ダイアログ・ボックス

「詳細サービス・オプション」ダイアログ・ボックスでは、詳細なCONNECT_DATAパラメータを構成できます。

その他のサービス設定

要素 説明
インスタンス名 アクセスするデータベース・インスタンスの名前を入力します。インスタンス名は、初期化パラメータ・ファイル内のINSTANCE_NAMEパラメータ値から取得できます。
セッション・データ・ユニット(SDU) セッション・データ・ユニットのサイズを入力すると、ネットワークへ送信されるパケットのパフォーマンス特性が変わります。SDUサイズを通常のトランスポート・フレーム・サイズの倍数で設定します。Ethernetフレーム・サイズの通常サイズは1,500ですが、使用可能なサイズは1,460です。暗号化が使用されない場合は1バイトが留保されます。アルゴリズムが使用するサイズは、暗号が使用されない場合は1,460+1の倍数、Ethernetの場合は1,460の倍数です
異機種間サービスを使用 異機種間サービスを使用してOracleサーバーからOracle以外のシステムにアクセスする場合は、このオプションを選択します。

Oracle RDB設定

要素 説明
Oracle RDB設定 Oracle RDBデータベースのファイル名を指定します。
サービスのタイプ Oracle RDBデータベースで使用するサービスのタイプを入力します。
グローバル・データベース名 Oracle RDBデータベースの識別子を入力します。

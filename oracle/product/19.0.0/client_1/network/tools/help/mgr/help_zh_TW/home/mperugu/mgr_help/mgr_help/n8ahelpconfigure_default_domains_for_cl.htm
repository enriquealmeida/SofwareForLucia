設定從屬端的預設網域

在從屬端經常從特定網域要求名稱的環境中，最好設定預設的網域。當設定檔中有設定預設網域時，預設網域會自動附加至任何不完整的網路服務名稱或服務名稱。例如，如果預設網域設定為 us.example.com，系統就會以 sales.us.example.com 來搜尋下列連線字串中的 sales：

CONNECT scott/tiger@sales

指定預設網域：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 設定檔。

  2. 從右窗格的清單中，選取命名。

  3. 選取方法頁籤。

  4. 在預設網域欄位輸入網域。

  5. 選擇檔案 > 儲存網路組態。

Oracle Context ändern

Wenn Oracle Net Manager gestartet wird, entsprechen die Objekte, die in dem Ordner Directory > Dienstebenennung angezeigt werden, den Oracle-Diensteinträgen, die in dem standardmäßigen Oracle Context enthalten sind, der während der Konfiguration der Directory-Verwendung mit dem Oracle Net-Konfigurationsassistenten festgelegt wurde. Sie können Einträge für einen anderen Oracle Context erstellen und ändern, für den Sie eine Berechtigung haben.

So ändern Sie den aktuellen Oracle Context:

  1. Wählen Sie Befehl > Verzeichnis > Aktuellen Kontext ändern. Daraufhin wird das Dialogfeld Neuen Kontext wählen angezeigt.

  2. Aus der Liste Verzeichnisbenennungskontext wählen Sie den Verzeichnisbenennungskontext, der den anzuzeigenden Oracle Context enthält.

  3. Wählen Sie aus der Liste Oracle Context den Oracle Context, und klicken Sie dann auf OK. Wenn Änderungen am aktuellen Oracle Context nicht gespeichert wurden, werden Sie in einem Speichern-Dialogfeld aufgefordert, die Änderungen zu speichern.

Anforderungen an die Verzeichnisbenennung bei Oracle Net Manager

Mit Oracle Net Manager können Sie Netzwerk-Route-Informationen für einen Datenbankdienst erstellen. Außerdem können Sie Net Service Names in einem Directory Server erstellen.

Damit Oracle Net Manager in ein Verzeichnis schreiben kann, müssen folgende Bedingungen erfüllt sein:

  • Das Verzeichnis muss mit einem Oracle-Schema konfiguriert sein.

  • Der Rechner, auf dem Oracle Net Manager ausgeführt wird, muss mit einem standardmäßigen Oracle Context für den Zugriff und das Schreiben von Einträgen konfiguriert sein.

Diese Konfigurationseinstellungen werden mit dem Oracle Net-Konfigurationsassistenten festgelegt, wie in Überblick über die Konfiguration der Verzeichnisbenennenung beschrieben. Oracle Net Manager verfügt nicht über diese Funktionalität.

Wenn der Directory-Zugriff nicht konfiguriert ist, sind der Ordner Oracle Net-Konfiguration > Verzeichnis im Navigator-Fensterbereich und das Menü Befehl > Verzeichnis nicht verfügbar.

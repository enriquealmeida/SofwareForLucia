Barra de Herramientas

La barra de herramientas, situada a la izquierda del Gestor de Red de Oracle, contiene iconos que corresponden a los objetos de red de las jerarquías de carpetas. Los botones de la barra de herramientas se activan según los objetos que aparezcan o se seleccionen en un panel. Mueva el mouse sobre un icono de la barra de herramientas para que aparezca la descripción de la función del icono.

La barra de herramientas contiene los siguientes iconos:

Icono Descripción
Agregar Seleccione esta opción para crear nuevos nombres de servicio de red, listeners y un Oracle Names Server para esta computadora.
Suprimir Seleccione esta opción para suprimir los objetos de nomenclatura de servicios y de listeners.
Verificar Conexión Seleccione esta opción para verificar la conectividad del objeto de nomenclatura de servicios seleccionado a un servicio de base de datos. Durante una prueba de conectividad, se realiza una conexión a un servicio de base de datos con la información del descriptor de conexión almacenada en un archivo tnsnames.ora o un servidor de directorios.
Ayuda Muestra la ayuda en pantalla.

Nota: Algunos comandos sólo están disponibles si selecciona el tipo de objeto adecuado. Por ejemplo, se desea crear un listener, debe seleccionar antes la carpeta Listeners .

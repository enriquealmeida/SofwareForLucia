Relativer Distinguished Name (RDN)

Der lokale, genaueste Eintragsname. Es sind keine anderen vollständigen Eintragsnamen vorhanden, mit denen der Eintrag eindeutig adressiert wird. In diesem Beispiel, cn=sales,dc=us,dc=example,dc=com,cn=sales, ist cn=sales der RDN.

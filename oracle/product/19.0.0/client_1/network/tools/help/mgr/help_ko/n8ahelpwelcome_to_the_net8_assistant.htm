Oracle Net Manager 시작

Oracle Net Manager를 시작합니다. Net8 Assistant는 구성 기능과 구성 요소 제어 기능을 결합하여 Oracle Net를 구성하고 관리할 수 있는 통합 환경을 제공합니다.

Oracle Net Manager 사용 대상

서비스에 클라이언트 접속을 설정해야 하는 네트워크 관리자라면 누구나 사용할 수 있습니다. Oracle Net Manager를 사용하여 네트워크 특정 영역을 구성하는 데 초점을 맞출 수 있습니다.

Oracle Net Manager의 새로운 기능

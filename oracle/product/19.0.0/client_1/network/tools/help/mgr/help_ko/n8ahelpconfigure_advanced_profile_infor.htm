고급 프로파일 정보 구성

프로파일은 다음 고급 기능으로 구성할 수 있습니다.

  • 접속 시간 초과 값

  • 로그 파일의 고유한 클라이언트 식별자

  • 소멸 프로세스를 정리하는 신호 처리기

  • 진행 중인 작업 중단

고급 프로파일 정보를 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창의 목록에서 일반 사항을 선택합니다.

  3. 고급 탭을 선택합니다.

  4. 설정할 옵션에 대한 값을 입력합니다.

  5. 파일 > 네트워크 구성 저장을 선택합니다.

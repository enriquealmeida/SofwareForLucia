Archivo tnsnames.ora

Archivo de configuración que contiene nombres de servicio de red asignados a descriptores de conexión. Este archivo se utiliza para el método de nomenclatura local.

El archivo tnsnames.ora debe residir en una de las siguientes ubicaciones:

  1. El directorio especificado por la variable de entorno TNS_ADMIN. Si la variable de entorno TNS_ADMIN no está definida como una variable en Windows NT, puede que esté en el registro.

  2. El directorio de configuración global del nodo. Para Sun Solaris, este directorio es /var/opt/oracle. Windows NT no tiene un directorio central.

  3. El directorio $ORACLE_HOME/network/admin en UNIX o el directorio ORACLE_HOME\network\admin en sistemas operativos Windows.

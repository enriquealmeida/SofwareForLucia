Registro de Serviço

Um recurso através do qual instâncias e dispatchers registram automaticamente informações com um listener. Como essas informações são registradas com o listener, o arquivo listener.ora não precisa ser configurado com essas informações estáticas.

Durante a inicialização do banco de dados, o registro de serviços fornece ao listener informações sobre:

  • Nomes de serviços para cada instância do banco de dados sendo executada

  • Nomes de instâncias do banco de dados

  • Handlers de serviços (dispatchers e servidores dedicados) disponíveis para cada instância

  • Informações de carga do dispatcher, da instância e do nó

Estes itens permitem que o listener direcione uma solicitação de cliente de forma adequada.

Essas informações de carga permitem que o listener determine qual dispatcher pode tratar melhor uma solicitação de conexão de cliente. Se todos os dispatchers estiverem bloqueados, o listener poderá criar um servidor dedicado para a conexão.

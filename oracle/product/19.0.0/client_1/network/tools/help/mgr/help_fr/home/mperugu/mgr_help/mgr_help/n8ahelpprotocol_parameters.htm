Paramètres de protocole

Le processus d'écoute, le serveur Oracle Names et Oracle Connection Manager sont identifiés par une adresse de protocole. Les clients qui veulent utiliser ces services doivent aussi être configurés avec la même adresse. Les paramètres utilisés par les protocoles Oracle pris en charge sont décrits ci-après.

Protocole Paramètre Description
IPC PROTOCOL Indiquez ipc.
IPC KEY Indiquez un nom unique pour le service. Oracle recommande l'utilisation du nom de service ou de l'identificateur système (SID) Oracle du service.
Named Pipes PROTOCOL Indiquez nmp.
Named Pipes SERVER Indiquez le nom de l'ordinateur serveur Oracle.
Named Pipes PIPE Indiquez le nom de canal pour la connexion au serveur de base de données. Il peut s'agit de tout nom arbitraire.
TCP/IP et TCP/IP avec SSL PROTOCOL Indiquez la valeur tcp pour TCP/IP, ou tcps pour TCP/IP avec SSL.
TCP/IP et TCP/IP avec SSL HOST Indiquez le nom d'hôte ou l'adresse IP de l'ordinateur.
TCP/IP et TCP/IP avec SSL PORT Indiquez le numéro de port du processus d'écoute.

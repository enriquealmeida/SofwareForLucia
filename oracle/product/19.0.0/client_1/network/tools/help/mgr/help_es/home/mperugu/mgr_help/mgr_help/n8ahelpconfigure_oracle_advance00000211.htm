Configuración de Oracle Advanced Security

Utilice el Gestor de Red de Oracle para configurar Oracle Advanced Security. Cada página de los separadores de Oracle Advanced Security permite configurar un juego de parámetros independiente. Oracle Advanced Security permite la comprobación del cifrado y la integridad de los datos, autenticación mejorada, conexión única. Oracle Advanced Security también proporciona gestión centralizada de usuarios en servidores de directorios compatibles con LDAP y conexión única basada en certificados; esta función utiliza SSL (Secure Sockets Layer ).

Para configurar Oracle Advanced Security:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione Oracle Advanced Security de la lista del panel de la derecha.

  3. Aparecen los siguientes paneles de separadores:

    • Autenticación

    • Otros Parámetros

    • Integridad

    • Cifrado

    • SSL

  4. Realice las modificaciones en la configuración.

  5. Seleccione Archivo > Guardar Configuración de Red.

Las modificaciones se guardan en el archivo sqlnet.ora.

Nota: Si Oracle Advanced Security no aparece en la lista, significa que no se ha instalado.

Para obtener más información, consulte Oracle Advanced Security Administrator's Guide (Guía del Administrador de Oracle Advanced Security).

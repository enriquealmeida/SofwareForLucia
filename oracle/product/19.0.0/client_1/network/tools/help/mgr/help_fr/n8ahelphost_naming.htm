Résolution de noms d'hôte

Méthode de résolution de noms permettant aux utilisateurs de se connecter à un serveur de base de données Oracle en utilisant un alias de nom d'hôte dans un environnement TCP/IP. Les noms d'hôte sont associés au nom global de base de données du serveur dans un service de résolution de noms existant, tel que DNS (Domain Name System), NIS (Network Information Service), ou dans un ensemble de fichiers /etc/hosts géré de façon centralisée.

Pour obtenir des informations sur la configuration de la méthode de résolution de noms d'hôte, reportez-vous au chapitre Configuring Naming Methods dans le Guide de l'administrateur Oracle10i Net Services.

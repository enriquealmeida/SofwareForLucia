Denominazione: Metodi

La scheda Metodi consente di specificare quali metodi di denominazione verranno utilizzati da un computer client e il relativo ordine di priorità. Il primo metodo di denominazione nella lista è il primo ad essere utilizzato per risolvere un determinato identificativo di connessione, nome di servizio di rete o servizio di database. Se il primo metodo non risolve il nome, viene utilizzato il metodo successivo nella lista.

Durante una richiesta di connessione, il client tenta di contattare il primo metodo di denominazione elencato per risolvere gli identificativi di connessione in un descrittore di connessione. Il client inoltrerà, quindi, la richiesta all'indirizzo del listener specificato nel descrittore di connessione.

Oracle Net fornisce il supporto per i seguenti metodi di denominazione:

  • Denominazione locale (TNSNAMES)

  • Denominazione delle directory (LDAP)

  • Denominazione host (HOSTNAME)

  • Denominazione di connessione semplice (EZCONNECT)

  • NIS (Network Information Service)

Pulsante <

Scegliere questo pulsante per spostare un metodo di denominazione selezionato dalla lista Metodi disponibili alla lista Metodi selezionati.

Pulsante >

Scegliere questo pulsante per spostare un metodo di denominazione selezionato dalla lista Metodi selezionati alla lista Metodi disponibili.

Pulsante Avanza

Scegliere questo pulsante per spostare un metodo di denominazione selezionato in alto nella lista Metodi selezionati. Oracle Net tenterà di utilizzare il primo metodo per risolvere un nome di servizio di rete o un nome di servizio di database. Se il tentativo non riesce, verrà utilizzato il secondo metodo e così via.

Pulsante Retrocedi

Scegliere questo pulsante per spostare un metodo di denominazione selezionato in basso nella lista Metodi selezionati.

Dominio predefinito

Immettere il nome del dominio della rete da cui proviene la maggior parte delle richieste client. Il nome può essere quello del dominio in cui il client risiede oppure uno da cui il client richiede con frequenza i servizi di rete.

Una volta impostato, il nome del dominio viene automaticamente aggiunto a qualsiasi nome non qualificato.

Export d'entrées Oracle Names vers un serveur d'annuaire

Oracle Names n'est plus pris en charge dans cette version. A l'aide d'une version 9.2 d'Oracle Names, vous devez exporter les objets de base de données et d'alias stockés dans un serveur Oracle Names vers un serveur d'annuaire, ou, pour un export indirect, vers un fichier LDIF (LDAP Data Interchange Format), que vous utiliserez ensuite pour charger le serveur d'annuaire. Lors de l'export des objets de base de données, seul l'enregistrement d'adresse (A.SMD) est exporté. Il contient les informations de descripteur de connexion de l'objet. Les objets de base de données sont exportés en tant que noms de service réseau. Les alias sont exportés en tant qu'alias de service réseau. Les données sont exportées à partir du domaine indiqué.

Pour plus d'informations, reportez-vous au chapitre Configuring Naming Methods dans le Guide de l'administrateur Oracle10i Net.

Pour obtenir des instructions complètes sur l'export de données à partir d'un serveur Oracle Names, reportez-vous au Guide de l'administrateur Oracle9i Net Services (à partir de la version 9.2).

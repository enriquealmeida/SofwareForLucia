Exportación de Entradas de Oracle Names a un Servidor de Directorios

Oracle Names ya no está soportado en esta versión. Con la versión 9.2 de Oracle Names, debe exportar los objetos de alias y base de datos almacenados en Oracle Names Server a un servidor de directorios o indirectamente a un archivo de formato de intercambio de datos de LDAP (LDIF), que se puede utilizar para cargar el servidor de directorios. Al exportar los objetos de base de datos, sólo se exporta el registro de direcciones (A.SMD). El registro de direcciones contiene la información del descriptor de conexión del objeto de base de datos. Los objetos de base de datos se exportan como nombres de servicio de red. Los alias se exportan como alias de servicio de red. Los datos se exportan desde un dominio especificado.

Para obtener más información, consulte el capítulo sobre configuración de métodos de nomenclatura, en Oracle10i Net Administrator's Guide (Guía del Administrador de Red de Oracle10i).

Para obtener instrucciones completas sobre la exportación de datos desde un servidor de Oracle Names, consulte Oracle9i Net Services Administrator's Guide (Guía del Administrador de Servicios de Red de Oracle9i) de la versión 9.2.

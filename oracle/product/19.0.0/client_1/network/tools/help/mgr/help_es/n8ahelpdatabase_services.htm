Servicios de Base de Datos

El separador Servicios de Base de Datos permite especificar los servicios de base de datos para los que el listener debe recibir solicitudes de conexión. El listener necesita esta información para conectar un cliente a la base de datos.

Las instancias Oracle9i y Oracle8i utilizan un registro de servicio dinámico para informar al listener sobre sus servicios de base de datos. Las bases de datos Oracle7 y Oracle8 versión 8.0 y las herramientas de Oracle Enterprise Manager necesitan configuración de servicio estático en el archivo listener.ora .

Haga clic en Agregar Base de Datos para agregar información de servicio de base de datos.

Nombre de la Base de Datos Global

Introduzca el nombre de la base de datos y del dominio de la base de datos, como se especifica en el parámetro SERVICE_NAMES o en los parámetros DB_NAME y DB_DOMAIN del archivo de parámetros de inicialización.

Directorio Raíz de Oracle

Introduzca la ubicación del directorio raíz de Oracle para la base de datos.

Nota: Windows NT ya conoce el directorio raíz de Oracle. Por lo tanto, no se necesita este campo para realizar una conexión a un servicio en Windows NT.

SID

Introduzca el identificador del sistema Oracle (SID) para la instancia.

Botón Agregar Base de Datos

Haga clic en este botón para configurar información de servicio para una base de datos. Aparece un nuevo separador Base de Datos en el que puede introducir información de servicio para otra base de datos.

Botón Eliminar Base de Datos

Haga clic en este botón para suprimir el separador Base de Datos seleccionado y su información de servicio.

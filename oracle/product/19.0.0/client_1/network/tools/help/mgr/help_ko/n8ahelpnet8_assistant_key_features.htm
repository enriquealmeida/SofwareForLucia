Oracle Net Manager의 주요 기능

Oracle Net Manager는 Oracle 네트워킹 환경을 구성하고 관리하도록 하는 도구입니다. Oracle Net의 다음 기능과 구성 요소는 Oracle Net Manager를 사용하여 구성하고 관리할 수 있습니다.

서비스 이름 지정

tnsnames.ora 파일, 디렉토리 서버 또는 Oracle Names Server에서 데이터베이스 서비스에 대한 네트워크 설명을 생성하거나 수정할 수 있습니다. 접속 기술자라고 하는 네트워크 설명은 클라이언트가 데이터베이스 접속 중에 접속 문자열에 사용하는 접속 식별자에 매핑됩니다.

리스너

리스너를 생성 또는 수정할 수 있습니다. 리스너는 데이터베이스 서비스에 대한 클라이언트 접속 요청을 수신하고 응답하는 서버 프로세스입니다.

프로파일

프로파일을 생성 또는 수정할 수 있습니다. 프로파일은 클라이언트가 Oracle 네트워크에 접속하는 방법을 결정하는 매개변수 모음입니다. 이름 지정 방법, 로깅, 추적, 외부 이름 지정 매개변수 및 Oracle Advanced Security에 클라이언트 매개변수를 사용할 수 있습니다.

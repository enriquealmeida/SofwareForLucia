일반 사항: 액세스 권한

[액세스 권한] 탭을 사용하여 데이터베이스에 대한 액세스를 허용 또는 거부할 클라이언트를 지정할 수 있습니다.

TCP/IP 클라이언트 액세스 권한 확인

데이터베이스 액세스를 허용 및 거부할 클라이언트를 지정하려는 경우 이 체크 박스를 선택합니다. 이 기능을 사용 안함으로 설정하려면 체크 박스를 선택하지 마십시오. 체크 박스의 선택을 해제하면 모든 클라이언트의 액세스가 허용됩니다.

액세스에서 제외된 클라이언트

데이터베이스 액세스를 거부할 클라이언트를 지정하려면 호스트 이름이나 IP 주소를 입력합니다. 동일한 행에 있는 항목을 구분하려면 콤마를 사용합니다.

액세스가 허용된 클라이언트

데이터베이스 액세스를 허용할 클라이언트를 지정하려면 호스트 이름이나 IP 주소를 입력합니다. 동일한 행에 있는 항목을 구분하려면 콤마를 사용합니다.

주: 액세스가 허용된 클라이언트는 액세스에서 제외된 클라이언트보다 우선합니다.

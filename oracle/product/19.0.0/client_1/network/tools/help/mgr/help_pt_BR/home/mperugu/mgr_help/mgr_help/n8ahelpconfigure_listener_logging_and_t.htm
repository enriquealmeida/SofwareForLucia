Configurando Log e Rastreamento de Listener

Você pode configurar o listener com parâmetros de log e rastreamento.

Em geral, as informações de log são armazenadas no arquivo listener.log localizado em $ORACLE_HOME/network/log para plataformas UNIX e em ORACLE_HOME\network\log para plataformas Windows.

Por padrão, as informações de rastreamento são armazenadas no arquivo listener.trc, localizado em $ORACLE_HOME/network/trace para plataformas UNIX e em ORACLE_HOME\network\trace para plataformas Windows.

Para ativar o log ou rastreamento:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Listeners.

  2. Selecione um listener.

  3. Na lista do painel direito, selecione Parâmetros Gerais.

  4. Selecione a guia Log e Rastreamento.

  5. Ajuste as definições de log e rastreamento conforme necessário.

  6. Escolha Arquivo > Salvar Configuração de Rede.

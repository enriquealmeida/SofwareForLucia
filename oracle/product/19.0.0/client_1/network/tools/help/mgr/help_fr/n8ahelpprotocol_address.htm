Adresse de protocole

Adresse correspondant à l'adresse de réseau d'un objet de réseau.

Lorsqu'une connexion est effectuée, le client et le destinataire de la demande, par exemple le processus d'écoute, le serveur Oracle Names ou Oracle Connection Manager, sont configurés avec des adresses de protocole identiques. Le client utilise cette adresse pour envoyer la demande de connexion à un emplacement d'objet de réseau particulier, et le destinataire "écoute" les demandes envoyées à cette adresse. Il est important d'installer les mêmes protocoles pour le client et le destinataire de la connexion, ainsi que de configurer les mêmes adresses.

Les adresses de protocole comprennent les éléments ADDRESS et ADDRESS_LIST.

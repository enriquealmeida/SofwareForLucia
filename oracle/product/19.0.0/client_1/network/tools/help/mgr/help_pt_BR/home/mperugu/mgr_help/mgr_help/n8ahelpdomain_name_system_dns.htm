DNS (Domain Name System)

Sistema destinado à nomeação de computadores e serviços de rede, organizado em uma hierarquia de domínios. O DNS é usado em redes TCP/IP para localizar computadores através de nomes amigáveis. O DNS resolve um nome amigável como um endereço IP, que é entendido pelos computadores.

No Oracle Net, o DNS traduz o nome do host usado em endereços TCP/IP para um endereço IP.

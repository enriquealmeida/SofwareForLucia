Archivo sqlnet.ora

Archivo de configuración para el cliente o el servidor que especifica:

  • El dominio del cliente que se agrega a nombres de servicio no cualificados o nombres del servicio de red

  • El orden de los métodos de nomenclatura que debería utilizar el cliente al resolver nombres

  • Las funciones de registro y rastreo que se van a utilizar

  • La ruta de las conexiones

  • Los Oracle Names Servers preferidos

  • Los parámetros de nomenclatura externa

  • Los parámetros de Oracle Advanced Security

El archivo sqlnet.ora debe residir en una de las siguientes ubicaciones:

  1. El directorio especificado por la variable de entorno TNS_ADMIN. Si la variable de entorno TNS_ADMIN no está definida como una variable en Windows NT, puede que esté en el registro.

  2. El directorio $ORACLE_HOME/network/admin en UNIX o el directorio ORACLE_HOME\network\admin en sistemas operativos Windows.

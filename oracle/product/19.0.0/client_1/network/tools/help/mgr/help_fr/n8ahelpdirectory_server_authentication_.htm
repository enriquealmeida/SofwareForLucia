Boîte de dialogue Authentification de serveur d'annuaire

La boîte de dialogue Authentification de serveur d'annuaire vous permet de définir des éléments d'authentification du contexte Oracle sélectionné dans le serveur d'annuaire.

Oracle Net Manager doit être authentifié auprès du serveur d'annuaire pour pouvoir effectuer des opérations sur le serveur d'annuaire, comme l'affichage, l'ajout ou la modification d'entrées Oracle Net dans l'annuaire.

· Utilisateur

Indiquez un utilisateur autorisé qui a accès à l'annuaire.

Si vous voulez ajouter ou modifier des entrées, indiquez un administrateur d'annuaire disposant de droits d'ajout et de modification sous la forme d'un nom distinctif (DN). Par exemple, c=US,o=EXAMPLE,ou=admin,cn=scott, représente le nom distinctif (DN) de l'administrateur Scott.

Mot de passe

Entrez le mot de passe de l'utilisateur.

Remarque : Pour afficher uniquement des objets et l'annuaire qui a été configuré avec des privilèges "anonymes", ne remplissez pas les champs Utilisateur et Mot de passe.

Par défaut, Oracle Net Manager tente de s'authentifier auprès de l'annuaire en utilisant l'une des méthodes suivantes :

  • Authentification native : Un nom utilisateur et un mot de passe de système d'exploitation Windows NT sont utilisés.

  • Authentification intense : Un certificat SSL (Secure Sockets Layer) est utilisé.

Caixa de Diálogo Autenticação de Servidor de Diretórios

A caixa de diálogo Autenticação de Servidor de Diretório permite que você autentique credenciais de autenticação para o Oracle Context do servidor de diretório.

O Oracle Net Manager deverá ser autenticado com o servidor de diretório para executar qualquer operação com o servidor de diretório, como, por exemplo, exibir, adicionar ou modificar entradas do Oracle Net no diretório.

Usuário

Informe um usuário autorizado que tenha acesso ao diretório.

Se quiser adicionar ou modificar entradas, informe um usuário administrador de diretórios que tenha privilégios de acréscimo e·modificação na forma de um DN ("distinguished name" – nome distinto). c=US,o=EXAMPLE,ou=admin,cn=scott, por exemplo, é o DN de um administrador chamado Scott.

Senha

Informe a senha do usuário.

Observação: Se você desejar simplesmente ver objetos e se o diretório tiver sido configurado com privilégios "anônimos", deixe os campos Usuário e Senha em branco.

Por default, o Oracle Net Manager tenta autenticar com o diretório usando um dos seguintes métodos:

  • Autenticação Nativa: São usados um nome de usuário e uma senha do sistema operacional do Windows NT.

  • Autenticação Rígida: É usado um certificado SSL (Secure Sockets Layer).

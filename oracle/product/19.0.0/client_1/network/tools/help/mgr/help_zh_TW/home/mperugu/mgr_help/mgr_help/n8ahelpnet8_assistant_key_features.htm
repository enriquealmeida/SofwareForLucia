Oracle Net Manager 主要功能

Oracle Net Manager 是一種可以協助您設定與管理 Oracle 網路環境的工具。您可以使用 Oracle Net Manager 來設定和管理以下的 Oracle Net 功能與元件：

服務命名

您可以在 tnsnames.ora 檔案、目錄伺服器或 Oracle Names 伺服器中，建立或修改資料庫服務的網路描述。網路描述 (稱為連線描述區) 會對應至從屬端在資料庫連線期間，於連線字串中使用的連線 ID。

監聽器

您可以建立或修改監聽器 (伺服器接收並回應從屬端要求連線資料庫服務的處理作業)。

設定檔

您可以建立或修改設定檔 (決定從屬端如何連線至 Oracle 網路的參數集合)。您可以使用命名方法、記錄日誌、追蹤和外部命名參數，以及 Oracle Advanced Security 的從屬端參數。

Oracle Net Services 簡介

Oracle Net Services 提供分散、異質電腦環境的整體企業連線解決方案。Oracle Net Services 可簡化網路組態與管理的複雜度、提供最大效能，以及改進網路診斷能力。

Oracle Net 是 Oracle Net Services 的一個元件，可以讓從屬端應用程式建立 Oracle 資料庫伺服器的網路階段作業。建立網路階段作業之後，Oracle Net 可以當成從屬端應用程式與資料庫伺服器的資料信使。它負責建立與維護從屬端應用程式與資料庫伺服器之間的連線，以及交換之間的訊息。Oracle Net 可以執行這些工作，因為它位於網路的每一部電腦上。

網路階段作業是透過監聽器的協助所建立。監聽器是存在資料庫伺服器上的個別處理作業。監聽器會接收內送的從屬端連線要求，並管理這些要求與伺服器之間的交通。

監聽器會中介從屬端要求，將要求交與伺服器。每次從屬端要求與伺服器的階段作業時，監聽器會接收實際的要求。如果從屬端的資訊符合監聽器資訊，監聽器便會允許伺服器連線。

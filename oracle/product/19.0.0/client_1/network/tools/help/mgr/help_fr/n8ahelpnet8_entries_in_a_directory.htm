Entrées Oracle Net dans un annuaire

Dans un annuaire, les entrées sont organisées en arborescence hiérarchique en fonction de leur nom distinctif (DN). Cette arborescence est appelée arborescence d'informations d'annuaire(DIT). Chaque entrée contient des attributs qui la décrivent.

Les entrées Oracle Net d'une arborescence d'informations d'annuaire peuvent être :

  • Un service de base de données créé au cours de la création de la base de données

  • Un nom de service réseau

  • Un alias de service réseau

Toutes les entrées Oracle Net apparaissent dans une sous-arborescence de contexte Oracle qui stocke les entrées Oracle. Cette sous arborescence est créée dans un emplacement, défini par l'utilisateur, de l'arborescence d'informations d'annuaire lors de l'installation personnalisée sur le serveur. Les clients sont configurés avec utilisation de l'annuaire et un contexte Oracle où rechercher les entrées.

Chaque entrée contient un ou plusieurs attributs qui la décrivent. Chaque attribut a un type et une valeur. Les entrées d'annuaire Oracle Net ont les attributs suivants :

  • Nom commun

  • Version de l'entrée

  • Descripteur de connexion

Nombre del Servicio

Representación lógica de una base de datos, que es la forma en que la base de datos se presenta a los clientes. Una base de datos se puede presentar como varios servicios y un servicio se puede implementar como varias instancias de base de datos. El nombre de servicio es una cadena que representa el nombre de la base de datos global, es decir, un nombre formado por el nombre de la base de datos y el nombre del dominio, introducido durante la instalación o creación de la base de datos. Si no está seguro de cuál es el nombre de la base de datos global, puede obtenerlo de los valores combinados del parámetro SERVICE_NAMES del archivo de parámetros de inicialización.

Los descriptores de conexión que utilizan los clientes para conectarse a una base de datos Oracle9i u Oracle8i se deben configurar con el parámetro SERVICE_NAME (sin S).

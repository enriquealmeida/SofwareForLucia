Alias de Servicio de Red

Nombre alternativo de un objeto de nomenclatura de directorios en un servidor de directorios. Un servidor de directorios almacena alias de servicio de red para cualquier nombre de servicio de red o servicio de base de datos. Una entrada de alias de servicio de red no tiene información del conector de conexión. En su lugar, sólo hace referencia a la ubicación del objeto para el que es un alias. Cuando un cliente solicita una consulta de directorio de un alias de servicio de red, el directorio determina que la entrada es un alias de servicio de red y realiza la consulta como si se tratara de la entrada a la que se hace referencia.

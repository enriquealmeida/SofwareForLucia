Utility Controllo del listener

Una utility inclusa in Oracle Net Services per controllare le varie funzioni del listener, ad esempio l'avvio, l'arresto e lo stato del listener. La utility Controllo del listener viene eseguita come riportato di seguito.

lsnrctl comando [nome_listener]

[listener_name] è il nome del listener definito nel file listener.ora. Non è necessario identificare il listener se viene utilizzato il nome del listener predefinito, LISTENER.

Per ulteriori informazioni, vedere il capitolo Listener Control Utility del manuale Oracle10i Net Services Reference Guide.

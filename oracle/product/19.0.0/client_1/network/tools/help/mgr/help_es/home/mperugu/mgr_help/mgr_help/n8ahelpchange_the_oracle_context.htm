Modificación de Oracle Context

Al iniciar el Gestor de Red de Oracle, los objetos que aparecen en la carpeta Directorio > Nomenclatura de Servicios son las entradas del servicio Oracle del Contexto de Oracle por defecto establecido durante la configuración del uso de directorios con el Asistente de Configuración de Red de Oracle. Puede crear y modificar entradas para otro Contexto de Oracle del que tenga autenticación.

Para cambiar el Oracle Context actual:

  1. Seleccione Comando > Directorio > Cambiar Contexto Actual. Aparece el recuadro de diálogo Seleccionar Nuevo Oracle Context.

  2. En la lista Contexto de Nomenclatura de Directorios, seleccione la entrada de directorio que contenga el Oracle Context que desea ver.

  3. En la lista Contexto de Oracle seleccione el Contexto de Oracle y, a continuación, haga clic en Aceptar. Si no se han guardado las modificaciones realizadas en el Oracle Context actual, un recuadro de diálogo denominado Guardar le solicita que guarde estas modificaciones.

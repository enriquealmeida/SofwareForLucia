Panoramica sul listener

I servizi coordinano le sessioni mediante un listener, un processo sul server che riceve le richieste di connessione per conto di un'applicazione client. I listener sono configurati per eseguire le operazioni di "ascolto" su indirizzi di protocollo per un servizio di database o non di database.

Gli indirizzi di protocollo sono configurati nel corrispondente file di configurazione del listener, listener.ora, per un servizio di database o non di database. Un servizio di database Oracle9i o Oracle8i registra automaticamente le informazioni presso il listener, ad esempio i nomi dei servizi, i nomi delle istanze e le informazioni sul caricamento. Questa funzione, denominata registrazione del servizio, non richiede la configurazione dei servizi nel file listener.ora. Altri servizi, come ad esempio i servizi di database Oracle8 e Oracle7 e gli strumenti di gestione di Oracle Enterprise Manager, richiedono invece la configurazione nel file listener.ora .

Dopo essere stato configurato, il listener può essere gestito con la utility Controllo del listener.

Exportando Entradas do Oracle Names para um Servidor de Diretórios

O Oracle Names não é mais suportado nesta release. Usando uma release 9.2 do Oracle Names, você pode exportar objetos de banco de dados e de apelido armazenados em um servidor Oracle Names para um servidor de diretórios ou indiretamente para um arquivo no Formato de Intercâmbio de Dados LDAP (LDIF), que pode, então, ser usado para carregar o servidor de diretórios. Ao exportar os objetos de banco de dados, somente o registro de endereço (A.SMD) é exportado. O registro de endereço contém as informações do descritor de conexão para o objeto de banco de dados. Os objetos de banco de dados são exportados como nomes de serviço de rede. Os apelidos são exportados como apelidos de serviço de rede. Os dados são exportados de um domínio específico.

Para obter mais informações, consulte o capítulo Configuring Naming Methods no Oracle10i Net Administrator's Guide.

Para obter instruções completas para exportação de dados de um servidor do Oracle Names, consulte Oracle9i Net Services Administrator's Guide da release 9.2.

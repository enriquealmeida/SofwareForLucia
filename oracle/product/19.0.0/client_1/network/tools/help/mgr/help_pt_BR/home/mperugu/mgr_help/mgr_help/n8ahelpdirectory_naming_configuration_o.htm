Visão Geral da Configuração de Nomeação de Diretório

Este tópico abrange os detalhes da configuração da Nomenclatura de Diretórios.

Configuração do Servidor Durante a Instalação

Durante uma instalação Personalizada do servidor, o Assistente de Configuração do Oracle Net solicita que você:

  • Selecione o tipo de servidor de diretório

  • Identifique a localização do diretório

  • Selecione ou crie um Oracle Context do qual este servidor pode pesquisar, criar e modificar identificadores de conexão

  • Configure a nomeação de diretório como o método de nomeação preferencial a ser utilizado

Se o acesso a diretório for configurado da forma adequada, o Assistente de Configuração de Banco de Dados Oracle tentará estabelecer conexão com o diretório e registrar uma entrada de serviço no diretório. Esta é configurada com um descritor de conexão, contendo informações de rede sobre o(a):

  • Endereço do listener

  • Nome do serviço de banco de dados

Configuração do Cliente Durante a Instalação

Durante todas as instalações do cliente, o Assistente de Configuração do Oracle Net solicita que você configure as seguintes definições de uso de diretório:

  • Especifique o tipo de diretório

  • Identifique a localização do diretório

  • Selecione um Oracle Context default do qual o cliente possa pesquisar identificadores de conexão

  • A nomeação de diretório como o método preferencial de nomeação a ser usado

Configuração com o Assistente de Configuração do Oracle Net após a Instalação

O Assistente de Configuração do Oracle Net também pode ser executado no modo independente para:

  • Configurar o cliente ou servidor (como descrevemos anteriormente)

  • Criar o esquema Oracle

  • Criar um Oracle Context

Para executar o Assistente de Configuração do Oracle Net no modo independente:

  • No UNIX, execute netca em $ORACLE_HOME/bin.

  • No Windows NT, escolha Iniciar > Programas >Oracle - NOME_DO_HOME > Administração de Rede > Assistente de Configuração do Oracle Net.

Configuração com o Oracle Net Manager.

O Oracle Net Manager permite:

  • Alterar o Oracle Context

  • Alterar credenciais de autenticação

  • Criar nomes de serviços de rede

  • Criar apelidos de serviços de rede

  • Configurar várias rotas para um listener através de vários endereços de protocolo

  • Configure informações avançadas sobre CONNECT_DATA para um nome de serviço de banco de dados ou de rede.

  • Migrar entradas de nome do serviço de rede de um arquivo tnsnames.ora para um servidor de diretórios

Para acessar a configuração de nomeação de diretório, no painel do navegador, escolha Nomeação do Serviço de Diretório.

Para obter mais informações, consulte o sistema de ajuda do Assistente de Configuração do Oracle Net.

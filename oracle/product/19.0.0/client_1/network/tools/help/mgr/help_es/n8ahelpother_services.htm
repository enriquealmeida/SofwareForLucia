Otros Servicios

El separador Otros Servicios permite especificar los servicios que no sean de base de datos para los que el listener recibe solicitudes de conexión. Los servicios que no son de base de datos incluyen procedimientos externos o Servicios Heterogéneos.

Nombre de Servicio Global

Introduzca el nombre y el dominio del servicio.

Nombre del Programa

Introduzca el nombre del programa ejecutable. Por ejemplo, extproc.exe.

Argumento de Programa Cero

Introduzca el primer argumento interno, que normalmente se utiliza como nombre de programa alternativo. Por ejemplo: argv0.

Argumentos del Programa

Introduzca los argumentos de la línea de comandos que se deben transferir cuando se inicia el servicio. Por ejemplo, -mode=5.

Entorno

Introduzca todas las variables de entorno que se deben establecer antes de iniciar el servicio. Las variables de entorno se especifican con el formato siguiente:

environment_variable=values

Por ejemplo, user=scott, start=immediate.

Recuerde que las variables van separadas por comas.

SID

Introduzca el nombre del programa o el identificador del sistema Oracle (SID) para el servicio.

Directorio Raíz de Oracle

Introduzca la ubicación del programa ejecutable.

Botón Agregar Servicio

Haga clic en este botón para configurar información de servicio que no sea de base de datos. Aparece un nuevo separador Servicio en el que puede introducir información para otro servicio.

Botón Eliminar Servicio

Haga clic en este botón para suprimir el separador Servicio seleccionado y su información de servicio.

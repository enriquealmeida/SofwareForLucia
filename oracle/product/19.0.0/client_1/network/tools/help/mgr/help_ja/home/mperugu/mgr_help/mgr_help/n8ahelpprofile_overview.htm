プロファイルの概要

プロファイルでは、クライアントまたはサーバー上で、Oracle Net機能を使用可能にしたり構成する際の作業環境を指定します。プロファイルを使用すると、次のことが可能になります。

  • 未修飾名に追加するクライアント・ドメインの指定

  • ネーミング・メソッドの優先順位付け

  • ロギングおよびトレース機能の有効化

  • 特定のプロセスを介したルート接続

  • 外部ネーミング用のパラメータの構成

  • Oracle Advanced Securityの構成

プロファイルにアクセスするには、ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「プロファイル」を展開します。

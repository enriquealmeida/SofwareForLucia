레코드 유형

Oracle Names Server에 저장된 네트워크 객체 정보 유형입니다. Oracle Names Server는 다음 레코드 유형을 지원합니다.

- A.SMD: 데이터베이스 서비스 주소와 네트 서비스 이름을 포함하는 네트워크 주소

- CNAME.SMD: 네트워크 객체에 대한 별칭

- DL.RDBMS.OMD: 전역 데이터베이스 링크

- DLCR.RDBMS.OMD: 링크 식별자

- NS.SMD: Oracle Name Server의 주소

- V1ADD.NPO.OMD: SQL*Net 버전 1 접속 문자열

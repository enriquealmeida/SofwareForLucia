配置监听程序地址

监听程序必须配置有一个或多个协议地址。监听程序从配置有相同地址的客户机那里接收连接请求。

要配置监听程序的协议地址，请执行以下操作：

  1. a. 在导航器窗格中，展开 Oracle Net 配置 > 本地 > 监听程序。

  2. 选择监听程序。

  3. 从右边窗格的列表中选择监听位置。

  4. 选择添加地址。将显示新的地址选项卡。

  5. 选择一个协议，然后为所选协议输入相应的协议参数信息。

    当配置监听程序对 Oracle Net 客户机的 TCP/IP 进行监听时，应该选择默认端口或 1521 端口。如果不这样做，则必须在初始化参数文件中配置 LOCAL_LISTENER 参数并通过命名方法解析监听程序名。

  6. 如果计算机有多个 IP 地址，而且希望监听程序监听所有可用的 IP 地址，请配置 TCP/IP 或使用 SSL 的 TCP/IP，并在主机字段中输入该计算机的主机名。

  7. 还可以单击显示高级设置指定相应的设置，以此指定发送和接收会话操作的 I/O 缓冲区空间。

  8. 选择文件 > 保存网络配置。

有关配置非默认端口的详细信息，请参阅 Oracle10i Net Administrator's Guide 中的 Configuring and Administering the Listener 一章。

有关配置 I/O 缓冲区空间的详细信息，请参阅 Oracle10i Net Administrator's Guide 中的 Optimizing Performance 一章。

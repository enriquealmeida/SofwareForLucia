Visión General de Perfil

Un perfil especifica preferencias para activar y configurar las funciones de Red de Oracle en el cliente o en el servidor. Con un perfil, puede:

  • Especificar el nombre del dominio que desea agregar a los nombres no cualificados.

  • Establecer prioridades en los métodos de nomenclatura.

  • Activar las funciones de registro y rastreo.

  • Dirigir conexiones a procesos específicos.

  • Configurar parámetros para la nomenclatura externa.

  • Configurar Oracle Advanced Security.

Para acceder a un perfil, desde el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

Structure de l'arborescence et dossiers hiérarchiques

Oracle Net Manager affiche une arborescence graphique d'objets réseau et les objets que ces derniers contiennent, organisés en hiérarchies de dossiers. Le panneau du navigateur vous permet d'afficher, de modifier, d'ajouter et de supprimer des objets dans chaque dossier.

Lorsque vous développez un dossier, vous voyez une liste imbriquée d'objets et de dossiers. Lorsqu'un objet est sélectionné, les informations le concernant apparaissent dans le panneau de droite d'Oracle Net Manager.

Répertoire

Affiche des noms simples, appelés identificateurs de connexion, configurés dans un serveur d'annuaire.

Remarque : le dossier Annuaire apparaît si l'accès à l'annuaire a été configuré. Pour plus d'informations sur la configuration de l'accès à l'annuaire et les autres exigences, reportez-vous à Exigences relatives à la résolution de noms d'annuaire pour Oracle Net Manager.

Local

Affiche les éléments du réseau configurés dans des fichiers de configuration locaux :

  • Noms de service réseau dans le fichier tnsnames.ora

  • Processus d'écoute dans le fichier listener.ora

  • Profils dans le fichier sqlnet.ora.

Serveurs Oracle Names

Affiche les serveurs Oracle Names.

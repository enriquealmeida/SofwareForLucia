配置高级连接数据选项

除了数据库的服务名或 Oracle 系统标识符 (SID) 以外，还可以选择使用以下选项配置连接描述符的 CONNECT_DATA 部分：

  • 连接特定数据库实例的实例名

  • 数据库的 Oracle 主目录，用于通过网络启动数据库

  • 指定缓冲区大小的会话数据单元大小

  • 连接专用服务器进程的专用服务器

  • 访问非 Oracle 系统的异构服务

  • 访问 Oracle Rdb 数据库的 Oracle Rdb 设置

要配置网络服务名的高级连接选项，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle Net 配置 > 目录 > 本地命名/服务命名。

  2. 选择网络服务名或数据库服务。右侧窗格显示当前目标服务和地址列表。

  3. 在服务标识框中，选择高级。将显示“高级服务选项”对话框。

  4. 输入适当的字段或选择适当的选项，然后选择确定。

  5. 在右侧窗格中，选择应用。

  6. 如果要对本地文件夹进行上述更改，请选择文件 > 保存网络配置。

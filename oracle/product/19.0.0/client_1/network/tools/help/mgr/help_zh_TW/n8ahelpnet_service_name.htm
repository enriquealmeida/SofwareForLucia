網路服務名稱

可解析成連線描述區的服務簡單名稱。使用者只要以連線字串傳送使用者名稱、密碼及網路服務名稱，即可針對要連線的服務起始連線要求：

CONNECT username/password@net_service_name

根據您需求的不同，可以將網路服務名稱儲存在不同的地方，包括：

  • 每一個從屬端上的區域組態檔 tnsnames.ora 檔案

  • 目錄伺服器

  • Oracle Names 伺服器

  • 外部命名服務，包括網路資訊服務 (NIS)

Nome di database globale

Il nome completo del database che lo identifica in modo univoco distinguendolo dagli altri database. Il nome di database globale è nel formato "database_name.database_domain", ad esempio sales.us.example.com.

La parte relativa al nome del database, sales, è un nome semplice da utilizzare per chiamare il database. La parte relativa al dominio del database, us.example.com , specifica il dominio che contiene il database e rende il nome del database globale univoco. Se possibile, il dominio del database dovrebbe rispecchiare il dominio di rete.

Il nome del database globale corrisponde al nome di servizio predefinito del database, specificato dal parametro SERVICE_NAMES nel file dei parametri di inizializzazione.

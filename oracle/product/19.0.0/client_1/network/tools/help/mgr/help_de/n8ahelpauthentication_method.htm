Authentifizierungsmethode

Eine Sicherheitsmethode, die Ihnen hohes Vertrauen in die Identität von Benutzern, Clients und Servern in verteilten Umgebungen gibt. Die Netzwerk-Authentifizierungsmethoden können auch die Funktion des Single Sign-On für Benutzer bereitstellen. Die folgenden Berechtigungsprüfungsmethoden werden in Oracle9i und Oracle8i unterstützt (je nachdem, ob Oracle Advanced Security installiert ist oder nicht):

  • CyberSafe

  • KERBEROS(v5)

  • RADIUS

  • SSL

  • Systemeigene Windows NT-Berechtigungsprüfung

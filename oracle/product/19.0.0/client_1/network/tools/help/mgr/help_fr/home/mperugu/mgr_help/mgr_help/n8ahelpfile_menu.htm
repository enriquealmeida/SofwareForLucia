Menu Fichier

Le menu Fichier permet d'effectuer des opérations d'ouverture, de rétablissement et d'enregistrement.

Ouvrir la configuration réseau

Choisissez cette option pour ouvrir des fichiers de configuration locaux.

Enregistrer la configuration réseau

Choisissez cette option pour enregistrer les paramètres de configuration courants à l'emplacement par défaut.

Enregistrer sous

Choisissez cette option pour enregistrer les paramètres de configuration courants à un emplacement autre que l'emplacement par défaut.

Rétablir la configuration enregistrée

Choisissez cette option pour n'enregistrer aucun des paramètres de configuration courants. Au lieu de cela, la dernière configuration enregistrée est récupérée.

Quitter

Choisissez cette option pour fermer Oracle Net Manager.

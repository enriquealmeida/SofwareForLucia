作業の概要

次のリストは、ネットワークに合せてOracle Advanced Securityを構成する手順を示しています。

  1. 認証方式を選択します。

  2. クライアントまたはサーバーのネゴシエーション時に使用する認証方式の優先順位を指定します。

  3. 選択した各認証方式の「その他のパラメータ」を構成します。

  4. クライアント側およびサーバー側の整合性(チェックサム・レベル)を構成します。

  5. クライアント側およびサーバー側の暗号化を構成します。

  6. クライアント側およびサーバー側のSecure Socket Layer (SSL)通信設定を構成します。

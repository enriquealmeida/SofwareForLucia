Creazione di un alias di servizio di rete

Note:

  • solo gli utenti che sono membri del gruppo OracleNetAdmins o OracleContextAdmins possono creare voci di alias di servizi di rete in una directory.

  • Gli alias dei servizi di rete non sono supportati con Microsoft Active Directory.

  • È possibile esportare gli oggetti alias esistenti da un server Oracle Names. Per ulteriori informazioni, vedere il capitolo Exporting Naming Data to a Directory Server del manuale Oracle10i Net Services Administrator's Guide.

È possibile creare alias di servizi di rete per gli oggetti di denominazione delle directory, ovvero i nomi di servizi di rete o i servizi di database, in un server delle directory. Un alias di servizi di rete memorizza il nome dell'oggetto a cui fa riferimento.

Quando un client richiede una ricerca all'interno di una directory di un alias del servizio di rete, la directory determina che la voce è un alias del servizio di rete e completa la ricerca come se si trattasse della voce di riferimento. Ad esempio, è possibile creare l'alias di servizio di rete salesalias per il nome di servizio di rete sales. Quando salesalias viene usato per connettersi a un servizio di database, come in CONNECT scott/tiger@salesalias, verrà effettuata la risoluzione e verranno utilizzate le informazioni del descrittore di connessione per sales.

Vi sono due utilizzi principali degli alias dei nomi di servizi di rete.

  • Utilizzare un alias del nome di servizio di rete per consentire ai client di fare riferimento a servizio di database o a un nome di servizio di rete mediante un altro nome.

  • Utilizzare un alias del nome di servizio di rete in un contesto Oracle per un servizio di database oppure un nome di servizio di rete in un contesto Oracle diverso. Ciò consente di definire una sola volta il servizio di database o il nome di servizio di rete nel server delle directory, ma di utilizzarlo al di fuori di altri contesti Oracle.

Per creare un alias del nome di servizio di rete, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Directory > Denominazione dei servizi > Alias.

  2. Fare clic sul segno più (+) nella barra degli strumenti oppure scegliere Modifica > Crea nella barra dei menu. Viene visualizzata la finestra di dialogo Crea alias di servizio di rete.

  3. Nel campo Alias di servizio di rete, immettere un nome per l'alias di servizio di rete.

  4. Nella casella del servizio di rete o del servizio di database, immettere le informazioni riportate di seguito sul nome di servizio di rete o sul servizio di database.

    • Contesto Oracle: specificare il contesto Oracle del servizio di database o del nome di servizio di rete selezionandone uno dalla lista o immettendone uno nel campo.

    • Nome: specificare il nome distinto (DN) del servizio di database o del nome di servizio di rete selezionandone uno dalla lista o immettendone uno nel campo.

  5. Scegliere Crea.

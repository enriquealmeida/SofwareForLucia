IP-Adresse

Mit der IP-Adresse wird ein Knoten auf einem Netzwerk identifiziert. Jedem Rechner in dem Netzwerk wird eine eindeutige IP-Adresse zugewiesen, die aus der Netzwerk-ID und einer eindeutigen Host-ID besteht. Diese Adresse wird typischerweise in der Punkt-Dezimal-Schreibweise dargestellt, wobei die Dezimalwerte jedes Bytes durch einen Punkt voneinander getrennt werden, z.B. 144.45.9.22.

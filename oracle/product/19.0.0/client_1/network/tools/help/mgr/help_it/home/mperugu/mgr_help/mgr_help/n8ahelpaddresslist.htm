ADDRESS_LIST

Una lista di indirizzi con caratteristiche comuni. Nell'esempio che segue, ADDRESS_LIST elenca due indirizzi con la funzione di bilanciamento del carico del client attivata:

(ADDRESS_LIST=

(LOAD_BALANCE=on)

(ADDRESS=

(PROTOCOL=tcp

(HOST=sales-pc)

(PORT=1521))

(ADDRESS=

(PROTOCOL=tcp)

(HOST=hr-server)

(PORT=1521)))

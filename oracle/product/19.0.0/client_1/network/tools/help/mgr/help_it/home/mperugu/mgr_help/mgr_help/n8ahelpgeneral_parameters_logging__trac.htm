Parametri generali: Log e trace

La scheda Log e trace consente di abilitare le funzioni di log e trace per il listener. Oracle Net Manager consente di specificare il nome e la posizione di scrittura dei file di log del listener.

Log disabilitato

Selezionare questa opzione per disabilitare la funzione di log sul listener.

Log abilitato

Selezionare questa opzione per abilitare la funzione di log sul listener. Se questa opzione è abilitata, è possibile specificare la posizione in cui verranno scritte le informazioni di log.

File di log

Immettere il percorso della directory e il nome del file in cui verranno scritte le informazioni di log del listener. L'estensione del nome di questo file è sempre log. Il nome predefinito è listener.log. La directory predefinita è $ORACLE_HOME/network/log sulle piattaforme UNIX e ORACLE_HOME\network\log sulle piattaforme Windows.

Trace disabilitato

Selezionare questa opzione per disabilitare la funzione di trace sul listener.

Trace abilitato

Selezionare questa opzione per abilitare la funzione di trace sul listener. Se questa opzione è abilitata, è possibile specificare il livello di trace desiderato e la posizione in cui le informazioni di trace dovranno essere scritte cambiando il file di trace predefinito.

Livello di trace

Dalla lista selezionare il livello di trace.

Off: il trace non è abilitato. OFF è l'impostazione predefinita.

Utente: il trace viene impostato su un livello appropriato per gli utenti. Consente di identificare le condizioni di errore provocate dagli utenti.

Amministratore: il trace viene impostato su un livello appropriato per gli amministratori del database. Consente di identificare i problemi specifici dell'installazione.

Supporto: il trace viene impostato su un livello appropriato per il servizio di assistenza clienti.

I file di trace possono raggiungere dimensioni notevoli. Si consiglia di disabilitare la funzione di trace quando non è in corso la risoluzione dei problemi della rete.

File di trace

Immettere il nome del percorso della directory e il nome del file in cui verranno scritte le informazioni di trace del listener. L'estensione del nome di questo file è sempre .trc. Il nome predefinito è listener.trc. La directory predefinita è $ORACLE_HOME/network/trace sulle piattaforme UNIX e ORACLE_HOME\network\trace sulle piattaforme Windows.

Parámetros Generales: General

El separador General permite configurar los parámetros generales para el listener.

Nombre del Listener:

Muestra el nombre del listener.

Tiempo de Espera de Inicio

Especifique el tiempo, en segundos, durante el cual el listener debe esperar antes de responder al comando START de la Utilidad de Control de Listener.

Nota: Este valor está obsoleto y no estará soportado en futuras versiones. Si necesita este parámetro para ejecutar el listener, notifíquelo a los Servicios de Soporte Oracle.

Guardar Configuración al Cerrar

Seleccione esta opción para que los cambios realizados por el comando SET de la Utilidad de Control de Listener sean permanentes.

Información de Contacto SNMP

Introduzca el texto que desea escribir en el archivo snmp.ora para utilizarlo con Oracle Enterprise Manager. Esta información se enviará a todas las conexiones SNMP y puede ser cualquier cadena. No influye en el proceso de creación de una conexión SNMP. Generalmente, esta información consiste en la información de contacto del usuario o del administrador del cliente.

ADMIN_RESTRICTIONS_listener_name

Restringe la administración en tiempo de ejecución del listener.

El valor por defecto es off. Si define ADMIN_RESTRICTIONS_listener_name en on, desactivará la función de modificación de parámetros de tiempo de ejecución en listener.ora. Es decir, el listener rechazará los comandos SET que alteren sus parámetros. Para cambiar los parámetros de listener.ora, incluido el propio ADMIN_RESTRICTIONS_listener_name, modifique manualmente el archivo listener.ora y vuelva a cargar sus parámetros (con el comando RELOAD) para que se apliquen los nuevos cambios sin necesidad de tener que parar y volver a iniciar el listener.

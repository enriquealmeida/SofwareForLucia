Nomenclatura Local

Método de nomenclatura que resuelve el nombre de servicio de red, almacenado en el archivo tnsnames.ora del cliente en un descriptor de conexión. La nomenclatura local es más aconsejable para redes de distribución simple con un número reducido de servicios que cambian en muy raras ocasiones.

File listener.ora

Un file di configurazione per il listener che specifica quanto segue:

  • il nome del listener;

  • gli indirizzi di protocollo sui quali il listener accetta le richieste di connessione;

  • i servizi (di database o non di database) per cui sta effettuando le operazioni di ascolto.

Il file listener.ora deve risiedere in una delle posizioni riportate di seguito.

  1. La directory specificata dalla variabile di ambiente TNS_ADMIN. Se la variabile di ambiente TNS_ADMIN non è definita come variabile in Windows NT, è possibile che si trovi nel registro.

  2. La directory di configurazione globale del nodo. Per Sun Solaris la directory è /var/opt/oracle. Windows NT non dispone di una directory centrale.

  3. La directory $ORACLE_HOME/network/admin nei sistemi operativi UNIX o la directory ORACLE_HOME\network\admin nei sistemi operativi Windows.

Nota: i database Oracle9i e Oracle8i non richiedono l'identificazione del servizio di database a causa della registrazione del servizio. Tuttavia, la configurazione statica del servizio è necessaria se si intende controllare il database mediante Oracle Enterprise Manager.

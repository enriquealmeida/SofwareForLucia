Schlüsselfunktionen von Oracle Net Manager

Oracle Net Manager ist ein Tool, mit dem Sie die Oracle Networking-Umgebung konfigurieren und verwalten können. Folgende Oracle Net-Funktionen und -Komponenten stehen zur Konfiguration und Verwaltung mit Oracle Net Manager zur Verfügung:

Servicebenennung

Sie können Netzwerkbeschreibungen für Datenbankdienste in einer Datei tnsnames.ora, einem Directory-Server oder einem Oracle Names-Server erstellen oder ändern. Netzwerkbeschreibungen, die als Verbindungsdeskriptoren bezeichnet werden, werden Connect-Bezeichnern zugeordnet, die Clients in ihren Verbindungszeichenfolgen während einer Datenbankverbindung verwenden.

Listener

Sie können einen Listener erstellen oder ändern, einen Prozess auf dem Server, der Verbindungsanforderungen von Clients für einen Datenbankdienst empfängt und auf diese reagiert.

Profil

Sie können ein Profil erstellen oder ändern, d.h. eine Sammlung von Parametern, mit denen bestimmt wird, wie sich der Client beim Oracle-Netzwerk anmeldet. Sie können Clientparameter für Benennungsmethoden, Logging, Tracing, externe Benennung und Oracle Advanced Security konfigurieren.

Instradamento delle richieste di connessione

È possibile configurare i client in modo che le richieste di connessione vengano sempre dirette a un processo del server dedicato o a Oracle Connection Manager.

Per instradare le richieste di connessione, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Profilo.

  2. Dalla lista presente nel riquadro di destra, selezionare Generale.

  3. Selezionare la scheda Instradamento.

  4. Selezionare il metodo di instradamento che si desidera adottare per le richieste di connessione.

  5. Scegliere File > Salva configurazione di rete.

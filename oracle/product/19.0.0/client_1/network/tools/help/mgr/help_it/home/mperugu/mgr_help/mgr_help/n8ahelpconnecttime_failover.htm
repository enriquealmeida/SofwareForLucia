Failover della fase di connessione

La richiesta di connessione di un client viene inoltrata a un altro listener se il primo non risponde. Il failover della fase di connessione è abilitato dalla registrazione del servizio, in quanto il listener sa se un'istanza è in esecuzione prima di tentare una connessione.

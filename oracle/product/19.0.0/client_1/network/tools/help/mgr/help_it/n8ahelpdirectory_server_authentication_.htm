Finestra di dialogo Autenticazione server delle directory

La finestra di dialogo Autenticazione server delle directory consente di impostare le credenziali di autenticazione per il contesto Oracle nel server delle directory.

Oracle Net Manager deve ricevere l'autenticazione dal server delle directory per poter eseguire qualsiasi operazione su tale server, come la visualizzazione, l'aggiunta o la modifica delle voci Oracle Net nella directory.

Utente

Immettere un utente autorizzato con accesso alla directory.

Per aggiungere o modificare le voci, immettere un utente amministratore delle directory che disponga dei privilegi di aggiunta e modifica come nome distinto (DN). Ad esempio, c=US,o=EXAMPLE,ou=admin,cn=scott è il DN per un amministratore di nome Scott.

Password

Immettere la password per l'utente.

Nota: se la directory è stata impostata con privilegi "anonimi" e si desidera visualizzare semplicemente gli oggetti, lasciare vuoti i campi Utente e Password.

Per impostazione predefinita, Oracle Net Manager tenta di effettuare l'autenticazione con la directory utilizzando uno dei seguenti metodi:

  • Autenticazione nativa: vengono utilizzati un nome utente e una password del sistema operativo Windows NT.

  • Autenticazione dettagliata: viene utilizzato un certificato SSL (Secure Sockets Layer).

進階服務選項對話方塊

您可以使用「進階服務選項」對話方塊，來設定進階的 CONNECT_DATA 參數。

其他服務設定值

元素 描述
執行處理名稱 輸入資料庫執行處理的名稱來存取。執行處理的名稱可自初始化參數檔的 INSTANCE_NAME 參數值取得。
階段作業資料單位 (SDU) 輸入階段作業資料單位大小，來變更與經由網路傳輸之封包相關的效能特性。請將 SDU 大小設為正常傳輸框架大小的倍數。乙太網路框架的正常大小為 1500，但可使用的部分為 1460。不使用加密時，會保留一個位元組。在不使用加密時，使用的演算法是 1460 的倍數加 1，用於乙太網路時，則為 1460 的倍數。
使用異質服務 選擇 Oracle 伺服器以透過異質服務存取非 Oracle 系統。

Oracle Rdb 設定值

元素 描述
Oracle Rdb 設定值 指定 Oracle Rdb 資料庫的檔案名稱。
服務類型 輸入 Oracle Rdb 資料庫要使用的服務類型。
全域資料庫名稱 輸入 Oracle Rdb 資料庫的 ID。

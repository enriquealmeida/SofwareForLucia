디렉토리 서버 인증 대화상자

디렉토리 서버 인증 대화상자를 사용하여 디렉토리 서버에서 선택한 Oracle Context에 대한 인증서를 설정할 수 있습니다.

디렉토리에서 Oracle Net 항목 보기, 추가 또는 수정 등의 디렉토리 서버 작업을 수행하려면 디렉토리 서버로 Oracle Net Manager를 인증해야 합니다.

사용자

디렉토리에 대한 액세스 권한이 있는 사용자를 입력합니다.

항목을 추가 또는 수정한 다음 제공된 디렉토리 관리자 사용자를 입력하려는 경우 DN(식별 이름) 형식으로 권한을 추가 또는 수정합니다. 예를 들어, c=US,o=EXAMPLE,ou=admin,cn=scott는 Scott라는 관리자에 대한 DN입니다.

비밀번호

사용자 비밀번호를 입력합니다.

주: 디렉토리가 "익명" 권한으로 설정되어 있고 객체를 단순히 보기만 하려는 경우 [사용자] 및 [비밀번호] 필드를 비워 둡니다.

기본적으로 Oracle Net Manager는 다음 방식 중 하나를 사용하여 디렉토리를 인증합니다.

  • 원시 인증: Windows NT 운영 체제 사용자 이름과 비밀번호를 사용합니다.

  • 강화된 인증: SSL(Secure Sockets Layer) 인증을 사용합니다.

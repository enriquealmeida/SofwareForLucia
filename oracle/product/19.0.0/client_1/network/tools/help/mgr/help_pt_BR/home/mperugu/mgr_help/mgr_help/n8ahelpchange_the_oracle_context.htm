Alterar o Oracle Context

Quando o Oracle Net Manager é iniciado, os objetos exibidos na pasta Diretório > Nomeação de Serviço são entradas de serviço Oracle contidas no default Oracle Context estabelecido durante a configuração do uso do diretório com o Oracle Net Configuration Assistant. Você pode criar e modificar entradas para outro Oracle Context para o qual você tenha autenticação.

Para alterar o Oracle Context atual:

  1. Escolha Comando > Diretório > Alterar Contexto Atual. A caixa de diálogo Escolha um Novo Contexto é exibida.

  2. Na lista Contexto de Nomeação de Diretório, selecione uma entrada que contenha o Oracle Context que deseja ver.

  3. Na lista Oracle Context, selecione o Oracle Context e, em seguida, escolha OK. Se as modificações não forem salvas no Oracle Context atual, uma caixa de diálogo Salvar solicitará que você salve as alterações.

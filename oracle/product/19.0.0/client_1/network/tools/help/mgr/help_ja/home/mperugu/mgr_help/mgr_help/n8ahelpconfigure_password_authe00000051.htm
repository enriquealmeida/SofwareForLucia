リスナーのパスワード認証の構成

リスナーは、セキュリティ機能を指定して構成できます。これによって、構成変更の保存やリスナーの停止など、リスナー制御ユーティリティから使用する権限を伴う操作には、パスワードが必要になります。Oracle Net Managerでは、暗号化されたパスワードを設定できます。

リスナー用のパスワードを設定する手順は、次のとおりです。

  1. ナビゲータ・ペインで、「Oracle Netの構成」→「ローカル」→「リスナー」を展開します。

  2. リスナーを選択します。

  3. 右ペインのリストから「一般パラメータ」を選択します。

  4. 「認証」タブを選択します。

  5. 「リスナー操作にパスワードが必要」を選択します。

  6. 「パスワード」フィールドにパスワードを入力し、それを「パスワードの確認」フィールドに再入力します。

  7. 「ファイル」→「ネットワーク構成の保存」を選択します。

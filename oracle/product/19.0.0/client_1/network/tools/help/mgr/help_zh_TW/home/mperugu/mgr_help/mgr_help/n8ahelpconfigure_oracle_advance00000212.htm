設定 Oracle Advanced Security 認證方法

集中、安全的認證方法可讓您在分散式環境中高度信任使用者、從屬端和伺服器的識別身分。網路認證方法也可為使用者提供單一登入的便利。

您可以使用 Oracle Net Manager 設定下列認證轉接器：

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • Windows NT 原生認證

注意：如果您已經設定 TCP/IP 與 SSL 協定，那麼依照預設，也會一併設定「安全通訊端層 (SSL)」。您在此頁籤選擇的認證方法會覆寫 SSL 的認證功能。請參閱 SSL 頁籤來設定其他 SSL 選項。如果您使用 TCP/IP 與 SSL 協定，請勿使用任何其他認證方法。

設定認證方法：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 區域 > 設定檔。

  2. 從右方窗格中的清單選取 Oracle Advanced Security。

  3. 選取認證頁籤。

  4. 從可用的方法清單中，選取您要使用的認證方法，然後選擇向右箭號按鈕 (>)。

  5. 根據您要使用的方法來排序認證方法。在選取的方法清單中選取認證方法，然後使用升級或降級按鈕，在清單中上下移動選擇的項目。

  6. 選擇其他參數頁籤來設定下列認證方法的其他參數：

    • Kerberos

    • CyberSAFE

    • RADIUS

  7. 選擇檔案 > 儲存網路組態。

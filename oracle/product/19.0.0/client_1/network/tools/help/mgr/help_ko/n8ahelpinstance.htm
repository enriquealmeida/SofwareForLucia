인스턴스

SGA(시스템 글로벌 영역)와 Oracle 백그라운드 프로세스의 결합입니다. 컴퓨터 유형에 관계없이 데이터베이스 서버에서 데이터베이스를 시작할 때 Oracle은 SGA라는 메모리 영역을 할당하며 하나 이상의 Oracle 프로세스를 시작합니다. 인스턴스의 메모리와 프로세스는 관련된 데이터베이스의 데이터를 효율적으로 관리하며 데이터베이스 사용자에게 서비스를 제공합니다. 인스턴스에 연결하여 병렬 서버 데이터베이스 내의 정보에 액세스할 수 있습니다.

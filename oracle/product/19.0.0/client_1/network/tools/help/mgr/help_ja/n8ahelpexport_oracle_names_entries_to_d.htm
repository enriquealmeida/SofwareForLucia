ディレクトリ・サーバーへのOracle Namesエントリのエクスポート

このリリースでは、Oracle Namesがサポートされなくなりました。Oracle Namesリリース9.2を使用する際は、Oracle Names Serverに格納されているデータベース・オブジェクトと別名オブジェクトをディレクトリ・サーバーにインポートするか、ディレクトリ・サーバーへのロードに使用できるLDAP Data Interchange Format (LDIF)ファイルに間接的にエクスポートする必要があります。データベース・オブジェクトのエクスポート時には、アドレス・レコード(A.SMD)のみがエクスポートされます。アドレス・レコードには、データベース・オブジェクトの接続記述子情報が含まれています。データベース・オブジェクトはネット・サービス名としてエクスポートされます。別名はネット・サービス別名としてエクスポートされます。データは指定したドメインからエクスポートされます。

詳細は、Oracle10i Net管理者ガイドの「ネーミング・メソッドの構成」の章を参照してください。

Oracle Names Serverからのデータのエクスポート方法については、リリース9.2のOracle9i Net Services管理者ガイドを参照してください。

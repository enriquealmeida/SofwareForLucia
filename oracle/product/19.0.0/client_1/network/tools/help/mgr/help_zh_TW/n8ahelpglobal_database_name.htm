全域資料庫名稱

可唯一識別資料庫的完整名稱，用來將該資料庫與其他資料庫做區分。全域資料庫名稱的格式為 "database_name.database_domain"，例如 sales.us.example.com。

資料庫名稱當中的 sales 是用來稱呼資料庫的簡單名稱。資料庫網域當中的us.example.com則指定資料庫所在的資料庫網域，使得全域資料庫名稱成為獨有的名稱。Oracle 建議您儘可能讓資料庫網域對應網路網域。

全域資料庫名稱是資料庫的預設服務名稱，由初始化參數檔中的 SERVICE_NAMES 參數所指定。

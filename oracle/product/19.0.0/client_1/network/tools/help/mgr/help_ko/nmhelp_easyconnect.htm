쉬운 접속 이름 지정

아무런 구성 없이 클라이언트가 데이터베이스 서버에 접속할 수 있게 하는 이름 지정 방법 입니다. 클라이언트는 호스트 이름, 선택적 포트 및 서비스 이름으로 구성되는 간단한 TCP/IP 주소를 사용합니다.

CONNECT username/password@host[:port][/service_name]

쉬운 접속 이름 지정 방법 사용에 대한 자세한 내용은 Oracle10i Net Services Administrator's Guide의 Configuring Naming Methods 장을 참조하십시오.

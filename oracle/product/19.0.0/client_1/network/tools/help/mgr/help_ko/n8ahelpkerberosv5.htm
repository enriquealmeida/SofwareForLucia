KERBEROS(V5)

KERBEROS(V5) 인증 방법을 사용하려면 다음 매개변수를 설정해야 합니다.

요소 설명
서비스 인증 서비스 이름인 KERBEROS를 입력합니다.
인증서 캐시 파일 Kerberos 인증서 캐시 파일에 대한 전체 경로입니다. 기본값은 운영 체제에 따라 다릅니다.
구성 파일 Kerberos 구성 파일에 대한 전체 경로입니다. 이 파일은 기본 KDC(Key Distribution Center)에 대한 영역을 포함하며 호스트를 KDC 호스트에 매핑합니다.
Realm 번역 파일 Kerberos 영역 변환 파일에 대한 전체 경로입니다. 이 파일은 호스트 이름 또는 도메인 이름을 영역에 매핑합니다. 기본값은 운영 체제에 따라 다릅니다.
키 테이블 Kerberos 주체/보안 키 매핑 파일에 대한 전체 경로입니다. 이 경로는 Oracle 서버가 해당 키를 추출하고 클라이언트에서 수신한 인증 정보를 해독하는 데 사용됩니다. 기본값은 운영 체제에 따라 다릅니다.
허용 시간 차 Kerberos 인증서가 만료되는 제한 시간입니다. 기본값 = 300 초

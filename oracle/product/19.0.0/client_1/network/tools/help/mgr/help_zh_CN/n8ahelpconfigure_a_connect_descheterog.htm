为异构服务配置连接描述符

异构服务使您能够从 Oracle 服务器访问非 Oracle 系统。要建立与非 Oracle 系统的连接，Oracle 服务器需要通过网关上的监听程序启动代理进程。为了使 Oracle 数据库连接到非 Oracle 系统，请使用网络服务名配置服务器。

要为异构服务配置服务器，请执行以下操作：

  1. 确保已为异构服务配置了监听程序，如为监听程序配置异构服务中所述。

  2. 在 tnsnames.ora 文件中创建可用于 Oracle 数据库服务器与非 Oracle 系统之间的连接的网络服务名。请确保连接描述符信息与为此监听程序配置的信息相匹配。

  3. 在本地 > 服务命名文件夹中选择网络服务名。

  4. 在右侧窗格的服务标识框中，选择高级。将显示“高级服务选项”对话框。

  5. 选择用于不同种类服务，然后选择确定。

  6. 在右侧窗格中，选择应用。

  7. 选择文件 > 保存网络配置。

一般參數：一般

您可以使用一般頁籤來設定監聽器的一般參數。

監聽器名稱

顯示監聽器的名稱。

啟動等待時間

指定回應監聽器控制公用程式 START 命令前，監聽器等候的時間 (秒)。

注意：以後的版本將不再使用和支援此設定值。如果您需要此參數來執行監聽器，請通知「Oracle 客戶服務部」。

關閉時儲存組態

選擇此選項可建立監聽器控制公用程式 SET 命令所做的永久性變更。

SNMP 聯繫資訊

輸入任何要寫入snmp.ora檔案以用於 Oracle Enterprise Manager 的文字。此資訊會透過任何 SNMP 連線傳送，而且可以是任何字串。它不會影響建立 SNMP 連線的處理作業。此資訊一般包含用戶端使用者或管理員的聯繫資訊。

ADMIN_RESTRICTIONS_listener_name

限制程式實際執行時的監聽器管理。

預設值為 off。設定 ADMIN_RESTRICTIONS_listener_name=on 會停用允許在程式實際執行時修改 listener.ora 中參數的功能。也就是說，監聽器會拒絕可更改其參數的 SET 命令。若要變更 listener.ora 中的任何參數 (包括 ADMIN_RESTRICTIONS_listener_name 本身)，請手動修改 listener.ora 檔案，並重新載入其參數 (使用 RELOAD 命令) 以讓新變更生效，不需明確停止監聽器再重新予以啟動。

프로파일 개요

프로파일은 클라이언트 또는 서버의 Oracle Net 기능 활성화 및 구성에 대한 환경을 지정합니다. 프로파일을 사용하여 다음 작업을 수행할 수 있습니다.

  • 클라이언트 도메인을 지정하여 규정되지 않은 이름에 추가

  • 이름 지정 방법 우선순위 지정

  • 로깅 및 추적 기능 사용

  • 특정 프로세스를 통한 접속 경로 지정

  • 외부 이름 지정에 대한 매개변수 구성

  • Oracle Advanced Security 구성

프로파일에 액세스하려면 네비게이터 창에서 [Oracle Net 구성 > 로컬 > 프로파일]을 확장합니다.

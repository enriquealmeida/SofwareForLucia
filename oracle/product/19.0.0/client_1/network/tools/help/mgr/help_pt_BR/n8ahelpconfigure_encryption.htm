Configurando Criptografia

Criptografia é o processo de disfarçar uma mensagem tornando-a ilegível para qualquer pessoa que seja não o destinatário pretendido.

Para configurar parâmetros de criptografia:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

  2. Na lista do painel direito, selecione Oracle Advanced Security.

  3. Selecione a guia Criptografia.

  4. Selecione Cliente ou Servidor na lista Criptografia.

  5. Selecione um valor na lista Tipo de Criptografia para especificar o comportamento do cliente ou servidor ao negociar criptografia e integridade. Os quatro valores possíveis são os seguintes:

    Elemento Descrição
    aceito O serviço estará ativo se o outro lado da conexão especificar "necessário" ou "solicitado" e houver um algoritmo compatível disponível do outro lado; caso contrário, ele estará inativo.
    rejeitado O serviço não deve estar ativo e a conexão falhará se o outro lado especificar "necessário".
    solicitado O serviço estará ativo se o outro lado da conexão especificar "aceito", "necessário" ou "solicitado" e houver um algoritmo compatível disponível do outro lado; caso contrário, ele estará inativo.
    necessário O serviço deverá estar ativo e a comparação falhará se o outro lado especificar "rejeitado" ou se não houver um algoritmo compatível do outro lado.

  6. Informe a semente de criptografia no campo Semente de Criptografia: uma string aleatória de até 256 caracteres. Essa semente de criptografia é usada com o algoritmo de criptografia selecionado na próxima etapa.

  7. Na lista Métodos Disponíveis, selecione um algoritmo de criptografia e escolha o botão de seta para a direita (<) para movê-lo para a lista Métodos Selecionados. A ordem dos métodos nessa lista determina a preferência de negociação. Os algoritmos a seguir são suportados pelo Oracle Advanced Security:

    Elemento Descrição
    RC4_256 Tamanho de chave de 256 bits
    3DES168 DES triplo com uma opção de chave tripla (168 bits)
    RC4_128 Tamanho de chave de 128 bits
    3DES112 DES triplo com uma opção de chave dupla (112 bits)
    RC4_56 Tamanho de chave de 56 bits
    DES Tamanho de chave de 56 bits padrão
    RC4_40 Tamanho de chave de 40 bits
    DES40 Tamanho de chave de 40 bits

  8. Escolha Arquivo > Salvar Configuração de Rede.

Observação: Todos os algoritmos de criptografia e tamanhos de chaves do Oracle Advanced Security só estão disponíveis para uso nos EUA.

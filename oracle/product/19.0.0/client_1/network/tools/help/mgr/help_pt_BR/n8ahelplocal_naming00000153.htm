Nomenclatura Local

Um método de nomeação que resolve um nome de serviço de rede, armazenado no arquivo tnsnames.ora de um cliente, para um descritor de conexão. A nomeação local é mais apropriada para redes simples distribuídas com um pequeno número de serviços que se alteram com pouca frequência.

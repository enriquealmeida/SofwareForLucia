Configurando Integridade de Dados

A verificação da integridade de dados é o processo pelo qual uma quantidade é calculada com base no conteúdo de uma mensagem. O remetente da mensagem calcula o checksum. O valor do checksum é incluído como parte do pacote da mensagem. O destinatário da mensagem recalcula o checksum da mensagem ao recebê-la. Os dois valores de checksum deverão coincidir se a mensagem for considerada inalterada durante o trajeto.

Para configurar a integridade de dados:

  1. No painel do navegador, expanda Configuração do Oracle Net > Local > Perfil.

  2. Na lista do painel direito, selecione Oracle Advanced Security.

  3. Selecione a guia Integridade.

  4. Ative a integridade de dados do cliente ou servidor escolhendo a caixa de seleção Cliente ou Servidor, respectivamente.

  5. Especifique como o cliente e o servidor negociarão a integridade de dados. Você deverá selecionar Níveis de Checksum para o cliente e o servidor. As descrições dos níveis são as seguintes:

    Elemento Descrição
    aceito O serviço estará ativo se o outro lado da conexão especificar "necessário" ou "solicitado" e houver um algoritmo compatível disponível do outro lado; caso contrário, ele estará inativo.
    rejeitado O serviço não deve estar ativo e a conexão falhará se o outro lado especificar "necessário".
    solicitado O serviço estará ativo se o outro lado da conexão especificar "aceito", "necessário" ou "solicitado" e houver um algoritmo compatível disponível do outro lado; caso contrário, ele estará inativo.
    necessário O serviço deverá estar ativo e a conexão falhará se o outro lado especificar "rejeitado" ou se não houver um algoritmo compatível do outro lado.

  6. Clique em Arquivo > Salvar Configuração de Rede.

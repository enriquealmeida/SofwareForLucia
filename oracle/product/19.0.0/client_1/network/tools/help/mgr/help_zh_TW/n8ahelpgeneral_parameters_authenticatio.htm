一般參數：認證

您可以使用認證頁籤來設定監聽器的安全性。如此，從監聽器控制公用程式發出的儲存組態變更或停止監聽器等授權作業，就需要密碼。您可以使用 Oracle Net Manager 來設定加密密碼。

執行監聽器作業不需要密碼

選擇此選項可停用管理員工作的認證。

執行監聽器作業需要密碼

選擇此選項可啟用特定管理員工作的認證。然後您可以輸入必要的密碼。預設的密碼是 "oracle"。

注意：Oracle Net 允許每一個監聽器設定一個密碼。

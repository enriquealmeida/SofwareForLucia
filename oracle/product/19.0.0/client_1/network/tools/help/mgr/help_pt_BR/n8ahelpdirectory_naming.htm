Nomenclatura de Diretórios

Um método de nomeação que resolve um serviço de banco de dados ou nome de serviço de rede para um descritor de conexão, armazenado em um servidor de diretório.

Um diretório permite a administração central de serviços de banco de dados e nomes de serviços de rede, reduzindo o esforço associado à adição ou realocação de serviços. Embora os nomes de serviços de rede possam ser configurados como apelidos de serviço, o diretório pode fazer referência direta a um serviço de banco de dados sem usar um nome do serviço de rede. O serviço de banco de dados é automaticamente adicionado ao diretório como uma entrada durante a instalação.

일반 사항: 로깅

로깅 탭을 사용하여 해당 컴퓨터의 로깅 매개변수를 설정할 수 있습니다. 이 컴퓨터가 클라이언트인 경우에는 클라이언트 정보 필드를 사용하고, 이 컴퓨터가 서버인 경우에는 서버 정보 필드를 사용하십시오.

로그 디렉토리

로그 파일이 작성될 디렉토리 경로를 입력합니다. 기본값은 UNIX의 경우 $ORACLE_HOME/network/log 이고, Windows 플랫폼의 경우 ORACLE_HOME \network\log 입니다.

로그 파일

클라이언트의 로그 파일 이름을 입력합니다. 기본 이름은 sqlnet.log입니다.

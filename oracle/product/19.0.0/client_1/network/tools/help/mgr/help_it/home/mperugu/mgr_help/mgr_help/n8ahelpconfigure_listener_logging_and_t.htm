Configurazione delle funzioni di log e trace del listener

È possibile configurare le funzioni di log e trace di un listener.

Le informazioni di log sono in genere memorizzate nel file listener.log in $ORACLE_HOME/network/log sulle piattaforme UNIX e in ORACLE_HOME\network\log sulle piattaforme Windows.

Per impostazione predefinita, le informazioni di trace sono memorizzate nel file listener.trc in $ORACLE_HOME/network/trace sulle piattaforme UNIX e in ORACLE_HOME\network\trace sulle piattaforme Windows.

Per abilitare la funzione di log o trace, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

  2. Selezionare un listener.

  3. Dalla lista presente nel riquadro di destra, selezionare Parametri generali.

  4. Selezionare la scheda Log e trace.

  5. Definire le impostazioni di log e trace nel modo desiderato.

  6. Scegliere File > Salva configurazione di rete.

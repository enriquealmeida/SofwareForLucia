NIS (Network Information Service)

O protocolo cliente-servidor Yellow Pages (yp) da Sun Microsystems para distribuição de dados de configuração do sistema, como nomes de usuário e host, entre computadores de uma rede.

Para obter informações sobre como configurar o NDS como um método de nomeação externa, consulte Configure External Naming Methods and Prioritize Naming Method.

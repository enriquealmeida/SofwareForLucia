Activez le service de sécurité si l'autre machine le demande.

Ma machine ne souhaite pas utiliser le service de sécurité, mais celui-ci sera autorisé si l'autre machine le demande à l'aide du paramètre requis ou demandé. Si l'autre machine est paramétrée sur requis ou demandé, et qu'une correspondance d'algorithme est trouvée, la connexion se poursuivra normalement et avec le service de sécurité activé. Si l'autre machine est paramétrée sur requis et qu'aucune correspondance d'algorithme n'existe, la connexion prendra fin avec l'affichage du message d'erreur ORA- 2650.

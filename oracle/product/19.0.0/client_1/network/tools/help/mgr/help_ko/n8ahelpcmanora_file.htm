cman.ora 파일

Oracle Connection Manager 매개변수뿐만 아니라 수신된 요청과 로컬 및 원격 명령에 대한 프로토콜 주소를 지정하는 구성 파일입니다. Oracle Net Manager는 이 파일의 구성을 지원하지 않으므로 수동으로 구성해야 합니다.

cman.ora 파일은 다음 위치 중 하나에 있어야 합니다.

  1. TNS_ADMIN 환경 변수에 의해 지정된 디렉토리입니다. TNS_ADMIN 환경 변수가 Windows NT에서 변수로 정의되지 않은 경우 레지스트리에 있을 수 있습니다.

  2. 노드의 전역 구성 디렉토리입니다. Sun Solaris의 경우 이 디렉토리는 /var/opt/oracle입니다. Windows NT에는 중앙 디렉토리가 없습니다.

  3. UNIX의 $ORACLE_HOME/network/admin 디렉토리 또는 Windows 운영 체제의 ORACLE_HOME\network\admin 디렉토리입니다.

자세한 내용은 Oracle10i Net Administrator's Guide의 Configuring and Administering Oracle Connection Manager 장을 참조하십시오.

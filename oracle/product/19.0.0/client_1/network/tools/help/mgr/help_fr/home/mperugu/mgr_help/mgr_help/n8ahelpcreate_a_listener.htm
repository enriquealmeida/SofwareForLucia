Création d'un processus d'écoute

Un processus d'écoute doit exister sur le serveur pour que la base de données puisse recevoir des connexions.

Remarque : Un processus d'écoute par défaut appelé LISTENER est créé pendant l'installation.

Pour créer un processus d'écoute :

  1. Dans le panneau du navigateur, développez Oracle Net Configuration > Local > Processus d'écoute.

  2. Sélectionnez plus (+) dans la barre d'outils ou choisissez Modifier > Créer. La boîte de dialogue Choisir un nom de processus d'écoute apparaît.

  3. Acceptez le nom indiqué dans le champ Nom du processus d'écoute ou indiquez un autre nom. Par défaut, le nom du premier processus d'écoute est LISTENER. Par défaut, les noms des processus d'écoute suivants sont LISTENERn. Quel que soit le noeud sélectionné, le nom du processus d'écoute doit être unique.

  4. Cliquez sur OK.

  5. Exécutez les étapes 4 à 6 de la section Configuration des adresses d'écoute.

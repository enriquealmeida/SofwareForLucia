Configurazione degli indirizzi del listener

Il listener deve essere configurato con uno o più indirizzi di protocollo. Il listener riceve le richieste di connessione dai client configurati con gli stessi indirizzi.

Per configurare un indirizzo di protocollo per il listener, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

  2. Selezionare un listener.

  3. Dalla lista presente nel riquadro di destra, selezionare Posizioni di ascolto.

  4. Scegliere Aggiungi indirizzo. Viene visualizzata una nuova scheda Indirizzo.

  5. Selezionare un protocollo, quindi immettere le informazioni appropriate relative ai parametri di protocollo per il protocollo selezionato.

    Quando si configura il listener per l'ascolto su protocollo TCP/IP per i client Oracle Net, è opportuno scegliere la porta predefinita o la porta 1521. In caso contrario, è necessario configurare il parametro LOCAL_LISTENER nel file dei parametri di inizializzazione e risolvere il nome del listener mediante un metodo di denominazione.

  6. Se il computer presenta più indirizzi IP e si desidera che il listener esegua l'ascolto su tutti gli indirizzi IP disponibili, configurare TCP/IP o TCP/IP con SSL e immettere il nome host del computer nel campo Host.

  7. In alternativa, fare clic su Mostra avanzate per specificare le impostazioni relative alla configurazione dello spazio del buffer di I/O per le operazioni di invio e ricezione della sessione.

  8. Scegliere File > Salva configurazione di rete.

Per ulteriori informazioni sulla configurazione di una porta non predefinita, vedere il capitolo Configuring and Administering the Listener del manuale Oracle10i Net Administrator's Guide.

Per ulteriori informazioni sulla configurazione del spazio del buffer di I/O, vedere il capitolo Optimizing Performance del manuale Oracle10i Net Administrator's Guide.

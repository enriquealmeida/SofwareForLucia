Oracle Advanced Security 인증 방법 구성

중앙 집중화되고 안전한 인증 방법을 사용하면 분산 환경에서 사용자, 클라이언트, 서버의 신원에 대한 신뢰도가 높아집니다. 또한 네트워크 인증 방법은 사용자에게 Single Sign-On의 이점을 제공할 수도 있습니다.

Oracle Net Manager를 사용하여 다음 인증 어댑터를 구성합니다.

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • Windows NT 원시 인증

주: SSL 프로토콜을 사용하는 TCP/IP를 구성한 경우 기본적으로 SSL(Secure Sockets Layer)이 구성됩니다. 현재 탭에서 선택한 인증 방법은 SSL 인증 기능을 무효화합니다. 추가 SSL 옵션을 구성하려면 SSL 탭을 참조하십시오. SSL 프로토콜을 사용하는 TCP/IP를 사용하는 경우 다른 인증 방법을 사용하지 마십시오.

인증 방법을 구성하려면 다음과 같이 하십시오.

  1. 네비게이터 창에서 Oracle Net 구성 > 로컬 > 프로파일을 확장합니다.

  2. 오른쪽 창에 있는 목록에서 Oracle Advanced Security를 선택합니다.

  3. 인증 탭을 선택합니다.

  4. 사용 가능한 방법 목록에서 사용할 인증 방법을 선택한 후 오른쪽 화살표(>) 단추를 선택합니다.

  5. 사용하고자 하는 방법에 따라 인증 방법을 정렬합니다. 선택된 방법 목록에서 인증 방법을 선택하고 올림 또는 내림 단추를 사용하여 목록에서 선택한 항목을 위 또는 아래로 이동합니다.

  6. 다음 인증 방법에 대한 추가 매개변수를 구성하려면 기타 매개변수 탭을 선택합니다.

    • Kerberos

    • CyberSafe

    • RADIUS

  7. 파일 > 네트워크 구성 저장을 선택합니다.

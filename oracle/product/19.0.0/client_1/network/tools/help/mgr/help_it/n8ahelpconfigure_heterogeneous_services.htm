Configurazione di Heterogeneous Services per il listener

Heterogeneous Services consente di accedere a sistemi non Oracle da un server Oracle. Per stabilire una connessione a un sistema non Oracle, il server Oracle avvia un processo agente mediante il listener sul gateway. La configurazione di Heterogeneous Services per il listener implica la creazione di informazioni sul servizio per l'agente eterogeneo.

Per configurare un componente Heterogeneous Services per il listener, effettuare le operazioni riportate di seguito.

  1. Nel riquadro del Navigator, espandere Configurazione di Oracle Net > Locale > Listener.

  2. Selezionare un listener.

  3. Dalla lista presente nel riquadro di destra, selezionare Altri servizi.

  4. Scegliere Aggiungi servizio. Viene visualizzata una nuova scheda Servizio.

  5. Immettere l'eseguibile dell'agente nel campo Nome programma, il SID (ID di sistema) Oracle dell'agente nel campo SID e la Oracle home dell'eseguibile nel campo Directory Oracle Home.

  6. Assicurarsi che il server sia stato configurato con un descrittore di connessione per l'accesso al servizio eterogeneo, secondo quanto descritto in Configurazione di un descrittore di connessione per le procedure esterne.

  7. Scegliere File > Salva configurazione di rete.

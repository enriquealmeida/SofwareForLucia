一般パラメータ: ロギングおよびトレース

「ロギングおよびトレース」タブでは、リスナーのロギングおよびトレースを使用可能にできます。Oracle Net Managerでは、リスナーのログ・ファイルの書込み先の名前と場所を指定できます。

ロギング使用禁止

リスナーのロギングを使用禁止にします。

ロギング可能

リスナーのロギングを使用可能にします。使用可能にすると、ログ情報を書き込む場所を指定できます。

ログ・ファイル

リスナーからのログ情報を書き込むディレクトリ・パス名およびファイル名を入力します。このファイルのファイル名拡張子は、常に.logになることに注意してください。デフォルトの名前はlistener.logです。デフォルトのディレクトリは、UNIXの場合は$ORACLE_HOME/network/log、Windowsの場合はORACLE_HOME \network\logになります。

トレース使用禁止

リスナーのトレースを使用禁止にします。

トレース可能

リスナーのトレースを使用可能にします。一度使用可能にすると、必要なトレースのレベルを指定したり、デフォルトのトレース・ファイルを変更してトレース情報の書込み先を指定できます。

トレース・レベル

リストから、トレース・レベルを選択します。

Off: トレースが使用禁止になります。デフォルトは「オフ」です。

User: トレースがユーザーに適したレベルに設定されます。ユーザーが発生させたエラー状態を識別します。

Administrator: トレースがデータベース管理者に適したレベルに設定されます。インストール固有の問題を識別します。

Support: トレースがカスタマ・サポートに適したレベルに設定されます。

トレース・ファイルがかなり大きくなる場合があります。ネットワーク上の問題のトラブルシューティングを行わないときには、トレースをオフにしておくことをお薦めします。

トレース・ファイル

リスナーからのトレース情報を書き込むディレクトリ・パス名およびファイル名を入力します。このファイルのファイル名拡張子は、常に.trcになることに注意してください。デフォルトの名前はlistener.trcです。デフォルト・ディレクトリは、UNIXの場合は$ORACLE_HOME/network/trace、Windowsの場合はORACLE_HOME\network\traceになります。

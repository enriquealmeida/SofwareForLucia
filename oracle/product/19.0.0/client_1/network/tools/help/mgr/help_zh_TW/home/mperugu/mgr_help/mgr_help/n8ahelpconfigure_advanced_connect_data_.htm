設定進階連線資料選項

除了資料庫的服務名稱或 Oracle 系統 ID (SID)，您還可以使用下列選項來設定連線描述區的 CONNECT_DATA 區段：

  • 連線至特定資料庫執行處理的執行處理名稱

  • 經由網路啟動資料庫的資料庫 Oracle 本位目錄

  • 指定緩衝區大小的階段作業資料單位大小

  • 連線至專用伺服器處理作業的專用伺服器

  • 存取非 Oracle 系統的異質服務

  • 存取 Oracle Rdb 資料庫的 Oracle Rdb 設定值

設定網路服務名稱的進階連線選項：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 目錄 > 區域或服務命名。

  2. 選取網路服務名稱或資料庫服務。右窗格就會顯示目前的目的地服務和位址清單。

  3. 在服務識別方塊中，選擇進階。進階服務選項對話方塊就會顯示。

  4. 輸入欄位或選取適當的選項，然後選擇確定。

  5. 在右窗格中，選擇套用。

  6. 如果要將這些變更儲存至區域資料夾，請選擇檔案 > 儲存網路組態。

建立其他協定位址

一個資料庫服務可以被一個以上的網路路由或協定位址存取。

將網路協定位址新增至現有的網路服務名稱或資料庫服務：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 目錄或區域 > 服務命名。

  2. 選取網路服務名稱或資料庫服務。右窗格就會顯示目前的目的地服務和位址清單。

  3. 在位址組態方塊中，選擇加號 (+)。即會顯示一個新的位址頁籤。

  4. 選取一個協定，然後輸入選取之協定的適當協定參數。

  5. 使用向左和向右箭號按鈕，根據位址在位址清單中應有的位置來排列位址的順序。除非設定了多重位址選項，否則只會連線第一個位址。若要設定不同的順序，請參閱設定多重位址選項。

  6. 在右窗格中，選擇套用。

  7. 如果要對「區域」資料夾進行這些變更，請選擇檔案 > 儲存網路組態。

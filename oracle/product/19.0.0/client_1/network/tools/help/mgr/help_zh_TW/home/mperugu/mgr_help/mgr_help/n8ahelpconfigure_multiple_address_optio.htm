設定多個位址選項

資料庫服務若可由多個監聽器協定位址存取，請務必指定使用協定位址的順序。例如，您可以設定隨機選擇或循序嘗試協定位址。

為網路服務名稱或資料庫服務設定多個協定位址之後，可以設定下列選項：

  • 連線階段容錯移轉 - 選擇第一個協定位址，然後在第一個協定位址失敗時，容錯移轉至下一個協定位址。

  • 從屬端負載平衡 - 隨機選擇協定位址。

  • 來源路由 - 循序選擇所有協定位址。

設定多個位址選項：

  1. 在瀏覽窗格中，展開 Oracle Net 組態 > 目錄 > 區域或服務命名。

  2. 選取網路服務名稱或資料庫服務。右窗格就會顯示目前的目的地服務和位址清單。

  3. 在位址組態方塊中，選擇進階。位址清單選項對話方塊就會顯示。

  4. 如果此組態是用於 Oracle9i 或 Oracle8i 從屬端，請取消選取使用與 Net8 8.0 從屬端相容的選項。

  5. 選取協定位址清單的適當選項，然後選擇確定。

  6. 在右窗格中，選擇套用。

  7. 如果要對「區域」資料夾進行這些變更，請選擇檔案 > 儲存網路組態。

Oracle Internet Directory

Oracle 데이터베이스에서 응용 프로그램으로 구현된 디렉토리 서비스입니다. 이 서비스를 사용하여 분산된 사용자와 네트워크 자원에 대한 정보를 검색할 수 있습니다. Oracle Internet Directory 서비스는 개방형 인터넷 표준 디렉토리 액세스 프로토콜인 LDAP(Lightweight Directory Access Protocol) 버전 3과 Oracle 데이터베이스의 뛰어난 성능, 확장성, 견고성 및 가용성을 통합합니다.

자세한 내용은 Oracle Internet Directory Administrator's Guide를 참조하십시오.

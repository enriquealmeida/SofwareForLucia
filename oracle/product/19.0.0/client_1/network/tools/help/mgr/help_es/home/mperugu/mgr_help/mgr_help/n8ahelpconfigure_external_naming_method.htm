Configuración de Métodos de Nomenclatura Externa

Puede configurar los parámetros de cliente necesarios para los métodos de nomenclatura externa, incluyendo Servicio de Información de Red (NIS) en el perfil.

Para configurar los parámetros del método de nomenclatura externa:

  1. En el panel del navegador, amplíe Configuración de Red de Oracle > Local > Perfil.

  2. Seleccione Nomenclatura de la lista del panel de la derecha.

  3. Seleccione el separador Externo.

  4. Introduzca un valor en el campo apropiado para el método de nomenclatura externa que utiliza.

  5. Seleccione Archivo > Guardar Configuración de Red.

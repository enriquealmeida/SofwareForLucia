Enable Adobe Flash Player to fix Setup failure

This document helps you to solve a crash of GeneXus suite setups in Windows systems that have not enabled the Adobe Flash Player.

Symptom

When you execute a setup of GeneXus or any related product (*) in Windows Server 2016, 2019 or any Windows edition or version you get the error

Runtime error in setup script:

Source file: <...>
Line number: <...>

Library not registered.

Cause

The setup requires Adobe Flash Player to show animations with information about the features of the product, and Adobe Flash Player is not enabled on that system.

Solution

Enable Adobe Flash Player.

To do this, open (as an Administrator) the “Command Prompt (Admin) or the “Windows PowerShell (Admin)”, and then execute the following, depending on the operating system version

Windows Server 2016

dism /online /add-package /packagepath:"C:\Windows\Servicing\Packages\Adobe-Flash-For-Windows-Package31bf3856ad364e35amd6410.0.14393.0.mum"

Windows Server 2019 Build 17744

dism /online /add-package /packagepath:"C:\Windows\servicing\Packages\Adobe-Flash-For-Windows-Package31bf3856ad364e35amd6410.0.17744.1001.mum"

Windows Server 2019 Build 17763 (RTM)

dism /online /add-package /packagepath:"C:\Windows\servicing\Packages\Adobe-Flash-For-Windows-Package31bf3856ad364e35amd6410.0.17763.1.mum"

When it says “The operation completed successfully” you can close the Command Prompt

 

If you want to disable the component after the setup installation, execute the following

Windows Server 2016

dism /online /remove-package /packagepath:"C:\Windows\Servicing\Packages\Adobe-Flash-For-Windows-Package31bf3856ad364e35amd6410.0.14393.0.mum"

Windows Server 2019 Build 17744

dism /online /remove-package /packagepath:"C:\Windows\servicing\Packages\Adobe-Flash-For-Windows-Package31bf3856ad364e35amd6410.0.17744.1001.mum"

Windows Server 2019 Build 17763 (RTM)

dism /online /remove-package /packagepath:"C:\Windows\servicing\Packages\Adobe-Flash-For-Windows-Package31bf3856ad364e35amd6410.0.17763.1.mum"

When it says “The operation completed successfully” you can close the Command Prompt

(*) Scope

1) Setups of version 16 upgrade 11 or prior of

  • GeneXus
  • GeneXus for SAP Systems
  • GeneXus Japanese Edition
  • GeneXus Platform SDK
  • GeneXus Server
  • GXfow Production Environment
  • GeneXus Theme Editor
  • GeneXus Report Editor
  • GAM Deploy Tool
  • GAM Platforms

2) Setup of GXquery 4.0 Upgrade 1 or prior

3) Setup of GeneXus Protection Server version 9.7.5.12 or prior

Note: This problem has been solved in setups of GeneXus 17 or higher and newer setups of GXquery by avoiding to require Adobe Flash Player..

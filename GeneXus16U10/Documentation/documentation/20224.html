EmptyGridText property

Shows a text when the Grid has no data.

Samples

Consider an example where we want to show a list of clients, so we apply the Work With for Smart Devices object to the Client transaction and add an entry point in a Menu for Smart Devices object pointing to the WorkWithDevicesClient.Client.List(). If there are clients in the data base and we go to the WorkWithDevicesClients screen, the list will be shown; however, if we don't have any records the list will appear empty and sometimes the user may think there is a problem.

In sum, this property allows the programmer to show a text instead of an empty space.

Scope

Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
Controls: Grid

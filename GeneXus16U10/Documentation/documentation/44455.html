Android Base Style property (GeneXus 16 Upgrade 5)

To specify the base colors used in the Android layouts.

Values

Dark Black background, white forecolor fonts and black application bar color.
Light White background, black forecolor fonts and white application bar color.
Light with Dark Action Bar White background, black forecolor fonts and black application bar color.

Description

Through this property, you can determine to use a dark style or a light one in Android applications. 

It applies to the background, fonts colors and application bar color.

The Theme SD can be set with all its classes in order to customize the application design.

Notes

  • Default value as of GeneXus 15 Upgrade 6 is "Light with Dark Action Bar". 
  • Default value for previous versions is "Dark".

This property changes its name to "Base Color Scheme" since GeneXus 16 upgrade 6.

Run-time/Design-time

This property applies only at design-time.

Samples

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(Android)

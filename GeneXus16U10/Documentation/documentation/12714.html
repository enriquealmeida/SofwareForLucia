ToLower Method

Returns the specified character expression in lowercase letters.

Syntax

Character-Expression.ToLower()

Type Returned:
Character

Description

Converts all upper-case letters (A - Z) in the character expression to lower-case (a - z). All other characters in the character expression remain unchanged.

Note:                                                                                                                        
In Cobol and RPG for iSeries the character expression can be up to 256 bytes long.

Example

&TxtLower = &ArtistName.ToLower()
// &ArtistName = "Van Gogh, Vincent"
// Result: &TxtLower = "van gogh, vincent"

Scope

Objects: Procedures, Transactions, Web Panels, Work Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

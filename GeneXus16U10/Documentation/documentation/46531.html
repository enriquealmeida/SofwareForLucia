Ad Unit Id property (GeneXus 16)

Sets the Unit ID provided by AdMob in the SDAdsView control for Smart devices. AdMob is a specialized resource provided by Google that allows you to monetize your mobile apps.

Scope

Platforms: Smart Devices(Android, IOS)

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

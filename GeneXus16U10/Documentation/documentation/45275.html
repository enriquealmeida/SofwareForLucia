Newest Version

GeneXus 16 upgrade 9

This article is an overview of GeneXus 16 Upgrade 9 features (compared to GeneXus 16 upgrade 8) and what needs to be taken into account to adopt it.

It has been released on April 30th, 2020.

Download

 https://www.genexus.com/developers/DownloadCenter?,,,5885

Overview

Web

  • New Smart Table control (CSS grid) to facilitate layouts definition and alignment of controls
  • The standard classes used by the generated .NET and .NET Core applications are now open source on Github: https://github.com/genexuslabs/DotNetClasses! 

Mobile

  • HowTo: Augmented Reality in GeneXus applications
  • SD Image Annotations
  • Offline Data backup and restore
  • Android 10 (API 29) support added
  • iOS: XCode 11.4 support added, and generator and resources have been updated to corresponding Swift 5.2.2 language version

Security

  • GAM
    • GAM backoffice distributed as binaries includes a security fix (Sac #47393).
    • Wechat Authentication type support for Android.
  • GeneXus SFTP Module

GeneXus Server

  • The Console added support for the Chinese Language.

Deployment

  • Creation of Dockerfile without Docker for Windows

Testing

  • Unit Testing on Business Components
  • More information at Changelog GXtest

Compatibility

  • XCode 11.4 is required iOS Requirements
  • Alert: "Google Play will require that new apps target at least Android 10 (API level 29) from August 2020, and that app updates target Android 10 from November 2020." (ref.: Google Developers).  This implies that you must use GeneXus 16 Upgrade 9 or higher to be able to deploy to Goolge Play from those dates on.

    More information at  Android Requirements

Note: In .NET, a memory leak has been detected that affects stability in production environments. This has been fixed in GeneXus 16 upgrade 10. Please Check https://www.genexus.com/developers/websac?es,,,47725.

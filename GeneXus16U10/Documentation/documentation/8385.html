GetEncryptionKey Function

To generate an encryption key.

Syntax

GetEncryptionKey()

Type Returned:

Character - It is the key value to be used for the encryption. To generate a new key, the function must be run again.Â 

Description

This function generates a new encryption key with up to 32 character. It is used to initialize the encryption.

Example

&GetEncryptionKey=GetEncryptionKey()

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java

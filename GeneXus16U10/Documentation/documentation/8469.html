NewLine Function

To return the string needed to 'skip a line' on the platform that it runs.

Syntax

NewLine()

Type returned Character

Examples

Text = Text + Trim(&User) + '  has registered you to this meeting.' + NewLine()
TextBlock.Caption = 'Welcome' + Newline() + Username

Scope

Objects Procedures, Transactions, Web Panels, Panel for Smart Devices, Work With for Smart Devices, Work with for Web 
Data type Character, VarChar, LongVarChar
Runtime properties Caption
SD Generators iOS, Android
Languages .NET, Java

Delete function

To return True when the Transaction object is in Delete mode. Otherwise, it will return False.

Syntax

Delete

Type Returned:
Boolean (True or False)

Note

In the iSeries environment, when an error rule for a certain mode is defined (as in the example above, for Delete mode) the corresponding code to support this mode is not generated.

Example

A common use of this function is to include an error() rule denying access to a certain mode within Transactions:

Error('Orders may not be deleted') If Delete;

This rule prevents the user from accessing the delete mode and thereby eliminating Orders.

Rename Method

Renames the worksheet

Syntax

&ExcelWorksheet.Rename(Character)

Type Returned:
Boolean

Where:
Character
New value to rename the worksheet

Description

It renames the active worksheet to the value defined in Character

Scope

Extended Data Types: ExcelWorksheet data type

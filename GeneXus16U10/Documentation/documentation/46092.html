Copy Method

Creates a Copy of the current Excel worksheet

Syntax

&ExcelWorksheet.Copy(newName)

Type Returned:
Boolean

Where:
newName
New value to name the copy of the Excel worksheet

Description

It creates a copy of the current Excel worksheet with the name defined in newName

Scope

Extended Data Types: ExcelWorksheet data type
Languages: Java

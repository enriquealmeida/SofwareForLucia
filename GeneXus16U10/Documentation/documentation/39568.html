Log output property

It allows indicating the output for the Log level property.

Description

Values

ASP.NET trace The log is displayed on the ASP .NET trace (.Net only)
Console Appender The log is displayed on the Console Appender (Java only)
Event viewer The log is displayed on the Event viewer (.Net only)
File The log is generated as a file

Scope

Platforms: Web(.Net, .Net Core, Java)

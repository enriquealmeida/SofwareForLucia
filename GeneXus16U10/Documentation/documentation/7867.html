Default Theme property

Assigns a Theme for Web object at version level.

Description

For every Knowledge Base, by default, this property offered at version level is set with the 'Carmine' Theme for Web object.

So, every Transaction object and every Web Panel object inherit and uses that Theme, unless you assign a specific Theme for a particular Transaction or Web Panel by setting its Theme property.

You can assign to this property as well as to the Theme property associated with Transactions and Web Panels, any of the Themes defined in your KB.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)

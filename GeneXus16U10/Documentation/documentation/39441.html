Date format property (for Date/DateTime attributes/variables)

To select if accept and display two or four digits for the year.

Values

Year with four digits (99/99/9999) Accepts and diplays four digits for the year.
None Only offered for attributes or variables based on DateTime data type. Accepts and displays only the time part (no date).
Year with two digits (99/99/99) Accepts and diplays two digits for the year.

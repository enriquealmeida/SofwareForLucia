NextPage Method

Takes the user to the following group of registers when the automatic paging of Grids and freestyle Grids is used

Syntax

Grid.NextPage()

Values

0 Successful operation
1 The paging is not enabled in the Grid.
2 Already in the last page 

Description

This event is available when the Grid has base table and when it does not. The 'Rows' property must have a value different from 0.  Grids may be nested. 

This method efficiency is associated to the efficiency of the definition of the corresponding Grid navigation.  I.e.: if the Grid without paging has good response times, these times will be similar with paging.

Example

Event Following.Click
    MyGrid.NextPage()
EndEvent

Scope

Objects: Web Panels
Controls: Grids, Free Style Grids
Languages: .Net, Java, Ruby (up to GeneXus X Evolution 3)

Minimum Time Between Sends property (GeneXus 16)

Determines the time between calls to the synchronization Send services, which are in charge of sending the changes made in the device to the application server.

Scope

Objects: Offline Database
Platforms: Smart Devices (Android, IOS)

Description

This property applies to Offline Smart Devices applications in which an Offline Database Object has the Send Changes property set to "When connected."

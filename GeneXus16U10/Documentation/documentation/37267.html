Android Google Services API Key property

For indicating the API key in order to use Google services.

Description

This property must be set when the developer wants to include maps, use geolocation picker or embed YouTube videos in the application, all of them through Google services.

Values

A key of 39 alphanumeric characters long, obtained as explained in HowTo: Get an API Key from Google.

Notes

  • The device requires Google Play Services installed.
  • For maps, the property Android Maps API property must have Google Maps API v2.
  • In addition, to be able to have maps in the application, the API Key must have a billing account enabled. Otherwise, the grid will not be displayed as a map. More information here.
  • When modifying the value of this property, make sure to update the related Application Signing section (Key Store File, Alias, and passwords).

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(Android)

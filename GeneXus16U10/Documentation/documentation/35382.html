Header Class Property

The objective of this property is to allow setting the Cascading Style Sheet Class for the grid column headers.

Description

It's a property of the Grid Theme Class and its descendants, and its purpose is to allow setting the style to the header row of the grid.
At the same time, it must be assigned to any Theme Class that is below the node of the "GridRow" predefined class.

Example

Once you have associated the grid with the "GridLiteraryWork" class, you need to configure the Header Class property of the class with any class (GridRow or a descendant of it). In the example, the class is "GridHeadLWRow".

Then, configure the class as you wish depending on the results you want to have in the grid's header row.

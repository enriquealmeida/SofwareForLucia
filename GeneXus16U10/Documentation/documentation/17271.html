BPD End Events

Purpose

A End Event marks where a process ends. There are different types of this events to indicate the varying circumstances that can ended process.

Description

An End event is represented by a single circle with a corresponding icon inside.

The End events available are as follows:

End Events
None
Terminate

Signal

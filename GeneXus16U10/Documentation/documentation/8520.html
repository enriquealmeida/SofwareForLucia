WrkSt Function

To return the current Work Station Identification.

 

Syntax

WrkSt()

Type Returned:
Character

 

Description

 

Returns the current Work Station identification. In a PC environment, it returns the name of the machine (using API Windows functions) when the PSTATION environment variable is not set. Otherwise, this variable's value is used. This is the expected behavior whether there is network support or not.

Note:

In Client Server generator, if the variable PSTATION is not set, the machine identification is considered.

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Ruby, Java, RPG, Visual Basic, Visual FoxPro

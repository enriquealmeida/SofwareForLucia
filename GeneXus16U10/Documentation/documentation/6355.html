Theme Properties

The properties of a Theme are as follows: 

Name

It is the name of the physical file corresponding to the Theme (or Template if you save it through "Save as template...;" in this case, one file called theme.css would be saved in the KB folder).

If you change this property in a GeneXus theme installed by default, another theme will be created (with the new name) and the original GeneXus theme will be unselected and removed from the KB. If you need to use this theme again, you have to select it and GeneXus will re-import it.

Description

A functional description of the Theme.

Type

Read-only property.

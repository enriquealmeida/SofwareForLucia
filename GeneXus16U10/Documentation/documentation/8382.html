Decrypt64 function

To decrypt any text that has been encrypted with a key.

Syntax

 Decrypt64 (&text, &key)

Type Returned:
Character - It is the result of the de-encryption.  

Where:
&text
    Is the parameter to be decrypted.

&key
    Is the key to be used for the decryption, generated by the GetEncryptionKey function.

Description

It decrypts the value received in the first parameter, using the indicated key. 

Encryption is based on the Twofish algorithm using a default key of 32 characters.

Scope

Objects Procedure object, Transaction object, Web Panel object
Languages .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

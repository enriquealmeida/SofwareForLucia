Apple Maps API Key property

API Key for Maps for Apple. Depending on the selected API, may have to be generated from the application's Bundle Identifier.

Description

This property will be displayed when Apple Maps API property has a different value from its default (Apple Maps) in order to set the access key for displaying maps.

  • If Apple Maps API property has 'Google Maps' value, refer to HowTo: Get a Google Maps API Key for iOS to get its respective key.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(IOS)

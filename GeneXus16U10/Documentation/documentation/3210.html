KB Explorer

From the KB Explorer you can create new folders, modules, or objects, view their history, reorganize them, etc.

It has several nodes in which Tables, Documentation,Modules, etc, are sorted. You can access a certain object by typing its name in the "Open" cell.

It is possible to drag/drop and cut/paste objects simply by right-clicking on them. This makes it possible to move several objects at the same time, when the objects are saved in different folders.

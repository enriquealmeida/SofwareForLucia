AllowDrop property

Enables to add a new grid row with content dragged from another control. The grid has to be with _no_ base table.

Values

False
True

Description

Default value: False.

Setting this property on True, the user could drop information (dragged from another control) into the grid, that will be loaded automatically as a new line. 

Since this load is triggering in client-side, and there is no base table, you have to save this information (for example in a session) in order not to lose it the next post to server occurs. 

The drag & drop could be implemented between controls on different web components.

Scope

Platforms: Web(.Net, Java)
Controls: FreeStyle Grid, Grid

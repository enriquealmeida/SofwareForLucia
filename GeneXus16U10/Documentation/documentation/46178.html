Smart Devices Generator (GeneXus 16)

Easily Create Mobile Apps

GeneXus for Mobile and Smart Devices allows you to easily generate native mobile applications through of the Objects for mobile applications development, for Android and iOS (iPhone, iPads, iPod). Create smart and compelling apps once, deploy them everywhere.

Run and Improve

GeneXus automatically creates the native applications so you can run, test, and improve it.

Publish

With just one click, GeneXus publishes your native application to the Cloud or Market.

Requirements

Requirements are depending on the platform that will be used, but all three platforms can be used simultaneously.

Today function

To return the current date.

Syntax

Today()

Type Returned
Date

Notes

  • In the iSeries environment, this function returns the start date of the JOB, and not the SYSTEM DATE. This allows running a job with a date different from the current one, and using this date in the program. For using the actual SYSTEM DATE, see the Sysdate() function.
  • In PC environment returns de SYSTEM DATE, so Today and SYSDATE functions are equivalent.

Example

The following rule sets the current date value as the default date for an Order's date:

Default(OrderDate, Today());

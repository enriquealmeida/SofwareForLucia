Import a Propagation of Changes

To import the changes between two versions created with Change Defender technology you will have to:

  • Open a Client KB which already has imported a propagated version and select the Import dialog.
  • Select the xpz file containing the propagated changes generated with the Propagate Changes operation.

  • Press the Load button.
  • Check the output windows for import details

Notice than when importing you can create a Backup version to maintain the version status before importing so you can revert the operation.

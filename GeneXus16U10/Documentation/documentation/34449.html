ToBase64 function

Converts a string into a base64 encoded string

Syntax

ToBase64( s1 )

Where:
s1
Is a string

Type returned:

Character

Example

&MyBase64EncodedString = ToBase64(&MyString)

Availability

It is available as of GeneXus 15 Upgrade 4

Scope

Objects    Procedure object, Transaction object, Web Panel object, Smart Devices objects

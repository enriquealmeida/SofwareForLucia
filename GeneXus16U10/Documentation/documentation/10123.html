AddLines Method

Adds lines to a grid.

Syntax

control.AddLines( N )

where:

control is the grid/free style grid ControlName

N is the number of lines to be added.

This method doesn't validate; it only adds blank lines.

Scope

Controls: Grids, Free Style Grids
Objects: Web Transactions

References

Sometimes we might need to know the relationships existing between our Knowledge Base objects. The References menu option, listed in the contextual menu of each object or theme class, allows us to know about such relations. 

References Windows

The References window has two panels. On the left panel, we can see the objects that refer to the given object or class. On the right panel we see the objects or classes referenced by it. The following image shows references made from and to the AppMasterPage Web Panel.


Note. Names in italics indicate that the object may be deleted despite the the existing references to it.

Group References

The Group References by Type check box gives us a simple way to organize references by grouping objects by type in their own panels.

Select an Object

We can use the search bar to select one object from our Knowledge Base and display its references (we also have the three-point button to do this).

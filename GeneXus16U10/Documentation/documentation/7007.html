GetDataStore Function

This property is used to assigns a GeneXus Data Stores to a DBConnection variable.

Syntax

&data_type = GetDataStore(Exp)
 
Type Returned:
DBConnection      

Where:
Exp
   Datastore name. Character.

Description

When you have a DBConnection variable, you must assign a GeneXus data store to it. You do this by using the GetDataStore function. The parameter indicates a valid GeneXus data store name (e.g. ‘Default’ indicates the name of the default data store).

Scope

Data Types: DBConnection
Objects: Procedures, Transactions, Web Panels
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual Basic (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Synchronize with External Wiki Property

This property is available in all the documents and documentation that are part of every object. It controls whether or not the documentation will be synchronized with an External Wiki.

The default value for all documents can be set in the corresponding property in Preferences at the environment level. If most documents in the KB are to be synchronized with an External Wiki you should set the value to Yes in the property, and then at the documentation level establish any exceptions you want.

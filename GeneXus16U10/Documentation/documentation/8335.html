Now Function

To return a DateTime data type value corresponding to the current date and time.

Syntax

Now()

Type Returned:
Datetime

Description

Returns a Datetime value corresponding to the current date and time.
The resulting value includes milliseconds only if the assigned variable or attribute supports Milliseconds.

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), RPG, Visual FoxPro, Cobol

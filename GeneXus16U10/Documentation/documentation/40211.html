Enable Web UI Generation property

Enables web user interface generation for this instance.

Description

If set to FALSE, the Web objects related to the Conversational Flows instance are not generated.

For example, the main object <InstanceName>WebUI (main webpanel which calls the "PanelChatWeb" resource object) is not generated. The objects specified in the Generated Web Component property of the Flows are not generated neither.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows Instance

Apple Maps API property

API used to display Maps in iOS.

Values

Apple Maps Default value. Use Apple maps.
Google Maps Use Google maps. Must set Apple Maps API Key property with a Google key.

Description

This property allows the developer to use another maps provider different from Apple's maps (e.g. Google Maps). When you change its default value, theÂ Apple Maps API Key property will be displayed in order to set the access key.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(IOS)

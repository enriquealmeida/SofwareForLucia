Module/Folder property

You can indicate the Folder object or Module Object to which the GeneXus objects will belong.

Description

Folders establish hierarchies where each folder can contain GeneXus objects as well as other folders — but no Module Objects. Modules also establish hierarchies where each Module can contain GeneXus objects as well as other folders. Please read Modules for further details.

This information can be seen in KB Explorer.

Action property

Deprecated: Since GeneXus 16 upgrade 5 . Replaced by Chatbot User Input Redirections.

Action to take when the condition is evaluated.

Values

Is Required The chatbot will Require this parameter if the condition is true. (The chatbot requires all the parameters by default)
Redirect You can specify an existent Flow to jump if the condition is true.

Description

The Action property is used inÂ User input conditions to determine the behavior depending on the evaluation of a condition.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows

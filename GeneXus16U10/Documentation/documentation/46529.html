Default Action property (GeneXus 16)

Default event to execute when selecting a Grid for Smart Devices row. The default value calls the Detail.

Syntax

control. default Action

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Controls: Grid for Smart Devices

Run-time/Design-time

This property applies only at design-time.

Base Color Scheme property

Specifies the base colors used in the application layouts.

Values

Dark Black background, white forecolor fonts and black application bar color.
Light White background, black forecolor fonts and white application bar color.
Light with Dark Action Bar White background, black forecolor fonts and black application bar color.
Default Allows to use different default values, depends on OS preferences.

Description

Through this property, you can determine to use a dark style or a light one in applications. 

It applies to the background, fonts colors and application bar color.

The Theme for Smart Devices object can be set with all the classes in order to customize the application design.

Important: This property is visible only when the object has its Main program property set to True.

Run-time/Design-time

This property applies only at design-time.

Samples

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Objects: Menu for Smart Devices, Panel for Smart Devices
Platforms: Smart Devices(Android), Smart Devices(IOS)

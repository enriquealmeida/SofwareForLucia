MilliSecond Method

Returns a numeric value representing the milliseconds of DateTime data type argument.

Syntax

Datetime.Millisecond()
 
Type Returned
Numeric(3)

Example

msg(&DT1.ToString()) // ==> 09/09/99 02:35:30.230 PM
&Milliseconds = &DT1.Millisecond() // ==> &Milliseconds = 230

Scope

Objects: Procedures, Transactions, Web Panels
Languages: Web(.Net, Java), Smart Devices(Android, IOS)

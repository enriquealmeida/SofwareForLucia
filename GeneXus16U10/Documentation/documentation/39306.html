Precision property

Determines the precision of a DateTime field. Specifically, whether the DateTime field supports handling Milliseconds or not.

Values

Milliseconds Handles milliseconds precision (0 to 999)
Seconds Maximum precision is seconds, it does not handle milliseconds (default value)

Description

When Precision is set to Milliseconds, then the DateTime data type field supports storing and displaying the DateTime with higher precision than just 1 second; it then supports fractions of seconds likeÂ 3.5 seconds (three and a half seconds), 10.250 seconds (ten seconds with 250 milliseconds) or 5.005 seconds (5 seconds with 5 milliseconds). Such a field is often also called a Timestamp.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Objects: Domain, Attribute
Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
Controls: Attribute/Variable

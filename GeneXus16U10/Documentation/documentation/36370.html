Include Network Id in Client Information property

Deprecated: Since GeneXus 16 upgrade 6.

Set to true if you need to query the ClientInformation.NetworkId property (which provides the IMEI/MEID/ESN number associated to the user's network connection). Note that this information is not available for all SD platforms and versions.

Values

False
True

Run-time/Design-time

How to apply changes

To apply changes made by this property, do a Build All.

AddMinutes method

To add minutes to a datetime.

Syntax

Att | &var.AddMinutes(minutes)

Type Returned:
DateTime

Where:

minutes
   The number of Minutes added to the Att | &var.

Example

&DateTime = &DateTime.AddMinutes(10)  // Adds 10 minutes to DateTime variable

Scope

Objects Procedure object, Transaction object, Web Panel object

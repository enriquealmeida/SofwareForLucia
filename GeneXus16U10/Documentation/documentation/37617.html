Apple Bundle Identifier property

An identifier string that specifies the application type of the bundle. The string should be in reverse DNS format using only the Roman alphabet in upper and lower case (A-Z, a-z), numbers (0-9), the dot ('.'), and the hyphen ('-')

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(IOS)

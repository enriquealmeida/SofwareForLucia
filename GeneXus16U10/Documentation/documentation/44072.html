Chatbots Channels API

GeneXusChannels is a module which has the necessary services to be able to send messages through different messaging platforms (i.e: Facebook Messenger, Whatsapp).

An end user can send and receive messages through a channel, in turn he or she can have more than one channel of a different type, for example Facebook Messenger or Whatsapp.

When we talk about sending and receiving messages, it can be related to a Chatbot generated with the Chatbot generator, or to sending a message that is not directly related to a Chatbot. For example, the end users who have contacted us through Messenger, may receive a message like: "Don't forget! Tomorrow the GeneXus meeting begins, are you going to miss it?".

So, the Channels API helps the developers to send messages (text, images, video, etc.) to different channels.

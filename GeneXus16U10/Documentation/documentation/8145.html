Theme property (and Default Theme property)

To assign a specific Theme for Web.

Description

Every Web Panel object and every Transaction object offers the Theme property so that you can assign a Theme for Web to it.

If you do not assign an explicit Theme for Web to this property, it inherits the Theme for Web set in the Default Theme property (offered at version level). The Default Theme property by default is set with the 'Carmine' value, which is the name of a Theme for Web object automatically created in every Knowledge Base.

You can assign to both properties any of the Themes for Web defined in your Knowledge Base.

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)

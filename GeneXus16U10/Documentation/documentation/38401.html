Unit Test object

A Unit Test object represents a GeneXus Procedure, that will have the ability to run tests on GeneXus IDE by developers or automatically on different environments when using CI/CD capabilities.

You have two ways to create a Unit Test object:

1) In the same way that you create any GeneXus object.

2) In the KB Explorer you can right-click on the desired Procedure and select in the contextual menu: Create Unit Test.

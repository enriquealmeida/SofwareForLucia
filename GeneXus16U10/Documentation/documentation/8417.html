Second Function

To return a numeric value representing the seconds in a datetime argument.

Syntax

Second(Datetime)
 
Type Returned
Numeric(2)

Example

You only want to obtain the seconds part of a datetime variable or attribute.

&Sec= Second(Now())
// Now = 11/24/2010 02:42:39 PM
// &Sec = 39

Scope

Objects: Procedures, Transactions, Web Panels, Work Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

ToUniversalTime method

This method applies to DateTime data type data, allowing you to convert its value to Coordinated Universal Time (UTC).

The method assumes that the DateTime to convert is in the server's time zone.

Syntax

&DateTime.ToUniversalTime()

Type returned

DateTime

Example

&MeetingInUTC = &MeetingDateTime.ToUniversalTime()

Scope

Data type DateTime
Languajes .Net, Java, Ruby

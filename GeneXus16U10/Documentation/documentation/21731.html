InputHistory property

The Input History property has the goal of assisting the user of the application to enter information on a field. The way this is done is by remembering what has been entered to a field on a screen. By remebemebring this next time a user is entering information on that field the app will show the last values entered giving the user the oportunity to select one of the previous values and save them to input information.

Values

False
True

Description

Default value: True

Considerations

  • This property doesn't work for Date based fields.

  • When the IsPassword property is set to yes the Input History property is disabled.

  • A value is "saved" for the input history when an action is executed on the screen the field is on.

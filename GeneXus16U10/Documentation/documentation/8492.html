ServerTime Function

Returns the DBMS time in 'hh:mm:ss' format.

Syntax

ServerTime()

Type Returned
Character C(8)

Notes

The following warning is displayed when using the function in a Smart Device panel with the Connectivity support offline.

warning: spc0210: 'ServerTime()' not supported, using 'Time()' function instead.

Scope

Objects: Procedures, Transactions, Web Panels

Day method

Extracts the day number from a given date. The value 0 is returned if the received parameter is null.

Syntax

date-expression | datetime-expression.Day()

Type returned:
Numeric N(2)

Example

A Company wants to implement the following discount policy. All debts paid before the 21st of every month will have a 15% discount.

If PaymentDate.Day() < 21
   &Discount = Total * 0.15
else
   &Discount = 0
EndIf

Scope

Objects   Procedure object, Transaction object, Web Panel object

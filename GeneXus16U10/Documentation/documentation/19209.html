AddHours method

To add hours to a datetime.

Syntax

Att | &var.AddHours(Hours)

Type returned
DateTime

Where:

Hours
   The number of hours added to the Att | &var.

Example

&DateTime.AddHours(1)  // Adds 1 hours to DateTime variable

Scope

Objects Procedure object, Transaction object, Web Panel object
Languages .NET, Ruby, Java, RPG, Visual FoxPro, Cobol

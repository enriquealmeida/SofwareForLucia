Execute method

This method helps developers to execute an HTTP method in the defined URL path.

Syntax

&HttpClient.Execute(Method, URL)

  Parameters Method - The type of HTTP method used to retrieved data (described here). Only accepts HttpMethod domain values.
    URL - Relative URL path for the object which information wants to be obtained.
  Return The retrieved data is stored in HttpClient data type based variable,and it can be obtained using ToString or To File  methods.

Example

&HttpClient.execute(HttpMethod.Get, "/servlet/hclientes")
&Result = &HttpClient.ToString()

Scope

Objects Procedure object, Web Panels events  
Languages .NET, Java

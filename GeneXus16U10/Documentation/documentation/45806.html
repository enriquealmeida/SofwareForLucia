GXtest UI Commands - Verify

This command is useful to validate if a boolean value is true.

Verify(Value:boolean, [optional] StopExecution:boolean)

Works like an assertion but if the value is not true, then a screenshot of web page is taken and by default causes that test stops its execution.

Parameters:

Value: the boolean value to check.

StopExecution: optional parameter indicating if received parameter is not true then execution must be stopped or not. Default value is true.

Returns: 

This method doesn't return any value

 

Examples of use:

This command can be combined with several commands.

&driver.Verify(&driver.IsElementPresentByID("controlID"))

&driver.Verify(not &driver.AppearText("Not found"), false)

&driver.Verify(&driver.GetValueByName("controlName") = "Customer Name")

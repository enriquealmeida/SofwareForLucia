FromURL method

To change images and audios in run-time.

Syntax

FromURL(<StringExpression>)

Where: 

<StringExpression> is a string expression that must evaluate to a valid image/audio URL. This format is equivalent to the LoadBitmap function.

When used in transaction's attribute, GeneXus will automatically download the image or audio from the URL specified and store it in the database table. Any reference to the URL is saved. If you need only to store in the database a reference to an image URL or an audio URL, use ImageURI Property, AudioURI property, FileURI property or VideoURI property instead.

Examples

Procedure:

&image.FromUrl(!"http://images/myimage.jpg")

Data Provider:

{
   &MyImage.FromURL(!"http://images/myimage.jpg")
   MyImage = &MyImage
}

Scope

Data types: Image data type, Audio data type
Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
   

Notes

For Smart Devices application, you can use this method into user event as of GeneXus 16 upgrade 1

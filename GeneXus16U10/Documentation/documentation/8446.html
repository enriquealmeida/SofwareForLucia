Link Command

To call static pages or redirect to a static URL. It is equivalent to the Call command.

Syntax

Link( usr-pgm | ’url’ [{, parm}…] )

Where:
usr-pgm
    Is the name of the Web Panel to where it is going to redirect.

url
    Is the name of URL to where it is going to redirect.

parm 
    Are the parameters that the Web Panel or URL receives.

Description

The Link command is equivalent to the Call command for calling static pages or redirecting to a static URL. This command can be used within any event in a Web Panel, except the Load event. When this command is used there is an automatic redirecting to the URL specified in it.

Example

Event Enter
    Link(‘http://www.ARTech.com.uy’) 
EndEvent

Scope

Objects: Transactions, Web Panels
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Smart Devices

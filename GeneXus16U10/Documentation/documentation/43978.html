Newest Version

GeneXus 16 upgrade 6

This article is an overview of GeneXus 16 Upgrade 6 features (compared to GeneXus 16 upgrade 5) and what needs to be taken into account to adopt it.

It has been released on October 30th, 2019.

Download

https://www.genexus.com/developers/DownloadCenter?,,,5869

Overview

As announced at the 29th GeneXus Meeting, with this upgrade you can add WhatsApp and Facebook messenger channels to your applications. Furthermore, now you can train custom machine learning models with its Artificial Intelligence Module, it requires Java 8 or higher to build Java applications, and ships more build performance improvements, among other important updates.

Multi-experience

  • WhatsApp channel
  • Facebook messenger channel
  • Support for Dark mode in iOS apps

Security

  • GAM Apple Authentication type (required by Apple Store for iOS apps that use external authentication)
  • SSL Pinning Pin Set property

Artificial Intelligence

  • Custom models support (Microsoft, IBM)
  • More at GeneXusAI: Release notes

Testing

  • Introduction to UI Test for Smart Devices Automation
  • More at Changelog GXtest

Build

  • Options - Custom Build
  • New output window that supports higher volume and has better usability
  • Several performance improvements

Localization

  • Export Translations File adds support for XLSX and XLIFF formats 

Compatibility

iOS Requirements changed: XCode 11 or 11.1 is required. 

Note: XCode 11.2 is not supported in this upgrade. To use XCode 11.2 you need to install Beta version (build higher than 137019) or Preview Update build higher than 137073).
If you want to continue using V16U6 you need to have both XCode versions installed and change XCode version used by GeneXus when compiling.
 

As announced months before, Java 8 is required to build and run Java applications. 

Chatbots: Due to a change in the IBM Watson service platform that affects the .NET generator, it is necessary to follow the steps explained in SAC 46351 to keep backward compatibility.

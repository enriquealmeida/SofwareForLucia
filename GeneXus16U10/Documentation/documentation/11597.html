Default Master Page property

Default Master Page used for Web Panel objects and Transaction objects.

Description

The Default Master Page property is offered at version level and by default is set with the 'RwdMasterPage' value, which is the name of a Master Page object automatically created in every Knowledge Base.

Besides, every Web Panel object and every Transaction object offer the Master Page property, which by default inherits and uses the Master Page object set in the Default Master Page property (unless you explicitly change the Master Page property of the object).

You can assign to both properties any of the MasterPages defined in your Knowledge Base.

Values

None or any <Master_Page> name defined into the Knowledge Base. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)

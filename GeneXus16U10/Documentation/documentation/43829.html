GXtest versioning

GXtest is released for several versions of GeneXus. The following table shows the mapping between the different GeneXus versions and GXtest versions.

GeneXus GXtest Download link
From GX15 U7 to GX15 U10 4.1.X.Y   https://gxtest-downloads.s3.amazonaws.com/staging/GXtest_15U7-15U10.zip
GX15 U11 and GX15 U12 4.2.X.Y   https://gxtest-downloads.s3.amazonaws.com/staging/GXtest_15U11-15U12.zip
GX16 until U3 4.3.X.Y   https://gxtest-downloads.s3.amazonaws.com/staging/GXtest_16U0-16U3.zip
Latest stable version 4.W.X.Y https://gxtest-downloads.s3.amazonaws.com/staging/GXtest_Stable.zip
Beta 4.W.X.Y https://gxtest-downloads.s3.amazonaws.com/develop/GXtest_Beta.zip

Latest stable and beta version have version numbers according to it respective GeneXus version. For instance, GXtest version released for GeneXus 16 Upgrade 7 was GXtest 4.7.X.Y.

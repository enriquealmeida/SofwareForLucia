Authenticate users using LDAP property

Indicates whether users will be authenticated using LDAP.

Values

No The user does not use LDAP as authentication method. This is the default value.
Yes The user uses LDAP as authentication method. The application will request username/password, which afterwards will be validated with the LDAP server.

Scope

Platforms: Web(Java)

Description

Warning:Â This property is no longer used from GeneXus 15 onwards.

It only applies when the protocol used is RMI, CORBA, or DCOM. It does not apply to HTTP; in this case the webserver authentification can be used.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.

Size property

Sets the GeneXus connection pool size.

Description

The size of the pool is the maximum number of available connections to the database. This property becomes available only when the Unlimited size property is set to "False" value, otherwise, it doesn't apply.

The default value is 10.

Scope

Available at the datastore.
Datastore: Iseries, DB2 UDB, INFORMIX, MYSQL, ORACLE, POSTGRESQL, SQLSERVER

WriteStartElement Method

Scope

Extended Data Types: XmlWriter
Languages: .NET, Java, Ruby, Visual Basic, Visual FoxPro

Purpose

Starts a compound element.

Syntax

&DataType.WriteStartElement(ElementName)
 
Where:
ElementName
Name of the element to be created

Description

Starts a compound element. This element may have sub-elements, so it is valid to nest calls to WriteStarElement/WriteElement.

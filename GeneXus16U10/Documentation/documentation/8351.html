Rows Function

To obtain the return of the number of rows defined for an array.

Syntax

Rows(&array())

Type Returned
Numeric constant, the same used to define the array.

Description

By using this function, we avoid referring the array's dimensions within the program source. Modifying the array's dimensions would only cause the corresponding modification throughout the model, but there would be no need to modify the program's source where the array is used.

Example

The following example illustrates how to load a 2D-array whose elements will be the sum of their bidimensional positions (row and column):

&I = 1
Do while &I <= Rows(&m())
    &J = 1
    Do while &J <= Cols(&m())
         &m(&I, &J) = &I + &J
         &J = &J + 1
    EndDo
    &I = &I + 1
EndDo

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

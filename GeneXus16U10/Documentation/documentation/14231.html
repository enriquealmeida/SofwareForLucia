ToDate Method

To return a Date data type corresponding to a given DateTime data type.

Syntax

&datetime.ToDate()

Type Returned:
Date

Where:
&datetime
   Is the DateTime data type to be converted.

Example

&Date = &DateTime.ToDate()

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Cobol

Preview SD application property

Indicates an application that shows the user basic data about a task before the user takes it.

Scope

Objects: Business Process Diagram
Controls: Task

Description

It applies only when using the GXflow Client for Smart Devices.

When this property is used, the user views information about the task but doesn't take it. This information will help users to decide whether they can performÂ the task.

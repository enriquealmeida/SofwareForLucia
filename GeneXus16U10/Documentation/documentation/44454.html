Newest Version

GeneXus 16 upgrade 7

This article is an overview of GeneXus 16 Upgrade 7 features (compared to GeneXus 16 upgrade 6) and what needs to be taken into account to adopt it.

It has been released by December 20th, 2019.

Download

https://www.genexus.com/developers/DownloadCenter?,,,5875

Overview

It adds flexibility for high-demanding User Experiences in Web and Mobile, flexibility to model security aspects of applications, flexibility to define complex systems, improves scalability of GXflow, among other improvements and fixes.

Web

  • Canvas control also in Web

Mobile

  • MercadoPago Custom Checkout - To receive payments through the app without needing to store cards (credit, debit) information and PCI compliance. Users check out using your custom layouts without knowing MercadoPago is behind the scene. Get the sample.
  • Biometric Auth - To add biometric auth (fingerprint or face recognition) to your apps that use GeneXus Access Manager (GAM).  
  • Android
    • Dark mode in Android
    • Google Mobile Ads
    • Services URL can be changed in runtime
  • iOS
    • Select files from different sources

Security

  • GAM Remote Rest Authentication type (Oauth 2.0)

GXflow

  • Performance and scalability improvements
  • Standard Inbox is now in Chinese too

SAP

  • Fiori 3.0 is now the standard for the UI. A few conversion steps have been added to.

GX29 app - Dark mode support

  • #GX29 app has been updated on Google Play and Apple Store and supports now Dark mode. It is generated with this version and the code is open: GeneXus Meeting Open Source Project

Compatibility

  • GXflow Engine requires the execution of a reorganization that involves data transformation and temporary tables creation in some DBMSes. More information at SAC 45472 
  • iOS: This upgrade requires XCode 11.2 and supports 11.3. More information at iOS Requirements
  • Chatbot generator: Now the configuration file (GXCF_Chatbots.config) is only one, and includes the reference to all the instances of the KB. This file has to be taken to production. More information at SAC 46621

GetCurrentProximityAlert method

If a Proximity Alert has an action defined, an event is executed when the alert is fired, in that event we could need some of the information used to set that alert.
This method allow us to get the information related with the ProximityAlert which fired the action.

Return value

  • GeolocationProximityAlerts
    type: Collection of GeolocationProximityAlert SDT 
    An SDT with the information of the alert which fired the event.

NOTE: Only works when called from an action fired by a ProximityAlert.

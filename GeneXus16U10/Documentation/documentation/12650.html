Minute Method

Returns a numeric value representing the minutes of a datetime argument.

Syntax

Datetime-Expression.Minute()

Type Returned:
Numeric(2)

Description

Returns a numeric value representing the minutes given a datetime argument.

Example

If &DateTime.Minute() > 00
   &TxtMinute = "Past o'clock..."
EndIf

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3)

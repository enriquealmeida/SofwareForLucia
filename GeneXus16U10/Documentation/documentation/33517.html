Content Size Change property

This property applies to Form theme-class on Smart Devices generator and It allows the developer to change the behavior of the form when the virtual keyboard is displayed on the app.

Values

Default (default) On iOS devices is Resize.
On Android devices is Resize when it does not have controls that require scroll, in another case, its value will be Scroll.
Resize The keyboard's dimension is not included in the form, displaying the behind content completely as it was designed.
Scroll The keyboard's dimension is included in the form, adding scroll to display the behind content.

Example

Resize Scroll

Availability

This property is available as from GeneXus 15.

Scope

Objects Theme for Smart Devices 
SD Generators iOS
Languages .NET, Java

LTrim function

To return the specified character expression with leading blanks removed.

Syntax

LTrim(Character-expresssion)

Type returned:
Character

Where:
Character-expression
    Is the character expression from which you want to trim leading blanks.

Description

Returns a character string that results from removing the leading blanks from the character expression character-expression. This function is especially useful for removing the leading blanks inserted when you use Str() function to convert a numeric value into a character string.

Note

In Cobol for iSeries the character expression can be up to 256 bytes long.

Example

&Var = LTrim(' My character expression ') 
// Result of &Var: “My character expression”

Scope

Objects   Procedure object, Transaction object, Web Panel object

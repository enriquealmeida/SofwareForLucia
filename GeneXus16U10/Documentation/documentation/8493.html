Sysdate function

To return the system date.

Syntax

Sysdate()

Type Returned
Date

Note

In the iSeries environment, this function returns the SYSTEM DATE of the client’s machine, and not the start date of the JOB. See the today function to use the JOB's start date.

Scope

Objects   Procedure object, Transaction object, Web Panel object

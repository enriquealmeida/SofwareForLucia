Generate Android property (GeneXus 17)

Enables the Smart Device generator to generate the code corresponding to the Android platform.

Values

False
True

Scope

Platforms: Smart Devices(Android)
Level: Generator

Description

Default value: False

In addition, the True value will add the Android Specific group to the list of properties and the developer will be able to set the necessary properties for prototyping in this platform.

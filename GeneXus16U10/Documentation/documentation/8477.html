Previous function

To return the previously entered value of a given attribute.

Syntax

Previous( [Attribute] )

Type Returned:
Same as argument attribute

Description

Returns the value of attribute used in the last insertion in the current program session. The value is not stored and it is lost after the user leaves the Transaction program. If no attribute is specified, the attribute being assigned is assumed.

Example

Transaction Rule:

Default(ClientState, Previous());

The first insert Transaction of a working session will prompt the user with a null value in ClientState. If, for instance, the user types in 'LA' as the ClientState value and completes the insertion for the Transaction, the next insert screen will display 'LA' as the default value for ClientState, which the user may change. In other words, the default value will be the last value entered.

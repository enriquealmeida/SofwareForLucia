Business Component Check method

This method can be applied to a variable based on a business component type of a Transaction object.

It performs the same validations that are carried out when the Save method is executed, but without updating the database.

It is used when the developer wants to validate data (and mainly to give feedback to the user) before actually updating the database.
 

Syntax

&VbleBasedOnBCType.Check()

Where:

&VbleBasedOnBCType
      Is a -temporary and local- variable defined in a GeneXus object, based on a business component type of a transaction.

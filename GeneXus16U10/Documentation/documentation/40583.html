Platform Overrides property

Sets values of Theme, Navigation Style and Orientation for an application different from the default ones.

Description

Sometimes, in a Knowledge Base is needed to have different Smart Devices applications and set different configurations for them in the same platform, i.e., Slide navigation style is needed for Application1 and Flip for Application2, both in Android platform.

Navigation Style, Theme and Default Layout Orientation are properties specific for the Smart Device Platforms. If different values are needed for one of these properties, in different applications in the same Knowledge Base, Platform Override property dialog, of the application Main object, has to be used.

Notes:

  • If Images that vary depending on the Theme are going to be used, it is needed to add those Themes in the Additional Themes property of the platforms.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16 upgrade 2.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(Android, IOS)

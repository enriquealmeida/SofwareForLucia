GetString Method

The GetString Method of Web Components allows us to know the value of a Web Component variable within a parent Web Panel. It is also possible to overwrite an event associated with a Web Component button in the parent Web Object that contains it.

Description

This method allows a Web Panel that contains Web Components to query the value of controls within Web Components.

The name of the control to be queried is the only parameter and must be enclosed between quotes (if it is a variable, it must be prefixed by an ampersand).

Example

In a Web Component we have the &Var1 variable in an Edit Box displayed on the screen. If we include this Web Component in a Web Panel named Component1, from a Web Panel event we will be able to program the following:

&Variable = Component1.GetString("&Var1")

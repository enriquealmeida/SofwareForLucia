Theme property (for SD Platforms)

Assigns a specific Theme for Smart Devices to a certain SD platform.

Description

In the KB Explorer, under the Customization node, you can find the SD Platforms node. 

Under the SD Platforms node, several subnodes are offered, each one corresponding to a specific platform.

For each platform, the Theme property is offered to assign to it the desired Theme for Smart Devices object.

Run-time/Design-time

This property applies only at design-time.

Scope

Platforms: Smart Devices(Android, IOS)
Level: SD Platforms node > Platform

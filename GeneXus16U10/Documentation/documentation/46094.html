Empty Method

Sets the cell or cells as empty

Syntax

&ExcelCellRange.Empty()

Type Returned:
Boolean

Description

This method sets the cell or group of cells defined in the ExcelCellRange data type as empty.

Scope

Extended Data Types: ExcelSpreadSheet data type
Languages: Java

Availability

This data type is available since GeneXus 16 Upgrade 11.

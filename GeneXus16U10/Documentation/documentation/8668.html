AddItem method

To add a new item at the end of the object.

Syntax

control.additem(code, description [ , index ] )

Where:

index (value greater than 0)
    It is optional, and represents the position where the item is inserted. If no Index is specified, then the item is added at the end. It is not supported by Form.Meta or Form.MetaEquiv.

Note

Form.Meta and Form.MetaEquiv only apply to Web Interface.

Example

In this example, we load a Combo Box with all countries in the American continent. The first item will be 'All'. We use the additem method to load each item to the combo.

&Var.clear()
&Var.additem(0, 'All')
For each
    where ContinentCode = 'AME'
         &Var.additem(CountryCode, CountryName)
EndFor

In this example, we add a MetaTag to the Web Panel.

Form.Meta.additem(‘Content’, ‘GeneXus’)

Note

Form.Meta and Form.MetaEquiv are objects properties.

Scope

Objects Transaction object, Web Panel object
Controls Combo Boxes, Dynamic Combo Boxes, Dynamic List Boxes, List Boxes
Languages .Net, Java, Ruby, Visual FoxPro

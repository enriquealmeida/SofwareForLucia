Program Description Variable - Pgmdesc

To store the description of the active program.

Syntax

&Pgmdesc

Data Type: Character (31)


 Description

This GeneXus variable stores the active program's description when it is evaluated. The description is the one specified in the object's properties option.

Note

Although the Objects Description is currently supporting up to 256 characters, the variable continues with a 30 length to keep compatibility, so that no change is caused in the screens where it is used.

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Ruby, Java, RPG, Visual Basic, Visual FoxPro, Cobol

Indicator Selected Color property

Deprecated: Since GeneXus 17.

Sets the color of the selected page indicator in an SD Paged Grid.

Scope

Controls: Grid (Control Type: Paged Grid)

Description

This property can be set for Grids whose Control Type property = SD Paged Grid in up to GeneXus 16.

The Control Type property = SD Paged Grid is deprecated since GeneXus 17, so the property is deprecated too.

If you have a Grid control with its Control Type property = "SD Paged Grid" in a KB created with a version prior to GeneXus 17, it will continue working but at build time with GeneXus 17 you will see a warning message informing that the control type is deprecated.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build a main object.

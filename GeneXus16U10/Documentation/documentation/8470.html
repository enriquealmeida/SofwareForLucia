Time function

To return the time in the 'hh:mm:ss' format.

Syntax

Time()

Type Returned
Character C(8)

Notes

  • In the iSeries environment, this function returns the start time of the JOB. See the Systime function if you want to use the actual SYSTEM TIME. 
  • In a PC environment, it returns de SYSTEM TIME, so Time and SYSTIME functions are equivalent.  
  • In a Client/Server environment, this function returns the time of the PC.

Scope

Objects   Procedure object, Transaction object, Web Panel object

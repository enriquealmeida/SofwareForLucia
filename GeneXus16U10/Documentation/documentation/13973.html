Asc function

To allow  converting the first character of a string into the corresponding ASCII character.

Syntax

ASC(string)

Where:
   string is the string of regular characters.

Returned value:
   Numeric

Example

&AscValue = Asc("Hello") // &AscValue = 72

Scope

Objects Procedure object, Transaction object, Web Panel object, Work Panel object
Languages .NET, Ruby (up to GeneXus X Evolution 3), Java, Visual FoxPro (up to GeneXus X Evolution 3)

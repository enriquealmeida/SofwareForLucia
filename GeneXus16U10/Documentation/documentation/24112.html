Maximum Upload Size property

This property allows us to control the size of the images that are transferred to the server.

Values

Actual Size Images are transferred in its original size.
Large

Images size is controled by the value of the property Large Image Upload Size

This is the default value.
Medium Images size is controled by the value of the property Medium Image Upload Size
Small Images size is controled by the value of the property Small Image Upload Size

Scope

Applies to Image data type
Platforms Apple, Android

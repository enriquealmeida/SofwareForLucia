Generate For Smart Devices Property

This property is deprecated since GeneXus X Evolution 3 and has been excluded from GeneXus Salto. 
Use Responsive Web Design in GeneXus instead, and follow this guide to make it responsive.

It allows to generating objects based on the Work With Pattern for Web Mobile Applications (Smart Devices).

Values

False Objects will not be generated. This is the default value.
True

The Work With Pattern will generate a new set of objects to implement the same functionality of the Work With Pattern for Smart Devices. The new objects are as follows:

  • Work With web panel called WW<TRN Name>SD
  • View web component called View<TRN Name>SD
  • Master Page called AppMasterSD
  • Theme called GeneXusXSD

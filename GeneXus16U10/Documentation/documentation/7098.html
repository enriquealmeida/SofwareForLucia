SpellCheck Method

Runs the spell checker through the document. 

Syntax

&WordDocument.SpellCheck()

Type Returned:
Numeric
 
Note:
This method returns an error code, so it is possible to call it as a function (&Err = &WordDocument.SpellCheck()).

Scope

Extended Data Types: WordDocument
Languages: .NET, Java, Ruby, Visual Basic, Visual FoxPro

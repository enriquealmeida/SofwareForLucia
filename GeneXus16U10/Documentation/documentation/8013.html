Standard Functions Object property

This object property must be turned-on in some cases, in order to allow the use of specific functions, not supported in all generators.

Values

Allow non-standard functions Inhibits the syntax control mentioned above, at specification time.
Use Environment property value
Only standard functions The syntax control mentioned above will be done at specification time. This is the default value.

Description

The property is available at Knowledge Base and Generator level.

Scope

Objects: Procedure, Transaction, Web Panel
Platforms: Web(.Net, Java)

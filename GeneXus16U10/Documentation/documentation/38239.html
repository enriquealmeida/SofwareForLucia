Header property for Menu (ex Dashboard)

To specify an image header to the Menu (ex Dashboard) for Smart Devices object.

Description

The property applies to the root item of the Menu (ex Dashboard) object and allows the developer to display a header image on the application (through an Image object in the Knowledge Base).

Notes

  • In Android, this property is ignored when the Dashboard has the Control property with Tabs value.

 

Run-time/Design-time

This property applies only at run-time.

Header property (Menu for Smart Devices object)

Specifies an image header.

Scope

Objects: Menu for Smart Devices
Platforms: Smart Devices(Android)

Description

The property applies to the root item of the Menu object and allows the developer to display a header image on the application (through an Image object in the Knowledge Base).

Notes

  • In Android, this property is ignored when the Menu for Smart Devices object has the Control property with Tabs value.

 

Run-time/Design-time

This property applies only at run-time.

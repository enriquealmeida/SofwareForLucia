Knowledge Base Navigator

GeneXus Knowledge Base Navigator (KBN) is an iOS native application whose aim is to allow GeneXus's users to navigate through the iOS online applications they create using GeneXus. It allows traversing the functionalities that a GeneXus Knowledge Base application exposes: Entities and Relationships.

Downloading

You can download KBN from the Apple Store.

Considerations

  • Offline applications cannot be prototyped with KBN. This tool only interpreted metadata.

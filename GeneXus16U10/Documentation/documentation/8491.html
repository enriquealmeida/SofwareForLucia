ServerNow Function

To return a Datetime value corresponding to the current date and time of the DBMS.

Syntax

ServerNow()

Type Returned:
Datetime

Scope

Objects Procedure object, Transaction object, Web Panel object
Languages .NET, Java, Ruby, Visual FoxPro
   

Notes

The following warning is displayed when using the function in a Smart Device panel with the Connectivity support offline.

warning: spc0210: 'ServerNow()' not supported, using 'Now()' function instead.

Standard Functions property

Allows you to use specific functions that are not explicitly supported by GeneXus.

Values

Allows non-standard functions when specifying Functions not supported by GeneXus can be used.
Error on non-standard functions when specifying Only the standard functions supported by GeneXus can be used. This is the default value.

Description

This property is used when you want to use specific functions that are not explicitly supported by GeneXus or in all generators. The check will be made when the object will be saved.

All functions supported by GeneXus become standard functions, even if they are only available for some generators.

Note

The following is the complete table of values at different levels:

At Environment level  
Allows only standard functions on saving Only the standard functions supported by GeneXus can be used. This is the default value.
Allows non-standard functions on saving Functions not supported by GeneXus can be used.
At Generator level  
Error on non-standard functions when specifying Only the standard functions supported by GeneXus can be used. This is the default value.
Allows non-standard functions when specifying Functions not supported by GeneXus can be used.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Platforms: Web(.Net, Java)

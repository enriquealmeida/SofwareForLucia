Attachments Property

Lists the names of the files attached to the message.

Syntax

&DataType.Attachments

Type Returned:
StringCollection

Description

If a complete route is not specified when sending a message, the attached files will be searched for in the directory specified by the AttachDir property.

SMTPSession: Either the slash (/) or the backslash (\) can be used as directory delimiters in the route of the attached files. This will provide compatibility between the UNIX and Windows environments. 

This property is read only and cannot be assigned.

Scope

Extended Data Types: MailMessage
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

.Net Core Generator Requirements (GeneXus 16 Upgrade 7 or prior)

The following is the necessary requirement to build .Net Core apps.

  • .NET Core SDK 2.1

Optional: by default .Net Core generator runs web applications in Kestrel web server which is included in .net core SDK. In order to run the application with IIS working as a reverse proxy in front of Kestrel (*) install:

  • .NET Core Windows Server Hosting Bundle (a.k.a Runtime & Hosting Bundle)

(*) To configure the .Net Core App for IIS set the generator property Web Server = Internet Information Server.

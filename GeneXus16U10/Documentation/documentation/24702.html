Create new environment dialog

Purpose

Allows to create a new environment in the Knowledge Base, by selecting the Server Platform, the Language, the DataSource, and whether it is a Smart Devices environment or not.

Description

Through this dialog a new environment can be created in the KB by specifying the basic information needed for that purpose.

  • Server Platform
    • Local
    • On the Cloud
  • Smart Devices
    • Android
    • iOS
  • Language (Java, Csharp, .NET Core)
  • Datasource (any of the supported DBMS)
     

Through the Advanced tab, additional information can be configured.

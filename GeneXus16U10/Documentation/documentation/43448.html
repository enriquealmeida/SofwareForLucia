BPD Start Events (GeneXus 16 Upgrade 3 or lower)

Purpose

A Start Event shows where a process can begin. There are different types of these events to indicate the varying circumstances that can trigger the start of a process.

Description

An event is represented by a single circle with a corresponding icon inside.

The events available are as follows:

Start Events
None

Item method

Returns a reference to an element of a collection. The developer must specify the position of the desired element in the collection.

The first position is 1. If the specified position is less than 1 or greater than the number of elements, the program fails.

Syntax

Item(Position)

Where:
Position
    Is the position of the desired element in the collection. This parameter must be Numeric (between 1 and the value of the Count property). 

Examples

1) &CharacterTypeVariable=&CharacterTypeCollection.Item(10)

2) &Customer=&CustomersCollection.Item(20)
 

Note

It is not possible to assign a value to this property. Therefore, the following code  “&sdt.item(&i) = Att” is wrong.

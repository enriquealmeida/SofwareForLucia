Newest Version

GeneXus 16 Upgrade 11

This article is an overview of GeneXus 16 Upgrade 11 features (compared to GeneXus 16 upgrade 10) and what needs to be taken into account to adopt it.

It has been released on September 3rd, 2020..

Download

https://www.genexus.com/developers/DownloadCenter?,,,5895

Overview

This upgrade includes several improvements and fixes related to security and stability, and also some new features in different fields. It allows to use of Artificial Intelligence capabilities on the edge and getting the most out of the devices' power; the Chatbot generator adds Botmaker as a Whatsapp provider to allow multimedia in chats through Whatsapp; GXflow is now available in .NET Core using the data stores MySQL and Oracle.

Mobile

  • AI 'on the edge': Support for MLKit in Android. More information at GeneXusAI Overview

Security

GAM API allows to

  • physically delete of a User (and all its related information)
  • recover a logically deleted User

GXflow

  • Support for .NET Core added for the following DBMSes: MySQL, Oracle
  • New settings to improve performance in high scalability environments
  • Custom authentication and authorization panels configuration when using GAM.

Chatbots

  • The Chatbot generator requires licenses since this upgrade. See Sac 47859.
  • Support for Botmaker Whatsapp partner. See Chatbots using WhatsApp: How to set up Botmaker.
  • Read also: Howto: Integrate a new WhatsApp partner into a GeneXus Chatbot

Reporting

  • Grammar and editor improvements in Query object filters and expressions

Compatibility

Query Object

Warning: The Query Object includes several improvements in relation to it's grammar to create more expresive queries. If you create or save objects with this version they cannot be evaluated in previous versions (For more information check SAC#47809). Once objects are saved with this version the conversion cannot be undone, the following message may appear:.

Checking Import References...
error: Part descriptor not found: 'Query Version 3' (provided by package 'Artech.GXplorer.BL').

Chatbots

The AnalyzeResponse object changes its structure, so the resources of the chatbot have to be updated to avoid this error:
error src0216: 'GXOutputCollection' invalid property. (Procedure 'x' Source, Line: 279, Char: 57, Details)
See SAC 47860 for more information.

Other

  • PDF Reports with HTML require to set height and width (For more information check SAC#46755)

GAM

Some methods change their signature, for security fixes. See SAC 47901 for more information.

  • GAMRepository.IsRemoteAuthentication (State parameter is added)
  • GAMRepository.GetRemoteApplication (Client Id parameter is changed by State)
  • GAMRepository.RedirectToRemoteAuthentication (State parameter is added)
  • GAMRepository.GetGAMRemoteInitialProperties (State parameter is added)

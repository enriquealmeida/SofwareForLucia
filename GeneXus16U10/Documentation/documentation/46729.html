Which objects can be defined in a module? (GeneXus 16 Upgrade 11 or prior)

Not all objects can be defined in any Module. Below is a table that shows the objects and the Modules they can belong to.

Object Module
Image, Language, Theme, Table, Attributes and Files These objects are global to the Knowledge Base and don't belong to any Module.
WorkPanel, Menu and Menu Bar Win objects can only belong to the Root module because Win Generators don't support the Modules Feature.

Folder, Transaction, Procedure, WebPanel, Panel object, Data Selector, Structured Data Type, Diagram, Document, External object and Subtype Group, Domain.

 These objects will belong to the Module where they are defined and can be defined in any Module in the Knowledge Base.

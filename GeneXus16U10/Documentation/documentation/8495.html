GxRemove variable

To indicate which records will be deleted.

In transactions, GeneXus uses an automatic numeric variable in the first column of the second level grid in order to implement an easy record deletion mechanism.

Data Type:
Numeric(1)

Scope

Objects: Procedure object, Transaction object, Web Panel object

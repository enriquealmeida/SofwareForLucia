Hour format property

To select with which parts of the hour to work (hour, minutes, seconds, milliseconds or only some of them).

Values

Hour and minutes (hh:mm) Accepts and displays hours and minutes.
Hour, minutes and seconds (hh:mm:ss) Accepts and displays hours, minutes and seconds.
Hour, minutes, seconds and milliseconds (hh:mm:ss.fff) Accepts and displays hours, minutes, seconds and milliseconds.
Only hour (hh) Accepts and displays only hours.

Description

This property is offered for attributes and variables that are based on the DateTime data type in order to set with which parts of the hour to work.

Note

The possibility of handling milliseconds is available since GeneXus 15 Upgrade 11. So, this property starts offering the    "Hour, minutes, seconds and milliseconds (hh:mm:ss.fff)" value since the same version.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

SD Application property

Indicates the GeneXus Object for Smart Devices associated with a task; that is to say, the GeneXus object that will be invoked when executing the task in a Smart Devices Application.

Scope

Objects: Business Process Diagram
Controls: Task

Description

This property indicates the Smart Devices Object to be called when using the GXflow Client for Smart Devices and ignored when executing the GXflow Client.

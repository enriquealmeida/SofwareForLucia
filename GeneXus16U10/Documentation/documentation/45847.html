Information messages at Specification time

The following list shows the nfo messages that can appear at Specification time.

Code Message

nfo0001

This object has client-side events only, no data providers will be generated.

It indicates that the panel has no events to evaluate on the server and therefore it does not require generating a Data Provider. That is, all the panel's logic is resolved without calling the server.
   

Availability

Information messages are available since GeneXus 16 upgrade 10

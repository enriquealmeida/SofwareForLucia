Generators supporting modules

As of GeneXus 15 the following generators support Module Objects:

  • .NET Web
  • Java Web
  • Android
  • iOS

Note: When using modules and generating for Java the Java package name property must be set.

Generators not supporting Module objects:

  • Cobol
  • RPG

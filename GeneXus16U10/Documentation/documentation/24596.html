RemoveDiacritics Method

The RemoveDiacritics method returns the text without diacritic characters.

Syntax

&DataType.RemoveDiacritics()

Type returned
Character

Example

&Text = "áéíóú"   
&TextNoDiacritics = &Text.RemoveDiacritics()   // Result: "aeiou"
if &TextNoDiacritics = 'aeiou'
     //Success evaluation
endif

Availability

This feature is available as of GeneXus X Evolution 3 RC (codename: Tilo)

Scope

Objects: Procedures, Transactions, Web Panels, SDPanel
Platforms: Android, iOS
Languages: .NET, Ruby, Java

Note

This method is useful in conditions evaluated on client side. On the server side this could be solved by the DBMS, for example SQL Server.

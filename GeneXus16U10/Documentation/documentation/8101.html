Output Variable

Specifies the output device.

Data Type: Character (3)

Values

SCR Screen. This is the default value.
PRN Printer.

Â 

Example

Different Procedure object layouts are produced, depending on whether it is a screen or printer output. The built-in variable &Output may be used to print different lines or header depending on the output device.

If &Output = 'PRN'
   // (lines with more than 78 characters-width)
Else
   // (lines not more than 78 characters-width)
EndIf

The calling program calls a Procedure object which receives variable &Output as parameter. The output device is determined by the parameter ('SCR' or 'PRN'). This is how the same Procedure may be used to display the data on the screen or have it printed.

Rules:
Parm(&Output);

Scope

Objects: Procedure object

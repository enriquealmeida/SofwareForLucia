Null Function

To return True if the value of a given attribute or variable is empty.

Syntax

Null(Attribute | &Variable)

Type Returned:
Boolean (True or False)

Description

Returns true if the value of the attribute or variable is empty.

Example

In a system that handles customer information, we want to display a message when no address is found for a given customer.

Msg('There is no address for the specified customer') If Null(CustAdd);

The message will be displayed if the attribute CustAdd contains a blank string.

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

Delete row image property

To set the image name to be displayed for the first column in the transaction grids, which offers to delete a row.

Description

Each row of a grid in a Transaction has a column with a control that offers to delete the row.

This property allows the developer to set the image name to be displayed for this column. The default value is that of the DeleteRowImage image.

GXtest 4 Installation

GXtest 4 is a set of tools for test automation, that is IDE-compatible in GeneXus 15 versions and upper and included automatically inside GeneXus 16. 

We always recommend working with the latest GXtest release version on top of your default one. 

You can download the latest releases over the public channels here.

To install GXtest 4 capabilities on GeneXus IDE you can: 

Install it manually (3 steps)

  1. Download GXtest version you want to install (ie: for GeneXus 16 you can use the latest preview version from: https://gxtest-downloads.s3.amazonaws.com/staging/GXtest_Stable.zip)
  2. Extract all the files over your GeneXus IDE installation (ie:  C:\Program Files (x86)\GeneXus\GeneXus16)
  3. Run genexus.exe using /install parameter from your command line, to let GeneXus install the GXtest extension. 

Or using Ansible (alternative)

If you want to automate the installation of GXtest and have a little experience over Ansible, you can download and use this playbook:

https://github.com/abstracta/ansible-playbooks/blob/master/GXtest/install_gxtest_gx16.yaml

The command line should look like this:

> ansible-playbook -i hostmachinefile install_gxtest_gx16.yaml

Installing GXtest Recorder (for Google Chrome)

GXtest v4 recorder works on latest Chrome browsers and allows to record and playback user actions in any GeneXus web app (from GeneXus 8 and newer versions)

To install GXtest recorder, please visit the official Google Chrome store:

https://chrome.google.com/webstore/detail/gxtest-recorder/edbiefalppkhegcephbbcojpibobbdbd

Reload Method

Reloads all the possible values of a dynamic combo box from the corresponding table.

Syntax

Control.Reload( )

Description

Reads the corresponding table and loads all the values into the combo box.

This property applies only at execution time.

Scope

Objects: Transactions, Work Panels
Controls: Dynamic Combo Boxes, Dynamic List Boxes
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to Genexus X Evolution 3)

Note

This method is not supported in web environments: SAC#40258

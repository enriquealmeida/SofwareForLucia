Classpath property

Use this property to add custom resources/JARS to the classpath. Note that these elements are added to the beginning of the base classpath, so these take precedence over resources at Base Classpath property.

Description

TheÂ Classpath can be set using this property.

Compatibility

This property has changed its behavior since GeneXus 15 upgrade 10. It can be configured by the user with any of the resources needed to run the application, which will precede the Base Classpath. The resources and JARs used by GeneXus are automatically assigned to the Base Classpath property.

Scope

Platforms: Web(Java)

Main Platform property

When in the Smart Device generator we select more than one platform (i.e. Android and iOS), we must indicate the one to be considered upon pressing F5 for executing the application. Nevertheless, when we press F5, everything necessary for both platforms will be generated.

Values

Android This value is available when the value of the Generate Android property is True.
iOS This value is available when the value of the Generate iOS property is True.

Description

When we have only one single platform, it is assumed by the property by default.

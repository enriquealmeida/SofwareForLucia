RecordCount property (GeneXus 16 Upgrade 5 or prior)

To obtain, at runtime, the number of records of a Grid or Freestyle Grid load as a result of paging.

Description

The RecordCount property applies only to Grids with Base Table, and returns a number which is equal to or higher than 0, representing the number of records of the Grid base table that meet the selection conditions. It may return -1, if there is no navigation for the Grid base table or if the conditions are evaluated client side.  
 

Value Comment Suggestion
-1 The Grid does not have a base table or the conditions are evaluated client side. Attributes must be included in the Grid to indicate navigation.
Check the conditions if they are evaluated server side.
>= 0 Number of records of the Grid base table  

Run-time/Design-time

This property applies only at run-time.

Scope

Objects: Web Panel
Platforms: Web(.Net, Java)
Controls: FreeStyle Grid, Grid

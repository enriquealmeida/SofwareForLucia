Auto correction property

Indicates if the Characters, VarChars or LongVarChars fields will be auto corrected in edit mode.

Values

False
True

Description

Domains, Attributes, Variables or SDT members whose data type is Character, VarChar or LongVarchar. Its value is inherited in the case that a field is based on a domain or attribute.

Values

True: Automatically check spelling and indicate possible errors. In Smart Devices will also suggest words. This is the default value.

False: Spelling is not checked. It is used for fields that contain alphanumeric codes that are not necessarily words in a dictionary.

Scope

Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
Controls: Attribute/Variable

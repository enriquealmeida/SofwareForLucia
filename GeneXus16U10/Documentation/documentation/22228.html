Smart Devices Offline Applications

GeneXus offers the possibility to generate Offline Smart Devices applications in addition to Online Apps.

This feature solves common situations for apps in Points of Sales, Events apps or any scenarios of limited or no Internet connectivity. This feature allows applications to execute database transactions even if the device has not connectivity.

Definitely, GeneXus generates Smart Devices application that can run online, offline or in a mixed way.

Generating an offline application involves creating a local database in the device and generating the corresponding programs in native code for each platform (Java for Android, Objective-C for iOS, etc) as stated in Smart Devices Offline Applications Generation.

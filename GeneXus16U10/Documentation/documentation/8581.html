Do command

Calla a subroutine (defined using the Sub command). Parameter passing is not allowed. All variables from the same source program can also be used by the routine (i.e. they are global variables).

Syntax

Do ‘SubRoutine-Name’

Where:
SubRoutine-Name
      Name of the subroutine to call.

Scope

Objects    Procedure object, Transaction object, Web Panel object, Work Panel object, Work With object, Panel object

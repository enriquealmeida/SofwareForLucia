Additional Themes property

UI Additional Themes associated with this platform.

Scope

Generators: Android, Apple

Description

It allows adding additional themes to the specified platform (Android and iOS) and change them using the SetTheme function.

 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

Availability

This property is available since GeneXus 16 upgrade 1.

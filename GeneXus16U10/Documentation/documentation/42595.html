Test Target property

Defines the main Smart Devices object to use as test target.

Description

This property offered by the UI Test for SmartDevices object is to define the main object to be used as the test target. 

You must always assign to this property a main object (it is not possible to save UI Test for SmartDevices object with this property empty).

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 16 upgrade 5.

Scope

Objects: UI Test for Smart Devices
Platforms: Smart Devices(IOS)

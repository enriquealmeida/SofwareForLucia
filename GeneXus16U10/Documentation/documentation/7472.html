CtoD function

To convert a date-string into a date value.

Syntax

Ctod(mm/dd/yy | dd/mm/yy | yy/mm/dd)

Type returned:
Date

Where:
mm/dd/yy | dd/mm/yy | yy/mm/dd
    Must be a string. It depends on the property “Date Format on CTOD function”.

mm represents the month in 2 digits
dd represents the day in 2 digits
yy represents the year, it can be 2 or 4 digits. In case 2 digits are used the year limit is applied (First Year of 20th Century Property)

Example

CDOW(Ctod('01/21/14')) = Tuesday

Scope

Objects   Procedure object, Transaction object, Web Panel object, Work Panel object

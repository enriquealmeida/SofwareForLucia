Collection property (IsCollection checkbox)

Indicates whether the element has multiple instances or not.

Values

False
True

Scope

Objects: Domain, Structured Data Type, Variable
Generators: Android, Angular, Apple, Java, .NET, .NET Core

Description

Default value: False.

Note

The same effect is produced by clicking/unclicking the IsCollection checkbox.

In case that the variable corresponds to a Chatbot User Input, seeÂ Variable collection property behavior for User inputs.

Run-time/Design-time

This property applies only at design-time.

Systime function

Returns the system time in 'hh:mm:ss' format. In other words, returns the same as what the Now Function returns, but in string format.

Syntax

Systime()

Type Returned:
Character(8)

Note

In the iSeries environment, this function returns the SYSTEM TIME and not the start time of the JOB. See the Time function if you need to use the JOB's start time. In a Client/Server environment, this function returns the time of the application server.

Scope

Objects   Procedure object, Transaction object, Web Panel object

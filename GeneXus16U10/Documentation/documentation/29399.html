Use annotations for servlet definition property

It enables the creation of web applications valid for the Servlet 3.0 specification.

Values

No
Yes Default Value

Description

Notes:

  • If  Tomcat 7.x or a higher version of Tomcat is detected, the value of the property is Yes; otherwise, if Tomcat has a lower version (or it isn't installed), the value is No.
  • Servlet 3.0 annotations are not compatible with lower versions; an application generated with the property enabled will not work on an application server with a lower level specification support.
  • For more information check SAC #29992.

Scope

Platforms: Web(Java)

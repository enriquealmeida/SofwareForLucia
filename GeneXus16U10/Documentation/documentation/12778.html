Beep on errors property

Allows you to define if you want a beep when an Error Rule is executed at execution time.

Values

No No beep sounds when an error rule is executed. This is the default value.
Use Environment property value
Yes A beep sounds when an error rule is executed.

Scope

Objects: Transaction
Platforms: Web(.Net, Java)

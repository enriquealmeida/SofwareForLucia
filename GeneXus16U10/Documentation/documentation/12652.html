Hour Method

Returns a numeric value representing the time in 24-hour format.

Syntax

datetime-expression.Hour()

Type Returned:
Numeric(2)

Description

Returns a numeric value representing the time in 24-hour format given a datetime argument.

Example

If &HappyTime.Hour() > 20
   &TxtHour = "Is the After-eight time!"
EndIf 

Scope

Objects: Procedures, Transactions, Web Panels, Work Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

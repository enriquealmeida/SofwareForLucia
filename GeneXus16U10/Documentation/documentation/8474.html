PrintDocument function

To print a document (it will not open it if not necessary).

Syntax

&ret = PrintDocument(‘document’)

Type Returned
Numeric

If the PrintDocument function successfully executes the document named, it returns 0 (zero). If the PrintDocument function cannot start the named document, it returns a number other than 0, depending on the error.

The behavior of this function is the same as choosing "print" from the explorer with the right button menu. Therefore, if the "Print" option does not appear on the menu, this function will not be performed or it will show error.

Certain types of documents support the print (Word, PDFs, TXT), while others (Audio, Video, ZIP) do not.

Scope

Objects Procedure object, Transaction object, Work Panel object
Languages .NET, Java, Ruby, Visual FoxPro

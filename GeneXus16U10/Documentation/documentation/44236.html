Target domain

Enumerated domain indicating the training target model in the context of GeneXus Artificial Intelligence for custom models.

Values

Label The model will be trained for label inference (e.g. to be used with Classify procedure)
Region The model will be trained for region inference (e.g. to be used with DetectObjects procedure)

Scope

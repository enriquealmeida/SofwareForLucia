SetTheme function

Changes the application theme at runtime for any other theme included in the property Additional Themes property. It returns 1 when it can change the theme, or 0 otherwise; if the theme can't be changed the base theme is kept. The change persists as long as the application session is active.

Syntax

&Numeric = SetTheme("ThemeObjcetName")

Where

&Numeric: Variable of numeric data type of length 1.
"ThemeObjectName": Name of the theme to change.

Example

Event 'SetTheme'
   &num = SetTheme(!'SimpleAndroid')

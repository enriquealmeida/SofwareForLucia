CallOptions for Smart Devices

CallOptions are used to specify at runtime the transitions, behavior, and position in a call to a particular panel.

Basically, in the passage from one panel to another there are several elements of user experience that we must consider:

  • Transitions: Enter and Exit effect for the new panel. 
  • Type: Behavior of the new panel (Popup, Callout, etc.).
  • Target: Where the panel is going to be loaded.
